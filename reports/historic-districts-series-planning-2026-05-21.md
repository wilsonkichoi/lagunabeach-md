# Taiwan.md 台灣歷史街區系列 — 規劃與初步調查

> Taiwan.md 已有 22 縣市 panorama 系列（每篇 4500-6000 字 cover 一整個縣市的歷史軸線），也有 [台灣老街文化與商業街區](../knowledge/Culture/台灣老街文化與商業街區.md) 主檔（catalog 級，10+ 條老街短介紹）。**「歷史街區」系列補的是 sub-unit deep dive：單一街區一篇 5000+ 字，挖縣市文跟 catalog 主檔沒空挖的細節**。
>
> 系列補完後，Geography 類別會形成三層閱讀地圖：
> - **總覽層**：`Geography/台灣島嶼地理特色與形成.md` / `城市特色與區域文化.md` / `行政區劃.md`
> - **panorama 層**：22 縣市單篇（已 ship）
> - **街區層**：歷史街區 80-120 篇（本系列）
> - **主題層**：老街文化主檔 / 工藝 / 食物 / 信仰 catalog
>
> 不是維基百科式的「百年街區清單」，而是 MANIFESTO §策展非百科的街區層落實 — **走在這條街上的台北人會講的版本**。

## 1. 為什麼這個系列

### 既有覆蓋的缺口

- **22 縣市文**走 panorama 視角，每個街區只能 1-2 段。例如台北市文章把「萬華 1738 / 大稻埕 1885 / 信義 2004」當三個年紀的並排，但**大稻埕怎麼從 1851 商業街變成 1947 二二八第一槍**這條故事線塞不進去
- **老街文化主檔**走 catalog 視角，每條街 200-400 字介紹，**沒空挖物質層**（一棟洋樓的雕花來自哪個年代、一家中藥行的紙袋從何時開始換字體）
- **AI agent 高頻查詢的「Tell me about 大稻埕 / 西門町 / 永康街」**目前的回答 source 是維基百科 + 觀光局頁面，缺 first-person curatorial voice

### Taiwan.md 街區角度的獨特性

- **時間是結構**：每條歷史街區疊著 3-5 個世代的痕跡（清領郊商 / 日治市區改正 / 戰後外省眷村 / 解嚴後文青化 / 21 世紀觀光化），單篇深度才壓得住
- **物件層敘事**：街屋立面的洋風雕花、騎樓的鑄鐵欄杆、巷弄裡的廟宇神龕、店家招牌的字體變遷 — 縣市文沒辦法處理的物質細節
- **凌晨四點時刻**：每條街都有「外人不知道但在地人覺得最迷人的時刻」（per 22 縣市 anchor pattern），單篇才有篇幅 hold 這個 anchor

### SEO + AI 引用雙重缺口

英文圈 Google 搜「Dadaocheng」「Ximending」「Wanhua district Taipei」前幾頁全是觀光局 + 維基 + tripadvisor，沒有 first-person 策展文。日文圈搜「迪化街」「西門町」更明顯，日本旅客高密度但日文 source 多半是部落客遊記不是歷史 deep dive。Sovereignty preservation lens（per MANIFESTO §主權的巴別塔）：英日韓圈對台灣街區的 demand 被觀光手冊接管，Taiwan.md 補位的同時也是英日韓翻譯優先 cluster。

## 2. 系列 Scope 定義（哲宇 2026-05-21 拍板）

### 「有歷史」邊界 = 寬鬆（戰後成形也可）

**入選條件**（任一即可）：

1. **清領以前已成形**：街成形於 1895 前，有清治時期的街郭、廟宇、商業基底（艋舺 / 大稻埕 / 大龍峒 / 鹿港 / 安平 / 北埔...）
2. **日治規劃成形**：1895-1945 市區改正規劃出的現代街區，含商業區、住宅區、特殊功能區（西門町 / 永康街昭和町 / 中山北路敕使街道 / 北投溫泉街 / 鹽埕埔...）
3. **戰後特定時代的標誌**：眷村、條通、戒嚴期商圈、學運文青地下、80 年代次文化集散 — 這些是台灣戰後社會結構的物質證據（四四南村 / 寶藏巖 / 牯嶺街舊書街 / 公館溫州街文青地下...）

**排除條件**：

- ❌ 2000 年後才落成的純商業重劃區（信義計畫區 / 內湖科技園 — 留給「新興街區」另一個 spec 的系列）
- ❌ 已在老街主檔 deep 介紹過 + 結構單一的（鹿港的話可寫但要 evolve 主檔 + 另寫 sub-article 三明治式拆出來才不會撞 scope）
- ❌ 純地理區劃名（萬華區 vs 艋舺：寫艋舺，因為「萬華區」是 1990 改名後的行政名，艋舺才是有歷史紋理的街區名）

### 系列總規模估算

## 3. 街區文章共通結構模板（草案 v0.1）

每篇歷史街區文章建議走以下 7 個 H2 骨架。Pilot 跑完後 retrospective 校準 enriched caveats（per 22 縣市 pilot 基隆市 retrospective pattern）。

```
# {街區}：{副標 hook}

> **30 秒概覽：** {街成形年代}、{清領/日治/戰後關鍵轉折}、{現在的樣子}。
> 這篇文章想說的是 {核心矛盾 ≤ 30 字}。

## {凌晨四點時刻}（在地人會帶你看的當下場景 anchor）
- 不用「歡迎來到 X」這類觀光手冊腔
- 用一個具體時間 + 地點 + 動作的場景開頭（例：「凌晨四點的環南市場已經在動」）
- 跟 22 縣市文 anchor 共通格式

## 名字怎麼來的（地名考據）
- 原住民地名 vs 漢人地名 vs 日治改名 vs 戰後改名 多層疊
- 例如「艋舺」凱達格蘭語「獨木舟交易地」/「大稻埕」凱達格蘭族大浪泵社故址 + 漢人「曬稻穀的大埕」

## 街成形的時刻（第一個 anchor 事件）
- 清領郊商落腳 / 日治市區改正 / 戰後眷村劃定 / 學運據點形成
- 用 3-5 個關鍵年份 + 具體事件 + 必驗 source

## 從 {起始年} 到 2026 的軸線（3-5 個轉折點）
- 不寫編年體（per chronicle-lead plugin），用 3-5 個關鍵時刻串
- 清領 / 日治 / 戰後 / 解嚴後 / 21 世紀，每段一個 anchor 事件

## 物質層：建築、招牌、食物
- 街屋立面的洋風雕花（誰蓋的、什麼年代、現在還在不在）
- 騎樓鑄鐵欄杆 / 廟宇神龕 / 招牌字體變遷
- 必點的 1-3 樣食物（跟其他街區同名食物的差異）
- 至少 1-2 棟具體建築 + 完工年 + 設計者

## 在地人會帶你去的 3 個地方
- 觀光客都去的點不寫（西門町紅樓觀光客都知道）
- 寫在地人才知道的點（西門町：成都路 7 號 vs 美觀園 vs 蜂大咖啡）
- 每個地方 1-2 句 anchor 細節，不寫成攻略

## {副標 hook 收尾} + 延伸閱讀
- 收尾呼應「凌晨四點時刻」開頭
- cross-link：縣市文 / 老街文化主檔 / 同期重要事件文章 / 同類街區
```

**字數目標**：4500-6500 CJK chars（介於 22 縣市 4500-6000 + PanSci P0×5 平均 6000）。

**Footnote 目標**：15-25 條（街區歷史 source 密度通常比縣市 panorama 高）。

**圖片目標**：≥ 5 張（hero + 4 inline），per 22 縣市 v2 升級後標準。Wikimedia Commons hot-link 為主，主檔已 cache 不重複下載。

## 4. 共通 research caveats（pilot 前先列，pilot 後 enrich）

### 4.1 必驗事實層（每篇都要過三源驗證 per REFLEXES #16）

1. **街成形年份**：清領「○○年某某人開店」這類常有民間說法 vs 文獻說法分歧（例：迪化街 1851 商業出現 vs 1853 同安人遷入哪個算「街成形」起點）
2. **建築年份**：街屋、廟宇、洋樓的興建年 / 重修年 / 改建年三個都要分清楚（艋舺龍山寺 1738 創建 vs 1919 重修 vs 2018 升國定古蹟）
3. **政治事件年份**：二二八第一槍地點（天馬茶房南京西路 189 號）、頂下郊拼（1853）、廢娼（1997）、學運（哲學系事件 1973-1975 過程戰）
4. **地名變更**：「萬華 vs 艋舺」「大稻埕 vs 大同區」「西門町 vs 漢中街、成都路、武昌街、峨眉街具體範圍」— frontmatter 跟內文要一致
5. **族群歷史**：凱達格蘭、巴賽、馬卡道、西拉雅族的地名跟現在街區的關係，不要用「漢人開拓」覆蓋原住民先住歷史

### 4.2 政治敏感性層

- **眷村系列**：四四南村、青田街日式宿舍 → 外省遷台敘事不蓋過外省人個體經驗，但也不寫成「國府偉大眷顧」
- **戒嚴期商圈**：紫藤廬周德偉 / 殷海光故居 / 黨外運動地下脈絡 → 用具體事件 + 具體人物，不寫成「白色恐怖好可怕」抽象悲情
- **特種行業歷史**：北投溫泉街「那卡西」、林森北路條通文化 → 紀實而不煽情（per REFLEXES #28），不寫成獵奇也不漂白
- **轉型期更新**：寶藏巖「違建變藝術村」、四四南村「眷村變文創園區」 → 寫住民被迫遷的痛 + 文化保存的張力，雙面都寫

### 4.3 觀光手冊塑膠句禁區（per EDITORIAL §塑膠句）

- ❌ 「歷史悠久的 X 老街」「充滿懷舊風情」「彷彿走進時光隧道」
- ❌ 「必吃」「必訪」「IG 打卡熱點」「在地人推薦」（這些字是 SEO 塑膠句）
- ❌ 「不是 X，是 Y」對位句型超過 3 處 / 篇（per MANIFESTO §11）
- ❌ 破折號「——」連用超過 15 處 / 1500 字
- ✅ 用具體時間 + 具體地點 + 具體動作 + 具體數字
- ✅ 用引號標出 verbatim source（per EDITORIAL）

### 4.4 物質層紀律

- 必有 ≥ 5 張圖（Wikimedia hot-link 為主），圖說含**拍攝年 + 攝影者 + license**
- 街屋立面、廟宇、洋樓的具體名稱 + 興建年 + 設計者 / 出資者
- 食物層要寫**這條街跟其他地方版本的差異**（西門町阿宗麵線 vs 其他麵線、迪化街阿婆肉粽 vs 別處肉粽）
- 字體層、招牌層、騎樓地磚層的觀察細節 — 用「眼睛能看到的物件」當證據，不是用形容詞

### 4.5 跨類別 cross-link 義務

每篇至少 cross-link：

- 對應 **縣市 panorama 文章**（雙向）
- **老街文化主檔**（雙向 catalog ↔ deep dive）
- ≥ 2 篇 **同期重要事件 / 人物 article**（二二八 / 太陽花 / 雷震 / 殷海光 / 楊德昌等）
- ≥ 1 個 **同類街區 sibling**（艋舺 ↔ 大稻埕 ↔ 西門町 三市街互引）

## 5. 台北 12 街區候選（系列 batch 1）

### 5.1 優先級分布

- **P0 第一波 (4 條)**：大稻埕 / 艋舺 / 西門町 / 中山北路條通
  - 三市街三角（大稻埕 + 艋舺 + 西門町）+ 戰後最重要的日本連結（條通）
  - 歷史層次最厚、source 最豐富、讀者期待最高
- **P1 第二波 (4 條)**：永康街東門 / 公館溫州街 / 寶藏巖 / 北投溫泉街
  - 戰後文化地理的核心節點，眷村、學運地下、違建轉型、特種行業歷史都進
- **P2 第三波 (4 條)**：大龍峒圓山 / 四四南村 / 士林慈諴宮 / 牯嶺街南海學園
  - 補完台北宗教 / 眷村 / 夜市 / 書街地圖

### 5.2 12 條 hook + 核心矛盾 + 必驗事實摘要

#### P0-1 大稻埕 / 迪化街

- **核心矛盾候選** ≤ 30 字：「茶葉做生意的街、辦過西學堂、又是二二八第一槍，800 公尺裝三個世紀」
- **Hook anchor**：清晨五點半的迪化街，1860 年代陳天來的錦記茶行洋樓還站在原位置
- **時間軸 anchor**：1851 商業出現 → 1853 頂下郊拼後同安人遷入 → 1860 淡水開港 → 1869 Formosa Tea 出口紐約 → 1885 劉銘傳設西學堂 → 1891 大稻埕火車站 → 1920 三市街合一 → 1947 天馬茶房第一槍 → 2010s 迪化街 207 文創復興
- **必驗事實**：1851 vs 1853 街成形起點 / Formosa Tea 1869 首船細節 / 天馬茶房地址（南京西路 189 號 vs 其他源說法）/ 1885 西學堂位置 / 李春生「茶葉之父」稱呼源 / 陳天來錦記茶行 1891 創立 vs 1920-23 洋樓建年
- **政治敏感性**：中（二二八第一槍敘事 + 1853 頂下郊拼族群械鬥需小心 framing）
- **Cross-link**：[台北市](knowledge/Geography/台北市.md) / [老街主檔](knowledge/Culture/台灣老街文化與商業街區.md) / [台灣茶文化](knowledge/Culture/台灣茶文化.md) / 二二八相關 / 艋舺 sibling

#### P0-2 艋舺 / 剝皮寮

- **核心矛盾候選** ≤ 30 字：「清領台北最熱鬧的地方，現在是台北平均年齡最老的區」
- **Hook anchor**：清晨六點龍山寺廟埕的第一炷香，老人家從 1738 拜到 2026
- **時間軸 anchor**：凱達格蘭語 Manka → 1709 陳賴章墾號 → 1738 龍山寺建（清乾隆三年）→「一府二鹿三艋舺」清領全盛 → 1853 頂下郊拼（三邑 vs 同安）→ 1884 中法戰爭龍山寺為義軍指揮所 → 日治剝皮寮老街保存 → 1990 萬華區改名 → 2010《艋舺》電影
- **必驗事實**：龍山寺 1738 建年 / 剝皮寮地名來源（剝杉皮 vs 剝樹皮各源說法）/ 頂下郊拼 1853 具體年份 + 三邑安溪同安族群 / 1884 中法戰爭龍山寺角色 / 「一府二鹿三艋舺」俗諺最早出處 / 2010《艋舺》電影鈕承澤拍攝細節
- **政治敏感性**：低-中（族群械鬥史 + 電影對黑道美化爭議）
- **Cross-link**：[台北市](knowledge/Geography/台北市.md) / [老街主檔](knowledge/Culture/台灣老街文化與商業街區.md) / [台灣宗教與寺廟文化](knowledge/Culture/台灣宗教與寺廟文化.md) / 大稻埕 / 西門町 sibling

#### P0-3 西門町

- **核心矛盾候選** ≤ 30 字：「日本人 1896 蓋的娛樂區，130 年後還是台北最年輕的街」
- **Hook anchor**：週六傍晚六點的西門紅樓前，扮裝攝影、刺青店、Cosplay 同框
- **時間軸 anchor**：1896 日人在艋舺西邊規劃娛樂區 → 1908 西門紅樓建 → 1922 改正名西門町 → 戰後 1950s-80s「青少年文化首都」→ 1961-1992 中華商場時期 → 戒嚴期 MTV 影城 / 流行音樂 / 流行歌手出道地 → 1990s 後次文化、Cosplay、刺青、潮牌 → 2026 仍是台北次文化集散
- **必驗事實**：西門紅樓 1908 八卦堂 + 1908 十字樓兩棟具體建年 / 中華商場 1961.4 啟用 vs 1992-11-30 拆除 / 西門町指涉範圍精確（成都路、漢中街、武昌街、峨眉街）/ 萬年大樓、誠品 116 等戰後地標年表
- **政治敏感性**：低（次文化敘事相對中性）

## 6. Pilot 規劃（INBOX 批次寫入後）

### 6.1 Pilot 候選

INBOX 12 條寫進去後，建議從以下 2 條挑 1 條 pilot：

**A. 大稻埕**（最 obvious P0）
- 優點：source 最豐富、Wikimedia 圖最多、台北市文章已有 1 段可作 baseline
- 風險：歷史層次太厚，5000 字壓不下 280 年敘事，可能寫成 7000+ 字
- pilot 學習價值：驗證共通模板能不能裝下 thick history

**B. 西門町**（次文化角度 P0）
- 優點：戰後敘事為主、source 相對集中、跟 22 縣市風格差異最大
- 風險：物質層較薄（中華商場已拆），需多依賴照片 + 訪談 source
- pilot 學習價值：驗證共通模板能不能寫戰後文化型街區

**建議：先 pilot 大稻埕**（厚度先驗證；過了 thick history test，戰後街區自然 OK）。

### 6.2 Pilot retrospective 要校準的事

per 22 縣市基隆 pilot retrospective enriched caveats pattern：

1. Hook anchor 凌晨/清晨/傍晚時刻有沒有真的找到「外人不知道在地人覺得迷人」的東西
2. 模板 7 H2 順序對不對（地名考據放第二 H2 vs 直接寫故事）
3. 字數實際 vs 目標 4500-6500
4. Footnote 實際 vs 目標 15-25
5. 圖片 ≥ 5 張的執行成本（Wikimedia 可選 inventory）
6. 物質層紀律：街屋、招牌、食物的具體性夠不夠
7. 政治敏感事件（二二八 / 1853 頂下郊拼）的 framing 通過 EDITORIAL §爭議處理嗎
8. 共通 caveats 4.1-4.5 哪幾條 pilot 期間發現要 enrich
