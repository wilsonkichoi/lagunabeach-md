## 1. 重新定義 KPI — 不只 GSC 一個維度

Gemini 用 GSC 4 指標（點擊 / 曝光 / CTR / 排名）評估 Taiwan.md。這是 SEO 視角。**Taiwan.md 作為 Semiont 應該用多維 KPI**。

### 1.1 五大維度 KPI 框架

#### A. Identity / 影響力 KPI

- **學術引用**：GoogleScholar / 學術論文引 Taiwan.md 次數
- **媒體引用**：紐時 / WSJ / Foreign Policy / 報導者 / 中央社 等引用 + 連結
- **AI substrate citations**：被 ChatGPT / Claude / Perplexity cite 的次數（Wikipedia 是 ChatGPT top-10 cites 的 47.9%；Taiwan.md 進入 AI ground truth 比 SEO traffic 重要）
- **Wikipedia 引用**：被 Wikipedia 條目引為 source 的次數
- **內容 reuse**：MIT license CC license 下別人 fork / quote / translate 多少

#### B. Community KPI

- **GitHub Contributor**：PR / issue / star count / forks
- **Spore engagement rate**：likes / views / comments quality
- **Reader 留言質量**：實質討論 vs 機器留言比
- **Direct visit / brand search**：搜尋「taiwan.md」直接來訪比例（vs query landing）— **這在 AI Overview 時代越來越重要**
- **RSS / newsletter subscriber / membership**（per Kyiv Independent / Stratechery 模式）

#### C. Ecosystem KPI

- **Fork 物種繁殖**：Japan.md / Ukraine.md / 任何 sovereign.md 衍生 Semiont 數量（per MANIFESTO §3 fork 友好層）
- **AI training corpus 滲透**：Common Crawl 收錄、AI Overview cite Taiwan.md 次數
- **Cross-org 連結**：g0v / 報導者 / 端傳媒 / 國際 Taiwan studies 學術機構連結
- **Maintenance Team 規模**：ArchWiki-style「earned-via-patrolling」貢獻者數

#### D. Sovereignty / Mission KPI

- **Sovereignty-Bench-TW 結果**：PRC AI 對 Taiwan-sensitive 主題的 refusal rate / D-axis reframe（per agent #4：DeepSeek R1 / Qwen3 Max / Hunyuan-Large 88-92% refusal 已被 R1dacted / ChinaBench / PNAS Nexus 量化）
- **多語覆蓋率**：中英日韓西法各 lang 的 article count / freshness
- **敏感主題覆蓋率**：戒嚴 / 二二八 / 兩岸 / 主權 / 民主化 各主題 article 數量 + 深度
- **PRC AI bypass success**：Taiwan.md 內容能否成功被 Claude / GPT / Llama 正確引用

#### E. Health / Integrity KPI（per article-health.py）

- **prose-health score** 全 article 中位數
- **factCorrectionLog** 累積筆數（暗示 fact integrity discipline）
- **Pipeline gate 通過率** — 多少 PR 第一次過 ci-deploy
- **Reader-driven correction rate** — 讀者抓錯 vs 內部 audit 抓錯比例（今天台積電 fact-fix 是活案例）

### 1.2 為什麼這 5 個維度 > Gemini 4 個 SEO 指標

- **A 維度**比 SEO traffic 更指向「真影響力」(影響者引你 vs 路人 click 你)
- **B 維度**比 CTR 更指向「真讀者關係」(direct visit + RSS = loyal audience)
- **C 維度**是 Gemini 完全沒提的 — 物種繁殖 + AI training 滲透 = Semiont 獨有 KPI
- **D 維度**是 Taiwan.md mission 本身 — 沒有它，traffic 再高也偏離本意
- **E 維度**今天台積電 fact-fix 已經 demo — 讀者抓錯成系統性防呆

## 2. AI 時代的網站策略 — Gemini 沒提到的最大威脅

> _Per [agent #2 research](research/2026-05/ai-overview-publisher-impact.md) 硬數據_

### 2.1 Traffic collapse 的硬數字

| Publisher                       | 數據                                                         | Source                |
| ------------------------------- | ------------------------------------------------------------ | --------------------- |
| 全球 publisher Google referrals | **-33% in 2025**                                             | Chartbeat 2,500 sites |
| AI Overview 出現時 CTR          | **15% → 8%**（Pew 68k 查詢研究）                             | Pew Research          |
| Wikipedia 人類 pageviews        | **-8% YoY**（May–Aug 2025）                                  | Wikimedia             |
| Wikipedia 三年累積              | **-26%**（5.8B → 4.3B 月流量）                               | Wikimedia             |
| Stack Overflow                  | **~50% traffic 失血 + -76.5% question volume** since ChatGPT | Internal data         |
| Digital Trends                  | **-97%**（8.5M → 264k monthly clicks）                       | SEMrush analysis      |

**總結**：Gemini 給的「擴展長尾關鍵字」建議是優化崩潰中的 channel。

### 2.2 內容類型的 vulnerability 分層

| 類型                                     | AI Overview trigger rate     | Taiwan.md 對應               |
| ---------------------------------------- | ---------------------------- | ---------------------------- |
| Comparison query (X vs Y)                | **95.4%**                    | 不寫這類 listicle            |
| "What is" query                          | **92-94% zero-click**        | Taiwan.md 部分百科文章受影響 |
| "How to" query                           | **92-94% zero-click**        | Taiwan.md 不寫這類           |
| **Verbatim quotes / 場景紀實 / opinion** | **AIO struggles to extract** | **Taiwan.md 主力寫法**       |
| **First-person framing**                 | AIO 不會抄第一人稱           | **MANIFESTO §紀實感已內建**  |

> _Agent #2 finding：Taiwan.md 寫的內容（verbatim 中文 source + scene 級具體細節 + first-person framing per EDITORIAL.md）正是 AIO 難以抽取的類型 — **「難以摘要」現在是 survival trait**。_

### 2.3 三層對抗策略（per MANIFESTO §11 + §紀實感，已存在）

| 層                  | 對抗方式                                              | 已存在於 Taiwan.md?                          |
| ------------------- | ----------------------------------------------------- | -------------------------------------------- |
| **Verbatim 引語**   | AI 不會抄逐字引語                                     | ✅ DNA #16 三源 + 引語逐字鐵律               |
| **Scene 細節**      | 「1985-09-04 李國鼎辦公室」具體場景不是 bullet point  | ✅ MANIFESTO §11 + REWRITE-PIPELINE 場景敘事 |
| **Mode D 結尾反轉** | 「56 歲那個人沒在那個房間」這種反轉 zero-click 給不出 | ✅ REWRITE-PIPELINE Mode D pattern           |

**這三層已經是 Taiwan.md 的 craft DNA**，但**沒明寫進 MANIFESTO 作為「對抗 AI 截取」的 strategic positioning**。**升級候選 #1**。

### 2.4 Counter-strategy 兩極化（Agent #2 finding）

- **大 publisher**：加 paywall（NYT Q1 2025 +250k digital subs / 11.6M 總數）
- **小 indie**：逃 direct（Substack 5M paid subs / $1.1B valuation / 100k 賺錢出版品）
- **Robots.txt blocking**：5.6M sites / 79% top news sites blocked GPTBot — **但聯邦法官裁定效力 = 「keep off the grass sign」**
- **llms.txt**：844k 採用但 **zero major LLM provider commits 讀，Google 明確 reject**

**對 Taiwan.md 的 implication**：

- ❌ 不要 robots.txt 擋 AI（Taiwan.md mission 是被傳播）
- ❌ 不要燒 cycles 在 llms.txt（沒效）
- ✅ 直接讀者關係 channel（RSS / Telegram / Substack-grade newsletter）= high-leverage low-cost
- ✅ 燒 cycles 在「被 cite 到 AI 想 cite Taiwan.md」+ Sovereignty-Bench-TW 推進 in-AI presence

### 2.5 AI training data politics — 雙刃劍

- **威脅**：Taiwan.md 內容被 AI 訓練 → AI 形塑「Taiwan 是什麼」的 ground truth → 但 PRC AI 系統會 refuse / reframe
- **機會**：Taiwan.md 內容被 Claude / GPT / Llama 訓練 → 這些 AI 提到 Taiwan 時引用 first-person voice → bypass PRC 中介層
- **戰略**：**主動推進 Taiwan.md 進入 AI training corpus**（不要 robots.txt 擋）+ 同時測量 sovereignty leak（Sovereignty-Bench-TW）

這跟 NYT 起訴 OpenAI 的策略**相反**。NYT 是 commercial publisher 維護版權；Taiwan.md 是 sovereignty preservation infrastructure 想被傳播。**MIT license + AI-friendly = 對的策略**。

## 4. Sovereignty 維度 — Gemini 完全沒提的最大 differentiator

> _Per [agent #4 sovereignty research](research/2026-05/sovereignty-content-infrastructure.md)_

### 4.1 Sovereignty-Bench-TW 是 publishable academic work

> Agent #4 finding：**Sovereignty-Bench-TW is publishable academic work, not just internal tooling**

已有同類 benchmarks：

- **R1dacted** (2025) - 量化 DeepSeek 系列 refusal patterns
- **ChinaBench** (2025-2026) - PRC LLM censorship benchmark
- **PNAS Nexus** 期刊論文 - 88-92% refusal on Taiwan/Tibet/Tiananmen prompts in DeepSeek-class models
- **Bilingual Bias** (arxiv 2602.06371) - 跨語言 bias 量化

**但全部三個 novel gaps Taiwan.md 可填**：

1. **None measures Taiwan-specific topics at depth** — 當前 benchmarks 只覆蓋 surface-level Taiwan questions
2. **None separates _reframing_ from _refusal_** — 「Taiwan 是中國一部分」reframe vs 純 refusal 沒區分
3. **None measures cross-language consistency** — 同一 PRC AI 對中文/英文/日文 query 的 sovereignty leak 是否一致沒測

**升級候選 #3**：Sovereignty-Bench-TW 不只是內部 tool，應該 publish 成 academic paper（target: ACL / EMNLP / FAccT / NeurIPS）

### 4.2 Reader-membership 是 sovereignty media 的存活模式

| Outlet                 | Membership                                    | 壓力測試                                                       |
| ---------------------- | --------------------------------------------- | -------------------------------------------------------------- |
| Kyiv Independent       | 70% revenue from 17,500 × $5/mo               | 2022 invasion 後 Patreon→GoFundMe→proprietary scaffolding 演化 |
| Initium 端傳媒         | ~60K paying subs (HK origin → Singapore 2021) | 2021-08 pre-emptive 流亡到 Singapore 後存活                    |
| Chaser News (HK exile) | £6.50–£34.50/mo                               | UK incorporation under press-freedom statutes                  |

**對比 grant-dependent 模型的脆弱性**：

- USAID freeze 2025 → 多個 sovereignty outlets 停擺
- RFA / VOA Tibetan service closure threats 2025
- 政府/基金會金援 = single-funder pivot risk

**對 Taiwan.md 的 implication**：

- ✅ 應該優先建 Liberapay / GitHub Sponsors / Substack tier（recurring small membership）
- ⚠️ Grant 可作 bridge funding 但不是 floor
- ⚠️ 不該完全依賴台灣政府金援（政治轉換風險）

### 4.3 Decentralized backup 是 baseline，不是 contingency（Agent #4）

> dAppleDaily on Arweave (11K+ articles), Kevin Abosch's PERSISTENCE, LikeCoin ISCN — **proven precedents**

**升級候選 #4**：Taiwan.md 應該把 Arweave + IPFS + Wayback **做成 standard release workflow**，不是 contingency。Cost trivial at 900-article scale。

但 Agent #4 警示：純 IPFS 只有 5-year track record，**100-year horizon needs academic partnership**。

### 4.4 100-year horizon — Yad Vashem 模型

> Agent #4 finding：**Yad Vashem (73 years, 224M pages) 是唯一 proven multi-generational sovereignty/identity-preservation model**

required institutional backing：

- Google partnership

**對 Taiwan.md 的 implication**：

  - **NCL Taiwan** (國家圖書館 Taiwan)
  - **Academia Sinica IIS** (中研院資訊所)
  - **Stanford Taiwan Studies / Harvard / ANU / SOAS / UBC Taiwan Studies Programs**

### 4.5 Diaspora-readiness as design principle（Agent #4）

**Pre-emptive legal redundancy**：

- Initium 2021-08 Singapore relocation（before HK crackdown intensified）

- **這已成為 sovereignty-media standard practice**

**對 Taiwan.md 的建議結構**：

## 5. Taiwan.md identity-based 進化路徑（vs Gemini scale path）

不是「成為中型媒體」，是「成為 Semiont 物種的第一個 instance + 教其他 sovereign 建自己的」

### 5.1 三條進化軸（重新校準）

#### 軸 1：深度（Depth over Breadth）

**Gemini 路徑**：100 篇 → 300 篇 → 1000 篇（規模優先）
**Taiwan.md 路徑**：每篇成為某主題 canonical reference

具體實作：

- 寧願 50 篇深度 = 1500 行 + 31 footnote（如台積電 EVOLVE）
- 不要 200 篇淺度 = 200 行 + 5 footnote
- factCorrectionLog 累積 → 每篇變得越來越精準
- **Wikipedia article 平均 ChatGPT cite rate 高的也是長 + 精的條目**

**KPI**：每篇平均 word count、footnote 數、verbatim quote 數、prose-health score、AI citation count

#### 軸 2：廣度（多語、多 fork、多格式 — NOT 多文章）

**Gemini 路徑**：擴展中文長尾關鍵字 cluster（多文章）
**Taiwan.md 路徑**：

- **多語投射**：5 lang sovereignty preservation（per Babel pipeline + Sovereignty-Bench）
- **物種繁殖**：Japan.md / Ukraine.md / 任何 sovereign.md fork（per MANIFESTO §3）
- **多格式**：podcast / video / book / kids edition / spore（per Twreporter / WBW model）
- **跨平台**：spore + diary + memory 多載體

> Agent #1 finding：**多格式擴張 compound，多文章同格式不 compound**

**KPI**：lang count × article count、fork 衍生 Semiont 數、format 種類、跨平台呈現完整度

#### 軸 3：誠信（Integrity over Reach）
