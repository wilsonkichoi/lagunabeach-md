# Research Report: 台南市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：安平古堡（1624 荷蘭熱蘭遮城／1661 鄭成功接收）／赤崁樓（普羅民遮城）／鄭成功（1662 收復台灣 + 同年逝世）／台南小吃（牛肉湯 + 担仔麵 + 鹹粥 + 蝦捲 + 棺材板）／鹽水蜂炮（元宵節）／五條港（清領時期五大運河商業區）／國華街 + 永樂市場美食街／孔廟（1665 全台首學）／延平郡王祠／億載金城（1875 沈葆楨建）／神農街
- **世代差異**：
  - 戰前老世代知「府城」（1684-1885 台灣府城 200 年首府）
  - 戰後 50-70s 世代知「鄭成功遺址」（安平古堡 + 赤崁樓 + 延平郡王祠）+「鹽水蜂炮」
  - 80-90s 世代知「台南小吃」（觀光局推台南為「美食之都」+ 牛肉湯 + 担仔麵全國名聲）
  - 千禧後世代知「2010/12/25 縣市合併升格直轄市」+「國華街觀光 + 神農街文青」+「台南鐵道地下化」
- **外人 vs 在地人**：外人覺得「台南吃牛肉湯 + 看赤崁樓」就走人；在地人知「我們是台灣 400 年首府 — 1624 荷蘭時代開始」「五條港 18 世紀後因為港口淤積廢棄但街道格局留下」「鹽水蜂炮為什麼蜂湧而上？清光緒年間瘟疫祈福延續至今」「2010 縣市合併讓台南市跟台南縣變成同一個市但鹽水到佳里到玉井是三個世界」

### 多元面貌

- **主流敘事**：「美食之都 + 古蹟之都」（過度浪漫化）vs 「衰退中的老城市」（過度悲觀）
- **支線敘事**：
  - **1624-1662 荷蘭時代**：1624 荷蘭佔安平建熱蘭遮堡 + 1653 建普羅民遮城（赤崁樓）+ 38 年荷蘭統治 + 1662/02/09 鄭成功揆一交鎖匙
  - **1661-1683 鄭氏王朝**：1661 鄭成功登陸 + 1662 接收熱蘭遮城（東都 → 1664 東寧）+ 1662/06/23 鄭成功逝世（享年 38）+ 1683 鄭克塽降清
  - **1684-1885 台灣府城**：清領設台灣府於今台南 + 200 年首府地位 + 1885 台灣建省後省會遷台北
  - **400 年首府 vs 1885 失去首都地位**：從 1624 到 1885 共 261 年是台灣行政中心
  - **2010/12/25 縣市合併升格直轄市**：原台南縣 + 台南市合併為直轄市 + 37 區（含原台南市 6 區 + 原台南縣 31 鄉鎮市區）
  - **五條港 18-19 世紀**：南河港 + 南勢港 + 佛頭港 + 中港 + 北港五條運河 + 商業繁盛 → 19 世紀末淤積廢棄 + 街道格局保存至今（神農街 + 海安路）
  - **1665 孔廟「全台首學」**：鄭氏時期陳永華建 + 鄭克塽 1684 重建 + 至今台灣最早孔廟
  - **1875 億載金城**：沈葆楨建砲台抵禦日軍（牡丹社事件後）+ 1884 中法戰爭法軍砲擊
  - **鹽水蜂炮**：清光緒年間瘟疫祈福 + 元宵節炮城延續百年 + 國際觀光
  - **台南文學基地**：林懷民（雲門舞集 1973 在台南創立）+ 葉石濤（戰後台灣文學重要評論家）
  - **37 區**：原市區（中西、東、南、北、安平、安南）+ 原縣鄉鎮（永康、新營、鹽水、白河、柳營、後壁、東山、麻豆、學甲、佳里、西港、七股、將軍、北門、新化、善化、新市、安定、山上、玉井、楠西、南化、左鎮、仁德、歸仁、關廟、龍崎、官田、大內、龍崎、楠西）
- **fault lines**：
  - 原台南市 6 區（市區）vs 原台南縣 31 鄉鎮市（廣大平原 + 山區）— 2010 合併後行政整合
  - 古蹟文化（赤崁樓 + 安平 + 孔廟）vs 美食文化（牛肉湯 + 担仔麵 + 永樂市場）
  - 400 年首府身分 vs 1885 失去首都地位 + 1949 行政中心北移
  - 鄭成功（漢人英雄）vs 荷蘭時代（揆一交鎖匙）vs 平埔族西拉雅族（原住民）三重歷史
  - 五條港 18 世紀繁榮 vs 19 世紀末淤積廢棄
  - 鹽水蜂炮（清領祈福）vs 觀光化（國際旅客 IG 打卡）

### 想法跟感受

- **正面**：台南人對「400 年首府」「全台首學」有強烈自尊 / 對牛肉湯 + 担仔麵 + 蝦捲 + 鹹粥 + 棺材板的飲食驕傲 / 對 1624 荷蘭 → 1661 鄭成功 → 1684 清領 → 2010 直轄市的歷史厚度
- **負面**：1885 失去首都地位的失落 / 「老城市」「衰退」標籤的疲乏 / 觀光化讓台南小吃變成符號（永樂市場排隊文化）
- **矛盾感**：1624-1885 共 261 年是台灣行政中心 vs 1949 後行政中心北移 75 年 / 鄭成功是漢人英雄 vs 揆一交鎖匙是台灣第一個漢人政權的起點 / 2010 縣市合併讓鹽水到佳里到玉井變同一個市

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 西拉雅族（平埔族）千年居住 + 新港社 + 蕭壠社 + 麻豆社 + 目加溜灣社四大社
  - 1624 荷蘭佔安平建熱蘭遮堡
  - 1653 建普羅民遮城（赤崁樓）
- **關鍵轉折**：
  - 1624-1662 荷蘭時代 38 年（東印度公司）
  - 1661/04/30 鄭成功登陸鹿耳門 + 1662/02/09 揆一交鎖匙 + 接收熱蘭遮城（東都）
  - 1662/06/23 鄭成功逝世（享年 38）
  - 1664 東都改東寧 + 陳永華輔佐鄭經
  - 1665 孔廟建立（陳永華「全台首學」）
  - 1683 鄭克塽降清 + 1684 清領設台灣府於今台南
  - 1685 諸羅縣設於今台南北部
  - 1875 沈葆楨建億載金城（牡丹社事件後）
  - 1884-1885 中法戰爭法軍砲擊億載金城 + 安平
  - 1885 台灣建省 + 省會遷台北（台南失去首都地位）
  - 1895 日治：台南廳 + 1920 改台南州
  - 1945 戰後：台南市 + 台南縣分治
  - 2010/12/25 縣市合併升格直轄市
- **當代意義**：台南是台灣「400 年首府 + 200 年陪都」的縮影。1624-1885 共 261 年是台灣行政中心，1885-2025 共 140 年是「老首府」。2010 縣市合併讓 37 區從鹽水到佳里到玉井統一為直轄市

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 台南是台灣「最早被殖民」+「最早建立漢人政權」+「最早設府」的城市（1624 荷蘭 + 1661 鄭成功 + 1684 清領）
- 1885 台灣建省 + 省會遷台北是台灣行政中心北移的開始（台南失去首都地位 140 年）
- 鹽水蜂炮是台灣「祈福民俗百年延續」的代表案例（清光緒年間瘟疫起源至今 130+ 年）
- 台南小吃是台灣「飲食文化在地化」的縮影（牛肉湯 + 担仔麵 + 鹹粥 + 蝦捲 + 棺材板）
- 2010 縣市合併是台灣「行政升格」最具規模的案例（37 區 + 186 萬人）
- 對讀者：「為什麼台灣 400 年首府失去首都地位 140 年後仍是 186 萬人的直轄市？」這個 frame 拆開後讓人看見「歷史厚度」如何撐起當代
