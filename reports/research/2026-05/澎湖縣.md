# Research Report: 澎湖縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：跨海大橋（白沙到西嶼）／玄武岩（七美雙心石滬、桶盤、員貝、小門）／望安綠蠵龜／二崁古厝聚落／開臺天后宮（馬公 1593，全台第一座媽祖廟）／菊島／東北季風（仙人掌與堅韌植物）／花火節（年度盛事）
- **世代差異**：
  - 戰前世代知「媽宮」（1920 改名馬公前的稱呼）
  - 戰後 50-70s 世代知「漁業母港」（澎湖船團、遠洋漁業）
  - 80-90s 世代知「夏天 vs 冬天的二元結構」（夏季觀光 vs 冬季東北季風蕭條）
  - 千禧後世代知「拒博弈兩次」（2009 / 2016 兩次公投反對賭場）+「花火節 IG 打卡」
- **外人 vs 在地人**：外人覺得「夏天去澎湖玩、吃海產、看跨海大橋」；在地人知「澎湖冬天比夏天更澎湖」「我們選擇拒絕博弈不是不要錢，是不要變賭場」「90 座島嶼裡只有 19 座有人住」「我們從 1604 沈有容諭退荷蘭人開始就是台灣 + 大陸的中繼站」

### 多元面貌

- **主流敘事**：「夏天玩水的離島」「觀光的菊島」（過度單一化）vs 「人口外流的衰退離島」（過度悲觀）
- **支線敘事**：
  - 90 座島嶼（2005 普查確認，總面積 141 km²）只有 19 座有人住
  - 玄武岩柱狀地質：除花嶼為安山岩外其餘 89 座皆玄武岩，1740-800 萬年前火山噴發形成
  - 1604 沈有容諭退荷蘭人（明萬曆 32 年 → 紅毛城遺址，台灣最早被歐洲列強覬覦的地方）
  - 1622 荷蘭東印度公司佔澎湖建紅毛城 → 1624 被明軍驅趕至台灣本島（澎湖 → 大員/台南的歷史軌跡）
  - 1885 中法戰爭法軍佔澎湖（孤拔將軍病死於馬公）
  - 1895 日本人從澎湖登陸接收台灣（甲午戰爭後馬關條約）
  - 開臺天后宮（馬公 1593）：全台最早媽祖廟，比北港朝天宮（1700 左右）早 100 年
  - 2009/09/26 第一次博弈公投 56.44% 反對（13,397 同意 vs 17,359 反對）
  - 2016/10/15 第二次博弈公投 81.05% 反對（壓倒性更明顯）
  - 二崁聚落（西嶼）：傳統閩南建築保存示範區
- **fault lines**：
  - 觀光季 vs 淡季的二元經濟（每年 4-9 月 vs 10-3 月）
  - 兩次拒絕博弈的選擇：經濟翻轉 vs 文化保護的張力
  - 離島補助 vs 本島心理距離（觀光客覺得「離島」但澎湖人覺得「我們是中心」）
  - 大島（澎湖本島）vs 小島（東吉、南方四島、花嶼等）的資源分配
  - 年輕人外移到本島 vs 老一輩堅守的世代差異

### 想法跟感受

- **正面**：澎湖人對「拒絕博弈兩次」有強烈自尊（被視為「全國最有共識的反賭」案例）/ 對玄武岩、媽祖、跨海大橋的在地驕傲
- **負面**：「夏天熱鬧、冬天死寂」的二元節奏疲乏 / 年輕人外移 / 冬季東北季風蕭條期長達半年 / 漁業萎縮
- **矛盾感**：兩次拒絕博弈卻又依賴觀光（觀光帶來的環境壓力跟博弈帶來的可能差別只是程度問題）/ 90 座島嶼但只有 19 座有人住的「擁有而不能使用」

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 1740-800 萬年前火山噴發形成 90 座玄武岩島嶼
  - 宋朝（13 世紀）澎湖巡檢司設立（早於台灣本島開拓）
  - 1593 開臺天后宮建立（馬公，現存全台最早媽祖廟）
- **關鍵轉折**：
  - 1604 沈有容諭退荷蘭人（明萬曆 32 年）
  - 1622-1624 荷蘭佔澎湖建紅毛城 → 被明軍逼退至台灣本島（澎湖 → 台南的歷史軌跡）
  - 1885 中法戰爭法軍佔澎湖、孤拔病死馬公（1885/06/11）
  - 1895 日本人從澎湖登陸接收台灣
  - 1920 媽宮改名馬公
  - 1970 第一代跨海大橋完工（連白沙與西嶼）
  - 1996 第二代跨海大橋完工 2,494 米
  - 2009/09/26 第一次博弈公投 56.44% 反對
  - 2016/10/15 第二次博弈公投 81.05% 反對（更強烈）
- **當代意義**：澎湖是台灣「拒絕成為什麼」的最強烈案例。兩次拒博弈不是消極反對，是積極選擇「我們要保留什麼樣的生活」

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 澎湖是台灣歷史中「比台灣更早被開發」的地方（1593 開臺天后宮比台南任何一座漢人廟宇都早；元朝就有澎湖巡檢司）
- 澎湖是台灣「歐洲列強進入東亞」的第一站（1604 荷蘭 / 1622 荷蘭再來 / 1885 法國 / 1895 日本登陸）
- 澎湖兩次拒博弈的選擇是台灣「公民投票如何決定社區命運」的最強案例
- 對讀者：「為什麼一個面積只有 141 km²、人口只有 11 萬的離島，能撐起 400 年的歷史」這個 frame question 拆開後讓人重新看見「規模 ≠ 重要性」

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：1740-800 萬年前火山噴發 → 玄武岩 → 90 座島嶼（141.05 km²）/ 季風東北季風 10-3 月 vs 西南氣流 4-9 月 / 缺水缺電（地下水鹹化、依賴本島輸電）
- **誰在這裡生活**：宋元巡檢司漢人 / 明清福建漁民 / 1622 荷蘭人 / 1885 法軍短暫 / 1895 日本人 / 戰後外省軍人（澎湖防衛司令部）/ 馬公 + 湖西 + 白沙 + 西嶼 + 望安 + 七美 6 個鄉鎮市
- **人和地有什麼共生**：玄武岩 → 二崁古厝建材 + 雙心石滬漁法 / 季風 → 仙人掌與耐旱植物 / 海洋 → 漁業 + 媽祖信仰 + 觀光潛水 / 缺淡水 → 仰賴台灣本島輸水（2002 起海底輸水管）

### 切入點清單（候選 hooks）

1. **「全台第一座媽祖廟」開臺天后宮**（1593 馬公 → 比北港朝天宮早 100 年 → 澎湖是台灣信仰的起點）
2. **400 年的列強試金石**（1604 沈有容諭退荷蘭 → 1622 荷蘭再佔建紅毛城 → 1624 被趕往台南 → 1885 中法戰爭法軍 → 1895 日本登陸 → 澎湖是「進入台灣的後門」）
3. **玄武岩柱狀地質的世界級景觀**（桶盤、員貝、小門、七美雙心石滬 → 1,740 萬年前火山形成 → 全球地質遺產候選）
4. **兩次拒博弈公投**（2009 56.44% 反對 / 2016 81.05% 反對 → 全國最有共識的反賭案例 → 拒絕的不是賭場，是某種未來）
5. **跨海大橋的物理意義**（1970 一代 / 1996 二代 2,494m → 不只是連白沙到西嶼，是把 6 個鄉鎮市的離島變成「澎湖」這個單一概念）
6. **冬天的澎湖才是真澎湖**（東北季風 10-3 月 → 觀光淡季 → 在地生活 → 老人留下年輕人離開的時刻）
7. **二崁聚落的閩南建築保存**（西嶼鄉 → 玄武岩 + 咾咕石 + 紅瓦 = 澎湖最完整傳統聚落 → 2000s 文化資產保存示範區）

### 預期核心矛盾候選（待 Stage 1.4 收斂為 ≤ 30 字）

- A：兩次拒博弈，拒絕的不是賭場是「成為什麼」的決定
- B：90 座島嶼只有 19 座有人住，但每一座都是台灣的
- C：1593 開臺天后宮比台南漢人廟早 100 年，澎湖才是台灣的入口
- D：冬天蕭條半年，但澎湖人的時間感是 400 年

### 研究方向（要搜什麼可以驗證）

1. 澎湖 90 座島嶼具體分布：馬公 7 + 湖西 10 + 白沙 37 + 西嶼 3 + 望安 32 + 七美 1 = 90 座，但 19 座有人住，這 19 座具體名單
2. 玄武岩地質：1,740-800 萬年前火山噴發 + 花嶼為安山岩例外 + 七美雙心石滬玄武岩
3. 1593 開臺天后宮建立背景 + 跟北港朝天宮、大甲鎮瀾宮的歷史先後
4. 1604 沈有容諭退荷蘭人完整事件（明萬曆 32 年 → 沈有容碑 → 紅毛城遺址）
5. 1622-1624 荷蘭佔澎湖：建紅毛城具體位置 + 1624 被明軍逼退轉往台南 + 對台灣史的決定性影響
6. 1885 中法戰爭法軍佔澎湖：孤拔將軍 1885/06/11 病死於馬公的具體經過
7. 1895 日本登陸澎湖：馬關條約簽訂後從澎湖開始接收台灣的軍事行動
8. 2009/09/26 第一次博弈公投：13,397 同意 vs 17,359 反對 / 投票率 42.16% / 反對方領先 3,962 票 / 56.44% 反對
9. 2016/10/15 第二次博弈公投：81.05% 反對的具體票數 + 後續政策
10. 跨海大橋 1970 一代 / 1996 二代 2,494m 完整工程史 + 對澎湖六鄉鎮市的影響
11. 二崁聚落保存：1980s 廢墟 → 2000s 文化資產保存 → 觀光化的拉扯
12. 望安綠蠵龜保育：望安鄉產卵地點 + 復育成果 + 觀光衝擊
13. 七美雙心石滬：玄武岩堆砌的傳統漁法 + 雙心圖案的歷史
14. 西嶼鄉燈塔（漁翁島燈塔 1875）+ 西台古堡（劉銘傳建）
15. 馬公舊名「媽宮」1920 改名 + 馬公老街（中央街、北辰市場）
16. 桶盤嶼 + 員貝嶼玄武岩柱狀景觀 + 旅遊化
17. 澎湖海上花火節（2003 起年度盛事）跟觀光業關係
18. 澎湖跟金門、馬祖的離島補助制度比較
19. 澎湖海底電纜輸水（2002 起 + 2007 第二條）解決缺水問題
20. 仙人掌、天人菊（澎湖縣花）等耐旱植物 + 風塔（牆中藏的乘風）+ 玄武岩民居
