# Research Report: 宜蘭縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：雪山隧道（2006/06/16 通車，12.9 公里亞洲第二長）／蘭陽平原（沖積扇）／龜山島／太平山林業／三星蔥／宜蘭餅／童玩節（1996 起）／噶瑪蘭威士忌（金車 2008 蒸餾廠）／礁溪溫泉／黃春明小說／林義雄家族／民進黨「民主聖地」
- **世代差異**：
  - 戰前老世代知「噶瑪蘭」（原住民噶瑪蘭族 → 1796 吳沙拓墾 → 1812 噶瑪蘭廳 → 蘭陽平原開發 200 年）
  - 戰後 50-80s 世代知「東部後花園」（蘇花公路 / 北宜公路 9 彎 18 拐 / 蘭陽平原農業）
  - 90s-2000s 世代知「童玩節」（1996 起年度盛事，全國孩子的夏天記憶）
  - 千禧後世代知「雪隧後的衛星城」（2006 雪山隧道通車後從 3 小時車程變 43 分鐘，蘭陽平原農地變民宿）
- **外人 vs 在地人**：外人覺得「雪隧 43 分鐘到宜蘭、吃三星蔥油餅、住礁溪溫泉旅館」；在地人知「雪隧通車前後是兩個宜蘭」「我們的農地一條一條變成面海第一排民宿」「蘭陽平原冬天的東北季風比夏天的太陽更宜蘭」「我們是民主聖地但林義雄家滅門血案也發生在這裡」

### 多元面貌

- **主流敘事**：「台北後花園」「童玩節 + 礁溪溫泉的觀光縣」（過度單一化）vs 「雪隧後民宿氾濫的農地殺手」（過度負面）
- **支線敘事**：
  - 噶瑪蘭族（平埔族）原本是蘭陽平原原住民，1796 吳沙拓墾後被漢化擠壓
  - 黃春明文學（兒時的玩伴 / 鑼 / 看海的日子）—— 戰後台灣鄉土文學重要陣地
  - 林義雄家族：1980 林義雄家滅門血案（林雙胞胎女兒亮均、亭均遇害，母親游阿妹遇害）→ 慈林教育基金會 + 慈林新館
  - 林義雄反核（蘭陽平原 vs 核四廠位置在新北貢寮但影響宜蘭）
  - 民進黨「民主聖地」：陳定南（1981-1989 縣長）首位非國民黨縣長 → 游錫堃 → 劉守成 → 呂國華 → 林聰賢 → 林姿妙
  - 雪山隧道工程：1991 開工 / 2006/06/16 通車 / 13 年 / 13 死 3 傷 / 600 億 / 6 地震斷層 + 98 剪裂帶 + 36 湧泉
  - 太平山林業：阿里山、八仙山、太平山台灣三大林場之一
  - 蘇花改 + 蘇花安：蘇花公路改善計畫
  - 童玩節 1996 起源 + 2007-2009 暫停 + 2010 復辦
- **fault lines**：
  - 雪隧通車的雙刃劍：交通便利 vs 農地破碎化 + 民宿違法 + 移居人口湧入
  - 民主聖地 vs 林家血案的歷史傷口（同一個地方的兩種記憶）
  - 蘭陽平原稻田 vs 面海第一排民宿（土地用途的根本拉扯）
  - 噶瑪蘭族 vs 漢人移民 200 年的族群更替（今天噶瑪蘭族多遷至花蓮新社）
  - 童玩節舉辦 vs 暫停的政治經濟拉扯

### 想法跟感受

- **正面**：宜蘭人對「我們是民主聖地」有強烈自尊（陳定南反污染 / 反賓士車 / 反六輕到宜蘭 = 一連串地方自治經典）/ 對黃春明、林義雄、慈林的文化驕傲
- **負面**：「雪隧後我們不再是宜蘭」的失落（農地變民宿 / 蘭陽平原破碎化 / 房價漲翻）/ 林家血案的歷史傷口至今未解
- **矛盾感**：民主聖地誕生在血案發生的地方 / 雪隧帶來繁榮也帶來失序 / 童玩節是孩子的快樂但暫停過 3 年

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 噶瑪蘭族（平埔族）原住民居住數千年
  - 1796 吳沙率漳泉移民從頭城（烏石港）登陸開拓蘭陽平原
  - 1812 清廷設「噶瑪蘭廳」（蘭陽平原首次納入清版圖）
- **關鍵轉折**：
  - 1875 改噶瑪蘭廳為宜蘭縣（沈葆楨時期，跟基隆改名同一時期）
  - 1895 日治：太平山林業開發、宜蘭線鐵路 1924 全線通車
  - 1945 戰後：宜蘭縣行政體制延續日治
  - 1980/02/28 林義雄家滅門血案
  - 1981-1989 陳定南任縣長（首位民進黨縣長，反污染、反六輕到宜蘭）
  - 1996 第一屆宜蘭國際童玩節
  - 2006/06/16 雪山隧道通車（13 年工程 + 600 億 + 13 死 → 蘭陽平原從 3 小時變 43 分鐘）
  - 2007-2009 童玩節暫停
  - 2010 童玩節復辦
  - 2008 噶瑪蘭威士忌（金車）蒸餾廠投產
- **當代意義**：宜蘭是台灣「地方自治對抗中央」的最強烈案例（陳定南拒絕六輕 → 王永慶被迫到雲林麥寮蓋廠）。雪隧通車是「交通便利如何重塑一個地方」的最大實驗

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 宜蘭是台灣「地方主體性」的典範縣（陳定南時代奠定的反污染、反賓士車、反六輕 → 民進黨民主聖地）
- 林義雄家滅門血案是台灣戒嚴時期最重要的政治悲劇之一（至今未破案，凶手身分仍是台灣政治史的謎）
- 雪山隧道是台灣交通建設史最浩大的工程（13 年 600 億 + 13 死）—— 但通車後對蘭陽平原的衝擊也是最深的
- 噶瑪蘭族的歷史是台灣「平埔族被漢化擠壓」的縮影（今天噶瑪蘭族多遷至花蓮新社村）
- 對讀者：「雪隧之前我去宜蘭要 3 小時，之後 43 分鐘，這 2 小時 17 分鐘的差距如何改變一個地方？」這個 frame question 拆開後讓人看見「交通建設不只是工程，是社會工程」

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：蘭陽平原是蘭陽溪、宜蘭河、冬山河沖積而成的三角形平原 / 雪山山脈與中央山脈交會 / 龜山島（火山島）/ 太平山（中央山脈北段）/ 蘇花斷崖 / 東北季風 + 雪山隧道前的「葛瑪蘭」氣候孤島
- **誰在這裡生活**：噶瑪蘭族（平埔族）/ 1796 吳沙漳泉移民 / 日治日本人（林業 + 鐵路）/ 戰後外省移民 / 雪隧後台北移居者（民宿主、退休族、藝術家）
- **人和地有什麼共生**：蘭陽平原 → 水稻 + 三星蔥 / 太平山 → 林業 → 觀光 / 冬山河 → 親水公園 + 童玩節 / 礁溪 → 溫泉 / 蘭陽溪 → 河口沖積扇地形 / 龜山島 → 漁業 + 海洋觀光

### 切入點清單（候選 hooks）

1. **雪山隧道前後的兩個宜蘭**（2006/06/16 通車是分水嶺：之前 3 小時車程 → 之後 43 分鐘 → 從「東部後花園」變「台北衛星城」→ 蘭陽平原農地破碎化）
2. **陳定南 vs 王永慶**（1987 陳定南反六輕到宜蘭 → 王永慶被迫到雲林麥寮 → 一個地方政治如何阻擋一個資本巨擘）
3. **林家血案 1980/02/28 vs 民主聖地**（同一個地方的兩種記憶：滅門血案的歷史傷口 + 反獨裁民主轉型的搖籃）
4. **噶瑪蘭族的消失與回返**（1796 吳沙拓墾 → 漢化擠壓 → 1830s 噶瑪蘭族遷往花蓮新社村 → 今天噶瑪蘭族正名運動）
5. **蘭陽平原農地變民宿的拉扯**（雪隧後農地破碎化 → 違法民宿氾濫 → 蘭陽平原從米倉變地產投資標的）
6. **童玩節 1996 + 2007 暫停 + 2010 復辦**（公部門文化事業如何走過政治起伏 + 對宜蘭觀光的長期影響）
7. **黃春明的鄉土文學**（《兒子的大玩偶》《看海的日子》《鑼》→ 戰後台灣鄉土文學重要陣地）

### 預期核心矛盾候選（待 Stage 1.4 收斂為 ≤ 30 字）
