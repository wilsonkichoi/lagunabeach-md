# Research Report: 台灣美食總覽

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

不同世代的記憶圖層：

- **戰前世代**：日治時期的洋食模仿、酒家菜（台灣料理 1.0）、清酒與味噌
- **戰後第一代外省**：眷村山東饅頭、四川辣豆瓣、湖南臘味、江浙小籠包
- **嬰兒潮**：辦桌十二道、外省牛肉麵、辦公室便當
- **解嚴後 X 世代**：夜市黃金期、1983 春水堂發明珍珠奶茶、阿宗麵線排隊
- **千禧世代**：手搖飲遍地、晚上 24 小時早餐店、便利商店御飯糰
- **Z 世代**：米其林進台（2018）、新住民媽媽家鄉味、IG-food、Vegan 選項

共同 anchor 物件（不分世代都有）：
- 阿嬤的滷肉飯 / 便當盒裡的滷蛋
- 夜市那家排了一輩子的攤
- 出國想念的台灣味（多半是麵類 + 滷味）
- 早上趕著上學買的蛋餅奶茶

### 多元面貌

**主流敘事**：「台灣美食很厲害、夜市最有趣、珍奶征服世界」（觀光手冊版）

**被忽略的支線**：

- **族群層**：原住民飲食往往被「漢人主流敘事」邊緣化；新住民料理還沒進入「台灣味」的官方論述
- **階級層**：辦桌跟米其林是兩個極端，但講台灣美食常只講其中一邊
- **地理層**：北中南東離島各有獨特性，但常被簡化為「台南最美食」
- **時序層**：經濟起飛前的「窮人怎麼吃」（地瓜籤、麵線糊、菜脯蛋）被浪漫化但很少被認真記錄
- **食安層**：2011 塑化劑、2013 油品事件、2014 餿水油 — 信任崩塌時刻很少被寫進美食論述
- **產業層**：外籍移工漁工 / 餐飲業勞動條件 / 食材供應鏈 — 美食 narrative 背後的物理基礎

**fault lines**：
- 老一輩懷念「以前的味道」vs 年輕人覺得新口味才有趣
- 在地人嫌觀光夜市太貴太套路 vs 觀光客覺得驚艷
- 美食家追求精緻 vs 普通人覺得 70 元滷肉飯最幸福
- 健康趨勢 vs 重鹹重油的「真台灣味」

### 歷史脈絡（pre-search hypothesis）

- **形成期 1620s-1895**：原住民飲食底層 + 閩南 / 客家移民帶來的漢人飲食
- **日治轉折 1895-1945**：米飯標準化（蓬萊米培育）、洋食引入（蛋包飯、咖哩飯）、清酒、糖業
- **戰後重組 1945-1970s**：1949 約 120 萬外省移民帶來八大菜系 → 眷村菜誕生；美援帶來麵粉與沙拉油 → 麵食普及 + 炸物文化
- **經濟起飛 1970s-1990s**：辦桌全盛、夜市黃金期、1983 珍奶發明、早餐店連鎖化
- **解嚴後多元化 1987-2010s**：新住民移入帶來東南亞料理、咖啡精品化、餐飲連鎖品牌出海
- **米其林時代 2018-**：419 家餐廳上榜（2025 第八版）、Bib Gourmand 把夜市小吃納入國際視野
- **當代議題 2020s-**：永續餐飲、Vegan 選項、產地直送、食農教育

### 切入點清單（待搜尋驗證 / 反駁）

1. **「一天 24 小時的台灣味」軸** — 早晨豆漿油條 → 午便當 → 下午手搖 → 黃昏滷味 → 宵夜鹹酥雞 → 深夜麵店。讀者最容易代入，每個時段帶出一群文化議題。**最強候選**
2. **「四百年層積」軸** — 由古至今層層累積，從原住民石烤到米其林星光。歷史脈絡清晰但容易變編年體。
3. **「五個切片」軸** — 時間 / 族群 / 地理 / 場景 / 階級五投影。包容性最強但容易散亂。
4. **「一桌辦桌」軸** — 用辦桌十二道菜當縮影，每道菜帶出一個議題。優雅但範圍可能太窄。
5. **「混血證據」軸** — 從每道「最台灣」的菜挖出它的非台灣根源（牛肉麵的四川 / 珍奶的日本綠茶 / 鹹酥雞的美式炸雞）。最有 hook power。

**最佳組合**: 主軸用 #1（24 小時）當開場與 closure，中段用 #5（混血證據）+ #2（層積）+ #3（切片）展開。

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- **A**：「沒有一道菜是純粹台灣的，每一道菜都是最台灣的」— 最有 hook power、最有觀點、最 paradoxical
- **B**：「同一條街上路邊 70 元滷肉飯跟米其林三星可以平起平坐」— 階級扁平化的台灣味奇觀
- **C**：「四百年的移民史可以從一個便當吃完」— 將歷史層積具象化為日常物件

我傾向 **A**，因為它包容所有族群來源、所有時代、所有階級，而且讀者讀完會有「原來如此」的瞬間 — 牛肉麵不是台灣原生、珍奶有日本根、鹹酥雞學美式炸雞，但都因為在這座島上被吃了幾十年，成了最強的台灣記憶。

### 預想讀者帶走的那一件事

讀完這篇，讀者不再問「台灣有什麼好吃的」，而是會說：「台灣味是四百年的混血證據。我嘴裡的每一口都是一個祖先 / 移民 / 商人 / 阿嬤的記憶被島嶼煮過一遍的結果。」

具體一句話：**「台灣美食不是一種菜系，是一座島嶼把外來的食物吃成自己的方式。」**
