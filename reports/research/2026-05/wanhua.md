# Research Report: 艋舺 / 剝皮寮（歷史街區系列 Wave 1）

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

**大眾共有的 anchor**：

- 龍山寺廟埕坐了一輩子的老人家、廟前那座蘿蔔糕 / 廣州街口的青草巷
- 華西街夜市的蛇店招牌、剝皮寮的紅磚拱廊（教科書上、Google Map 觀光客打卡）
- 鈕承澤 2010 電影《艋舺》—— 但「電影艋舺」跟「真的艋舺」是兩個東西
- 「一府二鹿三艋舺」這句俗諺
- 萬華 = 遊民 + 茶室 + 早晨進香的老人家（台北人對「萬華」的刻板敘事）

**不同世代差異**：

- 戰前世代：1853 頂下郊拼祖輩血脈記憶（部分大同/大稻埕家族「我們是被打輸跑出去的同安人」）
- 戰後世代：華西街觀光夜市時期、剝皮寮拆遷風波
- 解嚴後世代：1990 改名萬華區的疏離感、艋舺青山宮夜巡、Lucky 飯店
- 後 2010 世代：鈕承澤電影 + 西門徒步區外溢的觀光客

### 多元面貌

**主流敘事**：「清領全盛 → 日治沒落 → 戰後邊緣化」線性下行
**支線敘事**：

- 1853 頂下郊拼把同安人推到大稻埕，**成就了大稻埕後來的茶葉黃金時代**——艋舺的「沒落」是大稻埕「興起」的對偶
- 1884 中法戰爭龍山寺當義軍指揮所——艋舺人不是被動受害者，是動員主體
- 剝皮寮 2003 年迫遷案的痛 vs 2009 年正式開放後變文創園區的便利——保存的代價是「真的住戶被請出去」
- 萬華遊民問題不是「敗壞」是台北社會福利結構縮影
- 廢娼運動（1997-2001）跟華西街的時間軸

**fault lines**：

- 老艋舺人 vs 新搬進來的文青咖啡店主
- 萬華街友議題的同情派 vs 排斥派
- 電影《艋舺》浪漫化角頭 vs 在地人對「角頭根本不是那樣」的反應
- 觀光客眼裡的「老街可愛」vs 在地人「我家被當成 museum」

### 歷史脈絡（pre-search hypothesis）

**形成期**：

- 1709 陳賴章墾號（合股組織，五人股東）— 第一張官府墾照
- 1738 龍山寺建廟（晉江、南安、惠安三邑人合資）— 街成形的精神中心
- 1740s-1850s「一府二鹿三艋舺」全盛 — 商船從福建運貨進淡水河，艋舺渡口集散

**關鍵轉折**：

- 1853 頂下郊拼 — 同安人輸了，神像（霞海城隍）抱去大稻埕，種下大稻埕茶葉時代的種子
- 1860 淡水開港 — 艋舺漸被大稻埕取代為茶葉外銷重鎮
- 1884 中法戰爭 — 龍山寺指揮所
- 1907 日治填淡水河道 + 河岸線後退 — 艋舺渡口功能消失
- 1922 日治市區改正 — 街道拓寬、字體被取代
- 1945 戰後與外省眷村疊加，舊艋舺敘事被「萬華 = 落後街區」覆蓋
- 1990 行政區「萬華區」正式定名
- 2010 鈕承澤電影
- 2003-2009 剝皮寮保存運動 → 開放

**當代意義**：

- 萬華區人口高齡化全台北市最嚴重之一
- 龍山寺仍是觀光 + 信仰 + 老人社交三合一空間
- 街友 + 廢娼後留下來的茶室 + 龍山寺前老人家三類人共享同一個廣場
- 北市府都更壓力 + 文資保存張力

### 切入點清單（待搜尋驗證 / 反駁）

1. **「平均年齡最老」hypothesis**：萬華區是不是台北市平均年齡最高？要搜內政部人口統計
2. **頂下郊拼跟大稻埕誕生的因果鏈**：1853 同安人遷出 → 1851 / 1853 大稻埕中街形成（時間軸要對得起來）
3. **剝皮寮地名 4 說**：剝杉皮 / 剝樹皮 / 福皮藔 / 北皮藔 — 哪個說法在學界比較被接受
4. **「一府二鹿三艋舺」最早出處**：俗諺的具體記載源頭（《淡水廳志》？《噶瑪蘭廳志》？）
5. **凱達格蘭語 Manka 詞源**：「獨木舟」vs「以獨木舟交易之地」哪個更精準
6. **鈕承澤《艋舺》電影爭議**：在地人對電影 framing 的反應、票房數據

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- A：「清領台北最熱鬧的地方，現在是台北平均年齡最老的區」（hook 直接命中）
- B：「被打輸的同安人離開艋舺去了大稻埕，種下台北兩個世紀的命運分岔」（內部對偶）
- C：「龍山寺 288 年沒走，廟前的老人家從 1738 拜到 2026」（時間錨點型）

→ 鎖定 **A 為主、B 為內部分鏡、C 作為 hook anchor 的 scene** — 三者層次互補

### 預想讀者帶走的那一件事

「**艋舺不是落後的萬華，是 1738 就在那裡的台北。當代的「萬華邊緣化」敘事是 1853 那場械鬥的延長線，輸的人離開、贏的人留下、留下的人 288 年來看著淡水河變了又變。**」

### Stage 1.4 核心矛盾鎖定

**Core contradiction (final, 30 字)**：「清領台北最熱鬧的地方，現在是台北平均年齡最老的區」

**Anchor variations 三層**：
- Description / 開場 / 結尾：anchor A（清領熱鬧 vs 全市老齡冠軍）
- 中段「街成形」H2：anchor B（被打輸的同安人離開→大稻埕誕生）
- Hook scene：anchor C（凌晨六點龍山寺廟埕第一炷香）

### Stage 1.2 結尾素材鎖定（2-3 候選）

候選 1：**敘事閉環式** — 凌晨六點龍山寺廟埕第一炷香的老人家。1738 那年也有人在這個位置上香。
候選 2：**時間跳躍式** — 1853 那場械鬥後同安人抱著霞海城隍神像走出八甲庄；2025 大稻埕霞海城隍廟前 30 萬人求月老。神像走了 800 公尺，命運走了 172 年。
候選 3：**翻轉式** — 萬華不是台北的「過去」，是台北最早的「現在」——只是這個現在沒被信義 101 那套故事接收。

→ **首選候選 1 + 嵌入候選 2 內部 scene**（首尾呼應 hook anchor + 中段帶 anchor B）
