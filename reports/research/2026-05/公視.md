# Research Report: 公視 EVOLVE

> Stage 0 觀點成型 + Stage 1 待 spawn agent 跑深度研究。本檔頭部是 §觀點成型，agent 完成後 append §媒體授權矩陣三表、事實素材庫、引語、爭議素材、結尾素材、參考資料清單。

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

公視在不同世代留下不同形狀的記憶：

- **6/7 年級**：兒少節目陪長大 — 《下課花路米》《水果冰淇淋》《流言追追追》《下課花路米：泰雅族篇》
- **解嚴後成年人**：「老三台之外有個沒廣告的電視台」這件事本身就是新鮮體驗，1998 開播當天的儀式感
- **8/9 年級**：跟著《一把青》《麻醉風暴》《通靈少女》《我們與惡的距離》《茶金》看見台劇能做到的深度
- **共通**：65 台、藍底白字 logo、晚間新聞那種「沒有插播廣告」的節奏

關鍵 anchor 物件 / 場景：1998-07-01 開播首日 / 2019《我們與惡的距離》播畢那夜社群長文討論 / 2023-05《公視法》修法三讀通過議場畫面 / 客家電視 / 台語台 / 原民台跟主頻共用大樓的「公廣集團」實體建築。

### 多元面貌

**主流敘事**：「9 億預算被綁 23 年，公視做出旗艦台劇是不可思議的事」— voicetank / mediawatch / 文化界長年論述。既有 v1 文章已抓到這條。

**支線敘事 / 被忽略角度**：

1. 「公視沒人在看」vs「公視討論度高」— 收視率歷史長年低（一般時段 < 0.5），但《與惡》集數平均收視突破 2.0、線上熱議度遠超無線台。**這個收視 vs 影響力 decoupling 是公視 26 年的另一個特徵**
2. 客家電視 / 台語台 / 原民台 vs 主頻不是同層級存在 — 客家有客家委員會單獨預算、台語台有獨立董事會、原民台 2017 起改制。「公廣集團」是聯邦不是統一機構
3. TaiwanPlus 是「公共媒體」還是「政府英文台」？— 2018 起 NHK 模式 vs RT-style 中間
4. 華視併入公廣 17 年仍「不公不民」— 還是商業電視台運作但帶公共任務
5. 董監事選任歷次卡關 — 藍綠都卡過，背後是「公共性」跟「政治化」拉鋸的結構張力
6. 國際比較：BBC 用「TV licence fee」家戶徵收 + 政府零干預 / NHK 用「受信契約」+ 半官方 / PBS 用「政府捐贈 + 民間募款」混合 / 台灣公視走「政府預算 + 臂距原則」最接近 PBS

### 大家對它的想法跟感受 fault lines

- **老一輩**：懷念兒少節目年代、「我家的電視 65 台」歸屬感
- **文化界 / 傳播學界**：長期支持，認為預算太少、應該擴大編制
- **黨派人士**（藍綠都有）：質疑董監事任命「綠化」或「藍化」
- **一般觀眾**：「我從不看公視」但《與惡》《茶金》紅了會推薦
- **媒體改革派 vs 商業電視業者**：拉鋸公視是否應該擴大、是否搶廣告
- **TaiwanPlus 受眾**：海外台灣人讚為「終於有公共英文台」/ 質疑者「政府宣傳」
- **Z 世代**：可能根本沒開過電視，公視主頻關電視台轉戰 YouTube 後關係重新建構

### 歷史脈絡 (pre-search hypothesis)

**形成期 1980s-1997**：

- 1980s 「廣電基金會」開始委製公共節目
- 1990s 初解嚴後媒體民主化進程 — 「媒體改革聯盟」推動
- 1993 立院初版《公視法》草案
- 1997 法通過，但條文加入 9 億元預算上限 + 董監事門檻嚴格

**關鍵轉折**：

- 1998-07-01 正式開播（晚 NHK 24 年、晚 BBC 76 年）
- 2006「黨政軍退出媒體」華視併入公廣集團
- 2007 客家電視併入
- 2014 台語台籌備 / 2017 原民台正式營運
- 2018 TaiwanPlus 啟動（先是 Marketing 計畫，2022 升級為 24/7 國際英文台）
- 2019《我們與惡的距離》引發社會集體療癒
- 2021《茶金》(1950s 茶業史) / 2022《一把青》重播（眷村記憶）
- **2023-05《公視法》修法三讀通過，9 億預算上限解除**（解開「23 年緊箍咒」）
- 多次董監事選任 deadlock（2010s 多次卡關）

**當代意義**：

- 在 streaming + AI + disinformation era，公視如何找到新 relevance？
- 預算解鎖了，任務該擴張到哪？
- 跟「資訊主權」「PRC AI 政策影響台灣聲音」對照（per CLAUDE Sovereignty Preservation lens）

### 對社會 / 歷史 / 環境 / 我們人生的關聯

公視 26 年的存在本身回答一個 democracy maturity 問題：**「我們夠成熟到撐起一個不為廣告也不為政府服務的公共空間嗎？」**

這個試紙是台灣的：

- **資訊主權結構性實驗** — 跟商業媒體（看廣告主臉色）+ 黨政軍時代媒體（看政府臉色）區分，是台灣資訊環境的第三條路
- **跟 Taiwan.md 精神親緣** — 公開、公共財、不為單一商業/政治利益服務的知識生命體（per MANIFESTO §3 知識是公共財）
- **跟解嚴後民主化 pipeline 的一部分** — 媒體民主化是台灣民主轉型的關鍵環節
- **跟「黨政軍退出媒體」運動連動** — 2006 華視併入公廣是這個運動的具體 implementation
- **對 Z / Alpha 世代的問題**：streaming 取代電視時代，公視怎麼跨平台、怎麼維持公共性？

### Society 類加重維度（per pipeline 類型加權矩陣）

當代意義 + 爭議 + 未完成的問題：

1. **當代意義**：公視是 streaming era 的「公共空間」實驗。BBC iPlayer / NHK Plus / 公視+ 平台是否能取代電視時代的「沒廣告的安靜空間」？
2. **爭議**：
   - 「臂距原則」實踐到位嗎？— 預算解鎖後政府如何不干預內容
   - 董監事政治化 — 7-10 次選任卡關背後的結構問題
   - TaiwanPlus 變相政府宣傳？— 是公共媒體還是 soft power
3. **未完成的問題**：
   - 華視 17 年「不公不民」中間態
   - 客家 / 台語 / 原民 vs 主頻關係
   - AI / streaming 衝擊下的轉型路徑
   - 「臂距原則」如何制度化（不只靠董事會自律）

### 切入點清單（待 Stage 1 搜尋驗證 / 反駁）

1. **「九億緊箍咒」hook**（v1 已用）— 預算政治史
2. **「《與惡》播畢那夜」hook** — 一部劇引發全台討論「正義是什麼」，公視能做到什麼商業台做不到

### 預期核心矛盾候選（待 Stage 1.4 收斂為單一）

- **A**：「政府給 9 億預算，卻把公視預算綁住 23 年」（v1 已用）
- **B**：「沒人在看的台，做出全民集體療癒的劇」（收視率 vs 影響力）
- **C**：「政府給錢但不能干預，誰來保證臂距原則？」（公共性結構張力）
- **D**：「華視併入公廣 17 年，還在不公不民」（轉型未竟）
- **E**：「26 年公共媒體實驗，是台灣民主成熟度的試紙」（meta-thesis 候選）

EVOLVE 方向：**v1 anchor 是 A（預算政治）。EVOLVE Round 補 B + C + D，把「九億緊箍咒」從單一矛盾擴成立體 fault-lines。E 是統合 thesis 候選。**

### 研究方向（要搜什麼可以驗證）

1. **公視收視率歷史** — Nielsen 數據 + 公視自己年報，2000s / 2010s / 2020s 主頻平均收視率變化
2. **《我們與惡的距離》《茶金》具體影響數據** — 收視峰值、社群討論量、學術引用、改編授權國際 reach
3. **董監事選任歷次爭議** — 2010s 卡關 case，藍綠都卡過的具體事件

6. **2023 修法後實際預算變動** — 解鎖後第一年 / 第二年 budget 比較
7. **客家 / 台語 / 原民台預算** — 跟主頻關係、獨立性程度
8. **臂距原則國際定義 + 台灣實踐** — BBC / NHK / PBS / 公視四元比較
9. **公廣集團員工人數 + 製作預算分布** — 跟 BBC / NHK 同等規模公司比
10. **媒體改革聯盟 1990s 推動史** — 「黨政軍退出媒體」運動的歷史脈絡
11. **公視兒少節目年代記憶** — 《下課花路米》《水果冰淇淋》收視率 + 文化影響
12. **公視+ streaming 平台** — 數位轉型 measurable progress

### 預想讀者帶走的那一件事

「我家電視的 65 台是台灣民主成熟度的試紙。它不為廣告也不為政府服務，這個 26 年的實驗每一年都在問『我們夠成熟到撐起一個公共空間嗎？』」

讀者讀完應該不只知道「公視缺錢」（v1 已給的訊息），而是看見公視這個機構在資訊主權 / 媒體民主化 / 公共性 / 信任結構幾個維度上承擔的角色。

### 故事素材（≥ 4 個）

1. **1998-07-01 開播首日**：晚 NHK 整整 74 年（NHK 1925 年廣播、1953 年電視）、晚 BBC 76 年。當天公視推出論壇節目《圓桌論談》與《民意高峰會》、紀錄片《全球部落》—— 後者成為台灣第一個常態紀錄片電視節目。從 1980 年代「公共電視製播小組」由新聞局官派、節目配合國家政策，到 1998 年正式落地，公視走了 18 年。
2. **2010-09-19 馮賢賢解職那夜**：公視董事會以開會人數不足卻仍決議解職總經理馮賢賢與執行副總經理鍾裕淵。馮賢賢在蘋果日報發表文章寫下「觀眾才是我們的老闆」、「堅持專業經營，守護公共利益，不就是公視該有的態度嗎？」—— 兩年後 2012-06-05 高等法院判公視敗訴確定，解職因明顯違反《公視法》屬無效。她贏了訴訟卻已不在位置上。
3. **2019-04-21《我們與惡的距離》大結局那晚**：第 9、10 集連播，收視率 2.91 與 3.4 達全劇峰值、創公視戲劇史第五高，PTT 台劇版湧入 9860 人同時在線討論。呂蒔媛為這部劇做 4 個月田野調查、訪問 40 多位專業工作者（律師、法官、立委、精神科醫師、媒體人）。她說：「關鍵就是公視站出來，公視願意做這個故事。」9 億預算撐起的東西，那一夜全台灣社群在討論「正義是什麼」。
4. **2023-05-26 修法三讀通過那天**：立院議場通過《公視法》部分條文修正案，刪除「政府捐贈經費應逐年遞減」與 9 億元上限、董監事門檻由 3/4 下修為 2/3、增設員工董事 1 席。時任文化部長史哲在記者會說「用 2 個月扭轉 20 多年的困境⋯⋯人人都是造浪者」—— 凍結 23 年的緊箍咒正式解開。但他沒講的下一句是：解開緊箍咒不等於解開政治化結構，這結構在 18 個月後（2025-01）就被新的多數黨用新方法重新瞄準。
5. **2025-01-16 → 17 砍預算之亂 36 小時**：國民黨立委陳玉珍 1/16 提案「全數刪除」公視 113 年度 23 億捐贈預算，理由是 TaiwanPlus 川普報導用詞偏頗、《聽海湧》竄改歷史、預算暴增缺乏合理性。編劇陳世杰、導演王小棣、林希璇等影視人在立院門外連署反對。1/17 國民黨團撤回稱「個人行為」，最終協商刪 1%（2,300 萬）、凍結 5.7 億（25%）。9 億緊箍咒解開 19 個月，新的瞄準方式啟動。
6. **2026-05-07 議場驅逐事件**：立法院教育及文化委員會開議前，國民黨立委羅智強要求公視董事長胡元輝離席，理由是 2025-05 任期屆滿後修法刪除延任條款；文化部長李遠當場抗議。胡元輝事後臉書發文：「公共媒體本就不應成為政治爭鬥的對象，更不能成為政治算計的犧牲品。」第 7 屆董事會原本延宕 957 天差點破第 4 屆 968 天紀錄才組成（2022-05-20 施振榮等 12 人通過審查），現在連任期結束都被當成政治施力點。

### 結尾素材鎖定（≥ 2 候選）

- **候選 1（時間跳躍 + 餘韻模式）**：「1998-07-01 開播那天到 2026-05-07 議場驅逐事件，相隔 28 年。中間有 23 年的 9 億緊箍咒、3 年的修法解凍、19 個月後新的政治瞄準。緊箍咒解開不等於拔掉孫悟空頭上的金箍 — 任何時候新的咒語都可以再唸一次。公視這個機構在那 28 年裡學到的事：獨立性不是法條保障就成立，是每一代公民社會的具體選擇。」
- **候選 2（問題模式）**：「9 億預算可以做出《與惡》《茶金》《通靈少女》。9 億可以孵化《人選之人》《聽海湧》。9 億可以餵養客家、台語、原民三個少數語言頻道。我們現在的問題不是『為什麼公視只有 9 億』，而是『把它調整到 23 億這一年，我們有沒有同時建立一套不讓政治力滲透的結構』。這個問題目前沒有答案 — 2025-01 與 2026-05 的兩次事件都在問同一句話。」
- **候選 3（翻轉模式）**：「我家電視的 65 台，曾經是『沒廣告的安靜空間』。現在年輕一代根本不開電視。但公視+ 5 年從 25 萬會員長到 122 萬，《通靈少女》串流數據在新加坡刷新 HBO Asia 紀錄。最有趣的反轉是：當公視終於有錢做更多事的那一年，質疑它『該不該存在』的聲音變大了 — 因為它變得更值得搶。」
