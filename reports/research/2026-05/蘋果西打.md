## 一、核心矛盾（給主 session 寫作用）

### 主候選（推薦使用）

> **「打了 60 年的國民氣泡飲，最後不是靠配方還清債，是靠賣掉高雄湖內 7222 坪地。」**

理由：

1. 數字錨點具體（60 年、7222 坪、地點明確）
2. 反差結構強烈：配方（產品價值）vs 土地（資產價值）
3. 直接點出 2024 EPS 8.71 元 / 稅後淨利 4.93 億的真實來源 — 一次性處分利益 6 億，本業營收僅 1.91 億
4. 不污衊 — 賣地是合法且有效的求生策略，但揭穿「品牌復興」敘事的真實面

### Backup 候選 1

> **「蘋果西打的 60 年裡，最危險的不是兩次沉澱物事件，是它從台北車站二樓金華百貨櫃台後面換來的那 80 萬美元。」**

理由：1995 孫幼英 80 萬美元（約 2300 萬）從賴比瑞亞商果是高國際買回配方/商標/銷售特許 — 接著她在 2013-2019 又用同一個公司「假進貨真付款」掏空 1.085 億被判 9.5 年。最諷刺的是：救蘋果西打的人，也是後來掏空它的人。但這個矛盾可能太狠，主 session 自行斟酌。

### Backup 候選 2

> **「蘋果西打不是被熱炒店冰箱養大的，是被三次外國商標所有權轉手 — 謝斯尼斯 → 果是高 → 賴比瑞亞商果是高國際 → 孫幼英 — 才終於完全『屬於台灣』。」**

理由：揭穿原文章「美台混血」這種扁平標籤背後，真實的跨國商標流轉 30 年才落地台灣個人手上。

## 四、配方真相 deep dive（critical correction）

### 美國配方公司的正確名稱

原文章寫：「美國芝加哥可士可公司（Cosco）獨家特製的 Apple Sidral 配方」

**正確版本（多源 verify）**：

| 來源                                      | 公司名稱寫法                                                                              |
| ----------------------------------------- | ----------------------------------------------------------------------------------------- |
| 維基百科 大西洋飲料條目                   | **謝斯尼斯（CosCo）**                                                                     |
| 維基百科 蘋果西打品牌條目                 | 未具體寫名稱，僅「由美國引進配方」                                                        |
| 報時光（UDN）                             | **可士可**                                                                                |
| ETtoday 1943180                           | **謝斯尼斯CosCo**（初源）→ **果是高**（1980 轉手）→ **賴比瑞亞商果是高國際**（1989 轉手） |
| CTWANT 108152                             | **謝斯尼斯CosCo**                                                                         |
| 蘋果西打官方網站 / Roomie / weeklyhistory | 完全未提具體公司名                                                                        |

**結論**：原文章「Cosco」拼寫**錯誤**（CosCo 大寫 C 中間），且「可士可」是某些媒體的中文音譯選擇而非官方名稱。「謝斯尼斯」才是中文媒體 + 維基百科最常見的版本。重寫時應使用「**美國謝斯尼斯（CosCo）公司**」並補充商標流轉鏈：

```
1965 謝斯尼斯CosCo (原研發公司)
→ 1980 美商果是高公司 (轉手)
→ 1989 賴比瑞亞商果是高國際公司 (轉手)
→ 1995 孫幼英以 80 萬美元買回（蘋果西打才真正屬於台灣）
```

### Apple Sidral vs Apple Sidra

- **Apple Sidral**：原配方名（西班牙語 sidral = 氣泡飲）
- **Apple Sidra**：台灣品牌名英文標示
- 兩個拼法在不同來源混用，原文章直接寫「Apple Sidral 配方」可保留，但需註明「Sidral 是西班牙語對氣泡蘋果汁的泛稱，墨西哥 Sidral Mundet 為著名同類產品」以避免誤以為是獨家專名

### Hard Cider vs Soft Cider vs 蘋果西打

- **Hard Cider**：發酵蘋果酒（含酒精）
- **Soft Cider**：未發酵純蘋果汁
- **蘋果西打**：碳酸水 + 蘋果汁/香料 + 焦糖色素 + 檸檬酸的氣泡果汁汽水（不含酒精）

§ verbatim 維基百科：「不含酒精，其成份含有蘋果汁、蘋果香料及糖漿等。」

§ verbatim Roomie：「自1965年就開始在台灣販售，蘋菓西打的生產靈感來自盛行於美加地區的飲品『西打』（Cider）。那是一種以蘋果汁加工製成的飲料」

### 「整顆蘋果包括果肉果核次級品榨汁」說法

**未驗證**。原文章寫「據傳使用整顆蘋果榨汁，包含果肉、果核，甚至次級或有瑕疵的蘋果」— 此說法在搜索中找不到大飲官方或新聞 source，疑似源自網路傳言或老一輩口耳相傳。重寫時應**移除**此段或標註「網路流傳說法，未經大飲官方證實」。

- 2023：天花板發黴滴水、管線生鏽斑駁 → **白色沉澱物**
