# Research Report: 高雄市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：高雄港（1968 中島貨櫃中心啟用 + 1969 起超越基隆港）／愛河（2000 後整治成功）／85 大樓（1997 高雄地標）／旗津／西子灣／中央公園／六合夜市／瑞豐夜市／美麗島事件 1979/12/10（新興區）／2010/12/25 縣市合併升格直轄市／柴山（壽山）獼猴／苓雅高雄展覽館／衛武營國家藝術文化中心 2018 開幕／85 大樓
- **世代差異**：
  - 戰前老世代知「打狗」（高雄舊名 + 1920 改高雄街 + 1924 升格高雄市）
  - 戰後 50-80s 世代知「重工業港都」（中鋼 + 中油 + 大寮拆船 + 林園石化）+ 「大寮拆船王國 1960s-1980s」
  - 80-90s 世代知「美麗島事件 1979/12/10」+ 「1979/07/01 升格直轄市」（第二個直轄市，僅次台北 1967）
  - 千禧後世代知「愛河整治 2000s 後成功」+ 「2010/12/25 縣市合併升格直轄市」+ 「衛武營 2018 開幕」+ 「85 大樓 1997」
- **外人 vs 在地人**：外人覺得「高雄吃六合夜市 + 看 85 大樓 + 愛河遊船」就走人；在地人知「我們是 273 萬人全台第三大都市 + 全台第二個直轄市」「大寮拆船全盛 1960s-1980s 是全球拆船王國第一 + 然後環保開放 1989 衰退」「中鋼 + 中油 + 林園石化是重工業 + 高雄是台灣戰後工業化的起點」「美麗島事件 1979/12/10 + 黃信介 + 施明德 + 呂秀蓮 + 陳菊都在這裡」「2010 縣市合併讓鳥松到甲仙到那瑪夏變同一個市」

### 多元面貌

- **主流敘事**：「重工業港都 + 美食天堂」（過度功能化）vs 「衰退中的工業城」（過度悲觀）
- **支線敘事**：
  - **1924/12/25 高雄街升格高雄市**（大正 13 年，市役所在哈瑪星）+ **1979/07/01 升格直轄市**（第二個直轄市）+ **2010/12/25 縣市合併**（原市 + 原縣合併為今高雄市）
  - **1968 中島貨櫃中心啟用 + 1969 起超越基隆港**（per cities-series-planning + 基隆.md verbatim）
  - **1960s-1980s 大寮拆船全盛**：全球拆船王國 + 拆解廢船提煉鋼鐵 + 1989 環保開放後衰退
  - **重工業三巨頭**：中鋼（1971 中鋼建廠）+ 中油（楠梓煉油廠 1968）+ 林園石化（1973 設立）
  - **1979/12/10 美麗島事件**：新興區（中山一路新興圓環 + 大港埔圓環）+ 黨外運動關鍵 + 全台戒嚴史最重要事件之一
  - **愛河整治 2000s**：1997 第一波清污 + 2000s 後成功 + 高雄市觀光形象轉型
  - **2010 縣市合併**：原高雄市 + 原高雄縣合併 → 38 區（原市 11 區 + 原縣 27 鄉鎮市區）
  - **38 區**：原市區（11 區）+ 原縣 27 區（涵蓋大寮 / 鳳山 / 仁武 / 燕巢 / 大樹 / 林園 / 美濃 / 茄萣 / 甲仙 / 桃源 / 那瑪夏等 + 山地原住民 3 鄉：桃源 / 茂林 / 那瑪夏）
  - **88 風災 2009/8/8** 那瑪夏 + 桃源 + 茂林山地原住民鄉重災（雖然小林村在原台南左鎮但桃源 + 茂林也重）
- **fault lines**：
  - 原市區 vs 原縣鄉鎮市區（2010 合併後行政整合）
  - 重工業 vs 衛武營藝術 vs 觀光（高雄三層產業時代）
  - 1979 直轄市升格 vs 2010 縣市合併（兩次升格 31 年）
  - 美麗島事件 1979/12/10 vs 今天觀光化
  - 大寮拆船 vs 環保開放（1989 後衰退）
  - 山地原住民鄉（桃源 / 茂林 / 那瑪夏）vs 平地 35 區

### 想法跟感受

- **正面**：高雄人對「第二個直轄市」「重工業港都」有強烈自尊 / 對中鋼 + 中油 + 林園石化 + 高雄港的工業驕傲 / 對愛河整治成功 + 衛武營 + 85 大樓的轉型自信
- **負面**：「重工業 = 污染」標籤的疲乏 / 大寮拆船王國衰退的失落 / 美麗島事件的歷史傷口
- **矛盾感**：1924 升格高雄市 vs 1979 升格直轄市 vs 2010 縣市合併 — 三次升格 86 年 / 美麗島事件 1979 跟同年升格直轄市同時發生 / 重工業港都 vs 衛武營藝術中心 — 兩種高雄

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 馬卡道族（平埔族）千年居住 + 西拉雅族部分
  - 1624 荷蘭時代後漢人移民開拓
  - 1684 鳳山縣設於今高雄（含屏東部分）
- **關鍵轉折**：
  - 1858 天津條約 + 1863 打狗開港（英法等列強通商口岸）
  - 1895 日治：打狗 → 1920 改高雄街
  - 1924/12/25 高雄街升格高雄市（大正 13 年）+ 市役所在哈瑪星（大田大正町）
  - 1945 戰後：高雄市 + 高雄縣分治
  - 1968 中島貨櫃中心啟用 + 高雄港擴建
  - 1969 起高雄港超越基隆港（per cities-series-planning + 基隆.md）
  - 1971 中鋼建廠（戰後工業化第一波）
  - 1973 林園石化設立
  - 1979/07/01 升格直轄市（第二個，僅次台北 1967）+ 小港鄉併入小港區
  - 1979/12/10 美麗島事件（新興區）
  - 1997 85 大樓完工（高雄地標 + 348m 全台第二高）
  - 2000s 愛河整治成功 + 高雄觀光轉型
  - 2009/08/08 88 風災 那瑪夏 + 桃源 + 茂林山地原住民鄉重災
  - 2010/12/25 高雄縣市合併升格直轄市 + 38 區
  - 2018 衛武營國家藝術文化中心開幕
- **當代意義**：高雄是台灣「重工業港都轉型 + 兩次升格直轄市」的縮影。1924 高雄市 → 1968 中島貨櫃中心 → 1971 中鋼 → 1979 直轄市 + 美麗島事件 → 2010 縣市合併 → 2018 衛武營 — 七個關鍵節點撐起高雄當代意義

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 高雄是台灣「戰後工業化」最具代表性的城市（1968 中島貨櫃中心 + 1971 中鋼 + 1973 林園石化）
- 1969 高雄港超越基隆港是台灣經濟重心南北翻轉的歷史節點（per 基隆.md cross-link）
- 1979/12/10 美麗島事件是台灣黨外運動最重要事件（5 大黨外領袖：黃信介 + 施明德 + 呂秀蓮 + 陳菊 + 林義雄）
- 1979/07/01 升格直轄市同年 + 12/10 美麗島事件 — 高雄的兩個 1979（行政升格 vs 政治壓制）
- 2010 縣市合併是台灣「行政升格」最具規模案例之一（38 區 + 273 萬人）
- 88 風災 2009 那瑪夏 + 桃源 + 茂林山地原住民鄉重災（per 屏東縣 sibling 8 月 8 日相同時期）
- 對讀者：「為什麼高雄同時是重工業港都 + 美麗島事件發生地 + 衛武營藝術中心？」這個 frame 拆開後讓人看見「三層時代」的城市
