# Research Report: 落日飛車 Sunset Rollercoaster

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **西門紅樓地下室 / 海邊的卡夫卡 / 信義誠品書店 broadcast** — 2010s 中後期的週末，台北學生族群第一次在獨立樂團 live house 聽到一支唱英文卻完全融進台北夕陽氣味的樂團
- **2017〈My Jinji〉爆紅那段** — Spotify / YouTube algorithm 把這首歌推到全球 indie playlist，台灣樂迷第一次在 IG 看到「美國朋友傳訊息問你這是台灣團嗎」
- **2023 (or 2019？需 verify) Coachella 大草地的中華民國國旗** — 在沙漠舞台前揮舞國旗的瞬間，台灣樂迷把「indie viral」轉成「主權能見度」
- **世代差異**：5-6 年級多半沒聽過 / 7 年級透過信義誠品書店 broadcast 或女巫店、海邊卡夫卡 live 認識 / 8-9 年級透過 Spotify〈My Jinji〉or YouTube algorithm 認識 / 9-10 年級透過 Coachella 國際舞台 + Hyukoh 合作回流

### 多元面貌

- **主流敘事**：「台灣最國際化的獨立樂團」「Coachella 神話」「City Pop 復興浪潮代表」
- **支線敘事**：
  - **詩意的算計**：選英語不是國際野心，是「讓聽眾自己去發現意義」（per hub 既有引語）
  - **JinJi Kikko 廠牌經營**：國國成立自己的 label 並簽其他亞洲樂團，是 ecosystem 工作者不只是 frontman
  - **亞洲橫向合作軸**：跟韓國 Hyukoh 合作《AAA》(2024) 是避開西方中心化的 Asian indie 互聯
  - **解散與重組的低潮**：2011 Bossa Nova 後一度解散，2015 才重組，這段消失的四年是反 K-pop 工業傳奇的關鍵反差
- **被忽略的角度**：
  - 主唱曾國宏曾在 Manic Sheep（更早期的台灣 shoegaze 樂團），落日飛車不是憑空冒出
  - 樂團名字來自 MySpace Photo Booth 隨手拍的照片背景（雲霄飛車和夕陽）— 起源無厘頭，反而是反工業敘事的縮影
  - 全英語演唱在台灣 indie 圈是少數派路線，多數選台語 / 國語 / 客語 / 族語（per hub「每一種語言選擇背後都是一個關於我是誰的回答」）
- **Fault lines**：
  - 「他們代表台灣」vs「他們已經是 global」
  - 「全英語 = 失去本土性」vs「在台北錄音、用台北氣味做出的音樂，語言只是 surface」

### 想法感受

- **樂迷**：榮耀 + 親密 + 共謀感（小團變大團的全程見證）
- **同代台灣音樂人**：羨慕 + 困惑「怎麼做到的」+ 認知到「international 的路有很多種」
- **文化論者**：「全球化的勝利」vs「英語霸權的隱性接納」雙重 framing
- **國際樂評**：NME / Pitchfork / Stereogum 詫異於「亞熱帶 city pop」這個概念的成立
- **K-pop / J-pop 觀眾交叉**：透過 Hyukoh 合作回流的韓國樂迷，覺得這是「跟 K-indie 平行的 T-indie」

### 歷史脈絡（pre-search hypothesis）

- **形成期**：2009 年台北 MySpace 時代成軍（主唱曾國宏 + 王少光 + ...初代陣容待 verify）
- **第一波**：2011 年首張專輯《Bossa Nova》（lo-fi / Surf Rock 路線，跟後期 City Pop 風格差很大）→ 解散
- **沉潛期**：2011-2014 解散 / 各自單飛
- **第二波**：2015 年重組 → 2016 (or 2017？需 verify) EP《JINJI KIKKO》international viral → 2018 Audiotree session → 2020 LP《Soft Storm》NME 亞洲年度第四 (Ned Doheny 合作)
- **巔峰**：2023 (or 2019/2022？需 verify) Coachella 登台 → 2024《AAA》(Hyukoh 合作) 36 屆金曲獎 7 項提名
- **當代意義**：證明台灣音樂可以「不靠 K-pop 工業體制」自然觸及全球；證明 indie 可以從 Photo Booth 一張背景照走到沙漠音樂祭

### 預想讀者帶走的那一件事
