## 觀點成型（編輯前思考）

### 此次 EVOLVE 的 framing 鐵律（哲宇 2026-05-22 拍板）

這是**比單一 thesis 更難寫**的 framing — 需要 disciplined journalism 而非 essay opinion，需要克制「下結論」的本能。

### 對台灣人的記憶 anchor

**世代差異**：

- **30-40 歲（1985-1995 出生）**：22K 那個總統 / 太陽花 3/24 創傷 / 馬英九迷因（鹿茸、便當、普羅旺斯永久屋、死亡之握）
- **40-60 歲**：兩岸破冰 / ECFA 紅利期 / 不沾鍋 / 哈佛博士 / 1998 北市長 / Long Stay 競選
- **60+ 歲 / 外省第二代**：清廉君子 / 蔣經國的英文秘書 / 馬鶴凌「化獨漸統」家訓
- **00 後**：聽過但不熟，主要透過 meme / 父母提及

**共同 anchor 畫面**：

- 2015/11/7 新加坡香格里拉與習近平握手 80 秒
- 2014/3/30 太陽花 50 萬人凱道
- 2013/10 TVBS 民調 9.2% 數字曝光那天

- 「23K 變 22K」薪資對話 meme
- 2023/3 訪中與習近平合照 + 西安拜祭黃帝陵

### 多元面貌

**主流敘事**：清廉但無能總統 / 兩岸關係黃金期締造者

**支線敘事**（每條都要寫進文章）：

1. **外省第二代政治菁英在台灣 democratization 中的 identity**（蔣經國拔擢 → 1988 大陸工作 → 1993 法務部長 → 2008 總統的軌跡）
2. **中產階級偶像 (1998-2008) → 「天龍國」標籤 (2008-2014)** 的形象崩塌
3. **法律人從政（哈佛 SJD）vs 政治判斷力**：「他用最不政治的方式打了一場政治仗」（王金平案）
4. **22K 經濟學爭議**：是 caused 薪資停滯，還是 symptom of structural shift（中國磁吸 + 全球化）？兩派並陳
5. **ECFA 實際經濟效益分歧**：出口統計 vs 受益分布 vs 中小企業未受益
6. **馬習會的歷史意義 vs 對台灣主體性的 implication**：兩派並陳
7. **卸任後 frame 轉變**（破冰者 → 「親中」標籤）：兩派評價並陳
8. **被忽略**：莫拉克救災 (2009/8) + 反思 / 黃金十年破滅 / 食安連環爆 (2011 塑化劑 + 2013 大統油 + 2014 餿水油)

### 歷史脈絡（pre-search hypothesis）

**形成期**：外省二代政治菁英 1950s-1970s 國民黨培養 → 蔣經國「青年人才」拔擢 → 1981 英文秘書 → 1988 大陸工作主管 → 1993 法務部長

**關鍵轉折**：

1. 1998 北市長 → 非主流(本省派)壓力下崛起
2. 2005 國民黨主席 → 2008 大選 → 兩岸開啟
3. 2012 連任 51.6% → 2013 反核四 + 9.2% 民調 + 王金平案
4. 2014 太陽花 + 九合一大敗 → 政黨輪替不可逆
5. 2015 馬習會 → 卸任前最後 legacy 衝刺
6. 2023 訪中 → 卸任後 framing 重塑

**當代意義**：

- 馬英九時代是台灣最後一次「中華民族認同」總統的可能性（蔣經國 → 李登輝 → 陳水扁 → 馬英九 → 蔡英文 漸進 Taiwanization 軌跡的最後一步回望）
- ECFA + 兩岸路線可被視為「香港化試驗」的台灣版本（2019 香港反送中讓這個視角 retrospectively 被看見，但不下結論）
- 22K 是新自由主義 + 全球化 + 中國磁吸效應的台灣集中爆發
- 太陽花是台灣公民社會成熟標誌 — 是回應馬英九時代矛盾累積

### 切入點清單（待 Stage 1 搜尋驗證 / 反駁）

每個切入點都要找到具體 scene anchor，不是論述句：

1. **「不沾鍋」的清廉真實感**：1993-2006 法務部長 + 北市長 + 特別費案的具體判決細節 + 家人不收政治紅包的場景
2. **2008 海嘯下接班的具體時刻**：金融海嘯背景 + 蕭萬長搭檔 + 「準備好了」競選 slogan + 就職演說具體段落
3. **22K 政策設計的房間裡發生了什麼**：勞委會 vs 主計處 vs 經建會 2009 春的內部協商紀錄（搜尋有沒有當事人 oral history）
4. **2010 ECFA 簽署當天**：江丙坤陳雲林簽字場面 + 在野黨抗議畫面 + 早收清單實際受益產業
5. **2013 王金平案記者會**：馬英九講話原文 + 司法獨立性爭議 + 國民黨內部分裂的具體 quote
6. **2014/3/23-24 行政院強制驅離**：當晚現場記者報導 + 受傷者口述 + 警方指令鏈 + 馬英九事後回應原文
7. **2015/11/7 馬習會 80 秒握手 + 雙方致詞**：場景細節（西裝顏色、握手時長、現場新聞鏡頭）+ 雙方致詞原文 + 兩岸後續解讀差異
8. **2023/3 訪中具體行程**：南京中山陵 + 西安黃帝陵 + 武漢長沙重慶具體日程 + 演講原文「兩岸同屬中華民族」上下文 + 台灣內部反應 + 國民黨內部評價
9. **卸任後的兩種詮釋並陳**：馬英九基金會動態 + 國民黨改革爭議 + 兩岸最後溝通管道論 vs 賣台論

### 預期核心矛盾候選（per 哲宇 framing：不收斂為單一，多個並存）

### 預想讀者帶走的那一件事

不是「馬英九好 / 不好」的判斷，是**對「複雜政治人物」的多面向理解能力**：

> 讀完後讀者能更精準說出「我認為馬英九的 X 政策有 Y 效果，但 Z 面向值得肯定，至於 W 我還在思考」這種多軸 nuanced view，而不是「他賣台」或「他清廉」這種單軸判斷。

對應 MANIFESTO §策展式不是百科式：策展是給空間讓讀者形成自己的看法，不是給結論。

**故事 1：2007/2/13 一日內被起訴與宣布參選**

2007 年 2 月 13 日，台北地檢署依貪污治罪條例起訴馬英九特別費案。同日下午，馬英九召開記者會宣布辭去國民黨主席並參選 2008 年總統。場景張力：上午被起訴、下午宣布最高政治職位的競選。一個法律人面對司法指控的應對方式，用「大水庫理論」抗辯——「公私帳混合，因金錢具流動性，只要進出總額平衡即無犯罪」——獲法院採納，最終一二三審全部無罪。

**故事 3：2014/3/23-24 行政院強制驅離當晚**

3 月 23 日晚 7:35 數百學生試圖佔領行政院。3 月 24 日凌晨警政署長王卓鈞向江宜樺報告勸離無效，江宜樺下令依法執行驅離。警方自凌晨 0 時起發動六波強制驅散行動，運用水砲與警棍。清晨 5 時全面驅離完畢，造成 150 多名抗議者與警察受傷流血。監察院歷經 9 年調查（2023 年發布報告）認為「警察機關以逾越比例原則之暴力手段執行驅離」並糾正行政院/內政部/警政署/北市警/北檢。江宜樺事後堅稱「我不僅不後悔，而且我覺得我再也沒有下過一個比這個更正確的決定」。馬英九與江宜樺後被以殺人未遂提告，2020/9/15 一審獲判無罪。
