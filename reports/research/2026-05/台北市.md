# Research Report: 台北市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：總統府 / 中正紀念堂 / 國父紀念館 / 101 大樓 / 信義計畫區 / 西門町 / 萬華龍山寺 / 北投溫泉 / 故宮 / 大稻埕 / 中正區政府機關密集 / 捷運 1996 起站 / 圓山大飯店 / 陽明山 / 三軍總醫院 / 台大 + 師大 + 政大
- **世代差異**：
  - 戰前老世代知「1920 設台北市」「1923 裕仁皇太子訪台」「1945 國府接收」「1947 二二八事件第一槍在天馬茶房」
  - 戰後 50-70s 世代知「1967 升格直轄市」「1968 行政區擴大納入士林北投南港內湖木柵景美」「1972 中山高通車」「1979 中美斷交」「1981 第一家麥當勞民生東路」
  - 80-90s 世代知「1990 行政區重劃 16 → 12 區」「1996 捷運通車」「2004/12/31 101 落成」「2010/05/22 花博」
  - 千禧後世代知「2017 世大運」「2020 疫情封城但台北沒封」「2024 罷免運動」
- **外人 vs 在地人**：外人覺得「101 + 故宮 + 鼎泰豐 + 西門町」就走人；在地人知「12 區裡有同個城的三種年紀：信義萬華北投」「萬華是台北最老的部分」「萬華 vs 信義是同個城的兩端」「北投溫泉開發比台北市建府還早」

### 多元面貌

- **主流敘事**：「天龍國 / 政治經濟中心 / 全台最富」（一種優越感的反射）
- **支線敘事**：
  - **首都的內部分層**：信義金融區 vs 萬華傳統 vs 北投溫泉 vs 文山木柵山區 — 12 區同一個城三種年紀
  - **1875 設台北府**（沈葆楨奏請，1885 台灣建省，劉銘傳首任巡撫）
  - **1920 設台北州 + 台北市**：日治大正九年地方制度改革，以艋舺 + 大稻埕 + 城內三市街為基礎
  - **1947/02/27 二二八事件第一槍**：天馬茶房（今南京西路 189 號附近）+ 林江邁 + 緝煙血案 → 02/28 行政長官公署廣場開槍 → 全台
  - **1949 國府遷台**：蔣中正 + 政府機關 + 約 120 萬軍民湧入，台北人口從 30 萬激增至 50 萬+
  - **1967 升格直轄市**：第一個直轄市，當時人口約 130 萬
  - **1968 行政區擴大**：原台北縣的南港 + 內湖 + 木柵 + 景美 + 陽明山管理局士林 + 北投 全部劃入
  - **1980 信義計畫區**：原為日治軍營區 → 重劃為金融商業中心 → 2004 101 落成 → 2014 信義 ATT4FUN
  - **1990 行政區重劃**：16 區 → 12 區（古亭分為中正 + 大安，景美併入文山等）
  - **1996/03/28 捷運木柵線通車**：第一條捷運（後改名文湖線）
  - **二二八紀念館 1997**：原台北放送局 → 2/28 公園 → 1997 立館
- **fault lines**：
  - 信義金融 vs 萬華傳統（同個城的兩端）
  - 北投溫泉開發 vs 凱達格蘭族（凱達格蘭大道命名爭議）
  - 1947 二二八 vs 1949 國府遷台兩種記憶（外省 vs 本省）
  - 1967 升格直轄市 vs 1968 擴大行政區（在地史 vs 政治擴張）
  - 中正紀念堂 vs 民主紀念館（2007-2008 命名爭議）
  - 萬華 vs 大同 vs 中正：三個老區的不同邏輯
  - 大稻埕茶葉時代 vs 西門町電影街 vs 信義金融的世紀更迭

### 想法跟感受

- **正面**：台北人對 1875 設府 + 1920 設市 + 1967 直轄市的歷史傲氣 / 對中正紀念堂 + 101 + 故宮的政治符號地位的複雜感受 / 對萬華 + 大稻埕老街區的深情 / 對 12 區內部多元的自覺
- **痛點**：「天龍國」標籤的反作用 / 房價全台最高把年輕人趕出去 / 信義 vs 萬華的內部不均 / 1947 二二八的歷史創傷至今未癒 / 1949 後的外省/本省身份分裂 / 中正紀念堂議題的解嚴後爭議
- **驕傲**：第一個直轄市（早其他都會 43 年）/ 第一條捷運（早其他城市 12 年以上）/ 第一個現代百貨（中華商場 1961-1992）/ 全台最早的現代城市規劃（劉銘傳時代）/ 民主運動的關鍵舞台（中正廟前 1990 野百合學運）

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- **政治集中代價**：總統府 / 行政院 / 立法院 / 司法院 / 考試院 / 監察院 + 國防部 + 外交部 + 中央銀行 — 中央集權結構下「台北 = 台灣」的全國視角
- **房價結構性問題**：信義區 + 大安區房價全台最高 + 年輕人外移到新北通勤 + 北漂他鄉與在地分歧
- **二二八記憶**：1947 第一槍在台北，二二八紀念館 1997 立館，2007 中正紀念堂改名爭議 — 民主化的反向工程
- **首都的內部分層**：12 區同一個城三種年紀，外人只看 101，在地人活在萬華大稻埕
