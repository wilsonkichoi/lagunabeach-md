# Research Report: 中山北路條通

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

**世代差異明顯**：

- **戰前一代（1920s-40s）**：記憶是「敕使街道」，1923 裕仁皇太子坐馬車走過的儀式之路；圓山台灣神社是日治台灣最高國家神祇場所
- **戰後外省一代（1949-）**：記憶是「中山北路」紀念孫中山，1949 後成為國府首都門面街，外交官邸、駐外大使館、總統府前的主軸線之一
- **韓戰越戰一代（1950s-70s）**：記憶是中山北路三段的美軍俱樂部、晴光市場舶來品、第一家麥當勞前的「美國租界」
- **解嚴後文青一代（1990s-）**：記憶是條通文化（一條通到九條通）、林森北路居酒屋、《華燈初上》電視劇取景地、媽媽桑文化
- **當代少年（2010s-）**：記憶是 Mura、吞兵衛、八條通商圈、晚上 9 點居酒屋暖簾掀開的日語跟台語混講

### 多元面貌

**主流敘事**：「日治殖民道路」+「戰後外省眷村式中正紀念」+「特種行業帶」三層疊。

**支線敘事**（被忽略的角度）：

- **凱達格蘭族原始地形**：圓山貝塚（4500-2000 BP 史前遺址）就在中山北路盡頭的圓山公園 — 漢人來之前已經有人在這條軸線生活
- **日治大正町高級住宅區**：條通巷弄不是天生的酒吧街，原本是日本官員宿舍（治安最好、孩童最多的高級住宅區），日本人留下的方格巷弄才造就戰後酒吧長進來的空間結構
- **美軍與日本人「轉手」歷史**：1950s-70s 是美軍消費場（晴光 + 中山北路酒吧），1972 後美軍撤了 + 日本經濟泡沫起，日商遷入接手同一條街
- **媽媽桑跨世代文化網**：日治台灣藝妓 → 戰後美軍酒吧吧檯 → 1970s 日商陪酒文化 → 2020 《華燈初上》戲劇化記憶 → 2020s 「條通祭」嘗試文化轉型

### 歷史脈絡（pre-search hypothesis，搜尋後修正）

- **形成期 1898-1923**：日治市區改正規劃「敕使街道」（前往台灣神社的儀式之路）；同期大正町作為日本官員宿舍區仿京都町割
- **1923 裕仁訪台**：皇太子（後昭和天皇）坐馬車走過敕使街道，這條路被「定錨」為日治台灣最高政治儀式空間
- **1937-1941 擴建**：日治後期擴建為 40m 五線道，戰時建材稀缺仍堅持完成 — 證明這條路在帝國敘事中的地位
- **1945-1949 改名期**：戰後接管，敕使街道改名「中山北路」紀念孫中山，明治橋改名「中山橋」 — 政權更迭，地名換相
- **1950-1979 美軍世紀**：韓戰開始 → 美軍駐台 → 中山北路三段成「美國租界」 → 晴光商圈、美軍俱樂部、酒吧（不是日式酒店，是 American Bar 風格）
- **1972 中日斷交 + 日本經濟泡沫**：日本商社改為交流協會台北事務所，但日本商人沒撤，反而 1970s-80s 日本經濟泡沫期更多日商來台 → 條通酒吧文化從 American Bar 變 Japanese izakaya
- **1979 美軍撤 → 1980s 條通文化成型**：晴光市場蕭條，日商接手同一條街的酒吧位置，「七條通」「八條通」「九條通」名稱在 1980s-90s 流行音樂與文學中固化
- **1996 凱達格蘭大道改名**：陳水扁市長把總統府前「介壽路」改名為「凱達格蘭大道」 — 中山北路一段端點同時做了一次「去殖民」的去名動作
- **2010s 後媒體放大**：《華燈初上》（2021-2022）戲劇化 1988 條通文化記憶，年輕世代開始重新發現條通

### 當代意義

這條路本質上是**台灣外交史與權力轉換的物質地景**：

1. 日本帝國（1898-1945）：從敕使街道到台灣神社
2. 中華民國（1945-）：中山北路紀念孫中山 + 五院首都門面
3. 美國保護國階段（1950-1979）：美軍俱樂部 + 晴光商圈 + 越戰 R&R
4. 後美軍 / 後斷交時期（1972-）：日本商社聚集 + 條通文化

**外人看見的是日式居酒屋帶**；**台北人活的是三個外國帝國依序疊上去的軸線**。

### 切入點清單（待搜尋驗證 / 反駁）

1. **「同一條街換過三個外國名字」**：敕使街道 → Zhongshan North Road → Tiao-tong → izakaya street（每個名字都是不同帝國的接管痕跡）
2. **大正町仿京都町割如何成為戰後酒吧街的物質基礎**：方格巷弄結構（不是天生的酒吧街，是日治規劃的住宅區意外給了酒吧空間）
3. **美軍 vs 日商「接力」的物質證據**：晴光市場 → 1979 蕭條 → 1980s 日商接手；老建築仍在
4. **媽媽桑文化的跨世代生存**：日治藝妓 → 戰後美軍吧檯 → 1970s 日商陪酒 → 2020s 條通祭
5. **凱達格蘭大道+ 敕使街道 = 一條路的兩個端點對稱**：1923 日治皇太子儀式之路，1996 改名為原住民族紀念之路

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- **A**：日本人鋪的去神社的路，現在是日本商社最密集的街 ⭐ (planning doc 既定)
- **B**：每換一個帝國，這條街就換一個名字，但同一條巷子的酒吧一直都在
- **C**：條通不是天生的，是日治高級住宅意外給了戰後酒吧文化的物質地基

**選 A**：planning doc 給的核心矛盾仍是最強 — 「日本人鋪 → 日本人回來」的時間迴圈在 30 字內最 sharp。**但 B 跟 C 作為次層編織進文章中段**（「換名字」這個物理動作做文章 spine 的呼應 anchor / 大正町方格的物質基礎做歷史推導的支撐）。

# Stage 1 取材（搜尋 + 驗證）

## Stage 1 高信心事實 (40+ searches done)

### 敕使街道 + 中山北路時間軸

- 1898 起日治時期「市區改正」規劃道路 → 敕使街道前身 15 公尺寬路
- 1901-10 第一代明治橋（Parker truss 鐵橋 + 木板橋面）由總督府土木技師十川嘉太郎設計
- 1901 台灣神社創建（圓山，劍潭山麓）
- 1912 明治橋橋面改為鋼筋混凝土
- 1922 台北市町名改正：原大正街臨敕使街道部分編「御成町」（東側其餘部分為「大正町」）
- 1923-04-17 日本皇太子裕仁訪台拜台灣神社，走敕使街道
- **1930-01-25 第二代明治橋鋼筋混凝土圓拱橋動工（為防 1923 關東地震教訓） → 1933-03-20 完工（總長 120 公尺、寬 17 公尺、車道 10 公尺 + 行人步道 3.5 公尺 × 2 + 花崗石欄杆 + 青銅燈柱）**
- **1936 起策劃敕使街道擴建（為紀念日本紀元 2600 年 1940 + 台灣神社擴建）**
- **1937-03-30 敕使街道擴建動工典禮**
- **1941-03-28 上午 11 時 台灣神社舉行竣工奉告祭。路寬 40m 五線大道 / 工程費 162 萬餘日圓 / 分隔島樟樹 / 人行步道楓樹 / 水銀路燈 / 電線地下化**
- 1944 台灣神社升格為「神宮」（戰爭末期增祀天照大神）

### 戰後接管

- 1945-10-25 上午 10 時 安藤利吉於台北公會堂（今中山堂）向陳儀投降，台灣光復
- 戰後敕使街道改名「中山北路」紀念孫中山；明治橋改名「中山橋」
- 1947-02-28 下午 1 時 群眾從台北車站走向行政長官公署請願，剛到中山路路口（今中山南路 / 北路口）就被衛兵以機槍掃射，造成 ≥2 死。同日台北市參議會在中山堂（前公會堂）召開緊急會議。1947-03-01 二二八事件處理委員會在中山堂組成
- 1949 國府遷台，中山北路成為首都中軸線；總統府位於中山北路一段南端的延伸（介壽路）
- 1952-05-10 圓山大飯店成立（蔣宋美齡領導「台灣敦睦聯誼會」接管原台灣神社地塊，孔令偉為首任總經理）

### 美軍世紀

- **1950-06-25 韓戰爆發**
- 1951 美軍顧問團 MAAG 進駐台灣，中山北路三段一帶成為「美國租界」；晴光市場成型（民國 40 年）
- 1953-05 美國駐華大使官邸啟用（中山北路二段 18 號，1926 年建維多利亞式洋樓，原為駐台北領事館 1949 升格總領事館 1953 改大使官邸）
- 美軍協防台灣司令部 USTDC（前身「臺灣聯絡中心」1955-04-26 創立 → 1955-11-01 改名 USTDC）總部設於中山北路三段圓山段（**現址為臺北市立美術館**）
- 美軍顧問團 MAAG 也在民族東路與中山北路間數公頃土地興建勤管司令部營區（**今中山美術公園**）
- 1953 天母美軍宿舍群（陽明山美軍宿舍）動工，中山北路 7 段 181 巷 23 號等
- 1959-08-07~09 八七水災，USTDC 動員 3 架直昇機 + 21 架塞考斯基 H-34（「飢餓行動」Operation Hunger）救災
- **1963-08-26 臺北美軍招待所成立**（1965-1972 越戰 R&R 期間招待數十萬駐越美軍）
- 1968 九台街改稱「林森北路」（前國民政府主席林森百歲冥誕 1868-2-11 出生 / 1943-8-1 逝世）

### 中日斷交與條通文化轉型

- **1972-09-29 上午 北京人民大會堂東大廳 田中角榮 + 大平正芳 vs 周恩來 + 姬鵬飛 簽署《中日聯合聲明》→ 同日中華民國以「漢賊不兩立」宣布與日本斷交**
- 1970s-80s 日本經濟泡沫期，日商大量來台設立分公司；條通巷弄從美式 Bar 轉為日式居酒屋帶（最盛時期 400+ 店家同時營業）
- **1979-01-01 美國與中華人民共和國正式建交（同日卡特政府與中華民國斷交）**
- **1979-02-22 華府宣布 3-01 取消美軍顧問團 MAAG 名稱**
- **1979-04-26 美軍協防台灣司令部 USTDC 舉行最後一次降旗典禮**
- **1979-04-28 最後一任協防司令林德少將 (Rear Admiral James B. Linder) 離台**
- **1979-05-03 最後一名美軍離台**
- 1979 後晴光市場逐漸蕭條

### 1990s 後

- 1985 莎羅曼庭婚紗攝影公司於中山北路二段成立，成為地區第一間婚紗攝影公司
- 1989-03 錢櫃 KTV 第一家店於林森北路開業（同年底拓展至 5 家）
- 1989 中山足球場（原圓山運動場）完工（造價 NT$3 億）— 場址前身：日治 1923 圓山運動場 → 戰後美援物資倉庫 → 美軍顧問團辦公地點 → 1979 後改建為足球場
- 1996-03-21 陳水扁市長將總統府前「介壽路」改名「凱達格蘭大道」
- 1997-02-04 林森公園 + 康樂公園範圍內住戶收到拆遷通知（前身為日治三橋町三板橋日人公墓，戰後成為違章建築 + 眷村，含明石元二郎總督墓 + 乃木希典母親墓）；1997-03 強制拆除，社區老兵翟所祥屋內上吊自盡 — 台灣首場大型反拆遷運動
- 1997-02-20 內政部把前美國大使官邸列為市定古蹟（第三級古蹟）
- 1997 廢娼條例（陳水扁市長任內）
- 2000 公娼正式廢除
- 2002-10-31 台北鐵路地下化萬板段全線完工
- 2002-11-10 光點台北（前美國大使官邸）正式開館（侯孝賢台灣電影文化協會經營，台積電文教基金會贊助修復）
- **2002-12-20 中山橋（原明治橋）拆除，切成 435 塊保存於再春游泳池原址**
- 2021-11-26 Netflix《華燈初上》第一部上線（以 1988 年林森北路條通文化為背景，七條通取景，光酒店原址為 Sugar Bar，林森北路 133 巷實景）
- 2024 條通商圈受疫情衝擊，台北市條通商圈發展協會推「台灣蘭桂坊」概念輔導業者轉型
