# Research Report: 桃園市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：桃園國際機場（1979/02/26 啟用，台灣對外大門）／半導體產業（桃園台積電 + 群創 + 友達）／移工聚集（東南亞外籍移工最多縣市）／客家比例全國最高之一（近 80 萬客家人）／大溪老街（巴洛克紅磚街屋 + 豆干）／中壢區（族群混居）／2014/12/25 升格直轄市最快的縣 / 龍潭茶 + 客家擂茶 / 大園 / 觀音
- **世代差異**：
  - 戰前老世代知「桃園廳」（日治 1901-1920 + 1920 改新竹州桃園郡 + 中壢郡 + 大溪郡）+「埤塘文化」（桃園台地數百埤塘灌溉）
  - 戰後 50-70s 世代知「客家移民縣」（中壢市 + 龍潭 + 楊梅 + 新屋 + 觀音五大客家鄉鎮）
  - 80-90s 世代知「桃園國際機場 1979 啟用」（取代基隆港成為台灣門戶）+ 「龍潭石門水庫 1964」
  - 千禧後世代知「2014 升格直轄市 + 移工縣 + 半導體 + 桃園機場捷運 2017 通車」
- **外人 vs 在地人**：外人覺得「桃園轉機 + 大溪豆干 + 中壢車站」就走人；在地人知「我們桃園 230 萬人是全國第五大都市」「中壢車站附近的東南亞商店街有 12 國移工」「大溪豆干跟新竹米粉、東港黑鮪魚一樣都是地方代表食物」「2014 升格直轄市那年我們從『桃園縣』變『桃園市』，6 個鎮市變區，但 14 鄉鎮市的多元性還在」

### 多元面貌

- **主流敘事**：「機場縣 + 半導體 + 移工」（過度功能化）vs 「客家庄 + 大溪老街」（過度地方化）
- **支線敘事**：
  - **2014/12/25 桃園升格直轄市**（從桃園縣升格，是全台升格最快的縣份 — 從 1950 設縣到 2014 升格僅 64 年）
  - **客家比例全國最高之一**（行政院客委會調查近 80 萬客家人，是全國客家人口最多的縣份；中壢區客家約 50%）
  - **桃園國際機場 1979/02/26 啟用**：取代基隆港成為台灣國際門戶 + 1979 桃園客運站啟用 + 1980s-2000s 兩次擴建 + 2017/03/02 機場捷運通車
  - **桃園台地埤塘文化**：日治前後數百埤塘灌溉系統 + 戰後 1964 石門水庫取代部分埤塘 + 今天仍有 2,800 個埤塘殘存
  - **5 大客家鄉鎮**：中壢 + 龍潭 + 楊梅 + 新屋 + 觀音（從苗栗北上至桃園的客家移民帶）
  - **大溪老街**：清領晚期茶葉 + 樟腦輸出港 → 1919 巴洛克紅磚街屋 + 1985 古蹟保存 + 大溪豆干 + 月光餅
  - **中壢市區**：1968 中壢中山高速公路通車 + 1977 中壢事件（許信良選縣長爭議）+ 1990s 中壢車站東南亞商店街形成
  - **石門水庫 1964/06/14 完工**：當時遠東第一大水庫 + 灌溉桃園台地 / 新北 + 大溪 / 龍潭 / 復興 / 觀音水源
  - **13 區**（2014 升格後）：桃園 / 中壢 / 大溪 / 楊梅 / 蘆竹 / 大園 / 龜山 / 八德 / 龍潭 / 平鎮 / 新屋 / 觀音 / 復興（山地原住民區）
- **fault lines**：
  - 機場 vs 在地（機場員工 / 旅客 vs 桃園在地居民）
  - 移工縣 vs 漢人主流（中壢車站附近東南亞商店街 vs 客家庄 + 閩南）
  - 客家 vs 閩南（中壢 / 龍潭 / 楊梅客家 vs 桃園市區閩南）
  - 1977 中壢事件（戒嚴時期黨外運動）vs 今天觀光化
  - 復興區（泰雅族）vs 平地 12 區（漢人主流）
  - 2014 升格直轄市 vs 1950-2014 桃園縣身分（區跟鄉鎮市的差異）

### 想法跟感受

- **正面**：桃園人對「全國第五大都市 + 機場縣」有強烈自尊 / 對客家庄 + 大溪豆干 + 龍潭茶 + 石門水庫的在地驕傲 / 對 2014 升格直轄市的成就感
- **負面**：「機場縣」「轉機站」標籤的疲乏 / 1977 中壢事件被觀光化的尷尬 / 移工縣的歧視性敘事
- **矛盾感**：升格直轄市最快 vs 13 區仍保留鄉鎮市多元性 / 客家近 80 萬全國最多 vs 中壢車站東南亞商店街 vs 機場員工三層人口結構

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 凱達格蘭族 + 泰雅族（復興山區）千年居住
  - 1740s 客家移民從廣東惠州、嘉應州、潮州移居北部
  - 1786 林爽文之役後客家義民從南部六堆北上桃竹苗
- **關鍵轉折**：
  - 1812 設淡水廳含桃園地區
  - 1875 沈葆楨改革 + 桃園地區屬新竹縣
  - 1895 日治：桃園廳 1901-1920 + 1920 改新竹州桃園郡 + 中壢郡 + 大溪郡
  - 1945 戰後：桃園縣設立（從新竹州分出）
  - 1950 桃園縣行政體制確立
  - 1964/06/14 石門水庫完工（遠東第一大水庫）
  - 1968 中壢中山高速公路通車（中壢交流道）
  - 1977/11/19 中壢事件（許信良選縣長 + 黨外運動關鍵）
  - 1979/02/26 桃園國際機場啟用（取代基隆港成為台灣門戶）
  - 1980s-1990s 桃園台地工業化 + 半導體產業興起
  - 2014/12/25 桃園升格直轄市（從縣升格 64 年）
  - 2017/03/02 桃園機場捷運通車
- **當代意義**：桃園是台灣「升格最快 + 機場縣 + 客家最大 + 移工最多」的直轄市。1979 機場 + 2014 升格 + 客家 80 萬 + 12 國移工 — 四個故事撐起桃園的當代意義

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 桃園市是台灣「對外大門」（1979 桃園機場取代基隆港，台灣 95%+ 國際旅客從這裡進出）
- 客家近 80 萬人**全國最多**（行政院客委會調查），桃園是當代台灣客家最重要的縣份
- 1977 中壢事件是台灣黨外運動關鍵節點（許信良選縣長爭議 + 戒嚴時期街頭運動）
- 2014 升格直轄市是「行政升格最快」案例（從縣升格 64 年）
- 移工縣（中壢車站附近東南亞商店街 + 12 國移工）是台灣「外籍移工融入」最具代表性的案例
- 對讀者：「為什麼桃園同時是機場縣 + 客家庄 + 移工縣 + 半導體 + 直轄市？」這個 frame 拆開後讓人看見「層疊性」的城市結構

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：桃園台地（大漢溪沖積扇 + 林口台地南端）+ 復興山區（雪山山脈北段 + 拉拉山）+ 沿海平原（大園 / 觀音 / 新屋 / 林邊）/ 13 區分平地（桃園 + 中壢 + 平鎮 + 八德 + 蘆竹 + 龜山）vs 客家庄（中壢 + 龍潭 + 楊梅 + 新屋 + 觀音）vs 沿海（大園 / 觀音 / 新屋）vs 山地（復興）vs 大溪
- **誰在這裡生活**：凱達格蘭族（西部平原，已漢化）+ 泰雅族（復興山區）+ 1740s-1900s 客家移民（中壢 / 龍潭 / 楊梅 / 新屋 / 觀音）+ 閩南（桃園市區 + 大園 + 蘆竹）+ 戰後外省（眷村）+ 1990s 後東南亞移工（越南 + 印尼 + 菲律賓 + 泰國，集中中壢車站）
- **人和地有什麼共生**：桃園台地 → 埤塘灌溉文化 + 1964 石門水庫 / 沿海 → 觀音工業區 + 火力發電 / 山地 → 泰雅族復興區 + 拉拉山水蜜桃 / 客家庄 → 大溪豆干 + 龍潭擂茶 + 客家粄條 / 機場 → 國際旅客 + 物流產業
