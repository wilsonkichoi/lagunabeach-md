# Research Report: 桃園市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：桃園國際機場（1979/02/26 啟用，台灣對外大門）／半導體產業（桃園台積電 + 群創 + 友達）／移工聚集（東南亞外籍移工最多縣市）／客家比例全國最高之一（近 80 萬客家人）／大溪老街（巴洛克紅磚街屋 + 豆干）／中壢區（族群混居）／2014/12/25 升格直轄市最快的縣 / 龍潭茶 + 客家擂茶 / 大園 / 觀音
- **世代差異**：
  - 戰前老世代知「桃園廳」（日治 1901-1920 + 1920 改新竹州桃園郡 + 中壢郡 + 大溪郡）+「埤塘文化」（桃園台地數百埤塘灌溉）
  - 戰後 50-70s 世代知「客家移民縣」（中壢市 + 龍潭 + 楊梅 + 新屋 + 觀音五大客家鄉鎮）
  - 80-90s 世代知「桃園國際機場 1979 啟用」（取代基隆港成為台灣門戶）+ 「龍潭石門水庫 1964」
  - 千禧後世代知「2014 升格直轄市 + 移工縣 + 半導體 + 桃園機場捷運 2017 通車」
- **外人 vs 在地人**：外人覺得「桃園轉機 + 大溪豆干 + 中壢車站」就走人；在地人知「我們桃園 230 萬人是全國第五大都市」「中壢車站附近的東南亞商店街有 12 國移工」「大溪豆干跟新竹米粉、東港黑鮪魚一樣都是地方代表食物」「2014 升格直轄市那年我們從『桃園縣』變『桃園市』，6 個鎮市變區，但 14 鄉鎮市的多元性還在」

### 多元面貌

- **主流敘事**：「機場縣 + 半導體 + 移工」（過度功能化）vs 「客家庄 + 大溪老街」（過度地方化）
- **支線敘事**：
  - **2014/12/25 桃園升格直轄市**（從桃園縣升格，是全台升格最快的縣份 — 從 1950 設縣到 2014 升格僅 64 年）
  - **客家比例全國最高之一**（行政院客委會調查近 80 萬客家人，是全國客家人口最多的縣份；中壢區客家約 50%）
  - **桃園國際機場 1979/02/26 啟用**：取代基隆港成為台灣國際門戶 + 1979 桃園客運站啟用 + 1980s-2000s 兩次擴建 + 2017/03/02 機場捷運通車
  - **桃園台地埤塘文化**：日治前後數百埤塘灌溉系統 + 戰後 1964 石門水庫取代部分埤塘 + 今天仍有 2,800 個埤塘殘存
  - **5 大客家鄉鎮**：中壢 + 龍潭 + 楊梅 + 新屋 + 觀音（從苗栗北上至桃園的客家移民帶）
  - **大溪老街**：清領晚期茶葉 + 樟腦輸出港 → 1919 巴洛克紅磚街屋 + 1985 古蹟保存 + 大溪豆干 + 月光餅
  - **中壢市區**：1968 中壢中山高速公路通車 + 1977 中壢事件（許信良選縣長爭議）+ 1990s 中壢車站東南亞商店街形成
  - **石門水庫 1964/06/14 完工**：當時遠東第一大水庫 + 灌溉桃園台地 / 新北 + 大溪 / 龍潭 / 復興 / 觀音水源
  - **13 區**（2014 升格後）：桃園 / 中壢 / 大溪 / 楊梅 / 蘆竹 / 大園 / 龜山 / 八德 / 龍潭 / 平鎮 / 新屋 / 觀音 / 復興（山地原住民區）
- **fault lines**：
  - 機場 vs 在地（機場員工 / 旅客 vs 桃園在地居民）
  - 移工縣 vs 漢人主流（中壢車站附近東南亞商店街 vs 客家庄 + 閩南）
  - 客家 vs 閩南（中壢 / 龍潭 / 楊梅客家 vs 桃園市區閩南）
  - 1977 中壢事件（戒嚴時期黨外運動）vs 今天觀光化
  - 復興區（泰雅族）vs 平地 12 區（漢人主流）
  - 2014 升格直轄市 vs 1950-2014 桃園縣身分（區跟鄉鎮市的差異）

### 想法跟感受

- **正面**：桃園人對「全國第五大都市 + 機場縣」有強烈自尊 / 對客家庄 + 大溪豆干 + 龍潭茶 + 石門水庫的在地驕傲 / 對 2014 升格直轄市的成就感
- **負面**：「機場縣」「轉機站」標籤的疲乏 / 1977 中壢事件被觀光化的尷尬 / 移工縣的歧視性敘事
- **矛盾感**：升格直轄市最快 vs 13 區仍保留鄉鎮市多元性 / 客家近 80 萬全國最多 vs 中壢車站東南亞商店街 vs 機場員工三層人口結構

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 凱達格蘭族 + 泰雅族（復興山區）千年居住
  - 1740s 客家移民從廣東惠州、嘉應州、潮州移居北部
  - 1786 林爽文之役後客家義民從南部六堆北上桃竹苗
- **關鍵轉折**：
  - 1812 設淡水廳含桃園地區
  - 1875 沈葆楨改革 + 桃園地區屬新竹縣
  - 1895 日治：桃園廳 1901-1920 + 1920 改新竹州桃園郡 + 中壢郡 + 大溪郡
  - 1945 戰後：桃園縣設立（從新竹州分出）
  - 1950 桃園縣行政體制確立
  - 1964/06/14 石門水庫完工（遠東第一大水庫）
  - 1968 中壢中山高速公路通車（中壢交流道）
  - 1977/11/19 中壢事件（許信良選縣長 + 黨外運動關鍵）
  - 1979/02/26 桃園國際機場啟用（取代基隆港成為台灣門戶）
  - 1980s-1990s 桃園台地工業化 + 半導體產業興起
  - 2014/12/25 桃園升格直轄市（從縣升格 64 年）
  - 2017/03/02 桃園機場捷運通車
- **當代意義**：桃園是台灣「升格最快 + 機場縣 + 客家最大 + 移工最多」的直轄市。1979 機場 + 2014 升格 + 客家 80 萬 + 12 國移工 — 四個故事撐起桃園的當代意義

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 桃園市是台灣「對外大門」（1979 桃園機場取代基隆港，台灣 95%+ 國際旅客從這裡進出）
- 客家近 80 萬人**全國最多**（行政院客委會調查），桃園是當代台灣客家最重要的縣份
- 1977 中壢事件是台灣黨外運動關鍵節點（許信良選縣長爭議 + 戒嚴時期街頭運動）
- 2014 升格直轄市是「行政升格最快」案例（從縣升格 64 年）
- 移工縣（中壢車站附近東南亞商店街 + 12 國移工）是台灣「外籍移工融入」最具代表性的案例
- 對讀者：「為什麼桃園同時是機場縣 + 客家庄 + 移工縣 + 半導體 + 直轄市？」這個 frame 拆開後讓人看見「層疊性」的城市結構

### 1. 2014/12/25 升格直轄市：完整記錄

**行政院公告（三源確認）**

- **2013/01/03** 行政院第 3330 次院會公告：桃園縣於 2014/12/25 改制為直轄市，名稱定為「桃園市」，成為臺灣第六個直轄市（第六都）
- 2014/12/25 鄭文燦（民主進步黨）當選第一屆桃園市市長，於市政府前廣場舉行就職典禮
- 六都格局：台北 → 高雄（1979）→ 台中 → 台南 → 新北（2010）→ 桃園（2014）
- 升格後：原桃園縣 1 市（桃園市）+ 2 縣轄市（中壢市 + 平鎮市）+ 11 鄉鎮 → 改為 13 個直轄市區

**面積 / 人口（三源確認）**

- 面積：**1,220.9540 平方公里**（Wikipedia 中文版 verbatim）
- 2026/04 人口：**2,356,795 人**（桃園市政府戶政統計）；人口成長率全台第一
- 13 個區：桃園 / 中壢 / 平鎮 / 八德 / 楊梅 / 大溪 / 蘆竹 / 龜山 / 龍潭 / 大園 / 觀音 / 新屋 / 復興

**從桃仔園到桃園市：地名演變**

- 乾隆年間，移民薛啟隆等人拓墾，區內遍植桃樹，被稱為「桃仔園」
- 1774 年（乾隆三十九年）：《臺灣府志》已載「桃仔園莊」
- 日治 1920 年：廢「桃仔園」稱，正式更名「桃園」

### 2. 1977/11/19 中壢事件：完整記錄

**事件過程（三源確認）**

- **1977/11/19** 投票日當天，桃園縣中壢市第 213 號投開票所（中壢國小）監選主任**范姜新林**（時任中壢國小校長），被證人邱奕彬等人指控舞弊作票
- 檢察官廖宏明獲報後卻將**證人**移送警局，讓被指控的監選主任繼續在場執勤——此為事件導火線
- 消息傳出，上百名民眾前往投開票所，後聚集超過一萬名民眾包圍桃園縣警察局中壢分局
- 晚間 7 點，警方朝民眾發射催淚瓦斯，黑暗中從制高點開槍射擊民眾，造成：
  - **國立中央大學學生江文國**（苑裡人）頭部中槍不治
  - **19 歲張治平**（中壢人）亦不治
  - **16 歲少年劉世榮**重傷
- 民眾燒毀中壢分局

**選舉結果（Wikipedia 中文版 verbatim + 報導者 verbatim）**

- 許信良以「二十三萬票對十四萬票」（Wikipedia），「大勝國民黨提名的歐憲瑜近 10 萬票」（報導者），高票當選桃園縣長
- 黨外陣營在該場「五合一選舉」囊括 4 席縣市長、21 席省議員、146 席縣市議員

**許信良直引（報導者 40 週年報導 verbatim）**

- 「即使作票，我都有辦法贏他啦，我信心滿滿。」
- 「擒賊先擒王，當時中壢有很多特務，我不離開的話，一定馬上被控制住。」
- 「我去跟警察握手，他們看到我都頭一歪，還有人吐口水。但他們不能拿我怎麼樣，我已經選上縣長了。」

**歷史意義**

中壢事件被認為是台灣民眾繼郭雨新事件後第一次自發性上街頭抗議選舉舞弊，開啟此後「街頭運動」之序幕，1979 年橋頭事件與美麗島事件的直接前身。

### 3. 桃園國際機場：從中正機場到台灣門戶

**啟用歷史（台灣光華雜誌 verbatim + 三源確認）**

- **1979/02/26**：桃園中正國際機場正式通航
- 命名：1979/02/15 行政院會議通過，正式命名「中正國際機場」；2006/09/06 改名「臺灣桃園國際機場」
- 第一航廈：1974 年動工，1978 年底完工，1979/02/26 啟用
- 第二航廈：2000 年啟用

**台灣光華雜誌（1979 年）verbatim**

> 「遠東最大的國際機場——桃園中正國際機場，經過工程人員四年半的辛勤工作，已經結出豐碩的成果——於今年二月廿六日正式通航，開啟我國另一座現代化的空中門戶」

> 「這座佔地一千二百公頃的國際機場，比日本東京近郊的成田機場還要大出一百三十五公頃。」

> 「這座遠東目前最大、最新穎、最完善的機場，已被視為世界著名機場之一。」

**機場捷運**

- 2006 年動工興建（原 1996 年以 BOT 模式籌備，後因多方因素停止）
- 歷經**六次延誤**，2017/02/02 試營運，**2017/03/02 正式通車**
- 全長 51.95 公里，全台最長捷運線
- 2023/07/31 中壢延伸線（環北至老街溪站）通車
- 預算從原規劃大增至**新台幣 1,277 億元**

**2024 年旅客量**

- 2024 年客運量 **4,492 萬人次**（疫情前 2019 年的 92%）
- 全球機場國際旅客排名**第 13 名**；貨運量全球**第 10 名**
- 2024 年航班架次 **24.8 萬架次**

### 4. 石門水庫：台灣第一座多功能水庫

**基本事實（三源確認）**

- 完工日期：**1964/06/14**，副總統陳誠主持竣工典禮
- 建設歷時：8 年（工期），參與人力 7,500 餘人
- **壩高 133.1 公尺**；**總蓄水量 309,120,000 立方公尺**（Wikipedia verbatim）
- 建設經費：**約新台幣 32 億元，其中美援 19 億元**

**「遠東最大」verbatim**

> 「石門水庫曾經是遠東地區最大的水庫」（Wikipedia 中文版 verbatim）

**功能與意義**

- 台灣**首座多功能水庫**（灌溉 + 發電 + 防洪 + 公共給水 + 觀光）
- 位於桃園市龍潭區，毗鄰大溪區、復興區、新竹縣關西鎮
- 1954–1956 年三個籌備委員會逐步建立，1964 竣工
- 石門大圳：1964/06 完工，從後池堰取發電後尾水，通過 18 公里導水隧道至八德廣豐，灌溉 22,000 公頃

### 5. 埤塘文化：千塘之鄉

**台灣光華雜誌《千塘之鄉》verbatim**

> 「隨著飛機緩緩降落桃園中正機場，映入眼簾的是一面面閃閃發亮的鏡子，這片星羅棋布在桃園台地上的陂塘（即埤塘），是見證先民把荒漠變良田的水利工程傑作，其面積之大、密度之高，世界罕見。」

> 「三十年來因為都市和工商發展，曾達上萬口的桃園陂塘一口口被填平，目前僅剩 2,800 口，近來在保護濕地潮流和被文化部門列入『世界文化遺產潛力點』的二大保護傘下，才逐漸受到重視。」

> 「一口陂塘還代表一個生產、生活、生態的系統。」

> 「例如桃園新屋鄉著名的范姜古厝和附近的新屋埤等多口陂塘，就是探索客家庄開發史的活素材。」

**埤塘數量演變**

- 1913 年大旱前：桃園台地上萬口埤塘，達到最高峰
- 日治初期：桃園廳埤圳數 6,685 處，全台各行政區中數量最多、密度最高
- 2026 年現存：**約 2,800 口**
- 桃園大圳（1916 年開工、1924 年完工通水）串連埤塘，285 口蓄水量 3,400 萬噸，相當於石門水庫 1/3

**地理成因**

桃園台地是古石門溪沖積形成的古石門沖積扇，因河川劫奪現象致使灌溉水源嚴重缺乏，早期墾民以開鑿埤塘儲水。

### 6. 客家人口：全國最多

**數據（三源確認）**

- 桃園市客家人口 **85 萬以上**（最新統計 91 萬以上，2022 年報導）；早期行政院客委會調查「近 80 萬」，全台最多
- 客家聚落集中：中壢 / 平鎮 / 龍潭 / 楊梅 / 新屋 / 觀音（南桃園帶）
- 中壢區客家比例約 50%

**義民爺信仰**

- 桃園平鎮褒忠祠（義民廟）：乾隆五十六年（1791 年）建，主奉義民爺，供奉全台第一座義民爺金身
- 信眾範圍：中壢 / 平鎮 / 楊梅客家人為主，轄境 13 大庄

**范姜古厝：客家移民的物理見證**

- 乾隆十六年（1752 年）：范姜氏兄弟五人先後渡海至桃園新屋開墾
- 1855 年（咸豐五年）：范姜族人建祖堂（今范姜祖堂），從廣東迎祖靈奉祀，居民稱「起新厝」，成為「新屋」地名由來
- 今為市定古蹟，是探索南桃園客家庄開發史的代表性建築

### 7. 大溪老街：清代商港到日治建築

**歷史脈絡（三源確認）**

- 清領：大溪（舊稱「大嵙崁」）是淡水河系最上游的河港，茶葉、樟腦、木材、煤礦的集散中心
- 1863 年滬尾（淡水）開港後，大嵙崁成為台灣最內陸的貿易港，吸引英、德、西班牙、漢人商人設店
- 清末劉銘傳設「撫墾總局」於大溪

**1919 年市區改正（verbatim）**

> 「在日據時期的明治維新時代，1919 年左右，日本總督府執行等同現代都市計劃的『市區改正』計劃……規定商店必須採『巴洛克式』建築」

> 「曾向日本人學習過泥匠技藝的陳旺來、陳三川兄弟將常用於廟宇的剪黏與交趾燒應用於牌樓厝的立面上，引來其他店家及居住人士仿效」

⚠️ 「巴洛克」嚴格說是台人自解：日方市區改正規定的是「洋式店面牌樓厝」，台灣匠師在執行時融入閩式廟宇紋飾，形成「台灣式洋風」，並非正統歐洲巴洛克，聚傳媒有相關考證。

**大溪豆干**

- 起源：民初福建漳州人林絨渡海來台，帶入五香黑豆干滷製技術（以紅糖烹煮成糖烏、加五香粉滷製）
- 老字號：黃日香（民國 13 年，約 1924 年）、萬里香、大房豆干、廖心蘭等
- 大溪豆干好吃的根本：大溪擁有良好的水質（大漢溪上游）
- 桃園市政府統計，今大溪豆製品工廠約 21 家

### 8. 中壢移工商圈：小東南亞的形成

**歷史（三源確認）**

- 1990 年代，政府開放移工入台，中壢以泰籍移工為主，形成「泰國街」
- 2000 年代後隨移工結構更迭，泰國餐廳被越南、印尼餐廳超越
- 中壢車站前後站（元化路、中平路、新興路）百家以上東南亞商店及攤販

**2023 年勞動部統計（一手來源 verbatim）**

「112 年底桃園市移工人數達 13 萬 2,158 人，占全國移工人數 75 萬 3,430 人之 17.5%，居全國之冠」（桃園市政府新聞稿 verbatim）

- 產業移工：11 萬 1,885 人（全國最多），以越南籍為主
- 社福移工：以印尼籍為主
- 製造業移工：10 萬 3,513 人，佔桃園產業移工 92.5%

### 9. 復興區：泰雅族的桃園

**基本事實（三源確認）**

- 面積約 351 平方公里，占桃園市總面積 **七分之二**，人口 1.3 萬人，桃園市面積最大、人口最少的區
- 居民**約七成為原住民族**，以泰雅族為主
- **台灣泰雅族人口最多的鄉鎮市區**

**拉拉山（達觀山）**

- 泰雅族語：「R'ra」，意為「美麗的」「讚嘆的」「眺望守望」
- 1975 年更名達觀山，2000 年代起逐漸恢復「拉拉山」慣稱
- 位於桃園市復興區與新北市烏來區交界，海拔 1,500–2,000+ 公尺
- 以巨木群、水蜜桃聞名

**泰雅族傳統文化**

- 苧麻編織 / 紋面文化（1930 年代後漸衰）/ 播種祭 / 摘穗祭 / 收穫祭

**角板山行館**

- 日治時期建設的角板山行館；戰後成為蔣中正夏季行館；今開放觀光

### 10. 1895 年乙未戰爭：桃園客家義民的抵抗

**三角湧之役（1895/07/12）**

- 1895/07/12：日軍 35 人運糧隊在三角湧（今新北市三峽）遭義軍伏擊，幾乎全軍覆沒（「隆恩埔戰役」）
- 同日：日軍 894 人部隊（由分水崙往新竹方向）在三角湧遭義軍伏擊，日軍傷亡兩百多人，史稱「分水崙戰役」——為日軍進台北後戰鬥最慘烈的一役

**大嵙崁之役（1895/07/12–16）**

- 地點：今桃園市大溪區大漢溪畔
- 義軍約 1,000 人，由武生江國輝為統領，呂建幫為副，李家棟為助
- 日軍日軍重要支援部隊在渡大溪時遭義軍設伏，苦戰數日，僅四人扮乞丐逃脫
- 後日軍增援後攻占大嵙崁，焚燬村落

**抗日三傑：姜紹祖（桃竹苗客家）**

- 姜紹祖（1877–1895），廣東陸豐縣人後裔，新竹北埔人，年僅 19 歲即招數百鄉勇抗日
- 參與大姑崁口阻擊（6 月）、新竹城保衛、桃園各地戰役
- 與吳湯興、徐驤並稱「客家抗日三傑」
- 1895/07/09 新竹城失守後被俘，拒降服毒自盡

### 11. 慈湖：蔣中正的最後行館

**歷史（三源確認）**

- 位置：桃園市大溪區，靠近復興區邊界
- 1955 年板橋林家無償捐予中華民國政府，蔣中正以此地景緻與故鄉奉化相似而流連
- 1959/06/13 洞口賓館落成；蔣中正命名「慈湖」，以思念其母王太夫人
- 1975/04/05 蔣中正逝世後，靈柩移置慈湖陵寢
- 大溪陵寢：蔣經國 1988/01/13 逝世，1988/01/30 安厝
- 2006 年：桃園縣政府整合慈湖陵寢 + 大溪陵寢 + 慈湖紀念雕塑公園 + 角板山行館，成立「兩蔣文化園區」
- 慈湖紀念雕塑公園：集中全台各地「去蔣化」移除的蔣中正銅像，為全台唯一以單一個人銅像設立的紀念公園

### 12. 桃園大廟景福宮

- 乾隆十年（**1745 年**）草創，墾首薛啟隆捐田產二十甲餘倡議建廟
- 主神：**開漳聖王**（陳元光），為漳州籍移民的護佑神
- 為桃園十五街庄的信仰核心，民眾習稱「大廟」
- 直轄市定古蹟
- 1809 年（嘉慶十四年）興築桃園城後，十五庄漳州人再合建景福宮

### 13. 桃園工業：全台工業產值第一

**數據（三源確認）**

- 台灣 500 大製造業，超過三分之一在桃園設廠
- 工業區：42 個（依產業創新條例報編完成）
- 工業產值：每年約新台幣 2–3 兆，**連續多年全台第一**
- 主要產業：TFT-LCD、半導體（日月光半導體中壢廠、桃園各科學園區）、航太維修、光電、汽車零組件、紡織染整

**黑貓中隊：冷戰下的桃園秘史**

- 1961 年：中美合作「快刀計畫」（Project Razor），在桃園空軍基地秘密成立空軍第 35 偵察中隊（暱稱「黑貓中隊」）
- 操作 U-2 偵察機，深入中國大陸、越南、朝鮮半島執行高空偵察
- 28 名飛行員，執行 220 次任務，10 人殉職，2 人被共軍擊落（生死不明）
- 1974 年「快刀計畫」終止；直至近年才逐步解密
- 2004 年台灣有 879 個眷村，其中桃園 **80 個**，全台最多

### 14. 沿海海客文化：觀音 / 新屋石滬群

**全台唯一海客文化地帶**

- 新屋、觀音等沿海鄉鎮以**客家庄為主體**，農漁兼作，發展出「牽罟」「石滬」「吹海螺」等海洋客家文化
- 永安漁港：**全台唯一以客家庄為主體的漁港**（中壢區漁會轄）

**新屋石滬群**

- 台灣本島現存最多且最完整的石滬群在桃園新屋
- 共 **10 座**（另一來源說 11 座），其中 3 座仍有漁民在使用
- 起源清朝，由漢人先民以徒手堆砌鵝卵石築成
- 運作原理：利用天然地形 + 潮汐，漲潮時魚群游入石滬，退潮後大魚被困

**永安海螺文化體驗園區**

- 2022/03/19 開園，占地 5.6 公頃，主體建築擷取海螺曲線意象
- 主題：「漁村客家故事館」，展示桃園濱海客家漁村特色
- 免費入館

### ⚠️ Fact Flags 待確認清單

1. ⚠️ **客家人口數字**：「近 80 萬」（行政院客委會早期調查）vs 「85 萬」（2022 年中時新聞）vs「91 萬以上」（2022 年另一來源）。寫作時用「逾八十萬」（較保守），標注統計年份及來源（客委會調查）。

2. ⚠️ **大溪老街建築是否屬於「真正」巴洛克**：聚傳媒考證文指出，嚴格說是「台灣式洋風牌樓厝」，非歐洲正統巴洛克。寫作時宜用「1919 年市區改正後發展出的洋風牌樓厝立面」，可附注「坊間常稱巴洛克」。

3. ⚠️ **石滬數量**：新屋石滬「10 座」（多數旅遊來源）vs「11 座（深圳 2 + 蚵間 9）」（濱海生態網）。採用**10 座**（更多旅遊官方來源採用），⚠️ 與 Wikimedia 圖片核對後再確認。

4. ⚠️ **1895 年大嵙崁戰役的義軍領袖**：多個中文來源出現「江國輝」（江國澤?）的名字，但「姜紹祖」在乙未戰爭的行動地點主要在新竹城一帶，大嵙崁之役的領導者需分開確認。

5. ⚠️ **1974/07/29 中山高速公路通車日期**：搜尋確認為 **1974/07/29**，全長 30.1 公里，三重至中壢段，此條可採信。

6. ⚠️ **桃園市眷村數量「全台最多」**：2004 年台灣 879 個眷村，桃園 80 個（排名第一）。此數字來自日文 Wikipedia 轉引，需以中文史料核對。

7. ⚠️ **機場捷運預算 1,277 億元**：Wikipedia 桃園機場捷運條目，需確認是否包含中壢延伸線。寫作時用「超過千億元」。
