# Research Report: 新竹市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：新竹科學園區（1980/12/15 完工，台灣第一座科學園區）／台積電 / 聯發科 / 聯電 / 友達 / 群創總部所在地／東門城（迎曦門，1829 石城東門遺留）／城隍廟與其廟口小吃（貢丸湯 + 米粉 + 摃丸）／黑蝙蝠中隊紀念館（1953-1974 冷戰時期偵察 148 人殉職 15 架飛機）／玻璃工藝博物館（日治矽砂 + 天然氣發展）／米粉（風城風大乾燥優勢）／18 尖山／清華大學 + 交通大學（陽明交大）
- **世代差異**：
  - 戰前老世代知「淡水廳治竹塹」（1725 淡水廳遷至此 + 1733 種竹為城 + 1829 石城完成）
  - 戰後 50-70s 世代知「冷戰前線基地」（黑蝙蝠中隊 1953 起新竹基地 + 1958 改空軍 34 中隊 + U-2 偵察）
  - 80-90s 世代知「新竹科學園區 1980」（IC 設計 + 半導體 + 台積電 1987 創立 + 聯電 1980）
  - 千禧後世代知「竹科男 + 房價 + 老婆稅」（科技人匯聚 + 學區 + 米粉貢丸成為觀光符號）
- **外人 vs 在地人**：外人覺得「新竹喝米粉湯吃貢丸 + 看竹科」就走人；在地人知「我們是 1733 種竹建城的全台北部最早漢人城市」「黑蝙蝠中隊 148 人從這裡飛去大陸再也沒回來」「玻璃跟米粉都是風城的副產品 — 風大才能吹乾米粉、矽砂跟天然氣才有玻璃」「我們省轄市 45 萬人但被全國認知壓縮成『竹科員工居住地』」

### 多元面貌

- **主流敘事**：「科技城新竹」（過度單一化）vs 「米粉貢丸觀光」（過度簡化）
- **支線敘事**：
  - **1733 種竹為城**（雍正 11 年）+ **1829 石城完成**：北部最早建城的漢人聚落
  - **1725 淡水廳遷至竹塹**（雍正 3 年，原淡水廳設於新莊，遷此後新竹成為北台灣行政中心 150 年）
  - **1875 沈葆楨改革**：新竹縣設立（從淡水廳分出）+ 1879 首任知縣到任
  - **1950 縣市分治**：新竹市獨立省轄市 + 新竹縣治留在新竹市暫時 + 1989 新竹縣治遷竹北
  - **黑蝙蝠中隊 1953-1974**（空軍 34 中隊）：冷戰時期 P-2V / RB-69A / U-2 偵察大陸 + 148 人殉職 + 15 架飛機損失 + 2009 黑蝙蝠中隊紀念館設立
  - **新竹科學園區 1980/12/15 完工**：台灣第一座科學園區 + 跨新竹市 + 新竹縣寶山 + 苗栗銅鑼 / 龍潭 / 宜蘭 / 高雄 / 台南
  - **玻璃工藝**：日治發現竹塹矽砂 + 天然氣 → 1925 玻璃廠興起 + 戰後春池玻璃 + 玻璃工藝博物館 1999 開館
  - **米粉**：風城風大乾燥 + 福建移民帶入 + 老鍋米粉 + 米粉寮歷史
  - **眷村**：黑蝙蝠中隊家屬眷村 + 空軍 11 大隊眷村 + 新竹眷村文化保存
  - **清華大學 1956 在台復校** + **交通大學 1958 在台復校**（陽明交通大學 2021 合併）：科技園區人才搖籃
  - **3 區**：東區 / 北區 / 香山區
- **fault lines**：
  - 1733 竹塹城歷史 vs 1980 科學園區現代（兩個時代的新竹）
  - 省轄市新竹市（45 萬）vs 新竹縣（57 萬）行政切割 + 1989 縣治外遷
  - 黑蝙蝠中隊冷戰前線 vs 戰後觀光化（紀念館 2009 設立才開始說這段歷史）
  - 玻璃 vs 米粉 vs 科技 — 三個產業時代的層疊
  - 眷村文化 vs 科技新貴（兩種「外來移民」的對比）
  - 清華 + 交大學生 vs 在地客家 / 閩南 / 道卡斯後裔

### 想法跟感受

- **正面**：新竹人對「台灣第一座科學園區」「北部最早建城漢人聚落」有強烈自尊 / 對黑蝙蝠中隊 + 玻璃工藝 + 米粉貢丸 + 城隍廟的在地驕傲 / 對清華 + 交大（陽明交大）的教育自信
- **負面**：「竹科員工居住地」標籤的疲乏（被科學園區框架壓縮）/ 1989 縣治遷竹北的失落 / 米粉貢丸觀光化的尷尬（食物變成符號）
- **矛盾感**：1733 建城歷史 vs 1980 科技起點 / 黑蝙蝠 148 人從這裡飛走但戰後 30 年沒人說 / 風大才吹得乾米粉，矽砂才做得出玻璃 — 自然條件決定產業

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 道卡斯族竹塹社（Pocaal）千年居住
  - 1718 王世傑率閩粵移民拓墾竹塹埔
  - 1725 淡水廳遷至此（從新莊遷來）
  - 1733 種竹為城（雍正 11 年）
- **關鍵轉折**：
  - 1829 石城完成（道光 9 年）+ 東門城（迎曦門）至今留存
  - 1875 沈葆楨改革：新竹縣設立 + 1879 首任知縣
  - 1895 日治：新竹廳 + 1920 改新竹州
  - 1925 玻璃廠興起（日治矽砂 + 天然氣發現）
  - 1945 戰後：新竹縣設立含現在新竹市區 + 新竹縣
  - 1950 縣市分治：新竹市獨立省轄市
  - 1953 黑蝙蝠中隊遷至新竹基地（冷戰前線）
  - 1956 清華大學在台復校 + 1958 交通大學在台復校
  - 1958 黑蝙蝠中隊改稱空軍 34 中隊（冷戰 U-2 偵察）
  - 1974 黑蝙蝠中隊解散 + 損失 15 架 + 148 人殉職
  - 1980/12/15 新竹科學園區完工（台灣第一座）
  - 1987 台積電創立於新竹園區
  - 1989 新竹縣治遷至竹北市（新竹市完全獨立）
  - 1999 玻璃工藝博物館開館
  - 2009 黑蝙蝠中隊紀念館設立（冷戰歷史終於被說出）
  - 2021 清華 + 交通大學合併為陽明交通大學
- **當代意義**：新竹市是台灣「冷戰前線 + 半導體起點」共存的省轄市。1733 建城 → 1953 黑蝙蝠 → 1980 科學園區 → 1987 台積電 — 四個時代的新竹

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 新竹市是台灣「北部最早建城」的漢人聚落（1733 種竹為城）
- 黑蝙蝠中隊 1953-1974 是台灣「冷戰前線」最具體的物理見證（148 人殉職 + 15 架飛機）
- 新竹科學園區 1980 是台灣「半導體王國」的起點（先有園區，後有台積電）
- 玻璃 + 米粉 + 貢丸是「風城」自然條件的副產品（風大乾燥 + 矽砂 + 天然氣）
- 對讀者：「為什麼一個 45 萬人的省轄市，撐起台灣半導體 + 冷戰歷史 + 北部最早建城 + 兩所頂尖大學？」這個 frame 拆開後讓人看見「規模 ≠ 重要性」的具體案例

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：頭前溪沖積平原（西部沿海）+ 18 尖山（東部丘陵）+ 香山濕地（西南沿海濕地）+ 風大乾燥（東北季風 + 海陸氣壓差）/ 3 區：東區（市區 + 18 尖山 + 清華）/ 北區（城隍廟 + 玻璃博物館 + 米粉寮）/ 香山區（沿海濕地 + 工業區）
- **誰在這裡生活**：道卡斯族竹塹社（Pocaal，已漢化）/ 1718 後閩粵移民 + 1725 淡水廳遷來後行政官員 / 日治日本人（玻璃 + 米粉 + 行政）/ 戰後外省（眷村 + 黑蝙蝠中隊家屬）+ 清華交大師生 + 1980 後科技新貴（含越南 / 印尼新住民）
- **人和地有什麼共生**：風大乾燥 → 米粉產業（老鍋 + 米粉寮）/ 矽砂 + 天然氣 → 玻璃產業 / 平地 + 東部丘陵 → 科學園區 + 清華交大校地 / 城隍廟 → 廟口貢丸 + 米粉 + 城隍信仰 / 黑蝙蝠中隊基地 → 眷村文化
