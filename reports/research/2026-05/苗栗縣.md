# Research Report: 苗栗縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：客家本色／油桐花季（5 月雪）／三義木雕／大湖草莓／南庄老街（賽夏族原鄉）／苗栗大千鐵路（山線豐原-竹南）／公館紅棗／銅鑼桐花／賽夏族矮靈祭（兩年一次）／苗栗台灣燈會 2011／劉政鴻 2015 財政危機／大埔事件（張藥房 + 朱馮敏 2013）
- **世代差異**：
  - 戰前老世代知「貓裡」（道卡斯族語「苗栗」原名）+ 1889 改名苗栗縣
  - 戰後 50-70s 世代知「客家本色」+「最後一個通電的縣 1981」+ 礦業（南庄、三灣煤礦）
  - 80-90s 世代知「三義木雕」+「大湖草莓」+「油桐花季」
  - 千禧後世代知「劉政鴻 2008-2014 把苗栗搞到 676 億負債」+「大埔事件 2013」+「客家圓樓、客家大院（被嘲笑的蚊子館）」
- **外人 vs 在地人**：外人覺得「苗栗看油桐花、買木雕、吃草莓」；在地人知「我們是 60% 客家比例最高的縣」「劉政鴻 8 年讓我們欠 676 億」「大埔事件張藥房強拆是在這裡發生的」「我們客家硬頸的本色被一個五星級縣長糟蹋了」

### 多元面貌

- **主流敘事**：「客家本色」+「油桐花縣」（過度浪漫化）vs 「破產縣」+「蚊子館縣」（過度負面）
- **支線敘事**：
  - 60% 客家人口比例：全台最高（22 縣市中無人能及）+ 1988 還我客家話運動的後援基地
  - 賽夏族矮靈祭（兩年一次，農曆 10/15）：南庄向天湖 + 北賽夏（五峰）跟南賽夏（南庄）兩個祭典
  - 苗栗 2015 財政危機：劉政鴻 2008-2014 兩任期負債從 202 億→676 億（增加 2 倍多）
  - 2013 大埔事件：苗栗大埔張藥房強拆 + 朱馮敏老婦自殺（2010 抗議 + 2013/07/18 強拆 + 朱馮敏 2010/08/03 喝農藥自殺）
  - 2011 台灣燈會在苗栗（劉政鴻花 5 億，被列為 2015 財政危機推手之一）
  - 2014 劉政鴻國慶煙火 1.8 億 + 苗栗國際藝術季 2.2 億 + 客家大院 8000 萬 + 客家圓樓 1.2 億
  - 2016/10/06 監察院通過劉政鴻彈劾案
  - 1981 苗栗全縣最後通電：戰後 36 年才全縣電氣化（中央山脈山地通電遲）
  - 油桐花 5 月雪：客委會 2002 起「桐花祭」國家級文化活動
  - 三義木雕：吳進寶 1918 起風氣 → 戰後三義木雕街
- **fault lines**：
  - 「客家硬頸」vs 「劉政鴻五星級縣長」（同一個縣的兩個自我形象）
  - 大埔事件：地方政府強拆 vs 公民抗爭 vs 朱馮敏自殺的歷史傷口
  - 客家文化保存（客家圓樓、客家大院）vs 蚊子館化（劉政鴻時代興建大批客家相關設施）
  - 賽夏族（南庄、五峰）vs 客家（公館、銅鑼、苗栗、頭份）vs 閩南（後龍、通霄、苑裡）三大族群比例
  - 山線（豐原-竹南經苗栗）vs 海線（後龍、通霄、苑裡）的城鄉差異

### 想法跟感受

- **正面**：苗栗客家人對「客家本色」「硬頸精神」有強烈自尊（油桐花祭 + 三義木雕 + 客家圓樓即使被批還是客家文化展示）
- **負面**：劉政鴻時代 8 年負債從 202 億→676 億（兩屆任期翻倍）的羞恥感 / 大埔事件張藥房強拆的歷史傷口 / 蚊子館化的尷尬
- **矛盾感**：客家硬頸如何選出一個讓縣破產的縣長？/ 客家文化保存的意圖 vs 蚊子館的結果 / 油桐花季的浪漫 vs 大埔事件的暴力

### 歷史脈絡（pre-search hypothesis）

- **形成期**：
  - 原住民道卡斯族（西部平原）+ 賽夏族（南庄、五峰山區）+ 泰雅族（部分山區）
  - 道卡斯族語「貓裡」（Pali）為苗栗原名
  - 清領 1683：苗栗屬諸羅縣 / 1731 屬淡水廳 / 1875 屬新竹縣
  - 1889 析新竹縣苗栗、苑裡、通霄、銅鑼、公館、後龍等地置「苗栗縣」（光緒 15 年）
  - 客家移民主流：18-19 世紀廣東惠州、嘉應州、潮州客家人移居
- **關鍵轉折**：
  - 1895 日治：太平山林業 + 三義樟腦業 + 苗栗山線鐵路 1903 通車
  - 1945 戰後：苗栗縣行政體制延續日治
  - 1981 全縣電氣化（最後一個全通電的縣）
  - 2002 客委會「桐花祭」國家級文化活動起步
  - 2008-2014 劉政鴻兩屆縣長：負債從 202→676 億 + 大規模燈會 + 蚊子館
  - 2010/08/03 朱馮敏喝農藥自殺（大埔事件抗議）
  - 2013/07/18 苗栗大埔張藥房強拆事件
  - 2015 苗栗縣財政危機暴露：歲出比率全國第一、債務 676 億
  - 2016/10/06 監察院彈劾劉政鴻
- **當代意義**：苗栗是台灣「客家文化保存的雄心」與「地方財政災難」的同時並存。劉政鴻時代的客家圓樓、客家大院是好意但變成蚊子館。「客家硬頸」的精神在這個縣選出了一個讓縣破產的人

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 苗栗是台灣「客家文化」最重要的據點之一（60% 客家比例 + 還我客家話運動後援 + 客家委員會 2001 成立後最大受益縣）
- 苗栗 2015 財政危機是台灣「地方財政自治」的最大警示案例（地方政府如何在 8 年內負債翻倍）
- 大埔事件是台灣「強制徵收」最具代表性的案例（張藥房強拆 + 朱馮敏自殺 + 都市計畫 vs 居民權利）
- 賽夏族矮靈祭是台灣最古老的原住民祭典之一（兩年一次 + 兩百年從未中斷）
- 對讀者：「為什麼一個 60% 客家比例、號稱『硬頸精神』的縣，會在 8 年內負債翻倍？」這個 frame question 拆開後讓人看見地方政治的複雜

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：丘陵地形（山多於平原）/ 雪山山脈西側 / 西部沿海有後龍、通霄、苑裡 / 苗栗山線（內陸客家庄）vs 海線（沿海閩南庄）/ 後龍溪、中港溪沖積 / 北部跟新竹縣交界、南部跟台中市交界
- **誰在這裡生活**：道卡斯族（西部平原）/ 賽夏族（南庄、五峰）/ 泰雅族（部分山區）/ 18-19 世紀廣東客家移民 / 閩南移民（沿海）/ 日治日本人（林業 + 樟腦 + 鐵路）/ 戰後外省移民
- **人和地有什麼共生**：丘陵 → 樟腦業（戰前）+ 茶葉 + 油桐 / 山林 → 三義木雕（樟木）+ 油桐花 / 客家庄 → 公館紅棗、大湖草莓、銅鑼苗栗一帶 / 賽夏族祭典 → 南庄、五峰

### 切入點清單（候選 hooks）

1. **「客家硬頸選出五星級縣長」的悖論**（60% 客家比例縣選了劉政鴻 → 8 年負債從 202→676 億 → 大埔事件 → 大量蚊子館 → 但他連任了第二屆）
2. **2013/07/18 大埔張藥房強拆**（朱馮敏 2010/08/03 自殺、張藥房 2013/07/18 強拆、張森文 2013/09/18 投水身亡 → 台灣強制徵收最暴力的歷史現場）
3. **客家圓樓、客家大院的雄心 vs 蚊子館**（劉政鴻時代蓋的客家文化設施 → 客家圓樓 1.2 億、客家大院 8000 萬 → 今天遊客稀少）
4. **油桐花祭 2002 起 vs 客家文化政策**（客委會成立後最大文化動員 → 5 月雪每年 5-6 月 → 銅鑼、三義、公館客家庄遊客高峰）
5. **三義木雕的源起**（吳進寶 1918 → 戰後三義木雕街 → 樟木 + 檜木 + 龍眼木 → 台灣木雕重鎮 → 對應日治樟腦業歷史）
6. **賽夏族矮靈祭的兩百年延續**（南庄向天湖 + 五峰 → 兩年一次農曆 10/15 → 北賽夏 vs 南賽夏祭典 → 賽夏族文化保存）
7. **1889 改名「苗栗」vs 道卡斯族「貓裡」**（光緒 15 年新縣設置 → 原住民地名漢化 → 苗栗縣行政體制 130 多年）

### 預期核心矛盾候選（待 Stage 1.4 收斂為 ≤ 30 字）

- A：客家硬頸的縣，選出讓縣負債翻倍的縣長
