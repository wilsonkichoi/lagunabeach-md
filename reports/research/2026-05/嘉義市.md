# Research Report: 嘉義市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：火雞肉飯（噴水雞肉飯、劉里長、阿宏師）／北回歸線標塔／阿里山森林鐵路嘉義車站起點／嘉義公園（中正公園）／嘉義文化路夜市／檜意森活村（嘉義舊酒廠＋阿里山林業營林宿舍）
- **世代差異**：
  - 戰前老世代知「諸羅城」（康熙 1704 木柵築城、乾隆 1786 林爽文之役改名嘉義「嘉其忠義」）
  - 戰後 50-70s 世代知「木材城」（戰後阿里山木材輸出顛峰，嘉義作為阿里山林業樞紐）
  - 80-90s 世代知「KANO」（1931 嘉義農林甲子園準優勝，2014 魏德聖電影復活這段記憶）
  - 千禧後世代知「火雞肉飯」與「最不像直轄市的省轄市」（人口 26 萬卻是獨立行政區，被嘉義縣包圍）
- **外人 vs 在地人**：外人覺得「嘉義就是吃雞肉飯」「阿里山入口」就走人；在地人知「文化路夜市的源頭其實在中央七彩噴水池下」「畫家陳澄波死在這座城市的火車站前」「諸羅城的東西南北門遺址今天還在」「嘉義公園的史蹟資料館是日治時期的史料館建築」

### 多元面貌

- **主流敘事**：「雞肉飯之都」「阿里山入口」（過度簡化）vs 「最被冷落的省轄市」（過度灰暗）
- **支線敘事**：
  - 全球第一座北回歸線標塔所在地（1908 日治為慶祝縱貫鐵路通車立第一代標）
  - 1931 KANO 神話：嘉義農林學校漢人 + 原住民 + 日本人混合隊伍打進甲子園準優勝
  - 1947 二二八嘉義事件：本省人接收水上機場、火車站、機場圍城戰，畫家陳澄波遇害於嘉義火車站前
  - 嘉義畫派（陳澄波、林玉山）：日治時期帝展入選率全台最高的城市之一
  - 阿里山森林鐵路（1912 嘉義至二萬坪 66.6km / 1914 延伸至阿里山）：日治為運木材而建，戰後變觀光鐵路
  - 1995 嘉義市改制為省轄市，跟嘉義縣切割但 geographically 被嘉義縣包圍
- **fault lines**：
  - 嘉義市 vs 嘉義縣（行政切割 1982 vs 地理一體 → 縣治在太保不在嘉義市）
  - KANO 是「日本人榮光」還是「台灣多元族群協力」（藍綠史觀分歧）
  - 陳澄波之死的歷史定位：1947 全國二二八敘事 vs 嘉義在地遺族記憶
  - 雞肉飯文化的擴散（全國連鎖 vs 嘉義本地）

### 想法跟感受

- **正面**：嘉義人對「我們嘉義」有強烈認同（雞肉飯 / 阿里山 / KANO / 陳澄波 四個 anchor 加起來足以撐起一個城市的自尊）
- **負面**：被遺忘感（人口從歷史高峰流失到 26.7 萬，是六都以外人口最少的省轄市）/「最後一個有電的城市」（1981 才全縣通電的鄉村記憶仍在）
- **矛盾感**：全球第一座北回歸線標塔在這裡，但這座城市卻是 22 縣市中最「中等」（不大不小、不南不北、不富不窮、不熱不冷）

### 歷史脈絡（pre-search hypothesis）

- **形成期**：1621 顏思齊、鄭芝龍登陸笨港（北港、新港一帶）→ 1683 清領 → 1704 諸羅縣城木柵建城（康熙 43 年知縣宋永清）→ 1786 林爽文之役諸羅城守城有功，乾隆改名「嘉義」（取「嘉其忠義」之意）
- **關鍵轉折**：
  - 1895 日治始政 → 1899 阿里山鐵路規劃 / 1908 北回歸線標塔（第一代，慶祝縱貫鐵路全線通車）/ 1912 嘉義至二萬坪鐵路通車 / 1914 延伸至阿里山
  - 1920 設嘉義街，1930 升格為嘉義市
  - 1931 嘉義農林甲子園準優勝（KANO 神話誕生）
  - 1947 二二八嘉義事件：水上機場圍城、嘉義火車站前處決陳澄波等本省菁英
  - 1950 嘉義縣市分治 → 1982 嘉義市升格為省轄市
  - 1995 嘉義縣治從嘉義市遷至太保市，地理與行政正式切割
- **當代意義**：嘉義市是台灣「中型城市的標本」——不夠大到變直轄市，不夠小到變鄉村，恰好停在 26 萬人這個檻位上 50 年。城市發展的時間感很特別

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- 嘉義市是台灣「南北分界」的物理座標（北回歸線穿過嘉義市西南 3.3 公里）。這座城市的緯度本身就在說「台灣的南北是被切開的」
- 火雞肉飯這道菜是「戰後台灣 + 美援雞肉」歷史的縮影（1950s 美國雞肉援助 → 台灣人發明火雞肉飯）
- 陳澄波之死是台灣畫家的歷史傷口（畫家為什麼會死在自己的城市的車站前？這個問題是台灣 20 世紀的縮影）
- 對讀者：「為什麼一座有北回歸線、阿里山、KANO、陳澄波的城市，卻是台灣最容易被略過的省轄市？」這個 frame 拆開後讓人重新看見一座「中等城市」的存在感

### 類型加權矩陣 — Geography 必想的問題

- **這片土地怎麼形成**：嘉南平原的東緣，阿里山山脈西麓，海拔 30-40 公尺平原地形，氣候是熱帶季風氣候（北回歸線南側），夏季悶熱多雷雨、冬季乾燥
- **誰在這裡生活**：原住民鄒族（阿里山一帶）/ 平埔族洪雅族（諸羅山社）/ 漳泉移民（清領）/ 日治日本人（林業 + 行政）/ 戰後外省移民（軍公教 + 空軍機場）/ 阿里山林業工人後代 / 嘉農畢業生
- **人和地有什麼共生**：北回歸線過境 → 全球第一座地標 / 阿里山林業 → 嘉義作為木材集散地的歷史（檜意森活村今天就是當年的林業營林宿舍）/ 嘉南平原 → 米倉 → 雞肉飯文化的基礎

### 切入點清單（候選 hooks）

1. **「全球第一座北回歸線標塔」的意義**（1908 日治為慶祝縱貫鐵路通車立 → 1912 颱風吹毀 → 1923 裕仁太子來台諭示改建 → 1926 第三代 → 今天看到的是第六代）
2. **「諸羅 → 嘉義」改名故事**（1786 林爽文之役諸羅城守城兩個月有功，乾隆親賜城名「嘉義」 — 全台唯一被皇帝賜城名的縣治）
3. **KANO 神話的當代意義**（1931 漢人 + 原住民 + 日本人混合棒球隊打進甲子園準優勝 → 2014 魏德聖電影復活 → 台灣多元族群協力的最早 case）
4. **陳澄波之死**（嘉義畫派代表，1947 二二八嘉義事件死於嘉義火車站前，全台日治帝展入選率最高的城市畫家死於戰後第一場政治悲劇）
5. **阿里山森林鐵路 + 嘉義作為木材城**（1912 通車 → 戰後阿里山林業顛峰 → 1989 林業政策轉向 → 今天變成觀光鐵路；檜意森活村是當年的林業營林宿舍變身）
6. **火雞肉飯的歷史**（1950s 美援雞肉 + 嘉南平原稻米 + 嘉義在地醬料 = 戰後台灣的食物創新；噴水雞肉飯 1949 開業）
7. **被嘉義縣包圍的省轄市**（1995 嘉義縣治從嘉義市遷往太保市，行政與地理的切割完成；嘉義市像一顆被嘉義縣的甜甜圈包圍的洞）

### 預期核心矛盾候選（待 Stage 1.4 收斂為 ≤ 30 字）

- A：全球第一座北回歸線地標，全台最容易被略過的省轄市
- B：被皇帝親賜城名的諸羅，今天變成「吃雞肉飯就走」的中繼站
- C：陳澄波死在這座城市火車站前，這座城市才開始有自己的畫派
- D：被嘉義縣包圍的城中之城（geographically encircled by its own former territory）

### 預想讀者帶走的那一件事

讀完這篇嘉義，會理解一座「中型城市」如何在 26 萬人的尺度上撐起四個全國級的文化 anchor（北回歸線、阿里山、KANO、陳澄波）—— 不是每座城市都需要 100 萬人才能站得住腳。嘉義是「中等規模如何深度經營」的台灣標本。
