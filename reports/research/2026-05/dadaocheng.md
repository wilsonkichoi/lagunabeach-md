## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **物件層**：迪化街南北貨店家紅紙黑字寫的「老字號 18XX」字樣 / 霞海城隍廟前那條年貨大街每年農曆春節前的人潮 / 永樂市場的布料攤 / 阿婆水煎包 / 加蛋的滋養豆奶
- **場景層**：清晨五點半的迪化街靜得像一個世紀前；除夕前一週的年貨大街擠到連空氣都香 / 屈臣氏大藥房 1917 年那棟巴洛克立面的浮雕花鳥；中藥行味道
- **跨世代差異**：
  - **70 歲以上**：阿公阿嬤輩記得這裡是茶行布行米行批發中盤 + 永樂座戲院聽歌仔戲 + 法主公廟拜拜
  - **40-60 歲**：1980 年代第二高架橋通車後迪化街沒落、1996 都市更新計畫差點拆迪化街、2000 年「都市再生前進基地」（URS）保存運動
  - **20-40 歲**：2010s「迪化街 207 博物館」「印花樂」「合興八十八亭」「ASW Tea House」等文創進駐 / 林五湖本館 / 民藝埕等品牌
  - **海外觀光客**：日韓客的「老台北一日遊」必排
- **句子層**：「永樂、霞海城隍、地藏王、慈聖宮」四個廟接成一條從南到北的軸線，台北人說「去大稻埕」其實是說「去廟旁邊的某條巷」

### 多元面貌

- **主流敘事**：「茶葉古城」「年貨大街」「文創復興」三段式觀光局論述
- **支線敘事**：
  - **公共衛生史**：1918 大正流感台灣分院在大稻埕，李春生家族曾出資籌建大稻埕醫院
  - **教育史**：1885 西學堂台灣第一個現代式學堂 / 1898 大稻埕公學校（今太平國小）
  - **政治史**：1921 蔣渭水太平町 199 番地大安醫院創立、台灣文化協會在此誕生、1923 治警事件、1947 二二八第一槍在天馬茶房
  - **女性史**：1922 文化協會推動的「女子青年會」/ 1925 蔡阿信第一位台籍女醫師接管大稻埕醫院
  - **音樂史**：1932 純純《望春風》/ 1934 古倫美亞唱片（在永樂町）/ 1940s 鄧雨賢《雨夜花》
  - **被忽略的角度**：
    - 1853 頂下郊拼的械鬥死亡（同安人逃難中至少一說 38 人死）
    - 凱達格蘭族大浪泵社（Tappari）漢人來之前已被「消失」
    - 1947 二二八後 38 年戒嚴期間，大稻埕從「政治起點」變成「沒人敢提的地址」
    - 2010s 文創潮帶來租金漲，老南北貨店漸退場
- **情感 fault lines**：
  - 老在地人 vs 文青遊客（南北貨老闆對 IG 打卡客的觀感）
  - 觀光發展 vs 保存運動（1996 拓寬計畫 vs 反拓寬運動）
  - 茶葉光榮敘事 vs 二二八沉痛記憶並存

### 歷史脈絡（pre-search hypothesis）

- **形成期**（1851-1860）：清領咸豐年間，幾戶人家在大稻埕北側「中街」開店，1853 頂下郊拼後同安人大量遷入，1860 淡水開港讓這個原本邊緣的小聚落變成北台灣對外貿易節點
- **關鍵轉折 1**（1860-1895）：茶葉時代 — John Dodd / 李春生 / 1869 Formosa Tea / 劉銘傳省會選址 / 1891 鐵路通車
- **關鍵轉折 2**（1895-1945）：日治三市街並立 — 城內日本人 / 大稻埕台灣人 / 艋舺舊神，1920 三市街合一為台北市，1921 文化協會、太平町通成為「台灣人的銀座」
- **關鍵轉折 3**（1945-1987）：戰後沉默 — 1947 天馬茶房第一槍、戒嚴 38 年大稻埕從政治起點變沉默地址、1980s 後因第二高架橋通車跟新興商圈興起而沒落
- **當代意義**（1996-2026）：保存運動 → 都市再生 → 文創復興 —「歷史層次最厚的台北 800 公尺」

### 切入點清單（待搜尋驗證 / 反駁）

1. **「茶葉做生意的街、辦過西學堂、又是二二八第一槍」三層歷史壓在 800 公尺**：核心矛盾候選之一，最 sharp
2. **「凱達格蘭族大浪泵社 → 漢人大稻埕 → 日治大正町太平町 → 戰後迪化街 → 21 世紀文創街」五次更名背後的權力史**：從地名變更看殖民疊代
3. **物質層 anchor**：陳天來故居（貴德街 73 號）/ 李春生大厝（已拆，僅存基督長老教會大稻埕教會）/ 屈臣氏大藥房（1917 巴洛克）/ 林五湖本館 / 霞海城隍廟（1859 同安人遷建）
4. **「茶葉之父」李春生：信仰、買辦、財富、現代化先鋒**的人物 anchor
5. **二二八天馬茶房的物理位置**：南京西路 189 號（法主公廟對面）— 今天那塊地是什麼狀態
6. **1996 拓寬計畫 vs 反拓寬運動**：保存運動分水嶺
7. **音樂時代的太平町**：1932《望春風》在永樂町（今迪化街南段）哥倫比亞唱片公司錄製

### 預想讀者帶走的那一件事

讀完後，當讀者下次走在迪化街，會抬頭看那些洋樓的浮雕，並意識到：腳底下踩過的這 800 公尺，是台灣茶葉貿易、現代化教育、台灣人政治意識、戰後沉默、保存運動，五層歷史一層疊一層的物理切片。觀光客拍的是建築立面，台北人活的是這條街每一塊磚壓著哪個世紀。
