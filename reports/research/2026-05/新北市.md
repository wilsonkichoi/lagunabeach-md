# Research Report: 新北市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：板橋（縣府）/ 淡水（紅毛城 + 老街 + 漁人碼頭）/ 瑞芳金瓜石 + 九份 / 三峽（祖師廟 + 老街）+ 鶯歌（陶瓷）/ 烏來（泰雅族 + 溫泉 + 內洞瀑布）/ 平溪天燈 / 八里左岸 + 十三行博物館 / 福隆海水浴場 / 野柳女王頭 / 林口高鐵 / 三重二重疏洪道 / 永和豆漿 / 中和韓國街
- **世代差異**：
  - 戰前老世代知「板橋林家 1853」「九份金礦 1890」「平溪 1929 鐵路通車」「淡水 1858 開港」
  - 戰後 50-70s 世代知「1947 二二八板橋林宅事件」「1970s 三重二重工業移民」「1985 金瓜石關閉」
  - 80-90s 世代知「2003 新莊副都心 + 2010/12/25 升格新北市」「永和河堤 + 中和韓國街移民」
  - 千禧後世代知「2017 環狀線通車」「淡水輕軌 2018」「新北流行音樂中心 2022」
- **外人 vs 在地人**：外人覺得「淡水 + 九份 + 平溪天燈」就走人；在地人知「新北是包圍台北的環狀都會」「29 區跨幅極大」「板橋人不覺得自己是新北人」「永和中和原本是台北的延伸」「我們撐住台北的睡眠 + 通勤 + 工廠」

### 多元面貌

- **主流敘事**：「全台最大人口的睡眠城」（一種被輕視的反射）
- **支線敘事**：
  - **環狀包圍台北**：29 區圍繞台北市 12 區，全台最大人口 401 萬（2025 統計）
  - **1875 設淡水縣**（含整個新北市 + 台北市萬華大同）
  - **1920/10/01 改台北州**（與台北市同步設立，新北全境是台北州外圍）
  - **1945/12/25 設台北縣**：21 市鄉鎮（戰後第一次行政劃分）
  - **2010/12/25 縣升格直轄市新北市**：與台中市 + 台南市 + 高雄市同期，但新北是縣升格直轄市
  - **板橋林家 1853 起家**：林本源家族第二代林維讓 + 林維源建林家花園 1893
  - **淡水 1858 開港**：天津條約後外國人通商，紅毛城 1629 西班牙建 → 1644 荷蘭 → 1684 清 → 1868 英租
  - **九份 1890 + 金瓜石 1890s 金礦熱潮**：日治時期亞洲第一金礦
  - **平溪 1929 鐵路通車**：原為運煤鐵路（基隆煤田延伸）
  - **2003 新莊副都心 + 林口重劃區**：人口接住北漂
  - **三峽 1738 祖師廟 + 鶯歌 1804 陶瓷起源 + 烏來原住民泰雅族**
  - **1979/12/10 美麗島事件後林宅血案 1980/02/28**：林義雄家在台北市信義路三段，不在新北
- **fault lines**：
  - 板橋（縣府傳統中心）vs 淡水（觀光 vs 通勤）vs 新莊副都心（後發新興）vs 永和中和（台北延伸）vs 三峽鶯歌（傳統工藝）vs 瑞芳九份（礦業遺址）vs 烏來（原住民部落）
  - 包圍台北的 29 區身份分歧：板橋 / 三重 / 新莊 / 中永和 vs 瑞芳 / 烏來 / 平溪 / 雙溪 / 貢寮（同一個直轄市卻三種年紀）
  - 北漂他鄉 vs 在地認同：401 萬人裡多少是新北原生，多少是北漂
  - 2010 升格 vs 既有縣治傳統：板橋的縣府身份還有沒有意義

### 想法跟感受

- **正面**：對 401 萬全台最大人口的存在感 / 對板橋林家 + 淡水紅毛城 + 九份金瓜石的歷史驕傲 / 對 29 區內部多元的自覺 / 對 2010 升格的進化感
- **痛點**：「台北的睡眠城」標籤的反作用 / 房價接住北漂但年輕人撐不住 / 板橋 vs 新莊 vs 淡水的內部不均 / 烏來 + 平溪 + 瑞芳的偏鄉問題 / 林家花園的歷史與當代脫節
- **驕傲**：全台最大人口 / 全台最複雜的環狀都會 / 包含 1853 林家 + 1858 淡水開港 + 1890 九份金礦 + 1929 平溪鐵路 4 個歷史軸線 / 原住民泰雅族烏來部落保留

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- **環狀包圍的政治經濟**：401 萬人撐起台北中央政府 + 通勤族 + 工業生產 + 觀光景點，但新北人在自我認同上又跟台北市分割
- **板橋林家百年**：1853 落腳 → 1893 花園 → 1947 二二八 → 今天景點化，林家是台灣本土資本主義的第一個案例
- **淡水 1858 開港的世界連結**：天津條約後新北最早接觸西方，紅毛城 + 牛津學堂 + 真理大學 1872 都從淡水開始
- **礦業遺址的記憶**：九份金瓜石 1890-1985 = 95 年的金礦時代，1985 關閉後變觀光景點，礦工後代散在台北縣
- **原住民泰雅族烏來**：唯一山地原住民鄉，泰雅族保留地，福山部落 + 信賢部落
