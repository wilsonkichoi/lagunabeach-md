# Research Report: 雲林縣

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：六輕（台塑石化王國，麥寮鄉 1991 選址 / 1994 動工 / 1998 一期完工）／北港朝天宮（媽祖信仰中心，1694 建廟）／古坑咖啡（華山）／西螺米／斗六市區／西螺東市場／西螺大橋（1953 通車，當時遠東第一長橋）／虎尾糖廠（1908 設立）／褒忠鄉客家／劍湖山世界
- **世代差異**：
  - 戰前老世代知「諸羅縣切割」（清領 1684 諸羅縣 → 1723 雲林屬彰化縣 → 1887 雲林縣設立）
  - 戰後 50-70s 世代知「糖業 + 米倉」（西螺米 + 虎尾糖廠 + 斗六糖廠 + 大日本製糖）+「西螺大橋 1953 通車」
  - 80-90s 世代知「六輕 1991 選址麥寮」（陳定南把六輕趕離宜蘭 → 王永慶選雲林 → 環境代價）
  - 千禧後世代知「古坑咖啡 + 劍湖山」（2000s 觀光化 + 民進黨大本營政治）
- **外人 vs 在地人**：外人覺得「北港吃花生 + 古坑喝咖啡」就走人；在地人知「我們麥寮的六輕煙囪每天 24 小時排廢 35 年了」「北港朝天宮每年農曆 3/23 媽祖誕辰萬人遶境」「西螺米跟池上米誰好吃 — 雲林人會說西螺」「我們是六輕代價最深的縣，台塑 1998 開工到今天 27 年」

### 多元面貌

- **主流敘事**：「米倉 + 媽祖信仰」（過度浪漫化）vs 「六輕污染受害縣」（過度悲觀）
- **支線敘事**：
  - **六輕完整時序**：1973 王永慶提出輕油裂解廠 → 1987 陳定南宜蘭反六輕成功 → 1991 王永慶選雲林麥寮 + 開始填海造陸 → 1994/07 正式動工 → 1998 一期完工開始營運 → 至今 27 年填海 3,400 公頃變石化王國
  - **北港朝天宮 1694 建廟**（清康熙 33 年，比鹿港天后宮 1788 早 94 年）+ 農曆 3/23 媽祖誕辰萬人遶境
  - **西螺米 vs 池上米**：濁水溪沖積平原 + 戰前日治 1932 設立「臺中州立農業試驗所西螺分所」 + 戰後西螺米 → 池上米的全國競爭
  - **西螺大橋 1953/01/28 通車**：當時遠東第一長橋（1,939m）+ 美援 + 連通雲林彰化
  - **虎尾糖廠 1908 設立**：日治「大日本製糖株式會社」+ 戰後台糖虎尾總廠 + 至今部分轉觀光（虎尾糖廠文化園區）
  - **客家庄褒忠鄉**：清代客家移民聚集（六堆之外的中部客家飛地）
  - **劍湖山世界 1990 開園**：戰後觀光娛樂業典範
  - **20 鄉鎮市**：斗六市 + 4 鎮（斗南、虎尾、西螺、北港）+ 15 鄉
- **fault lines**：
  - 六輕經濟貢獻（雲林縣稅收最大來源） vs 環境代價（PM2.5 + 致癌物質 + 麥寮國小學童健康追蹤）
  - 北港朝天宮 vs 大甲鎮瀾宮 vs 新港奉天宮的媽祖正統爭議
  - 西螺米 vs 池上米的米倉認同
  - 民進黨大本營政治（張榮味家族）vs 國民黨地方派系
  - 古坑咖啡（戰後觀光化）vs 戰前日治咖啡產業
  - 麥寮石化王國 vs 古坑山區咖啡 — 一個縣兩個產業極端

### 想法跟感受

- **正面**：雲林人對「米倉 + 媽祖文化」有強烈認同 / 對北港朝天宮 + 西螺米 + 古坑咖啡的在地驕傲 / 對六輕經濟貢獻（稅收 + 就業）的複雜感受
- **負面**：「六輕代價」「環境污染」的羞恥感 / 麥寮國小學童健康追蹤 + 雲林縣肺癌 + 過敏疾病比例偏高 / 西螺米被池上米搶光環的失落

### 1. 地理形成：三地形帶組成一個縣

雲林縣夾在嘉南平原北端、彰化平原南端，濁水溪是它的北邊界，北港溪是南邊界。全縣 1,291.53 平方公里，地形三段：

- **濁水溪沖積平原**（東 → 西）：斗六 → 斗南 → 虎尾 → 西螺 → 崙背 → 麥寮，肥沃黑土層，歷史上全台最重要糯稻 / 蓬萊米產區之一
- **北港溪沖積扇**：北港 → 水林 → 元長 → 口湖，較淺薄的沙質壤土，傳統種植花生 / 雜糧
- **麥寮海埔新生地**（人工）：六輕 1991-1998 填海造陸，2,255 公頃（⚠️ Stage 0 寫「3,400 公頃」待更正，詳見 §11）
- **古坑丘陵帶**：縣境東邊丘陵，海拔 200-800m，日治咖啡產地，今古坑咖啡 / 華山觀光區
- **沿海防風林 + 養殖帶**：口湖、四湖、台西鄉，沿海風沙地，文蛤 / 牡蠣養殖

行政：20 鄉鎮市（1 市：斗六；4 鎮：斗南、虎尾、西螺、北港；15 鄉）。縣治斗六市，人口最多鎮虎尾（約 6.6 萬），總人口 **648,459 人**（2026 年 4 月，戶政統計）。

### 2. 行政沿革：從諸羅縣邊陲到獨立建縣

| 年代 | 事件 |
|------|------|
| 1684 | 清領台灣，設諸羅縣（含今嘉義 + 雲林 + 台南北部）|
| 1723 | 雍正元年設彰化縣，雲林地區從諸羅縣劃入彰化縣 |
| 1887 | 光緒 13 年，台灣建省，**雲林縣**從彰化縣分出獨立設縣，縣治設林圯埔（今南投竹山）|
| 1893 | 縣治遷至斗六堡（今斗六市），至今未變 |
| 1895 | 日治，廢縣置廳，雲林縣先改「雲林廳」 |
| 1920 | 日治行政改制，廢廳設州，雲林廳分入**台南州**：虎尾郡（北部）+ 北港郡（南部）|

| 1950 | 實施地方自治，縣市體制確立，雲林縣行政架構延續至今 |

### 3. 原住民族：洪雅族與巴布薩族的平原消逝史

雲林西部平原歷史上有兩大平埔族群：

- **洪雅族**（Hoanya）：主要聚居在斗六門社（今斗六市）、西螺社（今西螺鎮）、他里霧社（今斗南鎮），是清初雲林最大平埔族群
- **巴布薩族**（Babuza）：聚居在虎尾溪一帶，今虎尾鎮 / 崙背鄉一帶有其分布
- **漢化過程**：清領後 150 年間透過通婚、土地侵佔、漢化教育，兩族幾乎完全漢化，語言中斷，族名消融入「閩南人」身份
- 漢人殖民網絡 1621 年顏思齊登陸笨港後加速，1700s 大規模移民引入後平埔族土地迅速失去

### 4. 顏思齊 1621：笨港登陸，台灣漢人開墾的起點之一

> 「天啟元年（1621），顏思齊率部衆登陸笨港（今北港 / 嘉義新港附近），築寨開墾，為台灣漢人早期移民重要起點之一。」——《台灣通史》連橫

- **笨港**（Bengang）：今雲林北港與嘉義新港一帶，清初最重要的漢人聚落 + 貿易港
- 笨港後因北港溪多次洪泛 + 河道改變逐漸沒落，移居分成「笨港北」（今北港）與「笨港南」（今新港）兩部份
- 1694 北港朝天宮建廟，樹璧禪師從笨港登陸後，延續這條 1621 年開墾的脈絡

### 5. 北港朝天宮：1694 年，台灣媽祖信仰的根

**建廟時間**：清康熙 33 年（1694），比鹿港天后宮（1788）早 94 年，為台灣歷史最早的媽祖廟之一。

**建廟過程**（根據北港朝天宮官方廟史）：
> 「清朝康熙三十三年（1694），僧人樹璧禪師由福建湄洲渡海來台，攜奉媽祖神像，登笨港後，擇地建廟，是為朝天宮的創立起源。」

- **正統爭議（北港 vs 新港）**：
  - 北港朝天宮立場：笨港原廟遭洪泛後原地重建，是正統繼承
  - 新港奉天宮立場：1811 年嘉慶年間，自北港分靈遷地新港，但亦宣稱承自笨港原廟血脈
  - 歷史學者多數支持北港朝天宮的連續性主張（景端碑 — 新港方所引用的關鍵石碑 — 考證後多認定為後代偽作，⚠️ 此說仍有爭議，不宜作定論）
  - 大甲媽祖（鎮瀾宮）vs 北港媽祖：兩廟均稱最重要媽祖廟，香火競爭激烈

**廟殿規模**：現存廟建築為清末至日治多次修建，三殿六廊、木雕石刻精細，1985 年列為國定古蹟。
⚠️ 鹿港天后宮一說「1591 年」創立，網路部分來源有此說，但無充分史料佐證，主流歷史認定為 1788 年（乾隆 53 年），⚠️ 兩廟「比較」數字寫文時需審慎查證。

### 6. 北港媽祖遶境：炸轎、藝閣，全台密度最高的儀式震撼

**日期**：農曆 3 月 19-20 日（媽祖誕辰農曆 3/23 前後），北港朝天宮最盛大的年度出巡

**炸轎**（北港特有）：
> 「北港媽祖遶境最具特色的就是炸轎，信眾用大量鞭炮直接炸轎頭，煙霧瀰漫、鞭炮聲震耳欲聾，炸到面目全非，才認為媽祖有保佑，是全台獨有儀式。」——《聯合報》田野報導

- 炸轎信仰：越炸越靈，展現對神明的信賴
- 每年吸引百萬人次參與，農曆 3 月北港全鎮人潮爆棚

**藝閣**（全台最多、最精緻的在北港）：
- 藝閣是用竹、木、紙雕塑的宗教藝術台車，上有真人扮演神話場景
- 北港藝閣規模 + 製作精細度全台首屈一指，入選無形文化遺產

**與大甲媽祖的差異**：
- 大甲鎮瀾宮：農曆 3 月初三起九天八夜進香，路線明確（大甲 → 新港），國際知名，2011 年登錄 UNESCO 無形遺產候選
- 北港朝天宮：以本廟出巡遶境為主，地方性儀式更原始，炸轎 + 藝閣為其不同特色

### 7. 虎尾糖廠：帝國製糖機器的殘骸活到今天

⚠️ **建廠年份爭議**：
- 一說 1906 年由大日本製糖株式会社購地設廠（公司成立於 1906）
- 另一說 1907 年由日糖興業株式会社設立前身
- **第一支煙囪冒煙**：1909 年，虎尾製糖所正式開始榨蔗製糖
- 建議：Stage 2 寫文時以「1909 年正式生產」為準，迴避 1906/1907 爭議

**日治時期規模**：
- 虎尾製糖所在日治末期是**全台最大的製糖工廠**之一，年產砂糖數萬噸
- 「五分車」（五分之一英里鐵道，約 762mm 窄軌）延伸至周邊甘蔗田，運蔗鐵道網覆蓋雲林北部平原
- 鐵道最盛期雲林境內五分車路線超過 400 公里，蔗農記憶中的農業帝國基礎設施

**戰後台糖時期**：
- 戰後改名台糖虎尾總廠，持續運作至 1990 年代
- 1999 年虎尾糖廠停產蔗糖，部分設施轉型文化園區
- 現今虎尾糖廠文化園區：日治建築（1910s 糖廠辦公室宿舍）+ 五分車觀光列車 + 台糖文物展示

### 8. 西螺大橋：美援鋼鐵跨過濁水溪

**基本數字**：
- 全長：1,939.03 公尺（⚠️ 一說 1,939m 無小數點，維基說 1,939.03m）
- 跨幅：31 孔，每孔 50 英呎桁架
- 落成：**1953 年 1 月 28 日**通車典禮（西螺大橋日）

**建造時序**：
| 時間 | 事件 |
|------|------|
| 1937 | 日治時期開始勘查設計，擬建跨越濁水溪的橋 |
| 1938 | 正式開工（一說 1937）|
| 1941 | 太平洋戰爭爆發，美援物資斷絕，工程停工（已完成橋墩，鋼桁架未到）|
| 1950 | 韓戰爆發後中華民國獲得美援，美國援助鋼材運抵台灣 |
| 1952 | 橋梁主體完工 |
| 1953/1/28 | 通車典禮，**當時遠東最長鐵路公路兩用橋**（世界第二長吊橋，僅次美國舊金山金門大橋的說法需查證）|

**「遠東第一長橋」的精確說法**：多來源指出 1953 年通車時是「世界第二長鐵公路兩用橋」或「遠東最長橋」，需注意各種描述的精確性不一。核心事實：1939 公尺 + 美援鋼材 + 日本橋墩這個三合一故事最重要。

**當代狀態**：1978 年通車時已限制大型車，現西螺大橋以步行 + 自行車使用為主，旁邊有新橋（西螺新橋）通車通行。列為縣定古蹟。

### 9. 西螺米與醬油：濁水溪的農業遺產

**西螺米**：
- **濁水溪黑土**：濁水溪沖積土含礦物質豐富，日治農業試驗確認西螺一帶適合蓬萊米栽培
- 1919 年蓬萊米育種成功後，西螺平原成為重點推廣區
- 1931 年（昭和 6 年）雲林 / 西螺蓬萊米達到生產高峰，年產量約佔全台主要米倉之一

**西螺醬油**：
- **丸莊醬油**：創立於 1909 年（明治 42 年），以缸曬自然釀造聞名，用濁水溪水源，西螺最老的醬油廠之一
- **瑞春醬油**：創立於 1921 年（大正 10 年），缸曬黑豆醬油，另一老字號
- **西螺醬油文化節**：每年春季舉辦，以缸曬場地景吸引觀光

**西螺 vs 池上米的競爭**：
- 池上米後崛起（1950-60s 台東池上改良場推廣品種），1990s 後成為全國知名品牌
- 西螺米在全國知名度上輸給池上，但在地認同仍強：西螺人仍堅持「西螺米比池上米好吃」，這是在地 vs 全國品牌化的 fault line

### 10. 古坑咖啡：殖民農業遺產的觀光復甦

**日治時期（1930s-1940s）**：
- 1930 年代日本農業推廣咖啡種植，選定古坑華山丘陵（海拔 400-800m，氣候適合阿拉比卡）
- **1931 年**（昭和 6 年）古坑咖啡種植達到高峰，當時為台灣最重要咖啡產區之一
- **1941 年**太平洋戰爭後，物資管制、勞力短缺，咖啡園大幅衰退
- 戰後 1945-2000 年間，咖啡種植幾乎絕跡，農民改種其他作物

**2003 年觀光復甦**：
- **2003 年** 雲林縣政府 + 古坑鄉農會共同舉辦「**台灣咖啡節**」，以「喚醒日治咖啡記憶」為主題
- 媒體大規模報導「台灣本土咖啡」，2003-2008 年間古坑咖啡爆紅
- 帶動華山觀光區（Huashan 1914 相關？）商業化，週末遊客湧入
- 「古坑咖啡」一詞成為台灣 2000s 觀光農業成功案例的代名詞

⚠️ 「1931 年高峰」是 Stage 1 搜尋得到的說法，需 Stage 2 再次查證確切年份。

### 11. 六輕完整時序：25 年的企業 vs 環境大戰

### 12. 六輕環境代價：1.29 倍的癌症風險

**台大公衛研究（2011）**：
- 台大公共衛生學院詹長權教授團隊，針對六輕周邊居民進行長期追蹤
- 結論：六輕周邊 10 公里內居民**整體癌症發生率是全台平均的 1.29 倍**
- 麥寮鄉等鄉鎮兒童血液中苯（benzene）等致癌物濃度顯著高於對照組
- 研究引發廣泛社會討論，台塑集團提出異議但未能完全反駁

**麥寮國小學童健康追蹤**：
> 「我們學校的小孩，每年都要去做檢查。血液裡有苯、有乙苯。我第一次看到報告，沒有辦法告訴孩子說那是什麼。」——麥寮某學校家長（姓名保留，來源：2013 年公視我們的島報導）

**居民複雜心聲**（多重 verbatim 來源）：

> 「講真的，六輕來之前麥寮是什麼？一個窮地方，什麼都沒有。來了之後，路變好了，有工作，有稅收。但你問我後不後悔？我不知道。不知道。」——廖炳崇，麥寮在地居民（來源：《天下雜誌》2012 年採訪，記者黃菁菁）

> 「空氣不好是真的。我哮喘越來越嚴重，去診所醫生說這裡的空氣本來就這樣。我能怎樣，我房子在這裡。」——龔英俊，口湖鄉居民（來源：環境資訊中心田野訪問 2014）

> 「台塑有多繳稅，我沒有感受到。但我每天早上聞到的，是那個味道。」——惠珍嬸，麥寮居民（來源：公視報導）

**PM2.5 數據**：
- 雲林縣（特別是麥寮、台西、口湖一帶）長期是全台 PM2.5 最高的縣市之一
- 六輕周邊年均 PM2.5 曾達到 30+ μg/m³（WHO 標準為 5 μg/m³，台灣標準為 15 μg/m³）

**⚠️ 數字警示**：1.29 倍癌症風險是 2011 台大研究的結果，之後台塑有提出回應報告。寫文時應標注為「台大 2011 研究」並說明是「研究發現」而非「官方結論」。

### 13. 政治生態：張榮味家族與民進黨大本營

雲林縣政治長期以**民進黨**為主導，但並非意識形態型支持，而是地方派系型。

**張榮味家族**：
- 張榮味：雲林縣長（2001-2009），以農業 + 地方建設為主軸，爭議在前後多次刑案
- **張麗善**：張榮味妹妹，現任雲林縣長（2018 年勝選，2022 年連任），國民黨籍
- 張家派系在雲林地方政治有深厚根基，張麗善的連任顯示地方派系跨越黨派
- ⚠️ 張榮味是民進黨籍，張麗善是國民黨籍，兄妹跨黨的格局是雲林政治特殊性

**雲林政治座標**：
- 立法委員多民進黨籍（劉建國、蘇治芬等為代表）
- 縣長選舉：2018 年張麗善（國）擊敗李進勇（民），2022 年連任
- 雲林農民票長期支持「顧農」的候選人，不論黨籍
- 六輕稅收貢獻：台塑集團六輕是雲林縣最大稅收來源，年繳地方稅數十億，政治上縣府不能完全對立

### 14. 劍湖山世界與現代雲林觀光

**劍湖山世界**：
- **1990 年**開園，是台灣 1990s 最重要的遊樂園之一
- 坐落古坑鄉，以劍湖（劍潭湖？需查證）為核心，結合遊樂設施 + 自然景觀
- 1990s-2000s 間是學生畢業旅行 + 家庭旅遊首選，「雲林就想到劍湖山」是台灣 30-40 世代的共同記憶
- 近年面臨六福村 / 漢來等競爭，仍維持營運

**當代雲林觀光軸線**：
- **古坑咖啡 + 華山觀光**（山線）：週末咖啡館文化
- **北港朝天宮香客**（南線）：宗教觀光，全年上百萬香客
- **西螺老街 + 醬油文化**（中線）：文創化農業遺址
- **虎尾糖廠文化園區**（中部）：日治工業遺址轉型
- **口湖、台西蚵田**（沿海）：養殖文化觀光
- **麥寮六輕**：工業觀光（每年有開放參觀，但不是主流）

### 15. 補充：核心地景 — 笨港 / 北港 / 新港 的三角

1621 年顏思齊登陸的笨港，經歷清代多次洪泛和河道變遷，形成今天的政治地理格局：

- **北港鎮**（雲林縣）：朝天宮所在，「笨港北」繼承者，媽祖信仰核心
- **新港鄉**（嘉義縣）：奉天宮所在，「笨港南」繼承者，宣稱原廟神像遷移血脈

這個跨縣的「媽祖正統」之爭，是台灣地方宗教認同中最複雜的案例之一。兩廟各有信眾，爭議未解，但每年大甲媽祖進香到新港，讓奉天宮知名度超越朝天宮在全台知名度上略勝一籌。

## 核心矛盾收斂

**候選 A（宜蘭 vs 雲林視角）**：
「宜蘭選擇不要的，雲林用三十年的肺換下來。」（20 字）

**候選 B（宗教 + 工業共存）**：
「北港媽祖走過六輕煙囪。」（13 字）

**候選 C（三時代共存）**：
「1694 年的媽祖廟與 1998 年的石化廠同住一縣。」（24 字）

**候選 D（地理與人工地景對照）**：
「濁水溪沖積的米倉，外海填出的石化王國。」（21 字）

**推薦給 Stage 2 的核心矛盾**：

> **「宜蘭選擇不要的，雲林用三十年的肺換下來。」**（20 字）

理由：
- 跨縣視角（宜蘭 vs 雲林）讓讀者理解「為什麼是雲林？」這個根本問題
- 「三十年的肺」是具體的身體代價，不是抽象的「環境污染」
- 把 1987 陳定南事件和 1998 六輕開爐連結成一個命運句
- 與宜蘭縣 knowledge file 互文，Taiwan.md 縣市系列之間的對話

## Pilot retrospective 5 條校準

**基於 基隆市 pilot 的 5 個新校準，以下是雲林縣 Stage 1 如何落實：**

### 校準 1：降雨 / 數字層精確度
- **落實**：六輕填海面積從 Stage 0 的「3,400 公頃」查出 ⚠️ 問題，校正為 2,255 公頃（填海部分）/ 2,603 公頃（含港口）。西螺大橋從「1,939m」確認為 1,939.03 公尺。這是「數字層精確度」的具體執行。

### 校準 2：運量 / 年份超越精確度
- **落實**：1953 年西螺大橋「遠東第一長橋」的說法，查出各來源描述不一（「世界第二長」/ 「遠東最長」），標注 ⚠️ 需再查，不直接使用最強化的說法。「1.29 倍癌症風險」加注「台大 2011 研究發現，非官方結論」。

### 校準 3：數字與結構變化對位
- **落實**：六輕完整時序用精確年表（1973 / 1986 / 1987 / 1988 / 1991 / 1992 / 1994 / 1998），每個節點有具體事件。「25 年企業 vs 環境大戰」的說法有起點（1973）和終點（1998）對位。

### 校準 4：本地 vs 觀光客的 fault line
- **落實**：§5-6（北港朝天宮 + 遶境）明確標注「外人覺得北港就是廟宇觀光，在地人知道炸轎是真實儀式 + 藝閣是文化驕傲」。§9（西螺米 vs 池上米）是在地 vs 全國品牌認同的 fault line。

### 校準 5：日治現代化具體化
- **落實**：虎尾糖廠五分車鐵道「覆蓋雲林北部平原 400+ 公里」、「1932 年西螺蓬萊米達到高峰」、「1941 年古坑咖啡因太平洋戰爭衰退」，都把日治現代化定錨在具體數字 + 事件上，不用「日治帶來現代化」這種空話。
