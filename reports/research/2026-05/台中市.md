# Research Report: 台中市

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **大眾共有 anchor**：太陽餅 / 鳳梨酥 / 逢甲夜市 / 第三市場 / 大甲鎮瀾宮（媽祖遶境起點）/ 台中歌劇院（伊東豐雄設計）/ 國立美術館 / 中央書局 / 一中街 / 東海大學 / 中興大學 / 中科 1994 / 大坑步道 / 高美濕地 / 谷關溫泉 / 武陵農場（部分跨南投仁愛 + 部分台中和平）
- **世代差異**：
  - 戰前老世代知「1887 台灣建省設台灣府於台中藍興堡（後遷台北）」「1896 日治台中縣」「1920 設台中州 + 台中市」
  - 戰後 50-70s 世代知「1945 設台中市 + 台中縣分開」「1972 中山高通車經台中」「1978 中港路（今台灣大道）拓寬」
  - 80-90s 世代知「1994 中部科學園區規劃 + 2003 投運」「2010/12/25 縣市合併直轄市」「2016 國家歌劇院落成」
  - 千禧後世代知「2018 世界花卉博覽會 9 月-2019/04」「2020 林佳龍卸任改胡志強再上」「2024 盧秀燕政績」
- **外人 vs 在地人**：外人覺得「太陽餅 + 逢甲夜市 + 歌劇院」就走人；在地人知「台中是中部生活圈樞紐」「我們橫跨平地 + 山地 + 海線」「2010 縣市合併後變得很大」「285 萬第二大但常被忽略」「大甲鎮瀾宮才是真正的中部精神」

### 多元面貌

- **主流敘事**：「中部生活圈樞紐 + 285 萬第二大」（一種 underrated 的反射）
- **支線敘事**：
  - **中部生活圈樞紐**：橫跨 29 區，從海線（清水 + 沙鹿 + 大甲）到平原（南屯 + 北屯 + 西屯 + 中區）到山地（和平鄉跨大雪山）
  - **1887 設台灣府於藍興堡（現台中市區）+ 1894 遷台北**：台中曾短暫做台灣府治
  - **1896 日治設台中縣** + **1920/10/01 設台中州 + 台中市**：日治大正九年地方制度改革
  - **1945 設台中市 + 台中縣分開**：戰後第一次行政劃分（台中市 vs 台中縣）
  - **2010/12/25 縣市合併升格直轄市台中市**：與新北 + 台南 + 高雄同期，29 區
  - **大甲鎮瀾宮媽祖遶境**：每年農曆三月台灣最大規模宗教活動（橫跨台中 + 彰化 + 雲林 + 嘉義 + 嘉義）
  - **1994 中科規劃 + 2003 中部科學園區設置**：台積電 + 友達 + 美光 + 台灣半導體南北軸線中部節點
  - **1955 東海大學設立** + **1988 國立美術館** + **2016/09/30 國家歌劇院落成（伊東豐雄設計，正式名稱臺中國家歌劇院）**
  - **2018 世界花卉博覽會（11/03-2019/04/24）**：后里 + 外埔 + 豐原三園區
  - **政治派系**：紅派 + 黑派 + 顏家系（顏清標清水）+ 賴清德 vs 林佳龍 vs 胡志強 vs 盧秀燕政權交替
- **fault lines**：
  - 海線（清水 + 沙鹿 + 大甲 + 龍井 + 大肚 + 烏日）vs 平原（中區 + 西區 + 北區 + 南區 + 東區 + 北屯 + 西屯 + 南屯）vs 山地（和平 + 新社 + 東勢 + 石岡）
  - 2010 縣市合併前 vs 合併後：原台中市菁英集中 vs 原台中縣 21 鄉鎮邊緣化
  - 顏家紅派 vs 黑派 vs 民進黨：政治三角
  - 太陽餅老店 vs 文創新店：傳統商業 vs 新興品牌
  - 中科 vs 一中商圈 + 逢甲夜市：產業轉型與文化保留
  - 大甲鎮瀾宮 vs 鹿港天后宮（彰化）+ 北港朝天宮（雲林）：媽祖三大廟

### 想法跟感受

- **正面**：對 285 萬第二大人口的存在感 / 對國家歌劇院 + 國美館的文化驕傲 / 對大甲鎮瀾宮 + 媽祖遶境的精神認同 / 對 29 區內部多元的自覺
- **痛點**：「常被忽略」的反作用 / 中科發展 vs 在地產業壓力 / 太陽餅老店分裂 + 文創店過多 / 顏家政治影響力的爭議
- **驕傲**：1887 曾短暫做台灣府治 / 中部第一的政治經濟文化樞紐 / 大甲鎮瀾宮全台最大規模遶境 / 國家歌劇院伊東豐雄世界級建築 / 中科是台灣南北半導體軸線的中部節點

### 對社會 / 歷史 / 環境 / 我們人生的關聯

- **1887 短暫做府治**：劉銘傳建省後選台中藍興堡為省會，但因瘧疾與基礎設施不足 1894 遷台北 — 台中差點就成首都
- **大甲鎮瀾宮的全台精神**：每年農曆三月遶境跨 5 縣市、走 9 天 8 夜、約 10 萬信徒，是台灣最大規模民俗活動，文化部 2010 列為國家重要民俗
- **921 地震記憶**：1999 台中縣東勢 + 石岡 + 大里重災，是除了南投以外受創最深的縣市
- **中科 vs 竹科 vs 南科**：1994 規劃時的「南北軸線中部節點」，2003 後與台積電 + 友達 + 美光形成中部半導體核心
- **國家歌劇院的世界建築**：伊東豐雄「壺中居」設計理念，2016 開館後成中部文化新地標，與台北兩廳院 + 高雄衛武營形成「北中南三大表演藝術中心」
