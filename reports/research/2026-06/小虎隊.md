# Research Report: 小虎隊（EVOLVE）

## 1. 觀點成型（Stage 0，編輯前思考）

### 對台灣人的記憶 anchor

- **聲音 anchor**：〈青蘋果樂園〉的合成器前奏 / 隊呼「小虎、小虎，神氣十足」/〈愛〉的合唱
- **畫面 anchor**：吳奇隆的後空翻 / 三人制服 / 卡帶與電台（買不起卡帶的人靠廣播「追」）
- **符號 anchor**：三隻虎暱稱（霹靂虎 / 小帥虎 / 乖乖虎）
- **世代差異**：
  - 戰後父母輩（如 78 歲阿公）：覺得吵、覺得在「學日本」，永遠是「別人的事」，用鄧麗君當規模參照
  - X 世代（如 51 歲老闆娘）：全盛期少女粉，床頭海報、明星週刊，現在回看「幾分是自己選的、幾分是媒體建構」
  - 千禧世代：抖音〈青蘋果樂園〉lofi remix 的碎片接觸，不知道母體
  - Z 世代（如 12 歲、22 歲）：零記憶，只認得「吳奇隆＝四爺/步步驚心」，問「他們是 BTS 那種嗎」

### 多元面貌

- **主流敘事**：台灣第一個（成功的）男子偶像團體 / 兩岸三地共同青春記憶 / 華語偶像工業先驅（紅孩兒→小虎隊→F4→S.H.E）
- **支線 / 被忽略**：
  - 進口零件論——日本少年隊的選秀+包裝模板、主打歌疑為日本翻唱、美式 LED 舞台
  - 飛碟唱片造神 vs 真實流行力（華視黃金時段壟斷曝光：紅才被說好，還是好才紅？）
  - 陳志朋的邊緣化（從一開始就邊緣，還是後來被推到邊緣？本人怎麼說？）
  - 三人西進後的主權歸屬（吳奇隆/蘇有朋/陳志朋後來主要在中國發展，這符號還屬於台灣嗎）
  - 東南亞華語圈流通時，「台灣」這個產地標籤隱不隱形（印尼移工 persona：當年以為是「中國歌手」）

- **感受 fault lines**：老一輩的「學日本」焦慮 / X 世代女性的青春再評估 / 批判者視懷舊為造神延伸 / 粉絲對陳志朋的平反聲量

### 切入點清單（待搜尋驗證 / 反駁）

1. **進口零件 vs 本土代謝**：少年隊選秀模板 + 主打歌日翻唱 + 美式舞台——幾乎全進口，台灣卻記成「自己的」。立體在「外來如何變成自己的」
2. **主權歸屬考題**：三人後來都西進中國 + 東南亞流通時台灣標籤隱形 →「這還是不是台灣的記憶」（Taiwan.md 核心 lens）
3. **第一 vs 第一個成功**：紅孩兒（1987 環球？）先行；「台灣第一個男團」是飛碟文案還是史學定位——誠實交代 framing
4. **造神 vs 流行力**：華視黃金時段壟斷曝光，紅才被說好還是好才紅
5. **解嚴因果**：1987 解嚴 → 娛樂鬆動 → 1988 出道，因果有多強（別過度套用）
6. **陳志朋的邊緣**：結構性還是個人運氣，本人 verbatim 說法
7. **製作端幕後**：製作人/編曲/舞台/「虎」形象決策——那群沒被記錄的人
8. **解散真相**：兵役兩度解散 vs 公司拆散 vs 各自選擇

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- A：零件全進口的偶像團體，台灣卻把它代謝成最堅固的自己的青春（本土化弔詭）
- B：台灣第一個男團，最紅的三個身體後來都過了海峽——這還是不是台灣的記憶（主權歸屬）
- C：飛碟造的神 vs 真實流行力——「第一個成功」，成功屬於誰

> **傾向收斂**：A 為主脊椎、B 為結尾轉折。小虎隊用進口零件組裝出台灣青春，而它最紅的身體後來都過了海峽——「這是不是台灣的」這個問題本身，就是台灣流行文化主體性的考題。Stage 1.4 fact-confirmed 後鎖定。

### 預想讀者帶走的那一件事

你以為小虎隊是「台灣自己的」，其實它的選秀模式、主打歌、舞台幾乎全是進口的——但台灣把進口零件代謝成最堅固的集體青春。「把外來變成自己的」這個本事，正是台灣流行文化（乃至台灣）的縮影；而當這個身體後來過了海峽，「它還是不是台灣的」就成了一道考題。

## 3. Findings by sub-topic（標信度）

### §A 出道事實鏈（信度高，中日英多源一致）

- 節目**原名《青春大對抗》**（1988-07-09 華視開播），**1989-02 改名《TV新秀爭霸站》**（高信度，TV新秀爭霸站維基逐字）。舊文只寫「TV新秀爭霸站」不精確。
- **張小燕＝製作人**（非主持人；主持人為曹蘭、湯志偉）。張小燕＋苗秀麗 1987 創**開麗創意組合**（經紀），飛碟唱片管**唱片發行（非全約）**。
- **小貓隊**：開播 6 人女子團體後縮為 3 人（小虎隊維基說「3 個女生」是後期狀態）；徐若瑄（徐淑娟）為後期成員之一。隊呼「**小虎、小虎，神氣十足**」。
- **三隻虎暱稱（一手中日英三源一致）**：吳奇隆＝**霹靂虎**（體育生、後空翻）、陳志朋＝**小帥虎**（舞蹈底、酷似張國榮）、蘇有朋＝**乖乖虎**（建中、會讀書）。命名出自經紀公司（開麗）。
- **出生**：吳奇隆 1970-10-31 / 陳志朋 1971-05-19 / 蘇有朋 1973-09-11。**年齡校正**：1988-07-09 時為 17/17/14（媒體常寫的「18/17/15」是 1988 秋天組團時的年齡；日本維基記 1988-10 正式組成）。寫作用「十幾歲的高中生」+ 出生年較安全，避免精確歲數踩雷。
- 蘇有朋＝台北市立**建國中學**生，1991 考上**台大機械工程系**（理工組第 5 志願）後休學。吳奇隆省立體專（柔道黑帶二段、跆拳道黑帶五段）。
- **製作端**：《逍遙遊》全碟製作人**陳秀男**（飛碟副總特助兼國內處協理），後期李子恆；飛碟「五陳二李」（陳志遠編曲 / 陳大力策劃造型 / 陳秀男製作 / 陳樂融詞 / 陳耀川曲 + 李壽全 / 李子恆）。「虎」形象＋三性格標籤＝經紀公司決策（無具名拍板者）。
- 周傳雄（小剛）為**傳聞**第四人（不擅舞 + 理念不合婉拒）——性質是「傳出」，無一手確認，寫作標「傳聞」。
