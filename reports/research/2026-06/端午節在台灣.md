## §觀點成型（Stage 0.6 — 編輯前思考）

### 六核心問題

1. **對台灣人是什麼樣的記憶**：端午又叫「五日節」，是「夏天正式開始」的身體記憶。集體記憶錨點：媽媽/阿嬤在廚房包粽、戰南北粽、午時立蛋、河邊看龍舟、門口掛艾草菖蒲、香包、「未吃五月粽，破裘毋甘放」這句台語俗諺。它是一個跟「家族廚房 + 季節轉換 + 平安」綁在一起的節日。

2. **多元面貌**：(a) 起源多元——屈原(忠)/伍子胥(怨)/曹娥(孝)/百越族龍圖騰祭祀(更早)/惡月防疫(實用)，不是單一英雄敘事；(b) 地域多元——北部粽/南部粽/中部粽/客家粄粽/鹼粽；(c) 台灣在地版本——鹿港龍王祭、大甲鐵砧山劍井午時水、各地龍舟賽；(d) 古今——防疫巫術 → 現代體育/環保賽事。

3. **想法感受**：現代台灣人的矛盾——醫療發達了、立蛋被物理學戳破了、午時水不會招財了，為什麼還照樣做？感受層在「儀式的真正功能不是它表面宣稱的那個」。

4. **歷史脈絡**：百越龍圖騰 → 戰國屈原敘事附會 → 漢魏「惡月/毒月」防疫 → 隨閩粵移民來台 → 日治/戰後在地化(午時水信仰、立蛋、戰南北粽) → 現代國際龍舟賽 + 環保轉型。台灣特殊性是核心：午時水(劍井)、立蛋、南北粽認同政治、鹿港古禮。

5. **社會關聯**：南北粽戰爭 = 台灣人的地域認同遊戲(同構於豆花鹹甜、肉圓蒸炸)；送粽禁忌(不送整串、剪繩、「吊肉粽」) = 對生死的敬畏；龍舟現代化 = 傳統如何轉型(國際化、企業組隊、環保)。

6. **類型專屬問題（Culture / 節慶）**：節慶文章的專屬問題是「這個節日怎麼在台灣土地上長出自己的形狀」。失敗模式 = 複述維基百科的中國端午起源編年。正解 = 在地人視角(EDITORIAL §飲食文化)，寫「端午節**在台灣**」獨有的東西。

### 核心矛盾（anchor，整篇貫穿）

> **端午節的每一樣習俗，本來都是古人對抗死亡的工具——艾草是殺蟲劑、香包是空氣清淨機、午時水是消毒水、立蛋是巫術、划龍舟是搜救/驅疫。科學早就一條條戳破了它們的原始功能（艾草防蚊有限、立蛋跟陽氣地心引力無關、午時水不會招財）。但在台灣，這個節日不但沒死，還活得比誰都旺。因為它在這座濕熱島嶼上長出的，從來不是防疫功能，是一套讓人每年確認「我們一家、這個社區、這座島，還在一起」的集體儀式。**

一句話 anchor 候選：**「藥效過期了，台灣人照樣年年服用」** / **「對抗的是瘟疫，留下的是團圓」**。

反向解釋編織（通行說法 → 反向）：

- 通行說法：端午節是紀念屈原。
- 反向：屈原是後世附會上去的標籤；端午本是農曆五月「惡月/毒月」的防疫節，骨子裡是一個關於「怕死」的節日。而在台灣，它又超越了防疫，變成確認集體存在的儀式。

## §6 Clean fact-pack（Stage 1 後合成 — 給 writer 的導航層；**不取代 §8 raw，writer 必讀 §8**）

> ⚠️ Writer 注意：本 §6 是導航層。§8 有逐字原文 + URL + 信度，下筆細節/引語逐字以 §8 為準。
> **核心 anchor（整篇貫穿）**：端午每個習俗本是古人對抗死亡的工具（艾草殺蟲、雄黃驅毒、午時水消毒、立蛋巫術、龍舟驅疫），科學一條條戳破了原始功能，但在台灣這節日活得比誰都旺——因為它長出的不是防疫功能，是讓人每年確認「我們一家、這社區、這座島還在一起」的集體儀式。**藥效過期了，台灣人照樣年年服用。**

### 推薦結構（場景/矛盾 H2，非編年體；目標 S 級 ≥4500 字 / ≥12 footnote / 圖+影片 ≥5）

**開場（推薦）**：用 **2025 年 5 月國中教育會考自然科** 切入——那年考了一題要學生**反駁**「端午立蛋」迷思（設計對照實驗）。一個國家級考試叫十五歲少年證明老祖宗的習俗在科學上站不住腳，這個畫面濃縮全文 anchor，且百分百台灣在地、反直覺。接 30 秒概覽。人物錨：**沈文俊（新北市永平高中理化教師）** 受訪解說（[CNA](https://www.cna.com.tw/news/ahel/202505180134.aspx) 🟡）+ 萬華國中王美玲稱這是「創新題……思考如何設計實驗、提供證據來反駁『端午立蛋』的迷思」。
（替代開場：午時劍井取水近千人 / 廚房包粽阿嬤。但會考開場最強。）

**H2-1「被屈原蓋過去的，是一個怕死的節日」**（多元起源 + 惡月本源；反向解釋編織）

- 通行說法→反向：端午≠單純紀念屈原。**《史記》寫了屈原傳卻沒記投江日期**；「五月五日」最早見南朝梁《續齊諧記》（晚《史記》約 600 年），唐代才普及。「現階段臺灣社會能從內在裡說出來的端午源起屈原故事，則大多來自戰後教科書的洗禮」（[鳴人堂](https://opinion.udn.com/opinion/story/9789/2495396) 🟡）。
- 端午本源是「惡月/毒月」防疫：戰國《史記·孟嘗君列傳》「五月子者，長與戶齊，將不利其父母」（不舉五月子禁忌）——孟嘗君當場反問「人生受命於天乎？將受命於戶邪？」（反直覺：戰國就有人嗆惡月迷信）。「惡月」直接定義最早見東漢應劭《風俗通義》、魏晉董勛《皇覽逸禮》「五月俗稱惡月」。**注意：《禮記·月令》查無「惡月」字樣（agent 直查 ctext 推翻）。**
- 多元起源（去單一英雄）：屈原（忠）/ 伍子胥（怨，《曹娥碑》「五月五日，時迎伍君」）/ 孝女曹娥（孝，《後漢書·列女傳》一手「漢安二年五月五日……娥年十四……遂投江而死」，父曹盱是巫祝迎婆娑神溺死 — 揭示五月五日早有水神祭祀）。
- 百越龍圖騰說要**謹慎**：聞一多〈端午考〉談的是「古代吳越民族」龍圖騰祭，**不是「新石器時代」**（舊文誇大，CUT「新石器時代」）；且聞一多有救亡政治動機、論證被黎蝸藤批「激情多於邏輯」（[上報](https://today.line.me/tw/v3/article/aG3YqoO) 🟡）。寫法：呈現為「一說」，不當定論。

**H2-2「掛在門口的殺蟲劑，喝下肚的砒霜」**（防疫工具包 = anchor 第一個 worked example）

- 艾草：台大昆蟲系薛馬坦研究——燃煙對埃及斑蚊有「擊倒作用」但「效果微弱、範圍有限」「不建議作主要驅蚊工具」🟢（台大生農）。即「有效但很弱」。
- **菖蒲反直覺**：「實際上，在台灣常見的避邪物中，幾乎無菖蒲的使用。經常將同樣有著刀劍造型般的香茅訛作為罕見的菖蒲」（[維基台灣端午](https://zh.wikipedia.org/zh-hant/台灣端午節) 🟡）——台灣人掛的「蒲劍」多是香茅/榕葉。
- **雄黃酒（最強）**：衛福部官方逐字「雄黃為含硫和砷的礦石，經加熱後會被氧化為劇毒成分三氧化二砷（As2O3）……建議端午節不要喝雄黃酒」🟢（[衛福部](https://www.mohw.gov.tw/cp-3213-23302-1.html)）。泛科學：氧化後「正是大名鼎鼎殺人滅口必備良藥的砒霜呀」🟡。以毒攻毒的驅五毒酒，本身就是毒。
- 五色線（長命縷）：《風俗通義》「五月五日，以五彩絲系臂，名長命縷……辟兵及鬼，命人不病瘟」（內政部宗教知識庫引 🟢）。避「五毒」=蛇、蠍、蜈蚣、壁虎（守宮）、蟾蜍。香包（艾草雄黃白芷）。「天中五瑞」（農傳媒 🟡）。

**H2-3「立起一顆蛋」**（[MERGE-IN 立蛋] — anchor 第二個 worked example，配 tw-versus）

- 迷信：「正中午（11–13 時）是一年陽氣最重的時刻，太陽引力與地心引力相當，立蛋特別容易」（[NOWnews](https://www.nownews.com/news/6445529) 🟡）。
- **物理（國科會官方 🟢，最強來源）**[科技大觀園](https://scitechvista.nat.gov.tw/Article/c000009/detail?ID=b425a2c1-8524-406c-98f3-78e82b99e762)：「只要蛋接觸面間擁有 3 個接觸點……就可以使蛋直立得起來」「找一個摩擦力大的地方……都可以幫助雞蛋立起來」「立蛋會不會成功，與太陽和地球間的引力是沒有關係的」「故 1 年 365 天隨時都能立蛋，不必特地選端午的日子」。技巧：撒鹽增摩擦、鈍端朝下甩使蛋黃下沉降重心（[Uho](https://www.uho.com.tw/article-63842.html) 🟡）。
