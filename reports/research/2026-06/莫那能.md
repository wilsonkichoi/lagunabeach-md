# Research Report: 莫那能 (Maljaljaves Mulaneng)

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- **舊世代（戰後第一代）的盲點**：絕大多數台灣讀者第一次認真聽到原住民詩聲，是 1989 年《美麗的稻穗》。在那之前的原住民形象是「山地歌舞表演」、是「番」、是地圖上的山脈陰影，不是詩
- **跨族群記憶差異**：對排灣族／台東達仁鄉的人，這是「我們家鄉那個眼睛看不到的姪兒寫的詩」；對都市讀者，是「第一次聽見原住民用漢字直接控訴的聲音」；對學院文學界，是「原住民漢語文學的元年」
- **核心記憶物件**：一本黑色封皮、棕色稻穗插畫的詩集，30 首詩，1989 晨星出版；以及那首被多人傳唱的〈來，乾一杯〉

### 多元面貌

- **主流敘事**：盲詩人逆境奇蹟 + 原住民文學里程碑（典藏／文學館／百年文學地圖等公部門框架）
- **支線敘事**：8 歲眼疾失明的具體歷史脈絡（醫療資源城鄉差距 / 公衛體系對山地部落的缺席）／按摩師職業選擇背後的社會結構限制／長期參與「原住民還我土地運動」街頭組織者身分（不只是詩人）
- **被忽略的角度**：〈來，乾一杯〉裡那句「台獨喊得震天響，原住民的土地誰來還？」對 1980s 末本土運動的批判——這個批判到 2026 年依然有效，但被原住民史的「先驅」框架包起來，少有人正面回應
- **fault lines**：(1) 把莫那能讀成「勵志盲詩人」是去政治化，把他讀成「街頭運動詩人」才接近文本；(2) 用漢字寫作對原住民既是 enabler 也是 erasure，這個 paradox 他自己終身在處理

### 歷史脈絡（pre-search hypothesis）

- **形成期**：1956-06-03 出生於台東縣達仁鄉排灣族部落 → 童年因外傷感染眼疾、缺乏即時醫療逐步失明 → 1980s 投身原住民運動 / 學按摩維生 / 在街頭朗讀詩 → 1989 晨星詩集出版
- **關鍵轉折**：(a) 失明時間點與 1960s 山地公衛體系的對位 (b) 1984 原權會（台灣原住民族權利促進會）成立前後的街頭抗爭參與 (c) 1989 詩集出版正好夾在解嚴 + 還我土地運動高峰之間 (d) 後續族語復振運動中的位置
- **當代意義**：2020s 原住民族正名 / 傳統領域劃設 / 族語振興政策的所有議題都是 1980s 街頭那批人的延長線，莫那能用詩把那些街頭口號的核心存進文學史

### 切入點清單（待 Stage 1 搜尋驗證 / 反駁）

1. **「看不見的人看見族人」雙重張力**：失明 + 寫漢字 = 兩種「看不見」（物理 / 語言）但他用詩把雙重看不見的族人帶到主流文化視野
2. **〈來，乾一杯〉的雙刃**：對台獨運動的批判同時也是對原住民自己內部「酒精問題」的痛切自省，這首詩不是寫給漢人聽的，是寫給族人聽的
3. **按摩師詩人的階級錨點**：「街頭按摩」職業的身體勞動性 + 黑暗中工作 + 客人多元接觸面 — 這個生計現實塑造了詩的物質感
4. **漢字書寫的悖論**：用「殖民者的語言」為被殖民者吶喊 — 跟瓦歷斯·諾幹 / 利格拉樂·阿𡠄 / 沙力浪後續族語復振世代形成 dialectic
5. **排灣族出身但詩裡少見排灣語**：跟林央敏（台語）/ 杜潘芳格（客語）走完全相反的書寫路徑，這是策略還是限制？

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- A: 漢字救了原住民也滅了原住民 — 一個盲人用殖民語言為被殖民者吶喊的悖論
- B: 看不見的人寫下被看不見的族人 — 雙重盲區的相互照亮
- C: 第一位原住民現代詩人也是第一位被原運批判過的詩人 — 「先驅」框架掩蓋的政治尖銳

收斂候選：**B + A 合成** — 「看不見的人寫下被看不見的族人 — 漢字救了原住民也滅了原住民」（30 字以內）

### 研究方向（要搜什麼可以驗證）

1. **失明歷史 verbatim**：8 歲還是其他年齡？外傷感染還是先天？哪間醫院／哪個 misdiagnosis？— 主流敘事很少給細節
2. **詩作 verbatim**：〈美麗的稻穗〉/〈來，乾一杯〉/〈如果你是山地人〉/〈落葉〉的關鍵段落，必須對原始來源（晨星詩集 1989 / 山海文化典藏 / 國立台灣文學館）逐字
3. **原權會時期角色**：1984 成立的原權會 / 1988 還我土地運動 / 1990 反核廢蘭嶼 — 莫那能在哪場哪場具體做了什麼，不是泛說「參與原運」
4. **詩集出版背景**：晨星出版社當時為什麼接？吳錦發推薦語？1989 年詩集第一版印量？
5. **2010s 後況**：莫那能在族語復振浪潮中的位置 — 是被新一代族語派視為「過渡期」還是「奠基者」？

### 預想讀者帶走的那一件事

讀完知道「原住民第一次用漢字為自己吶喊」這個歷史時刻是哪一年（1989）、哪本詩集（《美麗的稻穗》）、哪位詩人（馬列雅弗斯·莫那能），更重要的是：**那個吶喊裡有一句「台獨喊得震天響，原住民的土地誰來還？」這句話到今天依然在 2026 年的台灣有效**——莫那能不是一個被供奉的先驅，是一個還沒被回答的提問者。

#### A.3 失明過程 — ⚠️ 主要事實 verified 但精確年齡 single-source 衝突

**Stage 0「8 歲失明」假設徹底錯誤**。實情是：

- **童年起的視網膜色素病變（progressive，遺傳性）**：
  - 原視界 ([insight.ipcf.org.tw/article/407](https://insight.ipcf.org.tw/article/407)) verbatim：「國中時發現眼睛有視網膜色素病變，終會失明」
  - NMTL 辭典 verbatim：「24歲時，在貨運行工作時，因車禍導致半盲」
- **1979 年車禍致全盲（決定性事件）**：
  - 維基百科 verbatim：「1979年，莫那能因車禍腦震盪，在醫院昏迷了將近兩個月，醒來的時候右眼全盲、左眼視力0.2」
  - Observer-Taipei 2022 verbatim 莫那能本人自述：「1979年，我在貨運行工作時發生事故，昏迷近兩個月後醒來，但因腦震盪而近於全盲」
  - 「之後因家族性肺結核發作而失業」
- **車禍年齡 single-source 衝突**：
  - 維基：1979 ≈ 23 歲（1956 出生）
  - NMTL 辭典：「24 歲」
  - Observer-Taipei：「26歲時因車禍意外而失明」
  - 原視界：「27 歲正式走入無光的世界」
  - 一筆 search snippet 出現「1976 年車禍」── ❌ **排除為訛傳**（與維基/Observer 第一人稱衝突）
  - **建議**：主 session 用「1979 年、二十三歲」+ 註腳指出多源差異，或統一寫「二十出頭時」regress 到非精確表達

#### A.4 教育 — ✅ verified (≥3 sources)

- **學歷**：**國中畢業**（NMTL 辭典 + 維基）
- **臺東縣立大武國民中學**畢業（維基）
- **考上空軍機械學校但因視網膜色素病變健檢未過無法就讀**（維基 + 原視界 + 民報全部一致）verbatim from 民報 search snippet：「國中畢業，莫那能考上空軍機械學校，卻在健檢時發現眼睛有視網膜色素病變，終會失明」

#### A.5 童年家境 / 創傷史 — ✅ verified (≥2 sources)

從原視界 + chaiwanbenpost「台灣的荷馬」整理：

- **不到 6 歲母親因肺結核離世**（原視界 verbatim：「不到6歲，母親即因肺結核離世」）
- **國中時父親因伐木坐牢**（原視界 verbatim：「國中時，父親因伐木坐牢」）— 父親替人頂罪（chaiwanbenpost：「初中三年級時父親替人頂罪入獄」）
- **16 歲以後到都市打工**（原視界 verbatim：「16歲以前在故鄉放牛，16歲以後就到都市裡做牛做馬」）
- **1972 年 18 歲被人口販子拐騙身分證**（chaiwanbenpost：「1972年，18歲時被人口販子騙賣到都市做砂石工、搬運工等工作」）
- **1977 妹妹遭堂姊夫賣入雛妓**（多源 verbatim：「1977年，莫那能未成年的妹妹遭堂姊夫賣給娼寮成為雛妓。莫那能千方百計將妹妹救出之後，1984年，妹妹卻又因著家人重病、家計困難的因素而自願回到娼寮工作」）
- **救妹妹過程中曾遭歹徒毆打致視力惡化**（多源一致，搜尋 #14 verbatim：「莫那能曾為了救援遭人口販子賣入火坑的親妹妹而遭歹徒毆打，致使視力惡化」）— 這條意味著 1979 車禍**只是壓垮的最後一根稻草**，視力惡化是長期累積

- **按摩師**（在 1979 失明後）：1980s 起在台北經營「阿能按摩院」（數位典藏 catalog.digitalarchives.tw item 4339889 metadata 給的訪談地點：「台北市 阿能按摩院」）

  - 維基：「1989 年 8 月」
  - NMTL ikm ([db.nmtl.gov.tw/site2/ikm?id=527](https://db.nmtl.gov.tw/site2/ikm?id=527)) verbatim：「11月，莫那能出版第一本原住民漢語詩集《美麗的稻穗》」

- **2010 年人間出版社再版** — ✅ verified（不是哲宇 prompt 提到的「印刻 2014」── ❌ 排除）

### §C. 政治立場與原運參與 — ⚠️ Stage 0 重大誤判，必須改寫

**這是最關鍵的 falsification**：莫那能不是「被原運邊緣化的先驅」（Stage 0 觀點 C 的弱版本），他是**公開、長期、自覺的左統派 (左翼統一派) 知識分子**。原運參與是真，但他對「台獨」的批判**不是修辭**，是**政治立場**。這點 Stage 0 hypothesis 1-5 全部沒接到。

#### C.1 左統派身分 — ✅ verified (≥3 sources)

- **維基百科 verbatim**：「夏潮聯合會會長，長期主張臺灣原住民族也是中國人，主張兩岸統一所帶來的臺灣海峽和平對臺灣原住民族最有利」
- **2010-06-16 中國作家協會宣布莫那能、朱秀娟、陳映真為該會首 3 位臺灣會員**（維基 + 百度百科一致）
- **本人公開聲明 verbatim**（接受專訪）：「他支持兩岸統一，只是因為『兩岸越和平，對原住民越有利』，這和原住民的身分沒有關係」
- **觀察雜誌 2022 NO.103 莫那能口述 verbatim**：「民進黨並不關心底層人民與少數民族的權益，而且被『台獨』綁架走向分裂國家的死路」（這比〈來，乾一杯〉的詩句強烈得多，是本人 2022 年明說的立場）
- **觀察雜誌結尾本人 verbatim**：「執意要在漢人社會裡推動部落語言，不僅脫離了生活化的語言學習，而且會排擠其他教育資源，最終流於形式，是不可能有效果的」── 連族語復振政策都反對

#### C.2 啟蒙脈絡 — ✅ verified

- **1974 年**認識淡江大學王津平老師（也說 1977 初認識，年份有差，**⚠️ single-source 衝突**）
- 透過王津平認識**蘇慶黎、陳映真、陳鼓應**等左統派知識分子（搜尋 snippet verbatim）
- **陳鼓應**邀請莫那能到家中養病數月，**閱讀左翼思潮書籍**（原視界 verbatim：「陳鼓應邀請他到家中養病數月，莫那能看著房內許多左翼思潮的書籍」）
- 1978 年底首次參與政治活動：協助**陳鼓應、陳婉真**競選立委 / 國大代表（原視界 + 維基）

#### C.3 詩寫作起源 — ⚠️ Stage 0 「陳映真鼓勵點字寫作」太簡化

**真實順序**：

1. **1984-03**：在台中**楊渡**家飲酒唱歌即興，**李疾**將即興吟唱整理成詩 → 刊《春風詩刊》第一期「山地人詩抄」
2. 春風詩刊第二期刊〈來，乾一杯〉
3. 1979 全盲後在台灣盲人重建院學點字 + 按摩
4. 陳映真**鼓勵**用點字寫作（時間點未明）
5. 陳映真為 1989 詩集寫長文〈莫那能──台灣內部的殖民地詩人〉（多源 verbatim 此標題）

**重點**：詩**先是口述吟唱**，**李疾筆錄**才成書面文本 — 這個歐德賽式 (Homeric) 口傳源頭是「台灣的荷馬」這個 framing 的真正依據

#### C.4 原權會 / 原運 — ✅ verified

- **1984-12** 與胡德夫、夷將·拔路兒等成立**台灣原住民族權利促進會**，擔任「促進委員會」**召集委員**（維基 + 多源 verbatim）
- 同年起源：**1984-06-20 海山煤礦煤塵爆炸**，72 死多為阿美族礦工，胡德夫辦「山地而歌」音樂會募款，後決定成立原權會（chaiwanbenpost verbatim）
- **〈為什麼？〉**為海山煤礦罹難阿美族而作（多源確認）
- **「部落工作隊」召集人**（NMTL 辭典 verbatim）
- **參與的具體運動**（原視界 verbatim：「拉倒吳鳳銅像、還我土地運動、救援雛妓等運動場合，都能看到莫那能戴著墨鏡、拄著拐杖的身影」）：
  - 1985-09-09 嘉義吳鳳廟落成抗議 + 後續「不要神話要歷史」遊行
  - 1988-08-25 第一次還我土地運動大遊行（總指揮為原權會秘書長**多奧·尤給海**，非莫那能）
  - 1988-12-31 拉倒嘉義火車站吳鳳銅像
  - 1988-01-09 救援雛妓大遊行（**莫那能寫〈鐘聲響起時〉聲援**）
- ⚠️ **反核廢蘭嶼 1988** 由達悟族（夏曼·藍波安）主導，**搜尋無證據莫那能直接參與** → Stage 0 假設 ❌ 排除

### §D. 「第一位」claims 對位驗證

- **「第一位原住民漢語現代詩人」 = ✅ robust**（NMTL 公部門 verbatim：「莫那能出版第一本原住民漢語現代詩集」）

### §I. 核心矛盾收斂建議
