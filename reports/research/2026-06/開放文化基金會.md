## 1. 觀點成型（Stage 0，含 §探索搜尋紀錄 ≥20 query）

### 對台灣人的記憶 anchor

- 大多數台灣人**叫不出「開放文化基金會」這個名字**，卻用過、記得它託管/支持的專案成果：
  - COVID 口罩地圖（g0v，2020，3 天）
  - 「Cofacts 真的假的」LINE 查證機器人（2016）
  - vTaiwan / 總統盃黑客松 / 開源祭
- OCF 是那種「**你用過它的成果，但叫不出它名字**」的存在。OCF 自己的行銷企劃劉韋巖逐字講過這個尷尬：「大部分的人都很禮貌地聽我們解釋完，然後點點頭，走開的時候還是滿頭問號。」
- 世代/圈層差異：開源 + 公民科技圈內人記得 clkao、記得 OCF 是「黑客松背後幫忙報帳的人」；圈外人只記得成果，不知道後台是誰。

### 多元面貌

- **後台行政中心**：報帳、法務、HR、勞健保、申請補助、簽合約 — 志工不想碰的那一塊。OCF 自稱「開源社群的里長伯」「各社群的匯集點」「家」。
- **連結者 / 橋樑**：OCF 自述「連結科技與人權、社群與政府的橋樑」「連結科技專家與倡議工作者、國內與國外的網絡」。
- **數位人權倡議者**：企業數位人權 RDR / 開放政府觀察報告 / 封網爭議 / OONI / 東亞隱私報告。
- **國際串接者**：RightsCon 2025 台北在地夥伴、APC 會員、Code for All 創始成員、DRAPAC。
- fault lines（感受）：
  - 圈內感激它扛掉無聊行政（讓黑客能黑客）
  - **體制化張力**：anti-institution 運動長出 institution，背叛初衷嗎？（OCF 自己用「財團法人」不是「社團法人」）
  - **成長 vs 核心價值張力**（OCF 內部自己講的）：董事 KC 陳貴成逐字「我現在比較擔心的是，組織擴大，可能會因為營運壓力而造成核心價值的偏離。」
  - **獨立性張力**：接政府計畫（如數位部培訓、公共出資公共程式）同時監督政府別封網過頭。

### 歷史脈絡（pre-search hypothesis）

- 形成期：2012 g0v 誕生（fork 政府）→ 2014 太陽花後專案爆量、志工扛不動行政 → clkao 在 g0v 挖坑成立 OCF（解決研討會核銷問題）。
- 關鍵轉折：2016 Cofacts → 2020 COVID 口罩地圖（公民科技高光）→ 2022 數位部成立（OCF 角色從「補政府的洞」變「替社群對政府發聲」）→ 數位中介法爭議 → 2024 詐欺條例封網條款 → 2025 RightsCon 台北 + 開源祭十週年。
- 當代意義：從「寫 code 改變社會」延伸到「守望數位人權、網路自由」。OCF story 自述：「開源文化就是這樣，一點一滴滲進生活，從邊陲走向主流。」

### 切入點清單（待搜尋驗證 / 反駁）

1. **口罩地圖背後沒被看見的後台**（具體場景開場）：大家記得 3 天奇蹟，沒人記得是誰幫專案報帳簽約。OCF 行銷自己都被問到「滿頭問號」。
2. **一張報不掉的研討會收據**（誕生 instance）：一個拒絕體制的運動為什麼需要一個財團法人。財團法人 vs 社團法人的反直覺選擇。
3. **幫「沒有人」開發票**（託管模式）：g0v「沒有人是萬能的」vs OCF 那本必須有人簽名的年報。
4. **里長伯不只報帳，也替里民去敲區公所的門**（從行政到倡議，**取代舊版「長出牙齒」watchdog 軸**）：里長伯做最無聊的事，也替社區去跟上面講話 — 這是 OCF 自己的「橋樑/里長伯」框架，不是被豢養的看門狗。
5. **一行指令讓一個網站從台灣消失**（封網/DNS RPZ，最強具體素材）：守望網路自由，用「界線該畫在哪」的持續辯論，不是 watchdog。
6. **拿政府的資源，守望政府的手**（中立性張力 + OCF 內部的 KC 自我懷疑）：不迴避，用 OCF 自己的「攤開來給人檢查」+ KC 的 mission drift 焦慮。
7. **國際串接 + 東亞數位人權**（呼應 MANIFESTO §主權的巴別塔）。

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- A（spine）：拒絕被體制收編的開源運動，靠自己長出一塊會報帳的體制才活下來（體制化悖論）。
- B（開場 instance）：大家記得口罩地圖，沒人記得是誰幫它報帳開發票（看不見的後台）。
- C（誠實的內部張力）：從幫社群報帳，長成國際提到亞洲數位人權就想到的名字 — 但它自己最怕的是長大後偏離初衷（KC 逐字）。

→ **收斂方向**：A 當 spine，B 當開場，C 當第三樂章誠實 counterpoint。里長伯/橋樑/守望 = 取代 watchdog 的敘事 anchor。

### 七個品質維度 anchor 自評

- 溫度：8D94 辦公室（八德路一段 94 號 4 樓）、開源祭那件 T-shirt、滿頭問號的路人
- 人味：clkao / 李欣穎 / KC 陳貴成 / 劉韋巖 / honki / bobchao — 真人逐字（OCF story 提供）
- 故事：研討會收據 → 財團法人 → 里長伯 → 守望封網，因果鏈
- 策展：里長伯框架（取代 watchdog）；體制化悖論
- 觀點：通行說法「公民科技靠黑客熱血」→ 反向「靠有人做最無聊的報帳」
- 體驗：讀者帶走「熱情免費、行政不行；有人把最不性感的事扛十年」
- 歷史/社會關聯：台灣公民科技為何沒曇花一現 + sovereignty（網路自由 vs 隔壁的封網國家）

### 預想讀者帶走的那一件事

台灣公民科技能撐十年不曇花一現，不是因為黑客特別熱血，是因為有人願意做最不性感的那部分（報帳、法務、申請補助），讓熱血有地方著陸 — 而這個里長伯後來也替整個社區去敲了政府的門。
