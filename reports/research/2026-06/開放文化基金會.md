# Research Report: 開放文化基金會（OCF / Open Culture Foundation）

## 觀點成型（編輯前思考）

### 對台灣人的記憶 anchor

- 大多數台灣人**叫不出「開放文化基金會」這個名字**，但用過、記得它託管的專案成果：
  - COVID 口罩地圖（g0v）
  - 「真的假的」LINE 查證機器人（Cofacts）
  - vTaiwan 線上政策討論（UberX / 萊豬 / 同婚相關）
  - Disfactory 違章工廠回報地圖
- OCF 是那種「**你用過它的成果，但叫不出它名字**」的存在。
- 世代/圈層差異：開源 + 公民科技社群圈內人記得 clkao、記得 OCF 是「黑客松背後幫忙報帳的人」；圈外人只記得成果，不知道後台是誰。

### 多元面貌

- **後台行政中心**：報帳、法務、HR、勞健保、申請補助、簽合約 — 志工不想碰的那一塊
- **數位人權倡議者**：台灣網路透明報告 / 企業數位人權 RDR / 開放政府觀察報告 / 封網爭議
- **政策看門狗**：數位發展部成立公民聲明、《數位中介服務法》草案工作坊
- **國際串接者**：APAC 開源社群、東亞隱私報告（TW/KR/HK）、RightsCon 類國際場
- 感受 fault lines：
  - 圈內感激它扛掉無聊行政（讓黑客能黑客）
  - 「體制化」張力 — anti-institution 運動長出 institution，是不是背叛初衷？
  - 中立性張力 — 接政府補助/標案 vs 監督政府別封網的看門狗角色

### 歷史脈絡（pre-search hypothesis）

- 形成期：2012 g0v 誕生（fork 政府）→ 2014 學運後專案爆量，志工扛不動行政 → clkao 挖坑成立 OCF
- 關鍵轉折：2016 Cofacts → 2020 COVID 口罩地圖（公民科技高光）→ 2022 數位部成立（OCF 角色從「補政府的洞」變「監督政府」）→ 數位中介法爭議
- 當代意義：從「寫 code 改變社會」延伸到「倡議數位人權、守網路自由」

### 切入點清單（待搜尋驗證 / 反駁）

1. **口罩地圖背後沒被看見的後台**（具體場景開場）：大家記得 72 小時奇蹟，沒人記得是誰幫專案報帳簽約
2. **clkao「挖坑」2014 成立的故事**：一個拒絕體制的運動為什麼需要一個基金會
3. **託管模式**：志工不想碰行政，OCF 把行政吸收掉 — 不可取代的「看不見的基礎建設」
4. **從寫 code 到倡議數位人權**：網路透明報告 / 封網爭議 / 數位中介法
5. **中立性張力**：接補助的基金會 vs 看門狗（這是 counterpoint，不迴避）
6. **國際串接 + 東亞數位人權**（呼應 MANIFESTO §主權的巴別塔）

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- A（spine 候選）：fork 掉政府的運動，最後得自己長出一個會報帳的體制才活得下去（體制化悖論）
- B（開場 instance）：大家記得口罩地圖，沒人記得是誰幫它報帳開發票（看不見的後台）
- C（第二樂章 / sovereignty）：接政府補助的基金會，同時是監督政府別封網的看門狗（中立性 × 數位主權）

### 研究方向（Stage 1 要搜什麼可以驗證）

- clkao「挖坑」的逐字引語 + 2014 成立場景細節（官方 / 訪談）

- **經費來源結構**（捐款 / 標案 / 補助 比例）— C 矛盾的事實基礎，最重要
- 網路透明報告的具體發現 + 封網/DNS blocking 的具體台灣案例
- 《數位中介服務法》OCF 的角色 + 唐鳳對話場景
- 董事會 / 執行長（現任？）/ 工作人員的具體人物 + 直引（人物文需 ≥ 2-3 句真人引語）
- 國際連結（APAC、RightsCon、東亞隱私報告 19 位受訪者）

### 預想讀者帶走的那一件事

台灣公民科技能撐十年不曇花一現，不是因為黑客特別熱血，是因為有人願意做最不性感的那部分 — 報帳、法務、申請補助 — 讓熱血有地方著陸。

### 🔴 Stage 1 三個關鍵更正（REFLEXES #31 主 session 重驗抓到）

1. **「台灣網路透明報告」是台灣人權促進會（TAHR）做的，不是 OCF**。TAHR 自 2014/9 起依《政府資訊公開法》要求政府揭露調取/限制狀況，報告 2015/2018/2020 三版。OCF 只在 2023 RightsCon 協助台權會實體參加。→ Stage 0 觀點誤把它列為 OCF 倡議，已移除。source: https://www.tahr.org.tw/issues/transparency-report（逐字核對）
2. **董事/創始成員是「李柏鋒」(pofeng)，不是「李伯峰」**。董事會逐字（ocf.tw/people）：高嘉良(clkao)、王潔英(Ijs)、李明哲(izero, 時任 KKBOX COO)、陳貴成(kcchen)、小路(lulu)、李柏鋒(pofeng)、李欣穎(singing)。
3. **OCF「基金會專案」≠ 託管 g0v/vTaiwan/Disfactory**。ocf.tw/p/ocf 列的是 OCF **自己跑的倡議計畫**（OONI / 企業數位人權 / 公共出資公共程式 / 資安陪伴 / 開放政府觀察等），g0v 只是「開源貢獻者快速通關」協辦。**託管關係要看「社群專案」** — 已確證 Cofacts 是 OCF 社群專案（募款走 ocf.neticrm.tw）。vTaiwan / Disfactory 的 OCF 託管關係未獨立確證，文章不宣稱 OCF 託管它們。
