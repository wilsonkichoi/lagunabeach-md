# 跨黨派的好政策 — 研究 SSOT

> 觸發：讀者 lov3ngine 在 Taiwan.md 第 83 天的 Threads 貼文下許願——他厭倦藍綠互打的媒體宣傳，想記得「真正對台灣好的、跨黨派的、長遠的善」。他自己舉了四個例子：解禁（解嚴）、十大建設、戒急用忍、前瞻計畫。
>
> 哲宇拍板（§自主權邊界 政治立場 human-in-loop）：
>
> - **框架**：「好政策如何活過它的政治」——有論點的策展，用民生/民主/民權/主權四把尺，看政策能不能活過自己的黨派出身。
> - **爭議近案**：全部等重納入（前瞻、戒急用忍、同婚不因政治仍滾燙而被 down-weight；SSODT 誠實寫爭議是非協商紀律）。

## 觀點成型（編輯前思考）

### 核心矛盾 anchor（整篇貫穿）

**當下我們用藍綠的尺量政策；時間用的是另外四把：民生、民主、民權、主權。一個政策能不能活過自己的黨派出身，不看它出生時是誰做的、有多少爭議，看它幾十年後對「住在這座島上的人」留下了什麼。而活過來的政策，出生時往往最有爭議。**

這個 anchor 要在 description + 開場 + 每個 cluster 收束 + 結尾各變奏一次。

### 這四把尺是誰的尺（方法論誠實——必須在文章前段處理）

> persona 領域專家（政治學者）最尖銳的挑戰：「『民生/民主/民權/主權』這四把尺的權重誰定？為什麼是這四個而不是永續性/財政可行性/世代公平？」

文章不能假裝這四把尺是天定的中性標準。誠實的答案：**這四把尺是讀者 lov3ngine 提的**（他原話：「真正有益於台灣的民生、民主和民權」+ 全文圍繞「主權的重要 mark」）。它們不中立——但它們有一個共同點：**都問「對住在這裡的人有沒有長遠的好」，不問「誰做的」**。文章用這四把尺，正是因為它們把「黨派」這個變數踢出了等式。把這個選擇攤在陽光下講，比假裝客觀更誠實。

### 後見之明是不是作弊（必須在文章前段處理）

> persona 工程師 + 學者：「四把尺是後來才知道符合，還是一開始就有？後見之明評斷公平嗎？」

誠實的答案：**後見之明就是這篇文章的方法，不是 bug**。lov3ngine 原話「事過境遷後我們回頭看」。當年沒有人拿這四把尺，當年只有藍綠（或威權年代的「擁護/反對」）。文章的整個重點就是：**時間是一把藍綠看不見的尺**。所以這不是作弊，是刻意換一個量度單位回頭看。要在文章裡 own 這件事。

### 不剝離政治脈絡（回應學者第二個挑戰）

> 學者：「『活過黨派出身』預設政策能從政治脈絡剝離——但土改/健保的執行方式跟政治意圖本來就纏繞，能切開嗎？」

答：文章**不剝離**。每個案例都保留它的政治脈絡——威權強徵、選舉算計、修憲角力、公投對抗。論點不是「忽略政治」，是「真正活下來的政策，靠的是它對四民的長遠效果，而不是它的黨派出身」。政治（爭議、黨派誕生）正是每個政策必須「活過」的那個東西。SSODT 把政治寫進去，不漂白。

### 多元面貌 / fault lines（SSODT 必須同時握住的兩端）

- **不能稀釋國民黨威權期功績**（persona 深藍退休公務員）：名單裡土改、九年國教、十大建設、解嚴、健保全是國民黨（含威權期）做的——文章本來就不該稀釋，要給足。
- **不能漂白善政策對特定族群的傷害**（persona 社運者 + 日本女婿岳父）：轉型正義被部分外省族群視為「清算」；土改有真實受害者（地主）；同婚當年社會撕裂。兩端都要寫。
- **「跨黨派」這個標籤本身會被挑戰**（persona 社運者）：同婚不是傳統共識型跨黨派，是靠大法官釋憲 + 專法繞過 2018 公投多數反對的民權里程碑。要誠實——它 illustrate 的是「民主（多數）與民權（少數保障）兩把尺打架」，不是「大家都同意」。
- **「換了人但沒比較好算好政策嗎」**（persona 計程車司機）：2000 政黨輪替後經濟差——但輪替本身是「民主」這把尺的成功（和平交接），不是哪一任的政績。要把這層分清楚。
- **移工 / 非公民算不算「民生」**（persona 印尼移工）：誠實標——健保確實納保移工（給付項目有差）；「民生」這把尺問的是「住在這裡的人」，但實作上常以公民為中心。一句誠實的 caveat。

### 讀者帶走的那一件事（體驗 payload）

> persona 國中歷史老師 + 高中生都要這個：「給我一把下次選舉/政策爭議能自己套用的尺。」

不是一份政策清單，是**一把可攜帶的判斷工具**：下次看到一個被罵翻天的政策，先別問「哪一黨做的」，問「它對台灣的日子、自由、權利、立足世界的能力，二十年後會留下什麼」。文章結尾要把這把尺交到讀者手上 + 回應「現在有沒有正在發生、二十年後會被這樣肯定的政策」。

### 結構（thesis-driven，4 cluster，全部等重）

- **開場**：lov3ngine 的疲憊（具體場景：藍綠飯桌 / 他自己的話）→ 提出四把尺 + own「這是讀者的尺」+「後見之明是方法」。
- **Cluster 1 — 威權的手，種出民生的果**：土地改革 + 九年國教 + 十大建設。三個都是國民黨威權期、手段有真實爭議（強徵地/加稅/舉債威權決策），但用民生這把尺量活過來了。
- **Cluster 2 — 把「誰能統治」交出去的那幾步**：解嚴 + 總統直選 + 第一次政黨輪替。民主+民權。蔣經國鬆手→李登輝飛彈下直選→2000 和平交接，把統治權從槍桿子改成選票。
- **Cluster 3 — 一張卡，一個道歉**：全民健保 + 轉型正義。健保（連戰倉促上路爭議→今日驕傲，民生）+ 轉型正義（李→扁→蔡跨黨，被視為清算的爭議，民權/民主）。
- **Cluster 4 — 還在被四把尺量的**：戒急用忍 + 同婚 + 前瞻。蓋棺難論定（主權vs經濟兩尺打架）+ 同婚（民主vs民權打架，靠憲法托住）+ 前瞻（塵埃未定，民生爭議 still live）。正面回應「近期政治 still live」「後見之明」「四把尺打架時誰贏」。
- **結尾**（先寫）：把那把尺交給讀者 + 閉環回 lov3ngine 的疲憊。餘韻 / 灰色地帶式。

### 工作標題候選（Stage 2 polish，冒號三明治 + 具體名詞 + 報導者腔）

1. 藍綠之外，還有四把尺：從一塊田到一張健保卡，台灣七十年那些活過自己政治的政策
2. 誰做的不重要，留下來的才算：台灣回頭看，那些活過藍綠的好政策
3. 一塊田、一張卡、一次直選：藍綠都罵過、台灣卻離不開的那些政策（注意：對戒急用忍/前瞻「離不開」會 overclaim，慎用）

→ Stage 2 自檢「遮住抽象詞後剩幾個具體名詞」。傾向 1。

## Stage 1 研究 brief（milestone 選定 + 已驗基本事實 + source map + 待深挖 / 反方）

> 4 cluster = 4 個平行 Sonnet 研究 agent。每 agent ≥ 20 search、4 源配額（中文一手≥10/英文國際≥5/反方≥3）、falsification-first、結構化 verification table、回報完整搜尋軌跡（不自摘要）。每 agent 順手記候選媒體（官方/CC 圖、官方影片）URL。

### Cluster 1（威權民生）— 土改 / 九年國教 / 十大建設

- **土地改革**：三七五減租（1949）/ 公地放領（1951）/ 耕者有其田（1953）。陳誠主導。地主補償 70% 實物土地債券（十年、年息 4%）+ 30% 四大公司股票（台泥/台紙/台灣工礦/台灣農林）。**爭議**：徵收地價過低、股票賤值、中小地主不信任賤賣 → 五大家族低價購入晉身工業資本家；違憲爭議大法官多次解釋。**待深挖**：受惠自耕農比例/數字、對 1960s 工業化的資本貢獻、地主一側的口述/個案、確切大法官解釋字號。
- **九年國教**：1967-12 蔣介石（經國安會）拍板，1968-09 實施，6→9 年，經費龐大→加稅平衡預算，奠定 1970s 經濟起飛中級技術人力。→ 十二年國教（2014-08，馬英九）。**待深挖**：當年加稅細節、就學率變化數字、跨黨延續性（誰決定哪個叫「一個政策」）。
- **十大建設**：1973 宣布 / 1974-79，蔣經國。桃機/台中港/鐵路電氣化/北迴鐵路/蘇澳港/中船/中鋼/石化/核電。鐵路電氣化~230.8 億（台鐵自籌 7.7+公債 62+國外借款 108.1+國際借款 41.5+關稅 11.5 億）。**爭議**：高速公路「為富人建設」、李國鼎口述「好大喜功」「浮誇」「歪打正著」。1976 經濟成長 13.86%/工業 24.4%/通膨 2.48%。**待深挖**：石油危機背景下的逆景氣投資邏輯、債務佔比、李國鼎原話出處。

### Cluster 2（民主三步）— 解嚴 / 總統直選 / 政黨輪替

- **解嚴**：1987-07-15，蔣經國。解除戒嚴 + 開放黨禁報禁。**主動 vs 被迫爭議**：1985 美參院/1986 美眾院決議施壓；黨外 1986-07-03「組黨十人秘密小組」、民進黨 1986-09-28 成立先聲奪人；蔣經國指示「採取溫和態度」、透過華盛頓郵報釋放、「唯有如此，才能與時代潮流相結合，才能與民眾永遠在一起」。**待深挖**：吳乃德/吳乃德等學者對「最不該感謝蔣經國」的論點、戒嚴令長度（38 年世界最長之一）、確切引語出處。
- **總統直選**：1996-03-23，李登輝。1994 國大三修通過直選。蘇志誠香港對中方稱「李登輝必須推直選因人民支持，否則直選變民進黨專利」。李連 54%，投票率 76.04%/1076 萬人。1996-03-08 中共對高雄基隆外海射飛彈、美兩航母戰鬥群。**待深挖**：委任直選 vs 公民直選之爭、飛彈危機時間線、投票率原始來源。
- **政黨輪替**：2000-03-18，陳水扁/呂秀蓮 39.3%。終結國民黨在台 55 年執政；新興民主國家難得的和平政權轉移；李登輝和平移轉。首位女性副總統。「少數執政」朝小野大考驗民主成熟度。**待深挖**：和平交接的具體場景/畫面、國際對「亞洲第三波民主」的評價、「換人但經濟差」的誠實處理。

### Cluster 3（補上某種公平）— 全民健保 / 轉型正義

- **全民健保**：1995-03-01，連戰內閣（KMT，李登輝任內）。連戰下令 3/1 如期開辦、準備只有 3 天、健保卡還沒印、資訊系統沒著落。部分國民黨立委擔心年底立委選舉「完蛋」、衛生署建議展延。葉金川 building。影響連戰兩次參選總統得票。今日近全民驕傲、健保 30 年高齡化/點值縮水/自費增加結構風險。**待深挖**：開辦覆蓋率/今日 99%+ 納保率、世界排名引用、葉金川原話、財務缺口數字、移工納保事實。
- **轉型正義**：1995 李登輝正式代表政府向二二八受難者家屬及全體國人道歉 +《二二八事件處理及補償條例》。2018 蔡英文促轉會（鎖定國民黨等 5 政黨、政治檔案收歸國有 → 被視為清算）。蔡「轉型正義非針對特定政黨」「目標是和解不是鬥爭」。吳俊瑩：李登輝對二二八較積極、處理白色恐怖太消極。**待深挖**：跨李→扁→蔡的具體里程碑（紀念碑/賠償/檔案/促轉會解散後）、外省族群「清算」一側的聲音、確切引語出處。

### Cluster 4（還在被量）— 戒急用忍 / 同婚 / 前瞻

- **戒急用忍**：1996-09-14 李登輝全國經營者大會。高科技/5000 萬美金以上/基礎建設三類對中投資戒急用忍。2000 陳水扁改「積極開放，有效管理」、2006「積極管理，有效開放」。**蓋棺難論定**：趙少康等批「害台灣經濟江河日下、原地踏步」vs 迫使高科技「根留本土」中美貿易戰下台灣受波及小。**待深挖**：主權（防經濟過度依賴）vs 民生/經濟（錯過西進紅利）兩把尺如何打架、學界正反評價、確切出處。
- **同婚**：2017-05-24 釋字 748（民法未保障同性結合違憲，限 2 年修法）。2018 公投第 12 案通過（不直接修民法、改專法）。2019-05-17 立院三讀《司法院釋字第 748 號解釋施行法》、5-24 生效。亞洲第一。**爭議**：社會撕裂、民進黨拖延、靠大法官、公投多數反對。**待深挖**：民主（多數公投反對）vs 民權（少數權利靠憲法托住）兩尺打架的誠實處理、跨黨派真相（哪些藍營立委支持/反對）、報導者等一手。
- **前瞻基礎建設**：2017，蔡英文。原 8 年 8800 億→改 4 年 4200 億、8 大項。軌道約佔一半→「台灣需要這麼多軌道嗎」。立法過程國民黨杯葛（丟水球撒紙）、混亂中通過初審。TVBS 民調 46% 不支持/28% 支持/75% 認為需重新檢視。**塵埃未定**。**待深挖**：軌道項目地方分配爭議、舉債特別預算合憲性爭議、執行至今（2026）成效/落地數字、正反一手。

### whats_excluded（誠實標：讀者會問但本篇不答）

- 移工/非公民完整權益分析（只在健保段一句 caveat，不展開——超出本篇 scope）。
- 執行者（公務員）視角的政策成敗（persona 文官提的好角度，但會稀釋 thesis；至多在健保段點一句「制度靠下面的人死頂」）。
- 個別政策的完整專文深度（本篇是策展 synthesis，每個 milestone 都有/將有獨立深度文，cross-link 過去）。

## §6 Clean Fact-Pack（交 Stage 2 writer）

> 四 agent ~135 search 蒸餾。每 milestone：✅ 可斷言（帶源）/ ⚖️ 爭議兩面 / 🚫 紅線（查證後不可斷言，幻覺鐵律）/ 🖼️ 媒體。**writer 只能用 ✅ 與 ⚖️，🚫 一律改安全措辭或不寫。**

### 寫作總綱（給 writer）

- **anchor（貫穿）**：當下我們用藍綠的尺量政策；時間用的是另外四把——民生、民主、民權、主權。活過自己政治出身的政策，出生時往往最有爭議。
- **四把尺要 own**：這是讀者 lov3ngine 提的尺，不是天定中性標準；共同點＝都問「對住在這裡的人有沒有長遠的好」，不問「誰做的」。
- **後見之明要 own**：當年只有藍綠（或威權的擁護/反對），沒人拿這四把尺。回頭看是刻意換量度，不是作弊——「時間是藍綠看不見的尺」。
- **不剝離政治**：每案保留黨派誕生 + 爭議（SSODT）。論點是「靠長遠效果活下來，不是靠黨派出身」，不是「忽略政治」。
- **不稀釋也不漂白**：名單一大半是國民黨（含威權期）做的——給足；同時誠實寫傷害（地主/被視清算的外省族群/同婚社會撕裂）。
- **體驗 payload**：結尾把四把尺交給讀者當「下次選舉/政策爭議自己套用」的工具 + 回應「現在有沒有 20 年後會被這樣肯定的政策」。
- **結構**：開場(lov3ngine 疲憊+四把尺) → C1 威權民生(土改/九年國教/十大建設) → C2 民主三步(解嚴/直選/政黨輪替) → C3 一張卡一個道歉(健保/轉型正義) → C4 還在被量(戒急用忍/同婚/前瞻) → 結尾(尺交給讀者+閉環)。結尾先寫。
- **級別 S**：~250-300 行；富文本 ≥3 種（timeline 跨七十年 / compare block 當年爭議vs今日 / stat block / pull quote）；媒體 0.7-1.2/1k CJK，長文朝 ≥8。tw-\* 視覺化模組（時間軸/對比卡/數據大字）很適合。

### C1 威權的手，種出民生的果

**土地改革（三七五減租1949/公地放領1951/耕者有其田1953，陳誠）**

- ✅ 三階段：1949 三七五減租（地租從超過收穫 50% 降到 37.5%）/ 1951 公地放領 / 1953 耕者有其田。1948 純佃農佔農戶 38.7%；地主補償 70% 實物土地債券（10 年、年息 4%）+ 30% 四大公司股票（台泥/台紙/台灣工礦/台灣農林）；地主可留水田 3 甲或旱田 6 甲。耕者有其田創設自耕農 194,823 戶、徵收放領 139,249 公頃（行政院農委會）。共有出租地被徵收比率 81.8% vs 個人有地 28.2%（中研院）。
- ⚖️ 正：解放農民購買力、近 20 萬戶得地。反：劉瑞華（清大）具名批「三七五減租是不合理的政策，地主承受數十年不合理損失」；地主補償系統性低估（土地以年收穫 2.5 倍計、股票市值偏低）；辜振甫（鹿港辜家、第三大地主）配合土改換股票後取得台泥、發展成和信集團（有文獻個案）。
- 🚫 「土改造就 80% 自耕農」無一手，改用「創設自耕農 194,823 戶」或「佃耕地從約 41% 降到 16%」。🚫「土改驅動經濟起飛」——Oliver Kim & Wang 2024（Berkeley/SSRN）因果研究：公地放領最多解釋 1950s 稻米增長的 1/6、耕者有其田沒提升農業生產力（反把女性勞力推入製造業）。**安全措辭**：土改是「分配正義」的成就（自耕農+地租減半），別把 1960s 工業化奇蹟直接歸因於它。🚫「五大家族低價購入」機制無一手，用「有記載個案顯示地主把股票賤售、部分家族藉此晉身工業資本家」。
- 🖼️ 1953 耕者有其田紀錄影片（國家電影及視聽文化中心，CC BY-NC 3.0 TW，[國家文化記憶庫](https://tcmb.culture.tw/zh-tw/detail?indexCode=Culture_Media&id=702725)）。

**九年國民義務教育（1968-09，蔣介石 1967-12 拍板）**

- ✅ 1967-06 國安會決定延長、1967-12 蔣介石拍板、1968-09-09 實施，6→9 年。國小畢業續讀國中比率 1966 年 59.04% → 1971 年 80.85%；1968-73 新設國中 254 所（+80%）。三年總經費約 36 億；為籌財源加稅平衡預算。→ 十二年國教（2014-08 全面實施，馬英九）。

**十大建設（1973 宣布/1974-79，蔣經國）**

- ✅ 1973-12-16 蔣經國宣布（財長李國鼎事前一無所知）。內容：桃機/台中港/鐵路電氣化/北迴鐵路/蘇澳港/中船/中鋼/石化/核電。中山高約 429 億（含國際貸款 81 億）、鐵路電氣化約 230.8 億。1974 石油危機：GDP 1.16%、工業 -4.5%、通膨 47.5%；1976 復甦：GDP 13.86%、工業 24.4%、通膨 2.48%。

### C4 還在被四把尺量的

**戒急用忍（1996-09-14 李登輝；2000/2006 陳水扁兩度改，2010 ECFA 實質終止）**

- ✅ 1996-09-14 全國經營者大會提出，限「高科技、5000 萬美元以上投資、基礎建設」三類對中投資。2000 陳水扁改「積極開放，有效管理」、2006 改「積極管理，有效開放」。台塑「海滄計畫」（逾 70 億美元）遭「停止股票交易、凍結資金、限制高層出境」三通牒被迫放棄。台積電守政策（2004 才赴陸設松江廠）；聯電經蘇州「和艦」迂迴（2005 和艦案）；張汝京帶百餘工程師赴陸創中芯。

- 🚫「戒急用忍→金融風暴受影響小」因果無法確認（更可能外債低/外匯足/台幣不可自由兌換）。🚫「根留台灣→貿易戰受惠」反事實難驗證。🚫 台商經開曼/BVI 第三地迂迴，政策「管制形式但執行有大量滲漏」。🚫 1997 GDP「0.4%」數字存疑（可能季度vs年度混淆）。**安全**：明標「至今正反評價兩極、蓋棺難論定」——這正是「主權 vs 民生/經濟兩把尺打架」的活教材。

**同婚（2017 釋字748 / 2018 公投 / 2019-05-17 專法，蔡英文）**

> 以下是 falsification 後**不能斷言**的 claim。每條對應 §6 安全措辭。違反＝幻覺鐵律失守。逐字核對引語、必驗數字、交叉驗證歸因。

1. 🔴 **不可寫**「土地改革造就 80% 自耕農」「土改驅動經濟奇蹟」→ 用「創設自耕農 194,823 戶」「分配正義成就」，經濟起飛歸因標 contested（Oliver Kim 2024）。
2. 🔴 **不可寫**「九年國教奠定經濟起飛人力（因此創造奇蹟）」→「學者認為有貢獻」。
3. 🔴 **不可寫**「十大建設帶來 1976 年 13.86% 成長」→「逆勢推進避免更深衰退；1976 復甦是出口與全球需求合力，建設為後續工業化奠基」。李國鼎語標「1993 事後口述」。
