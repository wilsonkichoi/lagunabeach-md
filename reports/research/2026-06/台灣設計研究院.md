# Research Report: 台灣設計研究院（TDRI）

## 觀點成型（編輯前思考）

> 總編輯視角：這是一篇 **Society 類組織深度文**。書寫對象是機構，但要避免寫成「機構簡介 / 年報摘要」。核心是「設計變成國家治理方法」這個賭注，用台灣人摸得到的物件（衛生所、選票、教科書、母校教室）當錨。

### 對台灣人的記憶 anchor

- **多數台灣人不知道「設研院」這三個字**，但摸過它的作品：改造後不像衛生所的衛生所候診區、學校「學美·美學」翻新的教室、跟中選會合作的選票與票箱、每年秋天巡迴不同城市的「台灣設計展」、金點設計獎。
- **世代差異**：
  - 老一輩記憶裡「Made in Taiwan」= 便宜代工、廉價貨。
  - 中生代記得 2011 台北世界設計大會、2016 台北世界設計之都。
  - 年輕世代記得 IG 上洗版的台灣設計展、改造後的母校。
- **物件候選**：改造後的衛生所候診區／美感選票／學校新教室／台灣設計展展場／松山文創園區裡的設研院本部。

### 多元面貌

- **主流敘事**：設研院把台灣設計推向國際（金點、iF、設計外交），讓城市變美。
- **支線／被忽略的角度**：真正吃重的工作不在 trophy，在「看不見的」公共服務再設計（衛生所、長照、戶政、健保、選務）。設計從「美化」升級成「治理方法」。
- **正面 / 負面 / 矛盾的感受 fault lines**：
  - 設計圈：終於有國家級平台、城市變美、母校變好看。
  - 質疑聲：設計展會不會是 expensive 政績工程／城市行銷？「美感」是不是中產品味霸權？「設計力即國力」是 inspiring 還是 inflated rhetoric？嘉年華（設計展）vs 深度服務（公共服務設計）哪個才是核心？

### 歷史脈絡（pre-search hypothesis）

- **形成期**：台灣代工史（OEM/ODM，世界工廠）→ 1979 外貿協會設計推廣中心起步推設計 → 2003 台灣創意設計中心（TDC，經濟部，松山文創園區）→ 2011 台北世界設計大會 → 2016 台北世界設計之都 → 2020-02 升格台灣設計研究院（TDRI）。
- **關鍵轉折**：升格不只是改名。mandate 從「設計推廣（promote design to industry）」擴成「設計研究 + 導入公共服務 + 社會創新」。預算一口氣拉到 5.3 億（2020）。
- **當代意義**：把「設計」從產業外銷工具，變成政府服務人民的方法（design as governance）。

### 切入點清單（待搜尋驗證 / 反駁）

1. **升格那一刻**（2019-10 蔡英文宣布 / 2019-12 蘇貞昌拍板 / 2020-02 揭牌）：改名背後 mandate 擴張 — 為什麼立體
2. **代工島的設計史**：從 MIT 便宜貨到「想被看見」的身分焦慮
3. **看不見的公共服務設計**（衛生所 / 學美·美學 / 選票 / 健保 / 長照）：對位 trophy 的隱形工作 — 最強切角
4. **金點設計獎想當「第四金」**（繼金馬金曲金鐘）+ 設計外交的野心
5. **台灣設計展作為城市治理工具**（每年換城市）：嘉年華 vs 深度服務的張力
6. **張基義其人**：建築人 / 前台東縣副縣長 / 交大建築所教授 → 為什麼相信設計能改變治理

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- **A（首選）**：設研院靠「被看見」存活（獎、展、國際排名），但它賭的是「設計好到讓你沒感覺」的隱形治理 — 好的公共服務設計會消失在無縫體驗裡，沒感覺就難證明預算值得
- **B（歷史底色）**：一個賭「設計即國力」的機構，但台灣當了幾十年連自己產品都由別人設計的代工島
- **C（爭議軸）**：把「美感」當國家政策，卻要面對「美感是不是中產品味霸權／政績工程」的質疑

→ 傾向以 A 當論點脊椎，B 當開場的歷史縱深，C 編進「爭議」段落（不補丁，編織）。

# Stage 1 研究結果 — Clean Fact-Pack（主 session 合成，3 平行 agent 去重）

## 🔒 核心矛盾鎖定（Step 1.4）

**設研院靠「被看見」證明自己（金點獎、百萬人次設計展、國際得獎、設計外交），但它賭最大的，是讓你在衛生所、選票、校園裡「沒感覺」的隱形治理 —— 而 2022 年捷運中山站那次，剛好證明被看見有多危險。**

歷史底色：台灣當了幾十年代工島，連自己的產品都由別人設計。設研院（與蔡英文喊的「Made in Taiwan → Designed in Taiwan」）賭的是台灣能不能自己設計——包括設計政府怎麼碰到人民。

## ⛔ 幻覺護欄（Stage 2/3 必守 — falsification 成果）

1. **法人性質 = 財團法人（公設財團法人），不是行政法人**。主管機關經濟部。
2. **「設計力就是國力」「Made in Taiwan → Designed in Taiwan」是蔡英文說的**（2020-09-30 新竹設計展開幕 + 2019-10-23），**不可掛在張基義名下**。張基義講的是「設計外交」「設計帶動公共服務」「設計在公部門變成 DNA」。
3. **美感教科書 = 美感細胞（2013 成立的獨立 NGO，3 個交大畢業生）**，不是 TDRI。TDRI 的校園案是「學美·美學」（空間設計）。不可混。
4. **健保卡 / 健保 / 長照 / 戶政 / 報稅 / 數位身分證 eID / 防疫口罩** —— 查無 TDRI 主導證據。不可寫成 TDRI 作品。（eID 是內政部後喊停；口罩地圖是 g0v + 唐鳳辦公室。）
5. **TDRI 揭牌確切日**（2020 年 2 月哪天）無官方一手可確認 → 只寫「2020 年 2 月」。
6. 張基義出生年（約 1963）無一手 → 不寫精確生年。

## A. 沿革 / 治理（footnote 來源）

- **1979-03-16**：外貿協會（CETRA / TAITRA）設「產品設計推廣處」，台灣第一個以國家名義推設計的機構；1987 擴為設計中心。來源：[維基/外貿協會](https://www.taitra.org.tw/)

- **2018-02-26**：張基義接任台創中心董事長（前任林榮泰屆齡退休）。來源：[工商時報 2018-02-26](https://www.chinatimes.com/realtimenews/20180226004141-260410)
- **2019-10-23**：蔡英文在全國設計論壇閉幕式宣布 2020 年台創中心升格台灣設計研究院。來源：[總統府新聞 108-10-23](https://www.president.gov.tw/NEWS/24937)
- **2019-12-05**：行政院蘇貞昌拍板，2020 年度預算 5.3 億元（舊台創 2019 為 4.24 億）。蘇貞昌語：「設計本身就是一種創新，更是促進產業升級重要的軟實力」。來源：[中央社 2019-12-05](https://www.cna.com.tw/news/firstnews/201912050118.aspx)

## C. 被看見：金點獎 / 設計展 / 國際

- **金點設計獎**：前身 1981「優良產品評選制度」→ 2009 更名金點設計獎 → 2014 國際化（全球華人市場）→ 2015 三獎分立（金點設計獎 / 概念設計獎 / 新秀設計獎）。TDRI 主辦/執行。2023：近 8,000 件、23 國/地區、25 件年度最佳。來源：[金點 about](https://goldenpin.org.tw/goldenpin/zh-TW/about)、[TDRI EN news/58](https://www.tdri.org.tw/en/news/58)
