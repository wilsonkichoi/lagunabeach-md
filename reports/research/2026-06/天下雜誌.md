# Research Report: 天下雜誌（CommonWealth Magazine）

## 1. 觀點成型（Stage 0，編輯前思考）

> Stage 0 探索 33 次搜尋（完整日誌見 §2）。模式 = **Fresh**（`knowledge/Society/天下雜誌.md` 不存在）。category = Society（媒體機構，與泛科學 / 報導者 / 公視 同層）。觸發來源 = 觀察者 directive，非 callout-triggered。

### 對台灣人的記憶 anchor

- **物件**：機場書店、便利商店、圖書館架上那本厚厚的財經雜誌；封面常是企業家或產業專題
- **年度儀式**：每年「天下兩千大企業調查」「天下 50 大集團」「幸福城市大調查」「縣市長施政滿意度」公布時上新聞——讀者也許沒讀過內文，但記得「天下排名」這四個字的份量
- **教育記憶**：教室裡發的「海闊天空」教改特刊、圖書館的天下教育專刊、「微笑台灣 319 鄉」的鄉鎮護照
- **世代差異**：
  - 戰後／解嚴世代記得它是「第一本敢把經濟寫成故事、不像政府文宣的雜誌」
  - 中生代記得它是「企業排行榜的權威裁判」
  - 網路世代可能先碰到換日線（海外青年）、獨立評論@天下、親子天下，未必知道母刊是誰

### 多元面貌

- **主流敘事**：女性報人傳奇（殷允芃）＋ 麥格塞塞獎 ＋ 創刊兩天售罄 ＋ 台灣第一本專業財經雜誌 ＋ 最受信任媒體 ＋ 數位轉型成功典範
- **支線 1（被忽略）**：天下是「四人創業、五年分家」的故事——它的純淨（台灣優先、新聞專業）正因為「企業經營＋大陸擴張」路線跟著高希均、王力行出走（去建遠見・天下文化）才被留下、被定調。分家不是失敗，是基因分流
- **支線 2（被忽略）**：天下不只是報導者，更是台灣商業與地方治理的**排名製造者／議題設定者**——兩千大、50 大、標竿企業、幸福城市、聲望調查。這個「定義什麼叫成功」的權力很少被檢視
- **支線 3（人味切口）**：殷允芃訪張愛玲聊了三小時、聯絡十幾年；訪遍全世界的人，最難開口的是自己——文學性的一面
- **感受 fault lines**：專業公信力 vs 與大企業的結構性親近 / 編輯獨立（2015 撤稿事件）/ 商業財經底色 vs 公益教育光環

### 歷史脈絡（pre-search hypothesis，Stage 1 驗證）

- **形成期**：1981/6 創刊。背景是 1971 退出聯合國、1979/1 中美斷交後台灣國際地位崩塌、戒嚴未解、經濟正要起飛的年代。⚠️ 斷交（1979）與創刊（1981）相隔 2.5 年，不可壓縮成「斷交那天」
- **關鍵轉折**：1986 首份大型企業排名（兩千大前身）＋ 四人團隊分家；1987 麥格塞塞獎 ＋ 同年解嚴；1998-2008 集團擴張（康健 / Cheers / 親子天下）；2013-2017 數位轉型（獨評＠天下 / 換日線 / 全閱讀付費牆）；2015 馬習會撤稿；2021 殷允芃交棒；2024/12 親子天下停刊紙本、2025 全面數位
- **當代意義**：天下是「台灣的成功想像如何被建構」的鏡子，也是紙本媒體在數位時代少數轉型成功的案例

### 社會 / 歷史 / 人生關聯

- 誰被排進兩千大、誰是標竿企業、什麼叫「幸福城市」——這些框架默默形塑台灣人對「成功」「進步」的集體想像
- 財經媒體在民主社會的角色：它能當第三方裁判，還是它本來就是場內玩家？
- 在 PRC 資訊戰、全球媒體信任崩跌的年代，一個「最受信任」的本土媒體意味著什麼

### 切入點清單（待 Stage 1 驗證 / 反駁）

1. **殷允芃在台灣最孤立的時刻，創辦一本「只談經濟」的雜誌**——在不能談政治的戒嚴島上，把經濟寫成台灣自己的故事，本身就是最政治的選擇。立體在「最不政治」裡藏著「最政治」
2. **「天下」這個名字 vs 財經／資本的內容**——若命名出自「天下為公」或「放眼天下」，跟報導私人資本之間有張力。⚠️ 命名由來必須 Stage 1 查證，查無則此切入點降級
3. **四人創業、五年分家**——純淨是分流的結果，不是天生
4. **排名製造者的權力**——兩千大 / 50 大 / 幸福城市，天下從報導者變成議題設定者
5. **2015 馬習會撤稿**——同一個「專業守門」動作，在雜誌世界是美德，在網路社群是侵犯言論
6. **數位轉型**——最在地的初衷，靠最國際（英文版 / 換日線）、最去紙本（付費牆 / 親子天下停刊）的手段延續

### 預期核心矛盾候選（待 Stage 1.4 收斂）

- **A（主候選）**：最受信任的位置，也是最靠近資本的位置——公信力與排名權力來自同一段距離（離台灣的資本與成功想像太近）
- **B**：在不能談政治的島上「只談經濟」，本身是最政治的選擇（用經濟敘事替失去國際承認的台灣，建立另一種主體性）
- **C**：天下的純淨（台灣優先、新聞專業）來自分流——「大陸擴張＋企業經營」路線出走後留下的結果

> **收斂方向**：A 當主 spine（最立體、最具策展翻轉性——挑戰「天下是中立第三方裁判」的通行說法），B + C 編成「起源」與「轉折」兩段的內在張力。anchor 詞：**「距離」**（接近資本＝公信力的來源，也是權力與風險的來源），整篇貫穿（description / 開場 / 中段呼吸 / 結尾各變奏一次）。

### 研究方向（Stage 1 要搜什麼可以驗證 / 反駁）

- 殷允芃完整生平 + 創刊精確背景 + 「天下」命名由來 + 4 vs 3 創辦人歸屬 + verbatim 引語
- 兩千大調查精確起始年 + 品牌沿革 + 50 大 / 幸福城市方法論爭議（陸之駿等批評 verbatim）
- 集團刊物 + 數位轉型精確數據（付費牆 / 數位廣告 / 續訂率 / 交棒）
- Reuters 信任度精確數字 + 馬習會撤稿事件全貌（黃丞儀 / 馬岳琳 verbatim）+ 結構性批評 + negative findings

### 預想讀者帶走的那一件事

下次看到「天下兩千大」「幸福城市排名」，會多想一層：這個排名是誰、用什麼方法、為誰製造的；以及一本媒體的「最受信任」，是靠多近的距離換來的。

### 七維度 anchor 自檢

- 溫度：殷允芃訪張愛玲三小時 / 首刷兩天售罄 / 交棒「凡事皆有時」
- 人味：殷允芃、高希均、王力行、張作錦、吳琬瑜
- 故事：四人創業→五年分家→基因分流；孤島用經濟談主體性
- 策展：排名製造者的權力很少被檢視（reverse「天下＝中立裁判」）
- 觀點：公信力與權力來自同一段距離
- 體驗：重新看待手上的「兩千大」「幸福城市」是誰為誰製造的
- 歷史社會關聯：台灣成功想像的建構 / 財經媒體在民主的角色

### §A 創辦人與創刊基因

- **殷允芃生平**〔高，≥2 源〕：1941 西安生，1949 來台（嘉義落腳、員林長大），北一女，成功大學**外文系**，美國愛荷華大學新聞碩士。記者職涯序列：費城詢問報（碩士後約兩年）→ 合眾國際社 UPI → 紐約時報 → 亞洲華爾街日報。**非 AP**。1976 獲十大傑出女青年。各段精確年份一手查無〔單源/未驗〕。
- **發起人是高希均**〔高〕：「在『書生報國』的意念下，高希均教授邀集了王力行、張作錦、殷允芃，在一九八一年共同創辦」（遠見官方歷史頁逐字）。高希均自述「我邀約了他（張作錦）、王力行及殷允芃討論創辦雜誌的可能性」（gvm.com.tw/article/68498）。
- **3 vs 4 創辦人分歧**〔分歧，雙方皆一手〕：天下雜誌維基 + 高希均維基稱「三位創辦人」（殷/高/王）；遠見官方 + 高希均悼張作錦文列 4 人（含張作錦，1981 任天下副總編輯）。最合理解讀：張作錦是副總編輯層級的核心參與者，天下官方論述定調三位主要創辦人。處置：寫「高希均邀集王力行、張作錦、殷允芃」+ 腳註揭露分歧。
- **辦刊動機**〔高〕：亞洲華爾街日報「把生硬經濟新聞寫得好看」「一般人都看得懂」的啟發；理想是「《經濟學人》的內涵、《時代》雜誌的寫法」。
- **命名**〔高，多源；非當事人逐字〕：刊頭「天下」取自孫中山墨寶「天下為公」；英文 Common Wealth 雙重意涵。
