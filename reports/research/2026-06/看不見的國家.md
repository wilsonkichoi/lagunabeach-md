# Research Report: 看不見的國家（Invisible Nation, 2023）

> Taiwan.md REWRITE-PIPELINE v7.0 Stage 1 SSOT。Fresh 模式，Art/紀錄片。對標研究所論文標準：4 agent fan-out，152 次搜尋，70 distinct 來源（中 22 / 英 45 / 一手 18 / 反方 8）。

## 1. 觀點成型（Stage 0，編輯前思考）

### 對台灣人的記憶 anchor

- 片名「看不見的國家」就是 2,300 萬人天天活著的處境：被排除在 UN/WHO 之外、奧運叫「中華台北」、邦交國剩 12、護照在國際櫃台被多看兩眼。
- 世代差異：解嚴前世代記得「不能說的話」；網路世代記得 2016 年 16 歲的周子瑜被逼著對鏡頭道歉。同一種「看不見」，兩種傷口。
- 一個更近的 anchor：2025 年 6 月這部片在台灣戲院賣破三千七百萬、登紀錄片影史第三——台灣人自己掏錢去看一部「美國人拍的、講台灣為何被世界看不見」的片。

### 多元面貌

- **主流敘事**：一部讓世界看見台灣民主的紀錄片，史無前例貼身拍在任總統。
- **支線 / 被忽略的角度**：(a) 它是一部「美國人的台灣」——外人的凝視；(b) 它最被質疑的是「access 換忠誠」——七年依賴蔡辦的配合，於是不敢得罪民進黨（Jay Liu）；(c) 它「斷代」——略過陳水扁、略過李登輝以前的黨外先行者（周玉蔻）；(d) 在台灣捲入「拿稅金拍片」假指控（已被製片闢謠）。
- **感受 fault lines**：感動（97% 觀眾分）vs 嫌它平（影評 80% 但多家嫌 conventional / hagiographic）vs 政治攻防（藍營匿名嘲諷 / 綠營內部嫌斷代 / PRC 沉默封殺）。

### 歷史脈絡（pre-search hypothesis — 已被 Stage 1 驗證）

- 形成期：Vanessa Hope 1995-96 在台北學中文，親歷 1996 首次直選 + 中國飛彈；2016 看蔡英文當選、聽見群眾喊「我們是台灣人」→ 種下拍片念頭。
- 關鍵轉折：2016 底遞案蔡辦 → 等半年 → 2017 年 5 月首拍 → 拍滿蔡英文兩任 → 2023-09 Woodstock 首映 → 2024 美國上映 → 2025-06 台灣上映。
- 當代意義：片名是雙關。「看不見」既是台灣的外交地位，也是這部片自己的方法與盲點。

### 切入點清單（已搜尋驗證）

1. **access 的雙面性**：七年五次專訪、總統專機、官邸、貓——但「不能問兩岸」是初始條件。讓片子成立的 access，也是批評者說它被「俘虜」的原因。✅ 立體
2. **外人的鏡頭**：為什麼是一個好萊塢製片世家的孫女、嫁給 Ted Hope 的美國人，來替台灣按下快門？✅ 立體
3. **片名作為方法**：「看不見」不只是控訴，是這部片在做的事——把不被看見的一格一格拍下來；而它自己也讓某些東西留在框外（陳水扁）。✅ 最立體，定為脊椎
4. **台灣自己買票看自己**：NT$37.71M、影史第三——一個被外交否認的社會，用票房確認自己存在。✅ 立體

### 預期核心矛盾候選（Stage 1.4 已收斂）

- A：台灣越自由民主，國際越拒絕看見它（片的論點）
- B：要被看見得靠外人的鏡頭，代價是被質疑「只拍一面」（access-capture）
- C：**看見是一個動作不是一種地位——連這部讓台灣被看見的片，都有照不到的角落**（A+B 的綜合，最深）

**→ 收斂為 C。** Anchor 句：「看見是一個動作，不是一種地位。」貫穿 description / 開場 / 中段 / 結尾。

### 研究方向（已搜尋）

- 片的事實骨架（導演 / 製作 / 發行 / 台灣上映）✅
- 片的內容（access / 受訪者 / 歷史弧 / 引語）✅
- 反方與爭議（影評光譜 / 斷代 / DPP propaganda / 稅金假指控 / PRC）✅
- 隱形的事實骨幹（2758 / 邦交國 / 中華台北 / 民主化日期）✅

### 預想讀者帶走的那一件事

被看見不是天上掉下來的地位，是有人願意按下快門、有人願意買票、有人願意在鏡頭前說「我們是一個獨立的國家」——一格一格爭來的動作。連這部片的盲點，都在提醒：可見性永遠是選擇，永遠不完整。

## 3. Findings by sub-topic

### §A 片的事實骨架（high_confidence 除另標）

- **片名**：中《看不見的國家》/ 英《Invisible Nation》。美國—台灣合製紀錄片。
- **導演**：Vanessa Hope（中文名**葛靜文**），美國人，生於紐約、居洛杉磯。芝加哥大學人類學與東亞研究學士（榮譽），哥倫比亞大學博士課程（**未完成博士學位** — 鏡週刊稱「東亞研究博士」為**誤**，ChinaFile/LRCCS 證實僅 coursework）。1995-96 在台北 Stanford/IUP 中文班，親歷 1996 首次直選。前作《All Eyes and Ears》(2015 Tribeca，講 US-China + 駐華大使 Jon Huntsman；**非主要講台灣**)。好萊塢製片 Walter Wanger + 女星 Joan Bennett 外孫女；夫為資深獨立製片 Ted Hope（《Manchester by the Sea》等）。
- **製作**：7 年（2017-2023）。首拍 2017-05（蔡就任約一年後）。製作公司 Double Hope Films / Seine Pictures / 100 Chapters。攝影 Laura Hudock。配樂 Wei-San Hsu。**無中華民國政府製作資金**（製片馮賢賢 = Sylvia Feng 公開聲明自籌）。Compton Foundation 啟動補助。
- **片長**：85 分鐘（戲院版）。single_source 衝突：HamiVideo 台灣版標 82 分；PBS 有 Extended Cut（確切片長 unverified）。安全引用 85 分。
- **首映 / 影展**：世界首映 Woodstock Film Festival 2023-09-29。IDFA 2023 歐洲首映（五場滿座）。CPH:DOX 2024。**得獎**：Sonoma 2024 評審團最佳紀錄片（high）；Cinema for Peace 2024 榮譽獎（high）；Sun Valley / Middleburg（single_source 待查）。Woodstock NYWIFT 最佳導演**提名未得獎**（敗給 No Accident）。
- **美國發行**：Abramorama，2024-05-31 紐約 Quad Cinema 首映。PBS 全國首播 2026-05-01。
- **台灣發行**：海鵬影業（Swallow Wings）。**2025-06-13 上映**（06-11 媒體首映）。65 院 → 加開 74 院。7 天破千萬、10 天破 1,800 萬、**最終 NT$37.71M（約 US$1.17M）**，**台灣紀錄片影史第 3、2025 年紀錄片冠軍**。HamiVideo 串流。
- **官方預告**：https://www.youtube.com/watch?v=RNUHWnt0bb0（Abramorama 官方，2024-04-23）。
- **台灣海報**：2025-04-28 公開，主視覺**玉山** + 蔡英文 + 蕭美琴/陳菊/鄭麗君/吳沛憶等女性政治人物。導演稱「我們看過最美的海報」。

### §B 片的內容與 access（high_confidence 除另標）

- **access 規模**：每家媒體都用「unprecedented（史無前例）」。遞書面案給總統府 → 等約半年回覆（英文訪談）/「一整年」（中文訪談，可能指到首拍的全程）→ 首拍 2017-05。**5 次專訪蔡英文**（Taipei Times）。
- **access 細節**：「每次跟她的專訪，都排在我離開台灣的前一天」（Hope，Taipei Times ✓）。**初始條件：不能問兩岸關係**（隨港、疫、烏俄情勢升級漸鬆動）。**保有剪輯權**、無政府資金。
- **拍到什麼**：總統府內專訪（蔡的頭在構圖上擋住背後蔣介石肖像）/ 總統專機上辦公 / 政務會議 / 官邸日常 / **蔡的貓「想想」**亂入（人性化名場面）/ 音樂節 / 同志遊行。
- **敘事手法**：observational，**無旁白、無字卡敘述**。史料 + 動畫地圖 + 受訪者談話交織。
- **歷史弧（片涵蓋）**：荷蘭 → 日治(1895-1945) → 國共內戰 / 國民政府 1949 遷台 → 白色恐怖 / 二二八(1947，realdemocracy 稱片涵蓋「February Incident」) → 1987 解嚴 → 民主化 / 李登輝 → 1996 首次直選（中國飛彈）→ 2000 陳水扁（**片中從未提及，見 §C 批評**）→ 2008 馬英九（受訪呈現反方視角）→ 2014 太陽花 → 2016 蔡英文 + 周子瑜事件 → 2019 同婚（蕭美琴在片中講述）→ 2019-20 香港 → 2022 Pelosi 訪台 → 烏俄戰爭類比 → **結尾：台灣人上民防課程**（沉重收束）。

- **史料 footage**：習近平 2019/2021 演說、蔣介石、蔣經國、鄧小平、李登輝、Clinton、Kissinger。
- **紀政場景**（single_source 但多源佐證）：拿出 1968 墨西哥城奧運銅牌，反問「中華台北是國家的名字嗎？不是。」（Variety「Chen Cheng」拼錯，正為**紀政**）

## 6. Clean Fact-Pack + Stage 2 操作規範（給 writer 的合成層）

**核心矛盾（脊椎，貫穿全文）**：看見是一個動作，不是一種地位。台灣的可見性不是被賦予的，是一格一格爭來的——而連這部讓台灣「被看見」的片，都有照不到的角落。Anchor 句「看見是一個動作，不是一種地位」在 description / 開場 / 中段 / 結尾各變奏一次。

**Title 候選（冒號三明治，報導者腔，writer 選一或改寫）**：

- 「看不見的國家：一個美國導演按了七年快門，拍一個世界決定不看的島嶼」

**Description（120-160 字，三段式：scene + 軌跡 + 核心矛盾 ending）**：
2025 年 6 月台灣戲院，一部美國人拍的紀錄片賣破三千七百萬、登上紀錄片影史第三。葛靜文花七年、五次貼身專訪蔡英文，把一個被聯合國除名、邦交國從 22 掉到 12、奧運只能叫「中華台北」的島嶼推到鏡頭前。但最被質疑的，是這部要讓台灣「被看見」的片，自己讓誰留在了框外。

**開場候選（克制，留懸念，writer 選一）**：

- A（推薦）：2025-06-11 台北媒體首映，放映前地面搖了起來（地震）；花七年拍台灣的美國導演葛靜文對著台灣說「你們不孤單」。——具體、有日期、人、動作 + 把「孤單/看不見」主題立刻翻轉。
- B：紀政拿出 1968 年墨西哥城奧運的那面銅牌，反問鏡頭：中華台北是國家的名字嗎？

**小標題候選（場景/物件/矛盾，禁編年體；5-7 個）**：

- 「離境前一天的專訪」（access 的縫隙）
