# Research Report: 複雜生活節 — 八年級「未成功者論壇」

## 1. 觀點成型（Stage 0，編輯前思考）

> Society 類型加權：當代意義 / 爭議 / 未完成的問題。

### 對台灣人的記憶 anchor

- **名字本身就是 anchor**：幾乎所有台灣人都聽過「簡單生活節」（李宗盛、張培仁，2006 起，商業文青地標）。「複雜生活節」靠這個對位讓人會心一笑——你不用參加過也懂這個玩笑。
- 對**八年級（民國 80 後 / 1991-2000 生）**的 maker / NGO / 獨立創作圈：這是「我們那場聚會」的集體記憶（250 人量級，非大眾事件）。
- 世代 anchor：「未成功者」「成長不是成功」這組詞，打中被「魯蛇 / 草莓 / 厭世代 / 22K」貼滿標籤的八年級。

### 多元面貌

- **主流敘事**：黃豆泥-中心的 web3 起源故事（他現在最有名 → 媒體回頭把複雜生活節寫成「FAB DAO 創辦人的青年時代」）。
- **被蓋過的支線（哲宇 directive 指向）**：**許皓甯**——共同總召、中醫師、得過多座現代詩首獎的詩人、獨立書店「引書店」老闆、「臨時動議有限公司」營運長、2018 時代力量北屯區議員參選人。他的 thesis 是「**創新來自邊陲**」（待驗證源）。命名那句話也是他說的。
- **兩個醫學生**：許皓甯（中國醫藥大學中醫系）+ 黃豆泥（西醫醫學系）——複雜生活節是兩個「本該當醫生」的人辦的。「系邊」（在自己系上邊緣）是他們的自我描述。
- **30 組參與者**橫切八年級 indie 生態：大學生玩簡報 / 臺大音樂節 / 秘密讀者 / 每天為你讀一首詩 / Hahow 好學校 / 移人工作室……
- **對照組**：簡單生活節（平靜而專注 / 大品牌 / 兩年一屆 / 李宗盛世代）。
- **後來的分岔**：有人「成功」了（黃豆泥→FAB DAO→哈佛甘迺迪學院民主訪問學者；III 講者唐鳳→數位部長；劉安婷→TFT），有人留在邊陲繼續複雜（許皓甯→診所+書店+參選+公司）。

### 想法跟感受 fault lines

- 浪漫化：「理想主義八年級的烏托邦/異托邦」。
- 批判角度（perspective scan 要查）：邀請制 / 臉書標註制——**誰被標、誰沒被標**？「工作狂 / 系邊 / 帶朋友無窮希望」的篩選，是不是只收了「正在往成功路上的未成功者」，真正的邊緣人不在場？是不是後來精英的 networking club？
- COVID 終局的惆悵：一個關於「聚集同類」的祭典，最後一屆（V）叫「自主隔離」，轉成 Podcast。

### 歷史脈絡（pre-search hypothesis，待驗證/反駁）

- 形成期：**2016**，太陽花（2014）後兩年的八年級自我組織能量；簡單生活節（2006，李宗盛世代）當對照樣板。
- 世代背景：2010s「厭世代 / 魯蛇 / 22K / 崩世代」論述貼在八年級身上。
- 同期生態：雜學校（ZA Share，2015，另類教育博覽會）/ TFT（2013）/ Hahow——社會創新 × 另類教育 × 斜槓世代浪潮。
- 五屆主題（待逐字驗證）：I 未成功者論壇 / II 加州旅館 / III 佛系青年（中山醫大，唐鳳·劉安婷·林依瑩）/ IV ㄇㄉㄈㄎ（台中伴畔旅店）/ V 自主隔離（2020，COVID→Podcast，11 月閉幕）。
- web3 轉向：黃豆泥 → FAB DAO（百岳計畫，與吳哲宇共同發起，2021-22）。
- 終局：COVID 2020 之後實體不再。

### 切入點清單（待搜尋驗證 / 反駁）

1. 兩個醫學生辦「未成功者論壇」——拒絕只當醫生 / 系邊 /「創新來自邊陲」
2. 命名作為世代宣言——複雜 vs 簡單，八年級拒絕被上一代的文青消費美學定義
3. 異托邦（傅柯 heterotopia）——兩天一夜的封閉異質空間，八年級短暫成為同類，再散回各自複雜的人生
4. 成長 vs 成功——拒談成功的論壇，參與者後來各自「成功」的反諷
5. 自主隔離終局——聚集同類的祭典，最後一屆變成隔離

### 預期核心矛盾候選（待 Stage 1.4 收斂，不預鎖）

- A：一場拒絕「成功」的祭典，由後來最「成功」的一群人辦／參加（成長 vs 成功）
- B：兩個醫學生用「複雜」反叛上一代的「簡單」，宣告八年級不要被定義
- C：「未成功者論壇」是八年級在平台資本主義收編注意力之前，最後一次純粹的自我組織
- D：邊陲的自我命名——「系邊」「未成功」「複雜」都是把貶義詞奪回來自稱

### 預想讀者帶走的那一件事

一個世代在被貼滿「魯蛇」標籤的時候，自己給自己辦了一場「還沒成功、但正在長大」的同學會——然後那些「未成功者」，長成了台灣現在的某一種樣子。
