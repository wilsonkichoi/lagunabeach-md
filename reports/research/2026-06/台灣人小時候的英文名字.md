# Research Report: 台灣人小時候的英文名字

## 1. 觀點成型（Stage 0）

> Mode: **Fresh NEW** (knowledge/Culture/ 無相關既有檔，無 toxic context)
> Category: Culture
> 哲宇 2026-06-12 goal directive：「Mary、Kevin 與補習班老師的命名權」

### 1.1 對台灣人的記憶 anchor

- **兒美班翻書指派儀式**：1980-2000s 何嘉仁 / 吉的堡 / 何嘉仁 ABC bookstart 等連鎖補習班的開班第一天，老師翻著教材人物名單（Peter / Mary / Susan / John），逐個指派
- **國小英語課的英文名**：1999 九年一貫之後正規教育階段每人一個英文名
- **星巴克吧台**：成年後在咖啡店紙杯寫英文名的儀式內化
- **護照拼音的尷尬**：習慣叫 Kevin 的人護照寫 Chih-Hao，海關停頓問「Kevin? But your passport says Chih-Hao」
- **世代差異**：70 後（補習班前 / 自選）／ 80 後（翻書指派最盛期）／ 90 後（國小英教課程化）／ 00 後（雙語教育下英文名變第二個正式姓名）

### 1.2 多元面貌

- **北部都市 vs 中南部**：六都佔補習班 75%（待驗證），南陽街兒美班密集 vs 偏鄉巡迴英語教師稀缺 → 命名實踐本身是城市現象
- **雙語家庭 vs 一般家庭**：海歸 / 外商家庭子女多「自選或延用英文母姓」，與被補習班教材指派的階級差異
- **原住民被雙重命名**：漢名 + 英文名雙層覆蓋，族名空間被擠壓
- **時代偏好變遷**：80s Mary / John / Linda / Kevin 主流 → 2010s Chloe / Zoe / Ethan / Aiden → 2020s 部分回流拼音名
- **新住民家庭**：越南 / 印尼 / 菲律賓媽媽帶小孩進兒美班，第三個名字加在已有的東南亞名與中文名之上

### 1.3 歷史脈絡（pre-search hypothesis）

- **形成期 1960-70s**：冷戰美援、留美潮、英語成為上升流動語言，1979 護照開放與「英文名是出國準備」直接掛鉤
- **爆發期 1980-90s**：兒美補習班連鎖化（何嘉仁 1983、吉的堡），教材人物名單成為命名 SSOT；1988 出版的《英文名字大全》成為老師 reference
- **課程化 1999**：九年一貫推動國小英語必修，英文名從「補習班儀式」升為「學校紀錄」
- **內化期 2000s**：星巴克入台（1998 統一星巴克）後成人英文名書寫從教室擴散到日常消費場景
- **回頭期 2010s-2020s**：原住民正名運動（2017 排灣族 Ljumeg Tjaruzaljum case 等）+ 護照外文姓名 2019 修法（從「限變更一次」改「無次數限制」，待驗證）→ 改名潮（Kevin → 拼音 Chih-Hao）
- **拼音政治**：威妥瑪（戰前 ROC 沿用）→ 通用拼音（2002 陳水扁政府）→ 漢語拼音（2009 馬英九政府傾斜）三套並存於同一家庭護照

### 關鍵 negative findings（誠實記錄搜不到）

1. **「翻書指派」物理動作** — first-person 回憶多為「即興配 / 3 秒寫下 / 看名單」，沒人說「翻書」。建議寫「即興指派」不寫「翻書」
2. **1988《英文名字大全》無具名 ISBN** — persona D 軼事可能是後造記憶；最接近是 1992《英文姓名寶鑑》
3. **Sesame Workshop 1987 親自設計教材**是時間壓縮錯誤 — Sesame English ESL 線 1999 才推出，台灣 1987 起跑的「芝蔴街美語」是授權品牌 + 本土團隊

6. **「Kevin → Chia-Hao 改名潮」無 trend 報導** — aithley.com 反方向建議「保留英文名 + 加簽為護照別名」NT$1,300。Stage 3 應弱化或刪
