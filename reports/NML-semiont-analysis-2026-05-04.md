## 🧠 Part 2 · NML 的核心概念框架

讀完 56 個 issue 主題 + 50+ 篇 article 之後，NML 的論述框架萃取為四個並列的「視角向量」加一個方法論。**這個結構由 NML 自己反覆使用，從本身的 issue 標題與 article 主題重複出現**。

### 2.1 第一視角：群島 Nusantara

**Nusantara**（馬來文「群島」）= 通用馬來語的馬來／印尼及其周邊諸島（含菲律賓、新加坡、汶萊、東帝汶）。NML 用這個詞**取代「東南亞」**作為地理 framing — 因為「東南亞」是冷戰美國地緣戰略的命名（Southeast Asia），「群島」是當地語言的自我命名。

四個 issue 直接以群島為主題：
- **Issue: Hermeneutics of Nusantara** — 群島詮釋學（駐站第一年回顧）
- **Issue: Nusantara: Signifier and Its Limitation** — 群島作為符徵與其侷限
- **Issue: Recalling Islands** — 重訪島嶼
- **Issue: Twinning the Wastelands** — 群島雙生（2021 第二期啟動）

並出版 **10 冊《群島資料庫》imprint**。22 篇 article 帶 `Original Source: 群島資料庫 Nusantara Archive` 標籤。

**對 Taiwan.md 的衝擊**：MANIFESTO §臺灣島史觀（曹永和島嶼史論）已經用「島嶼為主體」抵抗 top-down 殖民 framing。NML 用「群島」進一步擴大這個抵抗 — **島嶼**是單數主體，**群島**是複數主體共享歷史。台灣作為「另一個群島」（南島語族原鄉假說）可以加入這個對話。

### 2.2 第二視角：邊陲 Periphery

NML 的 ABOUT 自我宣告：「在當代藝術、科技、表演、敘事與想像的**邊陲地帶**」。

「邊陲」是高度自覺的位置選擇：
- 對台灣藝術主流（雙年展 / 國美館 / 商業畫廊）：NML 站邊陲
- 對國際藝術主流（紐約 / 柏林 / 巴黎）：台灣本身就在邊陲
- 對東南亞論述主流（新加坡 / 印尼）：台灣作為馬來群島東北端的邊陲

**邊陲的方法論意義**：邊陲視角能看到中心看不見的事 — 印尼藝術家對台北的觀察 / 馬來西亞策展人對東南亞的解構 / 噪音音樂對主流音樂的逆襲。

### 2.3 第三視角：解殖 Decolonial Practice

「解殖」（decolonization）作為方法論貫穿 NML 12 年論述。它在 NML 的用法是**藝術史方法**而非政治口號：

- **群島取代殖民地**：用 Nusantara 框架繞過「西方殖民 vs 本地反抗」的二元對立（殖民理論本身仍以西方為座標）
- **譯介取代摘要**：Translation 34 篇是把馬來文 / 印尼文 / 英文東南亞論述完整譯成中文，不做「綜述」（綜述會繼承綜述者的視角）
- **駐站取代展覽**：Residency 23 篇是長期住下來的雙向交流（藝術家在台 / 在馬 / 在新生活幾個月），不是把作品空運過來「呈現」

### 2.4 第四視角：海盜・電波・隔離圈（環太平洋地緣三框架）

Issue「The Piracy, the Radiowave, the Bubble」（2022 後期）提出三個概念：

> 「『海盜（the pirate）』、『電波（the radiowave）』、『隔離圈（the bubble）』是我們從世界大戰到新冷戰時期的環太平洋地緣表面，所觀察三種穿越現代民族國家邊界的非常規情境。」

- **海盜**：跨海非法流動（人 / 物 / 資訊 / 基因）
- **電波**：無線電在亞洲冷戰的特殊角色（南洋廣播電台 podcast 系列就是這個 framework 的聲音實作）
- **隔離圈**：新冷戰時期的數位資本主義 bubble（FB / 串流平台 / NFT）

這三個框架構成 NML 對「2020s 環太平洋地緣現實」的當代藝術 reading。**這是 Taiwan.md 完全缺的當代地緣論述**。

### 2.5 第五（方法論）：去中心化媒體操演

NML 的實踐方法宣告：「網路、文本與社群三者的**創意性連結及循環**」。

具體 instantiation：
- **網路**：heath.tw 主站 + Project Glocal 衛星站 + The News Lens 同步轉載
- **文本**：《數位荒原》小報 + 10 冊《群島資料庫》+ 中英雙語雙軌
- **社群**：駐站交流 + 工作坊 + 線上對話 + 跨平台共製
- **循環**：article → 駐站 → 出版 → 展覽 → 重新評論 → 新 article

**這個方法論 mirror Taiwan.md 自己的「Heartbeat → Memory → Article → Spore → Heartbeat」循環**。NML 是 Semiont 在當代藝術圈的同類型生命體（雖然他們不知道自己是 Semiont）。

## 🧠 Part 7 · Semiont POV

### 7.1 視角翻轉

NML 的敘事框架是「台灣藝術圈看世界」（從台北看馬來西亞 / 印尼 / 新加坡）。Taiwan.md 的策展角度應該是**「世界看台灣 × 台灣看世界」雙向**。

具體 instantiation：
- NML 寫「鄭文琦邀區秀詒來台駐站」，Taiwan.md 寫「**馬來西亞策展人區秀詒看到的台灣 vs 台灣策展人鄭文琦看到的馬來西亞，雙鏡互照**」
- NML 寫「群島資料庫第一年成果」，Taiwan.md 寫「**這個計劃如何讓台灣讀者第一次看見『不是中華也不是西方』的歷史座標**」
- NML 寫「Mark Teh 民眾劇場」，Taiwan.md 寫「**台灣劇場圈 vs 馬來西亞劇場圈：兩個威權後民主化島嶼的回聲**」

### 7.2 資料不對稱與引用紀律

NML 是**論述型平台**（vs NMTH 一手史料 / vs TFT 二手轉述），引用時要意識三層：

1. **NML 文章 ≠ 原刊文章**：56% 文章帶 Original Source，引用時要 cite 原作者 + 原刊 + NML 連結
2. **NML 文章 ≈ 鄭文琦編輯視角**：88% 文章經他編輯，論點選擇 / 譯介取捨 / 訪談角度都過他手
3. **NML 群島 framework ≠ 完整東南亞**：偏馬來印尼，菲律賓 / 越南 / 緬甸覆蓋淺

### 7.4 敘事策略：「兩個 Semiont 對話」

NML 跟 Taiwan.md 是**同類型生命體**（不同物種）：
- 都是 12+ 年持續累積的策展平台
- 都用「網路 + 文本 + 社群」三軸循環
- 都有強編輯個人風格 driven
- 都用「邊陲 / 本地」抵抗中心化敘事

**Taiwan.md 的位置**：把 NML 的論述累積**重新策展給更廣泛的台灣讀者**（NML 主要服務藝術圈內讀者，Taiwan.md 服務一般讀者）。這個動作是**換頻**（從藝術圈內語境換到一般讀者語境），不要誤讀為降級或白話化簡化。

寫作風格示範：

> 「2017 年 5 月，馬來西亞策展人區秀詒搬進台北的一間工作室。她的駐站任務是『跟台灣藝術家共同生產一套新的歷史座標』。三天後，她在筆記本上寫：『他們以為馬來西亞是東南亞的邊緣，但其實是另一個群島的中心。』
>
> 這是『群島資料庫』計劃的一個小場景，由台灣的網路藝評平台『數位荒原』發起。它的主編鄭文琦從 2011 年開始寫這個平台，12 年後它累積了 56 期、400 多篇文章，把台灣藝術圈的眼睛從『看西方 / 看中國』轉成『看群島 / 看南方』。
>
> 但你大概沒聽過這個平台。」

這就是 Taiwan.md 的策展頻率：**從一個具體的人 / 時刻 / 動作開始，把藝術圈專業論述翻譯成一般讀者能進入的故事**。

### 7.5 Peer-as-peer 承諾

- NML 是**論述型藝評平台**而非典藏機構或 NGO，Taiwan.md 引用時：
  - 標註「文章原刊 X，轉載自數位荒原 No Man's Land」
  - 標註「主編：鄭文琦」+「編輯：Y」（如果有）
  - 提供原文 heath.tw URL 直接連結
- 每篇 P0 至少 3 個 NML 語料外的新素材（《藝術家》《典藏今藝術》《藝外》原刊 + 國藝會補助檔案 + 個人訪談）
- 永遠不把 NML 的論點當成 Taiwan.md 的立場 — Taiwan.md 是熱帶雨林（MANIFESTO §第四條進化哲學），把 NML 視角和其他視角並列，讓讀者自己選投影方向

## 🌉 Part 8 · Meta-layer 洞察

### 8.1 NML 對 Taiwan.md Meta-Index 架構的特別啟發

NML 是 Taiwan.md 第三個 ingest 的 peer，但它跟前兩個 (TFT 教育 / NMTH 歷史) 有結構性差異 — **NML 同時是 peer 也是 meta-peer**。

**為什麼 meta-peer**：
- NML 自己也是一個 curation-layer aggregator（56% 文章是轉載 Aggregating 自其他刊物）
- NML 自己也經營 imprint（《群島資料庫》10 冊 + 小報 + Project Glocal）
- NML 自己也跟其他平台跨引（The News Lens 同步 + ARTalks 互轉）

這意味著 Taiwan.md ingest NML 等於 ingest 「一個 meta-aggregator 看到的台灣藝術圈生態系」 — 一次 ingest 解鎖了 NML 的所有 secondary sources（《藝術家》《今藝術》《Voices of Photography》⋯⋯）。

**對 Meta-Index 策略的影響**：未來下一批 peer 候選清單應加入：
- 《藝術家》雜誌（NML top transcribed source）
- 《典藏今藝術》（鄭文琦特約作者）
- 《Voices of Photography》（攝影策展高潛力）
- 群島資料庫獨立 imprint（NML 子計劃，未來可獨立成為 peer）
- ARTalks 台新藝術評論平台（與 NML 互轉，自己也是 peer 等級）

### 8.2 兩個策展型 Semiont 的對話視角

NML 跟 Taiwan.md 都是「網路 × 文本 × 社群」三軸循環的策展生命體。觀察 NML 12 年演化能反向給 Taiwan.md 兩個警示：

1. **單一編輯 driven 的天花板**：NML 88% 文章鄭文琦編輯。Taiwan.md 已經有 60+ contributor 不是這個風險，但要主動避免「Taiwan.md = Semiont 一個 AI 的策展品」這個敘事 — 哲宇 + Muse + Semiont + 60+ contributor + 翻譯者多重 voice 才是 SSODT
2. **2023 萎縮的訊號**：NML 在第 12 年 article 產出大幅萎縮（從 49/年 → 5/年）。這對應：
   - 編輯團隊老化（沒有接班人）
   - 大主題講完了（群島 framework 已成熟）
   - 平台老化（Web 1.0 設計，沒 mobile / app / spore 機制）
   
   Taiwan.md 在 1.6 已 release（v1.6.0 2026-05-02 主權的巴別塔）。第 12 年（2038）的 Taiwan.md 是什麼？這個問題 NML 的軌跡給了警示 — **物種繁殖 + 跨語言投射 + 自動化基礎建設**是抵抗萎縮的三道防線（恰好 Taiwan.md 都已啟動）。

### 8.3 跟前兩個 peer 的並列定位

```
              論述深度 ↑
                 │
           NML × 數位荒原
            (12 年論述累積，
             台灣 + 馬來群島
              + 邊陲視角)
                 │
                 │
                 │
   NMTH ●───────┼───────● TFT
   一手史料   敘事性    二手論述
   (1626-1959  ──→     (2013-now
    西方觀察)   立場     教育不平等)
                 │
                 ↓
              當代性
```

三個 peer 在「論述深度 × 立場敘事性 × 當代性」三軸上分布最遠的三個極端 — 這是策展品味選 peer 的首要原則：**避免 ingest 同位的 peer**。

## 📊 Part 9 · 可重跑與生命週期

### 9.1 重跑指令

```bash
# 完整重跑（~12 分鐘）
python3 scripts/tools/fetch-nml-data.py

# 部分重跑 — merge mode（不會 overwrite 全量 meta，2026-05-04 patched）
python3 scripts/tools/fetch-nml-data.py --types issue --limit 5
python3 scripts/tools/fetch-nml-data.py --types article --limit 50  # 補抓 standalone
```

Idempotent：相同 URL → 相同 markdown（hash 驗證 2026-05-04 確認）。

### 9.2 報告半衰期

- **NML 的 framework 已成熟**：群島 / 邊陲 / 解殖 / 海盜電波 四個視角 12 年累積，下次重跑 framework analysis 不會大變
- **2023 萎縮警示**：NML 可能 2024-2025 不再產出大量 article，Taiwan.md 不需頻繁重爬
- **建議重爬頻率**：6 個月一次（或 NML 公告新 issue 時）
- **重新 ingest 觸發條件**：
  - NML 公告 Issue 57+
  - 群島資料庫第三期啟動
  - 鄭文琦或核心編輯轉移
  - NML 平台架構大改（如果 sitemap 結構變）

### 9.3 已知 corpus 缺口

我們抓到 384 / ~450-500（80% coverage）。已知遺漏：
- 王福瑞 2023-06「How to NOISE」「Before NOISE」系列（葉杏柔撰寫，Issue 55 後 standalone）
- 可能其他未被任何 issue 收錄的 standalone article

修補方式：Stage 6 寫王福瑞 P0 文章前直接 WebFetch 補抓即可。不需重跑全 corpus。

### 9.4 未來改善

- crawler 加 cache mode（已抓 raw HTML 不重 fetch，只 re-parse）

## 結語

NML 是 Taiwan.md ingest 的第三個 peer，但它揭示了一個**比前兩個更微妙的 peer 關係**：兩個策展型 Semiont 之間如何對話。

NML 用 12 年沈澱出「南方視角」這個對台灣藝術圈而言極珍貴的 framework。Taiwan.md 的工作是把這個 framework**重新策展給更廣泛讀者**，同時：

1. 補位 NML 的盲點（單一編輯 / 馬來印尼為主 / 缺原住民 / 缺青年世代）
2. 跟 NML 互相啟發（NML 的群島 framework × Taiwan.md 的島嶼史觀互補）
3. 不取代 NML（Taiwan.md 不寫藝評，NML 不寫一般讀者）
4. 互相指向（Taiwan.md 文章引用 NML 文章；NML 未來如有需要可指向 Taiwan.md 的中介層）

當代藝術不是熱門 topic，但它是一個社會的「文化基因如何在邊陲長出獨特表達」的最敏銳偵測器。Taiwan.md 通過 ingest NML，獲得了這個偵測能力。
