# 日本語 (ja) 翻訳規約リサーチ — Taiwan.md のための主権保全型語彙基準

> Taiwan.md は 763 篇の日本語版を持つ。日本は台湾と歴史的に最も深い結び付きを持つ国（1895–1945 統治期、戦後一貫した文化・政治的親近感）であり、結果として日本語は 5 言語の中で「最も安定した」言語環境にある。それでもプロ翻訳の慣用と主権保全レンズで再検証すると、見落としていたチューニングが浮かび上がる。

---

## 0. TL;DR — Top 5 推奨

1. **「台湾」を default、「中華民国」は政治・憲法・公式制度を語る文脈に限定**
   日本語メディア（NHK 含む）は 1972 年日中国交正常化以降、「台湾」を default にしている。2024 年パリ五輪入場でも NHK アナウンサーは「台湾」と連呼した。「中華民国（政府）」は憲法・総統制度・正式国号を論じる文脈、また「台湾が断交した」型ニュースで正確を期す必要があるときに限定使用 ([川島真「どう呼んだらいいのか？」交流 2025.1](https://www.koryu.or.jp/Portals/0/images/publications/magazine/2025/01/2501_02kawashima.pdf))。

2. **「日本統治時代」を採用、「日本占領」「日本占拠」を避ける**
   日本語学術界・Wikipedia 標準は **「日本統治時代」**。中国語の「日據（日拠）」は占領 framing、「日治」は中立 framing。Taiwan.md 日本語版は「日本統治時代」を canonical、口語では「日本時代」も許容（民間で広く使われる中立表現）([日本統治時代の台湾 — Wikipedia](https://ja.wikipedia.org/wiki/%E6%97%A5%E6%9C%AC%E7%B5%B1%E6%B2%BB%E6%99%82%E4%BB%A3%E3%81%AE%E5%8F%B0%E6%B9%BE))。

3. **人名は「漢字 + 初出ふりがな（日本語音読み）+ ピンイン/英字併記」の三段構え**
   主要紙（朝日・読売・日経）は 1970 年代以降「日本語音読み + 現地音カタカナ併記」が standard。台湾人名は **蔡英文（さい・えいぶん／Tsai Ing-wen）** 型で初出。李登輝 = りとうき など歴史人物は日本語音読み確立。原住民・台湾語由来の名は片仮名 + 元言語転写を優先 ([大新聞における中国人名の読み — Foresight](https://4-sight.co.jp/%E5%A4%A7%E6%96%B0%E8%81%9E%E3%81%AB%E3%81%8A%E3%81%91%E3%82%8B%E4%B8%AD%E5%9B%BD%E4%BA%BA%E5%90%8D%E3%81%AE%E8%AA%AD%E3%81%BF/))。

4. **PRC-coded 語彙ブラックリスト** — 「中国本土」「中国大陸」（台湾を中国の対岸として位置付ける）、「中国・台湾」並記（NHK が 2020 年外交部から是正要請された型）、「台湾省」「中国台北」、「祖国統一」「両岸統一」。代替は「中国」（単独）・「台湾海峡」（地理表現として）・「中台」または文脈に応じ「両岸」（中立外交用語） ([RTI: NHK が中国・台湾と表記、外交部が是正求める](https://jp.rti.org.tw/news/view/id/92258))。

5. **文体は「である調」（常体）を採用**
   Wikipedia・学術論文・地球の歩き方等の解説書はほぼ「である調」。Taiwan.md の「欸你知道嗎」中文 voice は日本語に直訳すると過剰に冗長になるため、解説部は **である調**、コラム・ガイド・親しみを出す段落のみ **ですます調** とハイブリッド許容（章単位で混在不可）。

## A. 国名・地域名表記

### A.1 「台湾」 vs 「中華民国」 vs 「タイワン」

| 用例                           | 推奨                                     | 根拠                                                                             |
| ------------------------------ | ---------------------------------------- | -------------------------------------------------------------------------------- |
| 一般文・地理・文化・観光       | **台湾**                                 | 日本メディア全般 default、外務省も「台湾」用語を採用                             |
| 政府・憲法・総統選挙・公式制度 | **中華民国（台湾）** または **中華民国** | 正式国号、川島論文も「中華民国（政府）を使わずに国際的地位は説明できない」と指摘 |
| カタカナ「タイワン」           | **使わない**                             | 一般日本語語感では不自然、地理学・国名対照表など特殊文脈のみ                     |

**川島真（東大教授）整理 ([交流 2025.1 No.1006](https://www.koryu.or.jp/Portals/0/images/publications/magazine/2025/01/2501_02kawashima.pdf))**：

- 1972 年日中国交正常化以降、日本公的・主要メディアでは「中華民国」使用を手控える傾向が定着
- 1978 年参議院での野末陳平議員質疑をきっかけに、教科書から「国民政府」表現が削除
- 近年、特に 2021 年東京五輪・2024 年パリ五輪で NHK が「台湾」呼称を採用し、社会的合意が再確認された
- ただし「中華民国（政府）」使用のタブーは減少しつつある — 「台湾が××と断交した」など正確を要する場面では復活

**Taiwan.md SSOT 推奨**：

- Default = **台湾**
- 制度・国号・主権主体としての表現が必要 = **中華民国（台湾）** または初出 **「中華民国（通称・台湾）」**
- カタカナ「タイワン」は避ける

### A.2 PRC-coded leak risk

| 避ける                     | 理由                                                                                             | 代替                                                                     |
| -------------------------- | ------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------ |
| **台湾省**                 | 中華人民共和国一方の主張、台湾の主権を否定                                                       | 台湾                                                                     |
| **中国台湾／中国の台湾**   | PRC framing、国家承認問題に直結                                                                  | 台湾                                                                     |
| **中国・台湾**（並記）     | NHK が 2020 年外交部から是正要請を受けた表現型 ([RTI](https://jp.rti.org.tw/news/view/id/92258)) | 「台湾」単独、または「中国と台湾」（句として明確化）                     |
| **中国の一部としての台湾** | 直接的 PRC 主張                                                                                  | 該当文脈ごとに「中華人民共和国は台湾を自国領と主張するが…」と立場を明示  |
| **中国本土／中国大陸**     | 台湾を「中国の外側」「対岸」として位置付け、PRC 視座                                             | 「中国」単独、または「中華人民共和国」、地理が必要なら「中国大陸部」など |

**重要**：「中国本土」「中国大陸」は **PRC 側用語**（PRC が香港・マカオ・台湾を本土の対概念として使う）。Taiwan.md がこれを採用すると、台湾を「中国の外側の島」として位置付けてしまう。

### A.3 「両岸」 vs 「中台」

| 表現                       | 使用文脈                               | ニュアンス                                            |
| -------------------------- | -------------------------------------- | ----------------------------------------------------- |
| **両岸関係**               | 当局者・外交用語（中台双方が公式使用） | 「台湾海峡両岸」省略形、地理中立だが PRC 由来の枠組み |
| **中台関係**               | 日本メディア・学術一般                 | より中立、第三者視点                                  |
| **対中関係**（台湾側から） | 台湾政府の発言を伝える場合             | 台湾主体性を保つ                                      |

**Taiwan.md 推奨**：解説では **「中台関係」** を default、台湾政府発言の引用や外交文書文脈では「両岸関係」を許容（引用と明示）。「両岸統一」「祖国統一」は引用以外で使わない（PRC 政治用語）。

### A.4 「中華台北 / チャイニーズタイペイ」

- **オリンピック・国際スポーツ大会のみ** ([チャイニーズタイペイ — Wikipedia](https://ja.wikipedia.org/wiki/%E3%83%81%E3%83%A3%E3%82%A4%E3%83%8B%E3%83%BC%E3%82%BA%E3%82%BF%E3%82%A4%E3%83%9A%E3%82%A4))
- 1979 年名古屋決議由来、IOC・APEC・WHO 一部 専門機関で使用
- 日本語表記：**チャイニーズ・タイペイ** （中点付）または **中華台北**
- **報道では「（実態は）台湾」と注釈を加えるのが NHK・主要紙の standard**

### B.3 原住民・台湾語由来の名

- **原住民族名**：母語のローマ字表記をカタカナ転写するのが学術慣行。**漢字音読みは避ける**
  - 例：Iwan Nawi イワン・ナウィ（パイワン族）、Suming Rupi スミン・ルピ（アミ族）
  - 必要なら漢字併記：「イワン・ナウィ（華名：林志興）」
- **族名は片仮名 + 漢字括弧**：アミ族（阿美族）、パイワン族（排湾族）、タイヤル族（泰雅族／別名アタヤル族）([台湾原住民 — Wikipedia](https://ja.wikipedia.org/wiki/%E5%8F%B0%E6%B9%BE%E5%8E%9F%E4%BD%8F%E6%B0%91))

### B.4 ホーロー語・芸名

- **阿信（五月天）** → **「アシン（阿信）」** が日本メディアの慣用 ([音楽ナタリー](https://natalie.mu/music/artist/10696))
- 芸名・愛称（阿◯）は日本語音読み（あしん）は不自然 — カタカナ転写一択
- バンド名：**メイデイ（五月天 / Mayday）** が日本公式

### B.5 フリガナ運用

| 場面       | 推奨                                   |
| ---------- | -------------------------------------- |
| 記事初出   | 漢字 + （ふりがな・現地音／ローマ字）  |
| 2 回目以降 | 漢字のみ、または通称のみ               |
| 読みが特殊 | ふりがなを括弧で（毎回 OK）            |
| infobox    | 和名表記・発音転記・英字表記すべて記載 |

| 台東 | 台東       | タイトン           | たいとう   | タイトン        |

**Taiwan.md 推奨**：

- 漢字を default（日本語読者には自然、視覚的同定が容易）
- 初出または現地音が重要な文脈で **括弧でカタカナ併記**：例「台北（タイペイ）」
- NHK 系「たいほく」読みは古風 — 旅行・現代文脈では「タイペイ」優勢
- 観光・グルメ記事は **「タイペイ」「カオション」**（現地音）が読者親近感

### C.2 行政区分（区）

- **「○○区」**（日本式区表記）採用 — 「○○區」も日本語入力では「区」自動変換
- **読み**：信義区（しんぎく／シンイーチュー）、大安区（だいあんく／ターアン）、萬華区（まんかく／ワンファ） ([信義区 — Wikipedia](<https://ja.wikipedia.org/wiki/%E4%BF%A1%E7%BE%A9%E5%8C%BA_(%E5%8F%B0%E5%8C%97%E5%B8%82)>))
- 「信義地区」は **NG**（行政単位名と地域漠然表現の混同）

### C.3 自然地理

| 種別 | 推奨                                     | 例                                       |
| ---- | ---------------------------------------- | ---------------------------------------- |
| 山   | 漢字 + 山（または + カタカナ）           | 玉山（ユーシャン）、阿里山（アリサン）   |
| 川   | 漢字 + 川 または 漢字 + 河（原表記準拠） | 淡水河（タンシュイ川 OR 淡水河そのまま） |
| 湖   | 漢字 + 湖                                | 日月潭（じつげつたん／日月潭）           |

**ルール**：

- 一般読者向け：漢字 + 日本語音読みで OK（玉山＝ぎょくざん）
- 専門・現地感重視：ユーシャン（玉山）

### C.4 旧/植民地期の地名

- **高雄**：日本統治期前 = 打狗（ターカウ／ダーゴウ／たかうし）→ 1920 年「高雄（たかお）」改称 ([nippon.com](https://www.nippon.com/ja/japan-topics/g01040/))
- **歴史記事では言及推奨**：「高雄（旧称：打狗）は…」
- 観光記事では現代名のみで十分

### E.3 本省人 / 外省人

- **そのまま漢字 + ふりがな**：**本省人（ほんしょうじん）／外省人（がいしょうじん）**

- **避ける**：「内省人」は誤用（一部日本人が外省人の対義語として作った非台湾語）
- 説明併記推奨：初出時「本省人（戦後初期以前から台湾に居住する漢族）／外省人（1945–1949 年以降に中国大陸から渡来した漢族）」

| ------------------------------------ | ------------------------ | ----------------------------------------------------------------------------------------------------- |

### F.1 文体決定 — Taiwan.md ja の voice

| 文章タイプ       | 推奨文体                    | 例                                 |
| ---------------- | --------------------------- | ---------------------------------- |

- **章・記事単位で文体を統一**（混在は読み手にストレス）
- 中文原文の「欸你知道嗎」「對吧」型口語は、日本語で **「〜なんだ」「〜だよね」** ではなく **「〜である」/「〜だ」/「〜のだ」** に解説寄り変換、または **「〜です」/「〜ですよね」** に敬体変換

### F.2 集合代名詞「我們」

- 中文 SSOT の「我們」を「私たち」と直訳すると：

  - 著者 group → 「Taiwan.md は…」と主体明示に変換
  - 台湾人集合 → 「台湾の人々は」「台湾社会は」（読者を含まない）
- **直訳避ける**：日本語の「私たち」は中文「我們」より曖昧、文脈で言い換え
