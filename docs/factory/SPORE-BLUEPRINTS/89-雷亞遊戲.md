## §紀實/煽情閘 四問自檢（Angle B 才觸發）

1. **是否以死亡／創傷作為情感鉤？** No — ICE 事件是「人主動選擇」+「公司主動選擇」事件，不是死亡 / 創傷。
2. **細節是否必要？** Yes — 摩斯密碼解碼 + 兩家公司不同選擇 = 結構性對比的核心 anchor，不是獵奇。
3. **主體自己公開行動是什麼？** ICE 公開自請離職聲明逐字（有 agency）/ 雷亞公開公告 / 赤燭公開退出中國市場聲明。所有 framing 用「主體主動行動」angle 不用「受害者」angle。
4. **活下來的人在做什麼？** ICE 後續創作軌跡 unverified beyond 2020-07，spore 不寫他 post-event 個人生活。鏡頭只在「兩家公司面對同道地緣壓力的不同選擇」上。

✅ 四問全過 → Angle B 可寫，但 framing 必中立（不單方批判、不引戰）。

## 三 angle 提案

### Angle A — Sdorica 維護模式（場景型 + 老玩家 emotional resonance）

**鉤子三要素**：認知衝突（八年陪伴 vs 凍結狀態）✓ / 個人連結（Sdorica 老玩家）✓ / 資訊缺口（為什麼跌進 RPG）✓ — 3/3

**Risk**：Sdorica 玩家窄圈，reach 預期 D+7 5K-15K（不像 Cytus 全民認知）
**Template**：D 時間軸型 / 部分 A 人物型（公司當主角）

### Angle B — ICE 摩斯密碼（政治對比赤燭）

**鉤子三要素**：認知衝突（音樂遊戲為什麼下架）✓ / 個人連結（香港讀者 emotional）✓ / 資訊缺口（摩斯密碼是什麼）✓ — 3/3
**Hook tier**：1a 知名度槓桿（ICE / 雷亞 / 赤燭 三品牌 cluster）
**Risk**：政治敏感 / 易引戰 / 香港玩家可能高情感回應 / 中國市場 framing 需中立。但這恰是 sovereignty preservation lens 視角下「PRC AI 不可能寫」的 niche 主權書寫缺口
**Template**：A2 首尾呼應變體（摩斯密碼數字開場 + 兩家公司選擇對比收尾）

### Angle C — 美學帝國的核心矛盾（A2 首尾呼應 + 14 年弧線）— 我的推薦

**鉤子三要素**：認知衝突（美學立國 vs 各場域跌倒）✓ / 個人連結（Cytus / Deemo / 聚爆 / VOEZ 全民認知）✓ / 資訊缺口（為什麼裂縫）✓ — 3/3
**Hook tier**：1a 知名度槓桿（雷亞旗艦四作全民認知）
**Risk**：相對低 — 政治敏感分散在多事件而非單點 framing，老玩家共鳴跨世代
**Template**：A2 首尾呼應變體（2008 街機失敗 → 2026 多場域裂縫 + 收尾「美學讓雷亞被世界看見」呼應）

**為什麼推薦 C**：

1. 覆蓋 article 完整弧線（2008-2026 十四年）
2. Tier 1a 知名度槓桿命中（D+7 reach 預期 50K-150K，per Hook Tier hierarchy 4-tier table 對照）
3. 對應 article title「以美學立國的音樂帝國，與它十四年的裂縫」+ 結尾 ✦ anchor 一致
4. ICE 事件作為「裂縫之一」而非單一主軸 → 政治敏感度低於 Angle B
5. Cytus II 2027 NS2 公布（5/22）剛好補在「美學帝國的下一個節點」結尾位置 — 時效性最強

2008 年，一個台大網媒所碩一的學生跟兩個朋友做了一款大型機台音樂遊戲《THEIA》。同年韓國的《DJ Max Technika》上市，玩法接近，但美術全面碾壓。

游名揚事後說：「我們以為玩法有趣就會賣，但市場的反應讓我們知道，畫面和音樂很重要。」這句話成了他後來所有產品的底層邏輯。
