### Spore prose（Threads 主貼 / X 主文）

```
你知道嗎？🌾

1979 年冬天，新竹東郊一片稻田被推土機翻起。那年台灣人均所得還不到兩千美元，「半導體」三個字對多數人來說像外星語。

願意把國家賭在這片稻田上的，是一個叫何宜慈的工程師。史丹佛電機博士、在 IBM 待了快二十年，1974 年他走回台灣，後來成了新竹科學園區第一任局長。所謂「複製矽谷」，其實就是一批這樣的人，把矽谷的經驗裝在自己身上走了回來。

四十五年後，這片稻田跟後來的南科、中科，年營收合計衝破 5.8 兆，全世界最先進的晶片就在這三座園區量產。你的手機、你的車、全世界的 AI，都要等這座島出貨。

但賭贏的代價是，這座島成了全世界的命脈，也成了命門。一場乾旱、一次斷電、一張地緣政治的牌，都能讓全球供應鏈跟著屏息。台灣最強的護身符，同時是它最脆弱的靶心。

四十五年前那片稻田賭的是生存。賭贏之後，換成全世界開始賭它。
```
