## Prose Draft v1（target Threads，主貼純故事不含連結）

```
你知道嗎？🍎

熱炒店冰箱裡 60 年沒換過位置的那瓶金黃氣泡飲，它的商標前 30 年從來不在台灣手上。

1965 年菲律賓華僑李鴻略從美國謝斯尼斯買來配方，邀香港屈臣氏入股，廠設桃園平鎮。產品一開始叫「美國蘋菓西打」，1979 台美斷交後「美國」兩字被悄悄拿掉。商標之後在美國果是高、賴比瑞亞商果是高國際之間流轉。

1995 年，孫幼英自掏 80 萬美元，把蘋菓西打買回台灣。當年報導逐字寫：「考量大飲營收高達 95% 為販售蘋果西打。」

二十四年後，她從自己救回來的這家公司掏走 1.08 億。

KTV 包廂裡那杯玫瑰紅加蘋果西打、阿嬤辦桌時遞給孫子的解膩飲、Super Junior 圭賢在台南度小月喝完留下「好好喝喔，這個韓國沒有對吧？」一瓶氣泡飲，60 年裡把台灣的金融、外交、食安、文化都壓進同一個塑膠瓶。
```
