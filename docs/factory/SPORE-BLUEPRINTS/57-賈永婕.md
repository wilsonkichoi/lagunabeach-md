## 我的推薦

**Angle A（旗桿）** 首發。理由：

1. **唯一前所未見的 insider scene** — Honnold 攀岩是 1/25 全球 6.2M 人看過的事件，但「五根旗桿全換國旗」這個策展細節絕大多數讀者沒看到（Threads 上沒被廣傳）。資訊缺口最大
2. **賈本人 IG 限時 verbatim** 是強引語，且口語感強（「怎麼樣都可以拍得到，怎麼樣全世界都看得到」），不像 angle B 那條政治話語有公關文意味
3. **Scene-List-Scene 結構** 自然把她「藝人 → HFNC → 101 → Honnold」整條軌跡壓縮在 list 中段，省下空間給開場/結尾的具體 scene
4. **政治風險最低** — angle B 涉外省 / 北洋政府敏感，angle C 涉蔡英文時敘可能被解讀政治偏好；angle A 「全換國旗」是國家形象，跨黨派接受度高
5. **DNA #28 紀實節制達標** — 沒涉及死亡 / 創傷 / 未公開家庭事

**選 angle A**：直接 publish。
**選 angle B**：適合對「外省第二代 → 台灣人第一代」議題有興趣的讀者群（換日線 / TNL / 報導者風）。
**選 angle C**：適合 Covid 紀念日（2021-05 兩週年）等時事掛鉤時機。
