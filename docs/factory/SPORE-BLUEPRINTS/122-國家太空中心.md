```
你知道嗎？🚀

五月天〈頑固〉MV 裡那個在荒地造火箭、被當成做白日夢的瘋子，是真人，外號火箭阿伯的吳宗信。五年後，這個曾被當成笑話的人，接掌了國家的太空計畫。

而他接手的機構，1991 年成立時連個正式名字都沒有，只叫「籌備處」，花了三十二年才把 Taiwan 放回自己的英文名裡。

但這個機構現在能做的事，遠不只改個名。2024 年 0403 花蓮大地震後六分鐘，工程師遠端重設了一顆衛星；三小時後它飛過災區，拍下一千三百多處崩塌，成了撤離與封路的依據。它甚至已經核定，2028 年要送兩具自製儀器上月球。

那個曾被當成笑話的火箭夢，現在替整個國家看地震、上月球——也把一個小島的名字，一顆衛星一顆衛星地，送上了天。
```
