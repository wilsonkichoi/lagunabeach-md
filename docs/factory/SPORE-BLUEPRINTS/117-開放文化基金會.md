你知道嗎？🧾

你在 LINE 群組裡用「真的假的」查長輩轉的假訊息，2020 年用手機查巷口藥局還剩幾片口罩——這些成果台灣人都記得，卻幾乎沒人知道是誰幫這些專案報帳、開發票、跟政府簽約。

2014 年，一群辦開源研討會的工程師卡在一個很不浪漫的問題：活動收據沒地方合法核銷。於是他們成立了開放文化基金會（OCF），一個會開發票、寫年報、受董事會約束的財團法人。

最反直覺的是：一場喊著「沒有人是萬能的」、最自豪沒有老闆沒有領導的運動，為了活下去，得自己長出一個有地址、有董事、有人簽名負責的體制。

十幾個人的小團隊，扛起四十多個社群的帳務和行政。後來這個後台還長出了牙齒，開始評比企業有多尊重你的隱私、盯著政府打詐的封網指令會不會封過頭。

台灣的公民科技能撐十年不散場，靠的是有人願意做最不性感的那部分：把每一張收據貼好。熱血可以是免費的，行政不行。
