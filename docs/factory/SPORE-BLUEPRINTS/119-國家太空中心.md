```
你知道嗎？🚀

五月天〈頑固〉的 MV 裡，有一個中年男人在荒地裡造火箭，被當成做白日夢的瘋子。那是真人故事，原型叫吳宗信，外號火箭阿伯。

五年後，這個被當成笑話的人，接掌了國家的太空計畫。

而他接手的這個機構，1991 年成立時連個正式名字都沒有，只叫「國家太空計畫室籌備處」。花了三十二年，它才在 2023 年第一次拿到法律上的「法人身分」，順手把英文名從 NSPO 改成 TASA，刻意把 Taiwan 放進去。

一個連在運動場上都印不上自己名字的國家，先讓自己的太空署，把 Taiwan 放回了名字裡。
```
