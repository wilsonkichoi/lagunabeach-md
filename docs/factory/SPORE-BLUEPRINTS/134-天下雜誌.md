## Spore body（Threads 主貼 = X 主文）

```
你知道嗎？📰

一個台灣來的窮留學生，穿著一條借來的棗紅色牛仔褲，1968 年坐進了張愛玲在劍橋的房間，一聊就是三個小時。在她之後，再沒有人專訪過張愛玲；而她們兩個，後來還聯絡了十幾年。

她叫殷允芃，後來跑過紐約時報、亞洲華爾街日報，心裡一直擱著一個問題：為什麼台灣沒有一本大家都看得懂的財經雜誌？

1981 年，她在還沒解嚴、剛被世界推下舞台的台灣，把這本雜誌辦了出來。創刊號首刷一萬本，兩天賣光。她後來苦笑著說：「要害一個人，就讓他去辦雜誌。」

四十年過去，這本《天下》成了台灣最受信任的媒體之一，信任度五成七，而整個台灣對新聞的信任長年只有兩三成。它替企業排「兩千大」、替城市評「幸福指數」，那些名次你在便利商店架上、在新聞跑馬燈裡都見過，它就這樣默默替一整個社會定義了什麼叫「成功」。

可它最受信任的位置，剛好也是它離資本與權力最近的位置。那份公信力，和那份替別人打分數的權力，其實來自同一段距離。

八十歲那年，殷允芃把手上的股權交給信託，讓盈餘再投回雜誌，把資本和編輯室之間那道牆砌得更厚一點。然後她把雜誌交給六個專業經理人，自己回去做最初就在意的那件事——閱讀和教育。

當年那個穿著借來的棗紅色牛仔褲、去訪張愛玲的窮學生，創辦《天下》那年，收到了張愛玲寄來的一張賀卡。那是她們之間，最初、也最純粹的一段距離。
```
