## Hook 候選 3 angles

### Angle A（推薦）：消滅自己 — 哲學最強 + 對應 v1.6「使命設計成基礎建設」

**核心衝突**：創辦人主動設計「會讓自己變得不重要」的系統。讀者預期是「founder 會想留住 control」，實際是「founder 認為自己是最脆弱的部分要被淘汰」。

**情感弧線**：賴靜嫻問目標 → 哲宇答「消滅自己」→ 解釋為什麼（自己是最脆弱）→ Token donation 機制 + fork 證據 → 賴靜嫻 reaction 收尾。

**Tier**: 1a（哲學深度 + 既有 podcast 平台知名度槓桿）

**最強 detail**：「我是裡面最脆弱的部分」/ Token donation「成為台灣.md」/ 賴靜嫻「跟你聊變 60 了」reaction

### Angle B（候補）：越往外走越發現家就很好 — 場景生活型 + 普世共鳴

**核心衝突**：紐約 Lady M / 起司蛋糕 / fancy 都會 vs 早餐店蛋餅。為什麼 fancy 會輸給日常？

**最強 detail**：「我每次一回台灣，第一天早上就衝去早餐店吃蛋餅」/ 賴靜嫻自接「我早餐吃蛋餅，中餐吃蛋餅」/「越往外走，我越發現我們的家就很好」現在放在 Taiwan.md 簡介

**Tier**: 1b（具體性槓桿 — 早餐店蛋餅是讀者 visceral object）

### Angle C（候補）：西瓜 vs 乾乾的維基 — 認知顛覆型

**核心衝突**：知識庫一般以為應該越中立越好，哲宇說「不，要像西瓜，所有立場都裝進去 + 標誰講的」。

**最強 detail**：「不同立場、不同聲音都不去掉，但讓你知道是誰講的」/ 賴靜嫻維基百科助理「他是壞人耶為什麼要採訪他」笑話 / SSOT → SSODT 升級
