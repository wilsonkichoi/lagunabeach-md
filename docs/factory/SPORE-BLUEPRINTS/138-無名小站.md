## Spore body（Threads 主貼 = X 主文）

```
你知道嗎？🔌

1999 年的交大宿舍，六個資工系學生用系上報廢的零件，拼出一個叫「無名」的 BBS。一台撿來的電腦，最樸素的好處是：不想玩了，拔插頭就好，沒人有意見。

十年後它長成全台最大的部落格，250 萬會員、5 億張照片，2008 年甚至爬到母公司 Yahoo 頭上，當上全台第一大網站。創辦人簡志宇這時才發現，那個電源已經關不掉了，他得為兩百多萬人負責。

他沒說出口的是，電源其實早就不在自己手上。2007 年簽約那一刻，開關就轉給了 Yahoo。2013 年 12 月 26 日那晚，Yahoo 替他按下了開關，無名被永久刪除。今天用 Wayback Machine 去找，5 億張照片撈回來的，是一格一格打不開的空白方框。

台灣對自己最大的一份數位青春，零國家級保存，沒人覺得搶救它是誰的責任。

而最該讓人不安的，其實是現在：你放在 IG 的限動、FB 的相簿、雲端裡孩子的周歲照，跟當年那 5 億張一樣，都插在一個別人隨時能拔掉的電源上。
```
