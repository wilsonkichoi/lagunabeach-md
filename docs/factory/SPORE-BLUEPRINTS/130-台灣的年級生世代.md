## Spore body（Threads 主貼 = X 主文）

```
你知道嗎？🍓

1993 年「草莓族」第一次被印出來時，職場顧問翁靜玉罵的不是現在的年輕人——是當時剛進辦公室、二十幾歲那批，也就是 1960 年代出生、今天五十幾歲、最愛在臉書社團懷念養樂多和大同寶寶的五年級生。

三十年後，這個詞下移到七年級、八年級頭上，2008 年「野草莓運動」乾脆被罵的人反過來認領回去當運動名稱。最近，又戴到九年級頭上。被罵的人換了好幾批，標籤一字沒改。

但真正在五個世代之間移動的，不是抗壓性，是腳下那座「努力就有回報」的扶梯。台北房價所得比，從 2004 年的 6.4 倍漲到 2025 年的 15.41 倍——一個家庭不吃不喝十五年才買得起一間房。同樣的力氣，到不了同樣的位置。

下次想說「現在年輕人就是草莓」之前，先想一件事：這個詞最早罵的，很可能就是當年二十幾歲的你。
```
