## Spore body（Threads 主貼 = X 主文）

```
你知道嗎？📰

一家不接廣告、不接政黨資助、也不接政府標案的媒體，能活幾年？

《報導者》活了十年。2015 年上線的第一個月，定期定額捐款的只有 4 個人，創辦人何榮幸不敢讓同事知道這個數字。十年後，那 4 個人變成每個月大約 8,000 位，每人從自己的帳戶扣 200、300 元。

它靠這些錢，做了追到三個國家的《血淚漁場》、記者臥底八個月的《獵童風暴》。它的編輯室裡甚至沒有一塊顯示點閱率的螢幕，因為它的生死從來不靠流量。

而這些正好是演算法時代最沒人想付錢看的東西。一個一個陌生人，每月扣那 200、300 元，把調查報導從一門賠錢生意，救成了一塊公共財。
```
