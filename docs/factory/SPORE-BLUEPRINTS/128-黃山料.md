```
2014 年的倫敦伸展台，一個金門來的年輕人讓六套衣服走過台前，主題叫「金門1969」——軍裝剪裁、彈痕、印著《金門日報》的仿舊布料。那一年，他拿下倫敦畢業展時裝週的國際首獎。

回台灣，他投了一百多封履歷，沒接到一通面試。有家公司的人資告訴他，主管上網查不到他得獎的任何報導，懷疑他履歷造假。他後來想通一件事：「這個世界不是比誰有能力，而是比誰有說話的能力。」

於是他自己做媒體，再改行寫書，把句子寫到一行不超過 28 個字，要的就是讓人「像翻運勢書一樣，需要一句話的安慰」。連續兩年誠品年度暢銷冠軍，賣破 30 萬本。

這禮拜，一支影片把他的書評成「史上最難看」，一天 60 萬人點進去看。

2014 年穿著「金門1969」走秀的那個青年，和這禮拜被笑「沒料」的暢銷作家，是同一個人。他叫黃山料。
```
