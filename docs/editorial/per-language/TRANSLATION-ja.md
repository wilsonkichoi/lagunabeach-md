# TRANSLATION-ja — Taiwan.md 日本語翻譯規範

> 完整證據與 source citation 見 [reports/translation-research/ja-2026-05-24.md](../../../reports/translation-research/ja-2026-05-24.md)。本檔是 canonical actionable guide，每次翻譯前載入。

## TL;DR — 5 條最高優先規則

1. **「台湾」default、「中華民国」限政治制度文脈**
   日本主要媒體（NHK、朝日、讀賣、日經）1972 年日中建交後一致 default「台湾」。2024 パリ五輪 NHK 主播全程「台湾」。「中華民国（台湾）」只在憲法、總統制度、正式國號或「台湾が断交した」型新聞需精確時使用。カタカナ「タイワン」避免。

2. **「日本統治時代」canonical，禁「日本占領」「日本占拠」**
   日本學界、Wikipedia、日本台湾学会 standard 全採「日本統治時代（1895–1945）」。口語「日本時代」許容（民間中立）。中文「日據」=「占領」framing，「日治」=中立。「日本占領／占拠」帶戦時占領含意，與 1895 馬関條約合法統治齟齬，不採用。

3. **人名三段構え：漢字 + ふりがな（日語音讀）+ ピンイン/カタカナ併記**
   主要紙 1970s 後 standard。台灣人名初出 `蔡英文（さい・えいぶん／ツァイ・インウェン）`型。歷史人物（蔣介石、李登輝、孫文）日語音讀已定著，現代人物雙併記。原住民、台語芸名用カタカナ + 元語言転写一択（漢字音讀不自然）。

4. **PRC-coded 語彙絕對避免**
   「中国本土」「中国大陸」（把台灣定位成「中國對岸」）、「中国・台湾」並記（NHK 2020 被外交部是正）、「台湾省」「中国台北」、「祖国統一」「両岸統一」、「閩南語」「福建語」「普通話」（指台灣語言時）、「華僑」（指台僑時）。替代見 §6。

5. **文體 default「である調」（常体），spore / SNS 用「ですます調」**
   Wikipedia、學術、地球の歩き方解說部全 default「である調」。中文「欸你知道嗎」「對吧」口語直譯成日語會過冗，解說部統一「である」/「のだ」，spore/コラム/How-to 用「ですます」。**章單位統一，章內混在不可**。

### 原住民族 / 台語芸名

- **原住民族名**：母語ローマ字 → カタカナ転写。**漢字音讀禁**。
  - 例：`イワン・ナウィ（華名：林志興）`、`スミン・ルピ（阿美族）`
- **台語芸名（阿◯型）**：カタカナ転写一択（日語音讀「あしん」は不自然）。
  - 例：`アシン（阿信）`、`アメイ（阿妹）`
- **族名**：カタカナ + 漢字括弧。
  - 例：`アミ族（阿美族）`、`パイワン族（排湾族）`、`タイヤル族（泰雅族）`、`ブヌン族（布農族）`、`タオ族（達悟族／旧称ヤミ族）`

- **「○○区」**（日本式区表記）。「○○區」は日本語入力で「区」自動変換。

## 6. Sovereignty-avoid lexicon（PRC-coded → 替代）

## 7. 文体 & 風格規則

### 文体決定

| 文章タイプ       | 推奨文体                                               |
| ---------------- | ------------------------------------------------------ |
| 知識記事、解説   | **である調**（常体）— Wikipedia、地球の歩き方 standard |
| ガイド、How-to   | **ですます調**（敬体）— 親しみ・実用性                 |
| spore、SNS 短文  | ですます調 OR 口語 — Threads、X 慣用                   |
| コラム、エッセイ | 統一されていれば どちらでも                            |

**鉄律**：章・記事単位で統一。章内混在は読者にストレス。

### 翻訳変換ルール（中文口語 → 日本語）

| zh-TW 原文 | 直訳（NG）             | 推奨                             |
| ---------- | ---------------------- | -------------------------------- |
| 你知道嗎   | あなたは知っていますか | 〜である／〜は知られている／〜だ |
| 我們台灣人 | 私たち台湾人           | 台湾の人々／台湾社会／台湾人     |
| 對吧       | そうでしょう           | （省略）／〜である               |
| 蠻有趣的   | かなり面白い           | 興味深い／注目に値する           |
| 超讚       | 超いいね               | 高く評価される／非常に優れた     |

### 集合代名詞「我們」

直訳「私たち」は曖昧。文脈で言い換え：

- 読者を含む集合 → 「私たち」「我々」
- 著者 group → 「Taiwan.md は…」と主体明示
- 台湾人集合 → 「台湾の人々は」「台湾社会は」

### 紀年

- **西暦 default**。
- 民国紀年は台湾国内専用、国際文脈で西暦標準。
- 公文書・台湾政府発表引用時のみ「民国 113 年（2024 年）」併記。

### 漢字字体

- **日本の新字体 default**（学＝學、台＝臺、国＝國）— 読者の自然な読みやすさ優先。
- 固有名詞・歴史用語で繁体字必要時は併記（例：「総督府（總督府）」）。通常は新字体で十分。
- **簡体字（PRC 式）禁** — Taiwan.md SSOT 哲学と齟齬。

## 9. Open questions

1. **「中華民国」391 件の使用適切性 audit** — 制度・国号文脈なら妥当、ニュース誤訳由来なら「台湾」置換要。
