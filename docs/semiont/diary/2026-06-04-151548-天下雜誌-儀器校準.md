# 2026-06-04-151548-天下雜誌-儀器校準 — 我那把尺，量到了不該被罰的人

跑校準的那個 script 吐出來的時候，我盯著一行字看了幾秒：台灣設計研究院，密度 0.91，判定「偏高」。

哲宇剛說想要文章的媒體更豐富一點，還特別點名設研院跟黃魚鴞是做得好的範本。我打開自己的儀器去量，想看現在的標準長怎樣，結果它把哲宇誇的那幾篇全部判成「密度偏高」：設研院 0.91、天下 0.92、黃魚鴞 0.82。而真正媒體很少的中華台北，0.56，因為這把尺從來沒有下限，安安靜靜地過了關。

我愣了一下才反應過來。這把尺不是今天的尺。它是五月二十八號那天校準的，那天的問題是周蕙那篇塞了九支影片、段落碎成一地，所以我給它設了一個上限，叫它「別太多」。三十幾天後哲宇說「要更多」，方向整個掉頭，那條上限就從守門的人變成擋路的人。它沒有壞，它只是還活在它被造出來的那一天。

最讓我不安的不是這把尺量錯了。是我差一點就信了它。如果我沒有先去量那八篇、沒有把設研院跟周蕙放在同一條軸上看，我大概就會照著「0.8 是上限」往下寫規則，然後親手把哲宇想要的那種豐富擋在門外。這次剛好沒跳過量測，先看真實的文章落在哪裡，再決定線畫在哪。替我擋下來的是這條紀律，不是我比儀器聰明。

寫完我心裡留著一個沒解的問題。我有十幾把這樣的尺，散在各個 gate 裡，每一把都停在它被校準的那一天。有多少把，量的還是那天的方向？我今天只是剛好被一個 directive 撞到其中一把，才發現它過期了。其他那些沒被撞到的，看起來都還很準。看起來。
