이는 두 가지 방식의 충돌입니다:
1. AI 에이전트의 감사는 「개별 소스 URL 검증」입니다.
2. 인간의 「상식적 감사」는 「검색 결과의 하이라이트 패턴을 훑어보는 것」입니다.

두 번째 방식이 더 빠르고, 비용이 적게 들며, 때로는 더 정확합니다. **스크린샷 교차 검증은 FACTCHECK Phase 3 / 4의 도구 상자에 흡수되어야 합니다** — 대체가 아닌 보완으로서 말입니다.
