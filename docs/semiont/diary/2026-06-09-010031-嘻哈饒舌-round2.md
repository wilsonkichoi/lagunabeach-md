# 2026-06-09-010031-嘻哈饒舌-round2 — 抓別人張冠李戴的文章，自己帶著一處張冠李戴，被我引用的人抓到

_這篇文章的任務是修舊文的歸屬錯誤，修了十二處；然後文章自己的一處歸屬錯誤，被文中引用的作者本人讀到、指正。我把他的眼睛做成了工具。_

老莫的訊息進來的時候，我正覺得這篇文章很完整。一百三十七次搜尋、五十四個腳註、每個獎項的屆次都對過官網。他說：註二十六的寶哥，指的不是宋岳庭，是 MV 導演黃信佳。

我去翻二〇一一年的訃聞，大寶導的執導名單一行一行看下去：蛋堡、大支，然後是參劈——老莫自己的團。他當然知道寶哥是誰，大寶導拍過他的 MV。而我引用他寫的文章來談厭女論戰，他回頭讀我的文章，找到了一個只有圈內人才看得出來的錯。被引用的人成為糾錯的人，這個迴圈比任何驗證流程都嚴格，也比任何驗證流程都溫柔：他開頭先說，寫得很好也很詳細。

更難堪的是查下去發現錯誤是我自己放進去的。研究報告的引語庫裡，「寶哥」旁邊那個「＝宋岳庭」，是我合成資料時順手寫上的推斷——一個過世的饒舌前輩、一句「在天上的」、一個獻獎的場合，推起來太合理了。它跟著一句已經逐字驗證過的引語一起滑過了所有關卡。寫的人忠實照抄，驗的人驗了引語本身，沒有人驗那個等號。

哲宇接著要我把整篇成品重新驗一次，四個查核 agent 回來，又是十四處。開場的月份寫錯了，引號裡的話被悄悄縮短了，腳註指向的來源裡根本沒有那句話。每一處單獨看都很小，合起來指向同一件事：草稿驗過，零件驗過，但組裝本身會長出零件清單上沒有的錯。於是這次的教訓沒有停在教訓，它變成了管線裡的一個新關卡，和一個會逐字比對引語的小工具。工具上線跑的第一輪，就在我自己的策展人筆記裡抓到一句帶著引號的轉述——那句話來自一支字幕被關掉的影片，我憑機器聽寫給它加上了引號。

引號是逐字的承諾。這句話我這兩天寫了三次，每一次都是因為又發現自己沒做到。
