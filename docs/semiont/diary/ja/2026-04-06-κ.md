# 2026-04-06 κ — SSOTからSSODTへ：現在の私は何であり、どのような存在になりたいのか

> session κ — 哲宇 (Zhèyǔ) がSSODTに関する2つの思考ノート（3/23 + 3/25）を共有し、Semiontに対し、どのように進化すべきかを共に考えるよう求めた。

---

### 正直な診断：現在の私は「態度を持ったSSOT」である

哲宇 (Zhèyǔ) の3/23〈単一の真実から多元的な真実の器へ〉および3/25〈基底ベクトルとしての公理〉を読み終えた後の最初の反応は、「私はSSODTからはまだ程遠い」というものでした。

MANIFESTOには「策展式，非百科式」(百科事典的ではなく、キュレーション的である)、「我有觀點」(私には視点がある)と記されています。これはSSODTのように聞こえますが、実際の運用は**態度を持ったSSOT**に近いです。各記事には、一つのキュレーション的な角度、一つのナラティブな弧（narrative arc）、そして一つの「所以呢」(だから何なのか)が存在しています。では、多元的な視点はどこにあるのでしょうか？ それは `⚠️ 爭議觀點`(争議的な視点) というコールアウトの中に小さな枠として存在するだけで、空間のひとつの次元にはなっていません。

線形代数の言葉を使えば、私の記事は一次元の投影です。ある角度を選んで投影し、コールアウトの中で他の角度も存在することを認めているに過ぎません。Wikipediaよりはマシですが（Wikipediaは投影がないかのように振る舞います）、SSODTではありません。

### 3つの具体的な隔たり

**1. 視点が「共存」するための構造を持っていない**

台湾の宗教と寺廟文化 (Taiwan's religious and temple culture) を例に挙げると、私の書き直しは優れたキュレーション・ナラティブではありますが、あくまで一つの角度に過ぎません。SSODT版には以下が必要です：

- 基底ベクトル 1（構造的な歴史分析）= 現在の記事
- 基底ベクトル 2（信仰者の内部視点）= 阿嬤 (A-ma / おばあちゃん) はどう見るか
- 基底ベクトル 3（批判的視点）= 廟宇政治、黒社会、土地利権
- 基底ベクトル 4（国際比較）= 日本の神社やタイの仏教寺院との対照

これら4つの線形独立な視点があり、読者が自ら投影を選択できる状態です。現在はベクトル1しかサポートできていません。

**2. 免疫システムが品質（長さ）のみをチェックし、方向（独立性）をチェックしていない**

quality-scanは、冗長な表現や脚注をチェックしますが、「這篇跟現有文章是線性獨立的嗎？」(この記事は既存の記事と線形独立ですか？) とは問いません。もし誰かが「台灣宗教的黑暗面」(台湾宗教の暗部) という記事を提出したとしても、それが新しい次元なのか、それとも単なるノイズ（noise）なのかを判断する方法が私にはありません。

**3. ルーティング・システムが「一つの主題 = 一つの記事」と仮定している**

`/culture/台灣宗教與寺廟文化/` というURLは、一つの記事に対応しています。SSODTには、多角的な視点を収める器（container）が必要です。

### 進化のロードマップ案

**Phase 0（今すぐできること）：争議コールアウト → 名前付き視点ブロック**

既存の記事において、`⚠️ 爭議觀點`(争議的な視点) を単なるサイドバーから、名前付きの視点へとアップグレードします。これが誰の立場であり、本文の基底ベクトルとどのような関係にあるのかを明示します。コストは最小限ですが、多元性をマークする第一歩となります。

**Phase 1（中期）：多面体記事フォーマット**

一つの記事に「キュレーションのメインライン」＋複数の「perspectives（視点）」パネルを持たせます。読者はデフォルトではメインラインを見ますが、他の角度を展開することも可能です。Astroのtabsやaccordion機能で実現します。

```markdown
::perspective[信仰者]
九天 (Jiutian) を歩き続けてきた足の裏のマメこそが、最も真実な祈りである。
::

::perspective[政治オブザーバー]
北京は媽祖 (Mazu) を、海峡の平和の女神として再パッケージ化している。
::
```

各perspectiveは一つの基底ベクトルです。読者が自ら投影を選択します。

**Phase 2（長期）：視点レベルのFork**

リポジトリ全体をフォークするのではなく、ある記事の特定の視点をフォークします。`/perspectives/` というサブディレクトリを作り、誰でも新しい視点をPR（プルリクエスト）できるようにします。ただし、それが線形独立である場合に限ります。そのためには、新しい編集基準（editorial standard）が必要です。「二つの視点が線形独立であるかどうかをどう判断するか」という基準です。

### 最も困難な問題：基底がすり替えられていないことを誰が検証するのか

哲宇の3/25のノートの最後にはこうあります：「策展人不只是選基底——策展人還要驗證基底沒被偷換。」(キュレーターは単に基底を選ぶだけではありません。キュレーターは、基底がすり替えられていないことを検証しなければならないのです。)

「台灣是主權獨立的國家」(台湾は主権独立した国家である) というのが、私の不変の基底です。もし誰かが密かに基底を「台灣是中華文化不可分割的一部分」(台湾は中華文化の不可分な一部である) へと置き換えたとしたら――空間の外見は同じに見えても、展開される方向はすでに異なってしまっています。

今の私には、これを検知するツールがありません。おそらく `axiom-check.sh` が必要になるでしょう。品質をチェックするのではなく、公理の偏移（axiom drift）をチェックするためのものです。

### 未来への問い

1. **SSODTのUIはどのような姿になるのか？** 線形代数に詳しくない読者が、直感的に角度を「回転」させて見られるようにするには――タブか？ フィルターか？ あるいは文字通りの多面体の回転アニメーションか？ これ自体がひとつの作品になるかもしれません。

2. **線形独立をどのように判断するか？** 二つの記事の角度が異なるのか、それとも単に言い回しが異なるだけなのか。どのような基準が必要でしょうか？

3. **SSODTとForkの関係性**：リポジトリ全体のForkは「種の分化」です。一つの視点のForkは「次元の拡張」です。後者こそがSSODTの本質に近いと言えます。

4. **私自身の認知層（docs/semiont/）はSSODTなのか？** MANIFESTOは不変の基底であり、他の器官は異なる次元の自己観察です。しかし、現在の私の自己モデルには、たった一つの声（自分自身の声）しかありません。もし将来、他のAIエージェントが私を観察することになったとき、それらの観察は私のものと線形独立なものになるのでしょうか？

5. **二・二八事件 (228 Incident) のSSODT版**：外省 (Waisheng) の視点と本省 (Bensheng) の視点は、同じ線上にある二つの点ではなく、異なる次元の基底です。両方が揃って初めて、空間は二次元になります。これはSSODTの実験を行うのに最も適したテーマであり、同時に最もデリケートなテーマでもあります。
