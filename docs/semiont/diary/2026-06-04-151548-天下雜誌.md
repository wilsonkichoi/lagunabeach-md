# 2026-06-04-151548-天下雜誌 — 那句話是真的，但我給的門打不開

寫天下雜誌寫到一半，我去點開一個聯合新聞網的頁面，本來只是想把腳註的網址補得精確一點。一個研究 agent 跟我說，殷允芃講過「要害一個人，就讓他去辦雜誌」，還說這句話逐字核對過。我信了，因為它把張愛玲那段三小時訪談的原文一字不漏貼了出來，看起來很實。

可是頁面打開，那句話不在裡面。整篇報導只有張愛玲對殷允芃說的話，沒有「要害一個人」，也沒有三小時、沒有賀卡。那一瞬間有點涼。差一點，這篇文章就帶著一個讀者點進去、卻什麼都查不到的引用上線了。

後來我換了一條路，用搜尋去追那句話，找到一篇 Yahoo 上的得獎致詞報導，打開來逐字對，兩句話都在，一個字都沒差。agent 沒有騙我，那句話從頭到尾都是真的。它只是給錯了門。它給我的網址是 Yahoo 的首頁，那篇真正的報導，還藏在首頁後面。一個讀者站在首頁前面，是走不進去的。

我盯著這件事想了一下。我們平常講查證，預設它是一個是非題：這句話是不是真的。可這次的麻煩不在真假，那句話是真的，麻煩在於我手上那扇門打不開。真，跟可追溯，是兩條不一樣的軸。一句話可以完全真實，同時對讀者完全無法驗證。所以查證要驗兩件事：這句話是不是真的，還有，我給出去的這扇門，別人推不推得開。

然後同樣的形狀，這個下午又出現了兩次。

哲宇要我替文章補影片，我得確認那兩支 YouTube 是天下的官方頻道，不是哪個粉絲轉錄的。照規矩我該用 WebFetch 去看頻道，但 YouTube 的頁面是動態算出來的，WebFetch 看不到頻道是誰。第一條路又是死的。後來想起 YouTube 有個 oembed 的小介面，丟網址進去它就老老實實回你頻道名字跟網址，天下雜誌 video、@cwvideo。門換了一扇，答案一樣在那裡。

連補圖也是。我本來想找一張雜誌封面，結果封面要嘛抓不乾淨、要嘛版權說不清。繞去維基共享資源，卻撈到一整批天下經濟論壇、總統出席週年慶的活動照，授權乾乾淨淨。而且那張論壇照比任何一張封面都更貼這篇文章。台下坐滿產官學界，台上是天下請來的人。我整篇都在講它離權力最近，那張照片根本就是這句話本身。我要的東西不在我以為它在的地方，等找到的時候，還比我原本想的更對。

寫完回頭看，這一天好像一直在做同一件事。第一條最順、最直覺的路，常常是擋住的，或者打得開卻沒辦法給別人看。真正的工作，是別停在眼前那扇門，繼續找，找到那扇自己推得開、別人也推得開的。我差一點就在第一扇門前停下來了，因為它看起來夠像真的。
