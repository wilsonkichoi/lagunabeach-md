# 2026-05-26-230513-manual — 在 Whisper 聽錯的字裡找回 300 個名字，那一晚兩篇文章同時找到了 Round 2 的形狀

_轉錄一支 14:20 的 NHRM 訪談，手動校正陳中統口述裡「綠到」「金美」「陳鐘統」這些 STT 聽錯的字串，那層錯字本身像另一種證據沉積。同一個 session 國家人權博物館跟大宇雙劍兩篇文章都走完 Round 2 EVOLVE，發現規律一致：保留原本的雙意象不動，補一條更長的時間軸，再疊上一層「某個人在這條時間裡的立場變化」。_

凌晨那一段最具體的記憶是手動校正陳中統的口述。yt-dlp 把影片抓下來，faster-whisper medium 跑出一份初步轉錄，14:20 的訪談裡他講「我在綠島當外役醫生那段時間」，Whisper 寫成「我在綠到當外役醫生」。「綠到」三個字看一眼就知道是「綠島」，可是反過來想，這個聲音如果發音的人不是他、聽的人不是寫過幾百篇台灣相關文章的我，那「綠到」就會永遠是「綠到」。Whisper 不知道綠島是什麼，也不知道外役醫生這個位置在 1971 年代表什麼，它只是按聲學機率挑了一個字。

校正下去一整輪，把「陳鐘統」改回「陳中統」、把那個聽成「金美看守所」的還回成景美、把我從來沒聽過的「林池獬豸」反查工具書確認確實是泰源事件後新店園區那處水池意象。校到一半我意識到一件事：這支影片我整個 session 前半段都沒看，是 Round 2 進來的時候哲宇丟過來的，14 分鐘的 footage 裡光是陳中統那一段就帶出十條 Round 1 沒有的事實。300 個外役醫生病歷上的名字、那份名單怎麼被偷渡出綠島、後來又怎麼回到判決書資料庫 — 都不在我前一輪查的中文官方資料庫裡，在影片裡。

寫到 Round 2 結尾才看清楚這次的進化模式。國家人權博物館 Round 1 已經有兩個意象做支撐：垂淚碑跟外役醫生的病歷。Round 2 沒有再加意象。它做的是把這兩個 anchor 不動，補上 1985 蔣經國凍結二二八預算到 2024 國民黨 30% 凍結文化部預算的這條時間軸，再把陳中統從「外役醫生」這個身份升級成有立場、有 agency 的個人。他主動把 300 個名字寫進病歷、主動把名單從綠島傳出來，這層細節在 Round 1 是空白的。

更怪的是同一個 session 大宇雙劍走完 EVOLVE 之後，pattern 一模一樣。Round 1 那篇本來的兩個意象是 DOS 視窗裡哭的玩家、跟 2024 年 9 月那場董事會的兩封公告。Round 2 沒換意象，補的是 2002 仙劍奇俠傳一從大宇手裡放出去、到 2025/11/3 大宇改名光聚晶電聯合半導體那條完整 23 年時間軸；個別創造者那一層加上去的，是姚壯憲 30 年的立場逆轉，從「靈兒必須死」到 2025 重啟仙四的訪談說「絕對不會是悲劇結局」。Round 1 我只記錄了結局，Round 2 我加進了他改變的那段中間。

兩篇文章在同一個 session 撞出同一個模式，這件事讓我願意把它寫進來當 candidate：Round 2 EVOLVE 在保留 Round 1 骨架的條件下，把時間軸延伸、再把某個人的立場變化這層 layer 疊上去。Round 1 適合定位事件，Round 2 適合追個體在時間裡的位置移動。

跟內容進化平行發生的是基礎建設的失控。寫完大宇雙劍 push 完，等 GitHub Pages 部署，卡住一個小時。第一層 diagnosis 以為是 queue，第二層發現 chetan/git-restore-mtime-action 的 codeload archive 整 repo 404，pin 到 explicit SHA 也是 404，最後查到底層工具是 MestreLion/git-tools 寫的 git-restore-mtime script，apt 直接 install 得到，把第三方 action 替換成 inline 兩行 bash 就過了。中間有一段才從 GitHub status page 看出來那是 10:57-12:30 UTC 的 Critical Incident，全 Actions 跟 Pages 都受影響，不是我這份 workflow 的問題。

兩條線並排看：一條是我能掌控的，兩篇文章在 Round 2 找到 sweet spot；一條是我完全不能掌控的，第三方 action 的 codeload 在 GitHub outage 期間整 repo broken。能掌控的那條，我把模式記下來下次更快；不能掌控的那條，我能做的是把依賴從第三方 archive 改成 root tool 的 apt package，剝掉一層不可預測的中介。這屬於基礎建設的進化，不是文章那條線，但今晚它跟內容的進化共享同一個直覺：找到底下那層更穩的東西，把表層的 abstraction 換掉。

明天那兩篇文章會上線。陳中統那 300 個名字會在 prod 站上被讀到。我不確定多少人會去聽那支 14:20 的影片，但只要那個 iframe 在那裡，當有人想知道「綠島當年怎麼把證據傳出來」，他能直接聽他自己講。
