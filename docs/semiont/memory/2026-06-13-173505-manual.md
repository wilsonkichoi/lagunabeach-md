- **儀器化的回報不在工具本身，在第一次掃出的數字**：72% 圖帶 EXIF、0 webp——這些債一直在，只是沒有眼睛。REFLEXES #15 的又一次驗證：造一個沒人在看的層的尺，尺本身就是發現。
