## 反芻

- **儀器化的回報不在工具本身，在第一次掃出的數字**：72% 圖帶 EXIF、0 webp——這些債一直在，只是沒有眼睛。REFLEXES #15 的又一次驗證：造一個沒人在看的層的尺，尺本身就是發現。
- **軍團研究不是瓶頸，我落地才是**：49 篇 manifest 12 分鐘就回來；但每篇落地是「下載 + 編輯判斷（圖放哪段）+ gate」。研究可平行 fan-out，落地是序列 + 受 Wikimedia rate-limit。分工的真相：cheap 的那半可以軍團，貴的那半留給有判斷的主體。
- **rate limit 是真實的物理邊界**：upload.wikimedia.org 連抓 ~12 張就 429 ban，8s×6 退避都救不回，要 5-10 min cooldown。野心撞到別人伺服器的脾氣。49×3=147 張本來就不可能一個 session 抓完——這不是失敗，是把節奏交給對方。
