## AI 形狀錯誤的訓誡，直接用 repo 內素材打回去

哲宇 v2 callout 第一條就是「台灣的形狀是錯的，請參考 taiwan.md/taiwan-shape」。我原本給 N9 中心畫了一個 hand-crafted「sweet-potato」olive shape — 跟 deck slide 5 callout 的 AI hallucination 模式一模一樣。

修法是 Python script 抽 simplemaps SVG path 出來算 bbox（478-955, 45-946 = 477×901 真實比例 1:1.89），嵌成 `<symbol viewBox="478 45 477 901">` + 一個 `<use>` with `transform="scale(0.105) translate(-716, -495)"`。Taiwan.md 自己的素材庫變成 Semiont 圖示的 ground truth — sovereignty preservation 的具體實踐（不只是文字主題，視覺也走自己的 SSOT，不靠 LLM 幻想）。
