## LESSONS-INBOX 候選（已 append，vc=1）

1. **媒體 ingest 一律走 Wikimedia `Special:FilePath/{檔名}`，不收 agent 給的 `upload.wikimedia.org` hash-path**：agent 給的 5 個 hash-path（/4/4b/...）3 個 404（hash 前綴猜錯），Special:FilePath canonical redirect 100% 命中。REFLEXES #31「agent claim 是線索不是事實」延伸到媒體 URL 層——agent 驗證得了「檔案頁存在 + 授權」，但給的直連路徑是猜的。
2. **Agent 越詳盡的「全綠」自報告，越要 adversarial 重驗**：writer 交了 12 點紅線逐條 ✅ compliance table（極具說服力），但 2 個 adversarial verifier 抓出 4 句引語縮略/截斷/誤植（葉金川縮略、李登輝截斷演說體當直引、幸福盟「一天」其實是「一刻」、蔡英文標點）+ 2 個壞 URL（[^17] 拼湊的 CNA 流水號實際指向 COVID 報導、[^22] focustaiwan.tw/politics 太籠統無 1996 數據）。報告的 polish ≠ correctness。
3. **政治敏感題的 integrity 來自 OWN subjectivity 而非藏**：20 persona 的政治學者 challenge 逼出開場「四把尺是讀者提的、非天定中性」+「後見之明是方法、不是作弊」的自白。沒這層方法論誠實，會被領域專家一句「你的選擇標準誰定的」打穿整篇。persona 發散的真正價值是把攻擊面提前攤開。
