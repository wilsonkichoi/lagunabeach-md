# 대만 Game Jam 커뮤니티

## 30초 개요

대만 Game Jam 커뮤니티의 공개 역사는 2012년부터 시작된다. 해당 Faust Li가 일본에서 경험을 가져온 뒤 IGDSHARE와 함께 위안즈 대학교에서 최초의 공개 MIT Game Jam을 개최했다. 이후 커뮤니티는 Global Game Jam 타이베이 장소를 중심으로 점차 확대되며, 학생·개발자·업계를 잇는 비공식적 네트워크로 성장했다. 그중 가장 독특한 실험은 2015년에 창립된 KUSO GAME JAM이다. 이 행사는 좋은 게임을 만들 것을 요구하지 않으며, 오직 "나쁜 취향"에 대한 창작 충직을 정직하게 마주할 것을 요구한다.

**키워드**: Game Jam, IGDSHARE, Global Game Jam, KUSO GAME JAM, 인디 게임, 못난 작품

[Global Game Jam](https://globalgamejam.org/)은 세계 최대 규모의 연례 Game Jam 행사이며, 대만 장소는 2012년 전후로 차례로 설립되었다.

## 5분 심화 이해

### 이상한 질문으로 시작하기

2019년, 대만 게임 개발자 연차 대회 TGDF의 무대에서 스크린에 세 장의 사진이 차례로 떠올랐다: Diane Arbus가 촬영한 쌍둑이 흑백 사진, 1995년 홍콩 자체 제작 게임 HK97의 저해상도 스크린샷, 그리고 김이 모락모락 나는 두부 요리(臭豆腐).

발표자는 게임 강연답지 않은 질문을 던졌다: **나쁜 취향이란 무엇인가? 우리는 왜 그것이 필요한가?**[^1]

이 강연은 대만의 한 커뮤니티 행사인 KUSO GAME JAM을 옹호하기 위한 것이었다. 이 행사는 2015년부터 시작되어, 매년 참가자들에게 게임 디자인 수업에서는 거의 가르치지 않는 한 가지를 말한다: **나쁜 게임을 만들어도 괜찮다.**

### 위안즈 대학교에서 전 대만으로: 뿌리내리는 과정

2012년, 리자이(李家屹, 온라인명 Faust Li, 1979–2014)가 일본의 한 Game Jam에 참가하여 경험을 가져온 뒤, 귀국 후 IGDSHARE와 합동으로 "MIT Game Jam"을 개최했다. 장소는 타오위안의 위안즈 대학교였다. 이것은 대만에 기록된 가장 오래된 공개 Game Jam이다[^2]. MIT는 "Made in Taiwan"의 약자이며, 이름 자체에 현지 선언이 담겨 있다: 대만이 자체적인 게임 문화를 만들어야 한다. 이번 행사 규모는 크지 않았지만, 리자이는 겨우 2년을 더 보냈다. 2014년 그가 세상을 떠났을 때, 이 커뮤니티의 이후 성장을 보지 못했다.

Global Game Jam(GGJ)은 세계 최대 규모의 연례 Game Jam 행사로, 매년 1월 말 전 세계에서 동시에 진행되며, 주제가 통일되어 공개되고, 참가자들은 48시간 안에 무에서 유로 게임 하나를 완성한다. IGDSHARE는 이후 타이베이에 GGJ 제1 장소를 공식적으로 설립했으며, IGDSHARE의 Johnson Lin은 동시에 GGJ 아시아 지역 담당자를 맡아 대만 Game Jam 생태계 초기 가장 중요한 추진자 중 한 명이 되었다[^3].

GGJ 형식은 대만 개발자들에게 평소에는 얻기 어려운 조건을 제공했다: 강제 마감 기한. 게임 하나는 정상적인 개발 과정에서 무기한 "아직 만들고 있다"가 될 수 있지만, Game Jam에서는 48시간이 지나면 어떻게든 결과물을 제출해야 한다. 이 강제 완성의 압박은 많은 사람들이 처음으로 "게임 하나를 완성한" 경험을 하는 돌파구가 되었다. 타이베이 제2
