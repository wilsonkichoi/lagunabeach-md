# Mini Taiwan Pulse — 台湾交通リアルタイム3D可視化 🌐

> 📖 **詳細版記事**：本リソースは市民科技（シビックテック）の詳細研究記事にアップグレードされました。全文は [Mini Taiwan Pulse：データアナリストが台湾の交通の鼓動を呼吸する3D光跡に描き出す方法](/technology/mini-taiwan-pulse)（2026-04-19）をご覧ください。本ページはリソース一覧の索引項目として保持されています。

> **30秒概要：** 台湾の交通運輸のリアルタイム動態を3D光球と光跡に変換するオープンソースプロジェクトです。航空便が空に弧を描き、船舶が海面に尾を残し、列車が軌道を駆け抜けます――23の切り替え可能なレイヤーで、台湾の鼓動を「視覚化」できます。

## 注目すべき理由

多くの人が台湾の地図を見ると、静的な輪郭しか見えません。Mini Taiwan Pulse を使えば、**呼吸する島**を見ることができます。

本プロジェクトの野心的な目標は小さくありません。各政府機関に散らばるオープンデータ――航空便、船舶AIS、台鉄・高鉄の時刻表、地下鉄路線、人口統計、気象観測――をすべて同一の3D地図に統合します。単なる点のマーキングではなく、光球、光跡、彗星の尾のような視覚言語を用いて、データを動く風景に変換します。

> **📝 キュレーターノート**
> 台湾のオープンデータ基盤整備はアジアでトップクラスです（[世界オープンデータ指数](https://index.okfn.org/)で度々トップ10に入選）。しかし、「データの公開」と「データの可視化」の間には大きな溝があります。Mini Taiwan Pulse はこの溝を埋めています。

## 3層の鼓動

### 空 — 航空便光跡 ✈️

台湾全土14の空港、1,500便以上の航空便のリアルタイム動態をカバーしています。各航空機は発光する球体で表され、後ろには彗星の尾のようなグラデーション光跡が引かれます。高度の誇張倍率は調整可能（1x～5x）で、低空路線と高空路線の差を一目で把握できます。

データソース：FlightRadar24 API。

### 海 — 船舶追跡 🚢

台湾周辺海域の船舶位置は青藍色の光球で標示され、各船舶は30分間の尾の軌跡を残します。システムはGPSの異常ジャンプや無効なMMSIを自動的にフィルタリングし、表示されるすべての光点が実際の船舶であることを保証します。

データソース：AIS（自動識別システム）船舶位置データ。

### 陸 — 6大軌道システム 🚄

これが最も驚くべき部分かもしれません。6つの軌道システムが同期して運行されています：

| システム                 | 規模                                |
| ------------------------ | ----------------------------------- |
| 台鉄（台湾鉄路、TRA）    | 265路線、333列車、車種ごとに6色分類 |
| 高鉄（台湾新幹線、THSR） | 南北主線 + 支線                     |
| 台北捷運（MRT、TRTC）    | 8路線                               |
| 高雄捷運（MRT、KRTC）    | 紅線 + 橘線                         |
| 高雄軽軌（LRT、KLRT）    | 環状軽軌                            |
| 台中捷運（MRT、TMRT）    | 緑線 + 青線                         |

台鉄の処理は特に複雑です――OD軌道マッチング、彰化三角線のような分岐路線には、専用のエンジンが使用されています。

データソース：公開時刻表 + [OpenStreetMap](https://www.openstreetmap.org/) 軌道データ。
