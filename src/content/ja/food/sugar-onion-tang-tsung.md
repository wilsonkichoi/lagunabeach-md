# 糖蔥：白い気孔に隠された「抗争の糖」

> **30秒概要：** 糖蔥には葱は入っていません。それはかつて台湾の農民が植民地政府に対抗するための「偽装武器」でした。日治時代、台湾で生産された蔗糖は日本への外销が厳しく管制されており、現地の人々が糖を口にするには警察の検査を逃れなければなりませんでした。農民は糖液が冷える際の延展性を利用し、繰り返し引き伸ばして中空の管状構造を作り出しました。外見は葱白に酷似しており、菜籠に紛れ込ませれば検査を欺くことができました[^5][^6]。この「16大孔、16小孔」という精密な構造により、口に入れるとすぐに溶け、サクサクとして歯に付着しないため、台湾の歴史における最も草の根知恵が詰まった甘味となりました[^14]。

2024年の宜蘭伝統芸術センターでは、観光客が「糖蔥文化館」の前に集まっていました。濃厚なキャラメルの甘い香りと麦芽の清々しい香りが漂う中、ベテランの職人・卓創慶氏は重さ3キロ、温度120度の褐色の糖膏を木の杭に掛け、両手で素早く引き伸ばし、折りたたみ、再び広げていました[^12][^20]。

空気が糖体に強引に押し込まれるにつれ、現場では規則正しい「パタッ」という音が響きました。褐色の琥珀色の透明液がわずか10分で奇跡的に真珠光沢のある象牙色に変わり、最後に絹のような質感の葱管状に定型しました[^20]。この百年続く「甘味マラソン」は、1回1回の引き伸ばしが歴史の重みを帯びています。

## 潮汕から台湾へ：甘味の越境移動

糖蔥の歴史は清朝時代の広東・潮州、汕頭一帯まで遡ります[^2][^4]。明朝の潮州知府・郭子章は『潮中雑紀』の中で「潮の葱糖は、極めて白く極めてサクサクしており、滓が全くない[^15]」と記しています。閩粤移民の飲食文化の一環として、糖蔥は移民の波に乗って香港、福建へ伝わり、最後に台湾に根付きました[^2]。

古い潮汕の風習では、糖蔥は教育の役割も果たしていました。学童が入学初日に糖蔥薄餅を使って孔子像を祭る際、その「中空」の構造は「聡慧通竅」を意味し、心が通りが良く賢く学べることを象徴していました[^16][^17]。民間では入学初日から数日間の様子をユーモラスに歌った民謡も伝わっています。「初日は糖葱が甘く、二日目は竹篦が続き（先生に叩かれる）、三日目は町に逃げ込み、四日目は見つからない[^16]。」

## 禁止された甘味：農民の「偽装術」

台湾の糖業は日治時代に黄金期を迎えましたが、その甘味は台湾人のものではありませんでした。当時の日本政府は甘蔗保護政策を実施し、全ての白糖は優先的に日本へ輸出され、現地人が勝手に糖を作ったり隠し持ったりするのは違法と見なされていました[^5]。

一口の甘味を守るため、台湾の農民は極限の創意を発揮しました。彼らは煮詰めた後の糖液を繰り返し引き伸ばすと、無数の細かい気孔が生じ、外見が白く中空になり、切り分けると葱白そのもののようになることに気づきました[^5][^6]。農民はこれらの「偽装葱」を本物の青葱の籠に紛れ込ませて運び、日本の警察が巡回しても満籠の野菜しか見えず、貴重な糖分が隠されているとは知りませんでした[^5]。この「抗争の糖」は台湾の民間で広く普及し、「白糖葱」や「富貴糖」という別名が付けられました[^1][^5]。

## 16+16の精密工芸：物理と化学の交響

糖蔥の魅力は甘さではなく、「折るとすぐに折れ、口に入れるとすぐに溶ける」という独特の食感にあります。この物理的特性は、極めて複雑な内部構造に由来しています。

糖蔥を作るには、白砂糖、水、麦芽糖を特定の割合で混ぜ、正確に120°Cから130°Cまで加熱する必要があります[^3]。糖膏を木の杭に掛けて繰り返し引き伸ばすと、内部の糖分子鎖が強制的に再配列され、数万個の微小な気泡が包み込まれます[^3]。

| 製作段階       | 物理・化学変化                                     | 重要な目的                                               |
| :------------- | :------------------------------------------------- | :------------------------------------------------------- |
| **熬糖**       | 蔗糖分子が120-130°Cで一部転化                      | 糖液に十分な延展性と硬度を確保する[^3]                   |
| **冷却**       | 糖膏が「ガラス状態」と「ゴム状態」の臨界点に入る   | 糖体が半固体の黄金引き伸ばし期を捉える[^3]               |
| **拉糖**       | 空気を包み込み、糖体が光散乱原理により雪白に変わる | 多孔質構造を形成し、体積とサクサク感を増加させる[^1][^3] |
| **折りたたみ** | 16回折りたたんで蜂巢構造を形成                     | 16大孔と16小孔の精密な支えを構築する[^14]                |

合格した糖蔥の横断面には、「16の大孔があり、各々の大孔の周りに16の小孔が環状に並ぶ」という構造が備わっている必要があります[^14]。この精密な構造により、糖蔥は力が加わると極めて容易に折れ、噛み砕いた瞬間に気圧が解放されて生じる清脆な音は、どの機械で生産されたキャンディーも模倣できません。

## 糖芸の横断的比較：糖蔥と他の伝統糖菓

台湾の廟会や老街では、様々な伝統糖芸を目にすることができます。材料は似ていても、工芸と食感は全く異なります：

| 技芸名称   | 核心工程                            | 食感特徴                                             | 物理状態                                 |
| :--------- | :---------------------------------- | :--------------------------------------------------- | :--------------------------------------- |
| **糖蔥**   | 高温熬糖 + 16回折りたたみ引き伸ばし | サクサクとして歯に付着せず、口に入れるとすぐに溶ける | 硬くパリッとした中空管状[^18][^19]       |
| **龍鬚糖** | 熟糖膠 + 熟粉 + 万回延展            | 柔らかく、蚕糸のように繊細                           | 極細の糸状、中に餡が入っている[^18][^19] |
| **畫糖**   | 熬糖 + 銅板に流し込んで描画         | 硬くパリッとしており、ラリパップに類似               | 平面図案状[^19]                          |
| **吹糖**   | 糖塊をちぎり + 吹き込んで管で成形   | 薄くパリッとして中空                                 | 立体造形（例：干支）[^19]                |

## 老味道を守る職人たち

食品工業の発展に伴い、力が要り気候の影響を受けやすい手作りの糖蔥引き伸ばしは絶滅の危機にあります。現在台湾では少数の職人だけがこの技を堅持しています：

- **宜蘭五結**：卓創慶師傅、三代続く伝承者で、伝統芸術センターで毎日「甘味マラソン」を繰り広げています[^12]。
- **台南麻豆**：陳恆雄師傅、素手で40年以上糖を引き伸ばし続け、体力と温度の闘いだと強調しています[^11]。
- **三峽老街**：卓師傅（同系伝承）、古色古香な街並みの中でこの老味道を守っています[^7]。
- **淡水と嘉義**：独学や家伝の職人もおり、湿気と温度の狭間で純手作り製法を堅持しています[^9][^13]。

## 結語：甘味に込められた靭性

糖蔥の物語は、本質的に「靭性」に関する物語です。圧迫の中で生まれ、欠乏の中で成長し、最後に精緻な飲食の波の中で、機械では代替できない引き伸ばしの技だけを頼りに生き残りました。糖蔥を一口噛み、その清脆な折れる音を聞いたとき、その甘さの中には百年前の台湾農民が「甘味抗争」の中で屈しなかった草の根知恵が込められています。
