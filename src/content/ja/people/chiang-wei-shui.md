> **30秒概覽：**
> 蔣渭水（チャン・ウェイスイ）は1920年代台湾の社会運動の精神的支柱だった。医師として著名な〈臨床講義〉を著し、台湾社会が「知識的栄養不足症」を患っていると診断した。台湾文化協会（zh only — 臺灣文化協會）と初の合法政党・台湾民衆党（zh only — 臺灣民眾黨）を創立しただけでなく、大安医院（zh only — 大安醫院）を革命の診察室に変えた。彼の生涯は監獄、病院、料理屋の間を奔走し、最終的に40歳で夭折した。「同胞は団結しなければならない、団結こそが力だ（同胞須團結，團結真有力）」という遺言を残し、台湾の民主精神の重要な礎となっている。

1914年7月、台北総督府医学校の実験室で、数人の学生が致死性のコレラ菌を密かに培養していた。それは学術研究のためではなく、海を越えた暗殺計画だった。4年生の杜聰明（zh only — 杜聰明）が菌の培養を担当し、その背後の計画立案者と資金調達者は、後に「台湾の孫中山（zh only — 台灣孫中山）」と称えられる蔣渭水だった。彼らは病原菌を北京の水源に投じて、皇帝になろうとしていた袁世凱を毒殺する計画を立てた。最終的に実行には至らなかったこの「細菌暗殺」が、この医者のその後20年にわたる「国を医す」生涯の序章となった。[^1][^2]

### 医学校時代の反骨の遺伝子

蔣渭水は1891年に宜蘭（zh only — 宜蘭）に生まれ、幼少期に霊媒（乩童）を務めた経験があり、それが民間の苦しみと迷信への深い観察力を養った。1910年に優秀な成績で台湾総督府医学校に合格した。在校中は優秀な学生であると同時に、学生運動のリーダーでもあった。当時の医学校校長・高木友枝（zh only — 高木友枝）は厳格な教育方針で知られていたが、蔣渭水は何度も学生ストを主導して学校の不公正な扱いに抗議し、石延漢（zh only — 石延漢、後の台大医学院教授）らの支持を得て、民族意識を持つ抵抗運動を組織した。[^3][^4]

1915年に卒業した後、蔣渭水は宜蘭病院でインターンシップを1年間行い、その後すぐに台北大稲埕（zh only — 大稻埕）に大安医院を開設した。しかし彼の心に宿っていたのは、診察室の患者だけではなく、植民地支配の下で精神的に疲弊した台湾全島だった。

### 大安医院：社会運動の診察室

大安医院（現在の台北市延平北路の義美食品旧址付近）はすぐに医療機関の機能を超えた存在となった。医院の上階は台湾文化協会の準備委員会の事務所であり、《台湾民報》（zh only — 臺灣民報）の総配布所でもあった。蔣渭水は近くに「春風得意楼（zh only — 春風得意樓）」という料理屋も経営し、あらゆる人物と交流を深めた。「商売と仕事に忙殺されているが、血の中に宿る政治的な使命感が、大衆を啓蒙することこそが現状を変える唯一の道だと気づかせる」とため息をついたことがある。[^5][^6]

**📝 キュレーターノート：** 蔣渭水の最も魅力的な点は、その「複数の顔」にある。料理屋で紳士と交際し、病院で貧民を診察し、新聞社で論文を書いて批判した。運動を推進するにはスローガンだけでは足りず、実体的な拠点と安定した資金が必要だと深く理解しており、大安医院はあの時代最強の「地下司令部」だった。

### 〈臨床講義〉：台湾に対する最終診断

1921年11月30日、蔣渭水は《台湾文化協会会報》に全島を震わせた〈臨床講義（zh only — 臨床講義）〉を発表した。彼は医師の診断書の形式を模倣して台湾を擬人化した。

- **患者：** 台湾島
- **原籍：** 中華民国福建省台湾道
- **現住所：** 日本帝国台湾総督府
- **病名：** 世界文化の知的発達遅滞児
- **原因：** 知識的栄養不足

診断書の中で、台湾は外見は優れているが旧来の毒害を受けており、精神的生活が貧困だと率直に述べた。彼が出した処方箋には、正規学校教育（最大量）、補習教育（最大量）、幼稚園と読書会が含まれていた。この講義は文学的な比喩であると同時に、台湾文化協会のその後10年間の行動の設計図でもあった。[^7][^8]

### 文化啓蒙から政党政治へ

1921年10月17日、台湾文化協会が静修女子学校（zh only — 靜修女子學校）で成立した。蔣渭水は専務理事として、全島規模の講演会、夏期学校、芸術教育活動を推進した。1923年、「台湾議会設置請願運動（zh only — 臺灣議會設置請願運動）」への参加が「治安警察事件（治警事件、zh only — 治警事件）」を引き起こし、蔡培火（zh only — 蔡培火）らとともに投獄された。獄中で彼は《獄中記（zh only — 獄中記）》を書き、たとえ牢獄の中でも志を変えない楽観的な精神を示した。[^9]

1927年、文化協会は路線の相違により分裂した。蔣渭水は「台湾民衆党（zh only — 臺灣民眾黨）」の設立を推進した——台湾史上初の現代的な合法政党だ。彼は「民本政治の確立、合理的な経済組織の建設、不合理な社会制度の改除」という三大綱領を掲げた。さらに積極的に労働者運動に関与し、「台湾工友総連盟（zh only — 臺灣工友總聯盟）」を設立して、運動の重心を士紳階級から基層の民衆へと転換しようとした。[^10][^11]

### 遺産：同胞は団結しなければならない、団結こそが力だ

蔣渭水の生涯は逮捕・拘束が10度以上、投獄日数の合計は144日に及んだ。長年の奔走と牢獄の苦しみが彼の体力を使い果たした。1931年8月5日、腸チフスにより台北病院で逝去した。享年わずか40歳だった。臨終の前に彼はこう言い残した。「同胞は団結しなければならない、団結こそが力だ（同胞須團結，團結真有力）。」彼の葬儀には5千人以上が参列し、「大衆葬」と称えられ、日本統治時代の台湾で最大規模の政治的な集会となった。[^12][^13]

**📝 キュレーターノート：** 蔣渭水は「台湾の孫中山」と呼ばれるが、孫中山との最大の違いは、彼が常に法的な制度の中での「非暴力」抗争にこだわったことだ。医師の冷静さで社会を診断し、革命家の情熱で理想を実践した。あの暗黒の植民地時代に彼が出した「知識の処方箋」は、今も期限切れになっていない。

彼の物語を読み終えて、私たちは自問してみてもいいだろう——情報爆発でありながら断片化した現代において、台湾はまだ何らかの形の「知識的栄養不足」を患っていないだろうか。蔣渭水の〈臨床講義〉は歴史的な文書であるだけでなく、自己覚醒を求めるそれぞれの世代の台湾人の葛藤と渇望を映す鏡でもある。
