> **30秒概覽：** 蘇貞昌（スー・ジョンチャン）の政治生涯は、40年以上にわたる「突進」の連続だ。1979年の美麗島大裁判（zh only — 美麗島大審）の弁護士を出発点として、屏東縣長、台北縣長、そして2度の行政院長を歴任した。「電火球」のような強引なスタイルで知られ、アフリカ豚コレラの水際対策や「すべての教室にエアコンを（班班有冷氣、zh only — 班班有冷氣）」政策など卓越した実行力を示す一方、その覇気あるスタイルは二極化した評価を招いた。この「衝衝衝」の閣僚がどのように政治的低谷から逆襲し、論争の中で深い施政の足跡を残したかをたどっていく。

## 起源：戒厳令の影の下の法律家と民主への初心

1979年12月、台湾で美麗島事件（zh only — 美麗島事件）が勃発し、権威主義政府が大規模な逮捕を始めた。息をするだけでも重圧を感じるあの時代、開業してまもない弁護士の蘇貞昌は、姚嘉文（zh only — 姚嘉文）の妻、周清玉（zh only — 周清玉）の依頼を受け入れ、弁護士団に加わった[^1][^2]。

当時の法廷の雰囲気は張り詰め、蘇貞昌は姚嘉文や邱茂男（zh only — 邱茂男）らのために弁護した。それは単なる法廷での攻防ではなく、政治的な覚醒の場でもあった。この経験が彼を純粋な法律家から民主運動へと転じさせた。1981年、彼は故郷の屏東（zh only — 屏東）に戻り省議員選挙に立候補し、正式に政界への一歩を踏み出した。

📝 **キュレーターノート：** 蘇貞昌の政治の出発点は権力ではなく「リスク」だった。戒厳令下で政治犯の弁護を選んだ——この「不可能と知りながらやり遂げる」という性格が、後の強引なリーダーシップの心理的な基盤を築いた。

## 転換点：屏東の敗北から台北縣の「天地を揺るがす土下座」へ

蘇貞昌の政治の道は平坦ではなかった。1989年に屏東縣長に当選し、「百里の街路樹植樹（種樹百里）」計画を推進し、屏鵝公路（zh only — 屏鵝公路）を緑のコリドーに変えた[^3]。しかし1993年の再選挑戦では、対立候補の伍澤元（zh only — 伍澤元）と地元の有力者、鄭太吉（zh only — 鄭太吉）の誹謗攻撃に遭い、わずかな差で敗れた[^4][^5]。この落選は彼を一時的に権力の中心から遠ざけたが、同時に彼の粘り強さを鍛えた。

1997年、蘇貞昌は台北縣長選挙に打って出た。選挙前夜、民進党の選挙情勢が厳しくなる中、末期がんで体力の限界にあった立法委員の盧修一（zh only — 盧修一）が突然、決起集会の会場に姿を現した。万人の注目の中、盧修一はゆっくりと膝まずき、最後の力を振り絞って選挙人に蘇貞昌への支持を懇願した[^6][^7]。この土下座は台湾全土に衝撃を与え、同情票を引き出すことに成功し、蘇貞昌は3万票に満たない差でかろうじて勝利し、台北縣（現在の新北市）での黄金の8年間が幕を開けた。

| 時期   | 重要な出来事        | 影響と意義                                     |
| :----- | :------------------ | :--------------------------------------------- |
| 1979年 | 美麗島大裁判の弁護  | 政治的覚醒、民主的立場の確立                   |
| 1993年 | 屏東縣長の再選失敗  | 誹謗中傷を受け、政治生涯の低谷                 |
| 1997年 | 台北縣長当選        | 盧修一の「天地を揺るかす土下座」で情勢逆転     |
| 2019年 | 2度目の行政院長就任 | アフリカ豚コレラの水際対策に成功、実行力を証明 |

## 施政：「衝衝衝」で都市と国家を改造する

台北縣長任期中、蘇貞昌は「衝衝衝」のスローガンを具体的な実績に変えた。最も有名な決断の一つは、荒廃し汚れていた「二重疏洪道（zh only — 二重疏洪道）」を緑豊かな親水公園に改造したこと、そして「鶯歌陶磁博物館（zh only — 鶯歌陶瓷博物館）」と「鶯歌老街（zh only — 鶯歌老街）」の再生を推進し、衰退した工業地区を文化観光の重要拠点に転換したことだ[^8][^9]。さらに、淡水漁人埠頭（zh only — 淡水漁人碼頭）と八里左岸（zh only — 八里左岸）の改造も彼の任期中に基盤が築かれた[^10]。

2019年、蘇貞昌は2度目の行政院長に就任した。就任初日、彼はすぐに桃園空港に駆けつけ、高リスク地域からの入国者の荷物を100%検査するよう命じ、アフリカ豚コレラを国門の外で阻止することに成功した[^11][^12]。続いて、「すべての教室にエアコンを（班班有冷氣）」政策を推進し、1年半以内に全国の小中学校へのエアコン設置を完了させるよう求め、驚異的な行政効率を発揮した[^13][^14]。

📝 **キュレーターノート：** 蘇貞昌の施政ロジックは「問題を解決する」ことだ。途中の摩擦を気にせず、結果が達成されるかだけを気にする。この「結果志向」のスタイルが彼を効率の代名詞にする一方、「横柄」というレッテルも貼られることとなった。

## 課題と論争：強引なリーダーシップがもたらした二極化した評価

蘇貞昌の強引なスタイルは諸刃の剣だ。行政院長任期中、野党立法委員との激しい対立、部下への厳しい要求、政策推進における覇気は、しばしば「苛酷な官吏」や「傲慢すぎる」との批判を招いた[^15]。2020年以降の各種世論調査では、彼は「最も嫌われる政治家」リストの上位に頻繁に登場する一方、安定した支持率も持ち続けた[^16]。

また、屏鵝公路での「百里の街路樹植樹2.0（種樹百里2.0）」計画では、施工期間中の交通渋滞と高額な予算が、多くの民衆の不満と論争を招いた[^17]。

## 展望：「電火球」が残した政治的遺産

蘇貞昌は2023年に行政院長を退任し、4年にわたる任期を終えた。台湾の民選総統以来、最も長く在任した行政院長の一人となった。彼の政治的遺産は具体的な建設だけでなく、「大胆に決断し、果敢に実行する」という政治スタイルにある。

美麗島の弁護士から台湾政界の「電火球」へ。蘇貞昌の物語は、台湾の民主化過程における激しい衝突と絶えない進歩を映し出している。評価はどうあれ、彼の「衝衝衝」というスローガンは、台湾人の集団記憶に深く刻まれている。
