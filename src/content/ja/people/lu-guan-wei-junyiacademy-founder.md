# 呂冠緯：台湾のすべての子どもが自分のペースで学べるようにする

**30秒でわかる概要**  
台大医学部のエリートであり、医師免許も取得した呂冠緯が、2013年に白衣を着ることをやめ、無料の数学動画の制作を始めました。12年後、彼は「均一教育プラットフォーム」を台湾最大の非営利オンライン学習システムへと成長させ、累計528万人の登録ユーザー、月間27万人のアクティブ学習者を達成し、AIによって僻地の子どもたちと都市の子どもたちが同じスタートラインに立てるようにしました。[^1]  
これは慈善ではなく、彼が「教育」を台湾に最も必要なデジタルインフラと位置づけた結果です。

## 医師から教育者へ

2013年初頭、呂冠緯は台大医学部を卒業し、医師免許も取得していました。  
多くの人にとって、それはすでに人生の成功パスのように見えました。  
しかし彼は別の道を選びました——誠致教育財団に加入し、「均一教育プラットフォーム」の最初の数学動画の制作を始めたのです。

彼はこう言いました。「医師は一度に一人の患者しか救えませんが、教育は数百万人の家族の未来を変えることができる。」[^2]  
この決断は、後に多くの人々から「台湾教育界のエントロピーに対する逆転行動」と呼ばれるようになりました。

📝 **キュレーターメモ**：医学部出身の彼が「エビデンスに基づくアプローチ」を診察室から教室へ持ち込んだことは、単なるキャリアチェンジではなく、台湾の都市と僻地の教育格差に対する最も直接的な応答でした。

## 均一教育プラットフォーム

「均一」は「均等で、一流」という意味を込めて名付けられました。  
プラットフォームは小学校から高校まで、数学、理科、国語などの完全なカリキュラムを提供し、サイト全体が永続的に無料です。  
2025年末までの公式データによると：[^3]

| 項目                   | データ        |
| ---------------------- | ------------- |
| 登録ユーザー           | 528万人以上   |
| 月間アクティブユーザー | 27万人以上    |
| 動画数                 | 41,000本以上  |
| 練習問題数             | 108,000問以上 |
| 僻地学校のサービス割合 | 約25%         |
| 学習支援を受けた子ども | 14,500人以上  |

## 適性学習の理念

呂冠緯の最も核心的な信念は「適性学習」です：教育は子どもに合わせるべきであり、子どもが統一された進度に合わせるべきではない、という考え方です。

プラットフォームはAIを用いて学習履歴を分析し、Jutor（AI英語家庭教師）、10分診断システム、リアルタイム質問ノートなどのツールを開発しています。  
遅れを取り戻したい子どもは基礎に戻って復習でき、先に進みたい子どもはさらに先を探究できます。  
これは従来の教室ではほぼ不可能でしたが、均一プラットフォーム上では、すでに毎日27万人の日常となっています。[^4]

📝 **キュレーターメモ**：パンデミック後、台湾ではM字型の学習格差が生まれ、都市の子どもは加速し、僻地の子どもはさらに後れを取るようになりました。均一はAIによって「統一進度」の教室を「個別化されたトラック」へと変え、これこそがテクノロジーが本当に埋めることができる構造的なギャップです。

## 社会企業の実践と課題

均一は非営利組織であり、すべてのコンテンツが完全に無料で、資金はほぼすべて民間の寄付によって賄われています。  
呂冠緯は教育を「インフラ」と位置づけました——道路や水道のように、すべての子どもが無料で利用できる公共財であるべきだ、という考え方です。

しかし彼は課題を隠すこともしません：教育界のデジタルツールに対する懐疑、僻地の教員の高い離職率、そしてプラットフォームの独立性を維持するための継続的な資金調達。  
彼が歩んできた道は、最初から最も簡単な道ではありませんでしたが、12年の時間をかけて証明しました——台湾において、社会企業は単なるスローガンではなく、本当に数百万人の生活を変える力を持っているのだと。[^5]

📝 **キュレーターメモ**：多くの人々がまだ「教育改革」について議論しているとき、呂冠緯はすでに改革を毎日27万人が使う日常にしていました。これこそが台湾に最も必要な「切迫した忍耐」です。
