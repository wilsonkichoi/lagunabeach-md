> **30秒概要：** 台湾はプレートの境界に位置しており、地震はこの島の宿命です。本記事は、台湾独自に生まれた「地牛翻身」の伝説に始まり、清代の異象記録から921大地震の制度的転換までを辿り、台湾人が恐怖から「泰然自若」へと至る過程、そして揺れの中でいかに世界レベルの防災レジリエンスを築き上げたかを考察します。

えっ、知っていますか？台湾では、地震が起きたときの反応という、とても特別な「国民的運動」があるんです。

外国人が初めて台湾で地震に遭遇すると、たいてい顔面蒼白になり、人生を疑い始めます [^12]。でも私たち台湾人はどうするでしょうか。まず頭上の揺れる照明を見上げて、心の中で静かに評価します。「まあ、震度3くらいかな。」そして顔を上げずに麺を食べ続けるか、あるいはすぐにSNSを開いて投稿します。「地震！」 [^3] これは私たちが死を恐れていないわけではありません。ただ慣れすぎているのです。この「泰然自若」は、長年にわたる震災との共存によって鍛えられた筋肉の記憶であり、この島が私たちに授けた最初のサバイバルレッスンなのです [^6]。

## なぜ「地牛翻身」なのか？

台湾では、地震を「地牛翻身（ちぎゅうしんしん）」と呼ぶのが一般的です。この表現は実に興味深く、台湾独自に生まれた地動の伝説なのです [^4]。

世界各地には地震を説明する神話が数多くあります。日本では地下に大きなナマズがいると言われ、中国では鰲魚（おうぎょ）が身を起こすと伝えられています。しかし台湾では「牛」なのです [^4]。これは、先住民族の地下の動物に対する想像と、漢民族の移民がもたらした「地生毛」という異象が融合したものです [^2] [^8]。清代の文献である『重修臺灣縣志』には、地震の前後に「地生毛」という怪異な現象が記録されており、人々は地表に数センチの黒い毛や白い毛が生えるのを見て、地下に巨大な生物が活動しているのだと連想し、やがてこれらの毛は「牛毛」として想像され、「地牛」のイメージが定着していきました [^4] [^19]。

| 地震伝説の類型         | 代表的な文化・集団                 | 核心的な説明                                     |
| :--------------------- | :--------------------------------- | :----------------------------------------------- |
| **地牛翻身**           | 台湾漢人、ツォウ族、サイシャット族 | 地下の牛が体を揺らして震動を引き起こす [^4] [^7] |
| **大鹿の耳**           | タイヤル族                         | 海底の大鹿の耳が海面に出ると地震が起きる [^2]    |
| **地下人が石柱を押す** | アミ族                             | 地下の人が大地を支える石柱を押す [^4]            |
| **片手片足の人**       | プユマ族                           | 天地を支える人が動くと地震が起きる [^2]          |

牛は台湾人の文化の中で、非常に特別な地位を持っています。大きく、強く、それでいて素朴で実直です。伝統社会において、牛は農民の最も親密なパートナーであり、私たちは勤勉で忍耐強い精神を表すために自ら「台湾牛」と称することもあります [^4]。地震を地下で素朴な牛が寝返りを打ったことに例えることは、実は強い諦念と包容力を含んでいます。畢竟、牛は私たちの生命と密接に結びついており、無思慮ではあるものの、故意ではないことを知っているからです [^4]。

## 数百年にわたる揺れの記憶

台湾の地震史は、実は台湾人の生存史そのものです。17世紀には、すでにオランダ人の記録に台湾の地震の姿が現れています [^20]。

清代、地震はしばしば「天の戒め」と見なされ、政府は廟の再建を通じて民心を安定させました [^2]。日本統治時代に入ると、台湾は科学的観測の時代を迎え、1897年に台北測候所に初の地震観測機が設置され、100年にわたる精密な記録が始まりました [^20]。

| 歴史上の重大地震     | 時期   | 影響と意義                                                                             |
| :------------------- | :----- | :------------------------------------------------------------------------------------- |
| **嘉義大地震**       | 1906年 | 1,258名の死者を出し、日本統治政府に耐震建築の重視を促した [^20]                        |
| **新竹・台中大地震** | 1935年 | 台湾史上最も犠牲者の多い地震で、3,276名が死亡した [^20]                                |
| **921集集大地震**    | 1999年 | 戦後最悪の災害であり、台湾の防災と建築法規を根本から変えた [^13] [^16]                 |
| **0403花蓮強震**     | 2024年 | 25年ぶりの最大規模の地震であり、台湾の現代化された防災レジリエンスを試した [^14] [^15] |

1999年の **921大地震** は、私たちの世代の台湾人にとって共通の傷です。あの夜、102秒間にわたる揺れが2,415の命を奪いました [^13]。しかし、まさにその災害をきっかけに、私たちは世界レベルの災害医療体制の構築や、より厳格な建築耐震基準の確立を学びました [^16]。

> 「あの夜、102秒間にわたる揺れ、何百回もの余震、そしてその後に続く建物の崩壊、死、恐怖、悲しみ——それは台湾の戦後最大の自然災害であり、台湾人共通の記憶である。」 —— _BBC中国語ネット『921大地震20周年』_ [^12]

## 震災と共存する知恵

台湾はユーラシアプレートとフィリピン海プレートの境界に位置しており、地震はこの島の宿命です [^9]。私たちは地牛が身を起こすのを止めることはできませんが、彼と踊る方法を学びました。

小学校から、私たちには標準的な防災訓練があります。**「伏せて、守って、動かない」** [^11]。これはスローガンではなく、命を守るための反射行動です。台湾の国家警報システムは、地震波が到達する数秒前に、台湾全土のスマートフォンに同時に警報音を鳴らします。

私たちの地震に対する「泰然自若」は、実はこれらの準備の上に成り立っています。いつ逃げるべきで、いつ麺を食べ続けてよいか、私たちは知っているのです。これが台湾です。揺れの中で強さを学び、災害の中で助け合いを学んだ島です。次に地牛が再び身を起こすとき、私たちは心の中で知っています。この土地とその上にいる人々は、私たちが想像するよりもずっと強いのだと。
