# 勝利路の両面：装甲兵クーデターから「大駐車場」への湖口の記憶

1964年1月21日、湖口装甲兵第一師の副司令・趙志華が訓練場でピストルを取り出し「北上勤王」と叫んだとき、その背後にある勝利路はまだ静かな田與の小道でした。わずか2時間で未遂に終わったこのクーデターにより、湖口は台北の上層部の目には「反乱」の代名詞となりました。しかし、塀の向こう側に住む湖口の人々にとっては、この巨大な施設はまったく異なる姿をしていました。

物質的にまだ豊かではなかった当時、湖口営区は単なる禁域ではなく、地域社会の「福利センター」でもありました。

当時の営区内には映画館、プール、さらには住民が生活物資を買いに通う購買会がありました。そこで育った子どもたちにとって、夏の記憶はプールの湯気と、外よりも安い軍用品を扱う購買会のことでした。軍と民の境界は当時一枚岩ではなく、共生する近隣関係でした。

> 📝 キュレーター注記：軍事施設が日常の消費の場になると、塀は政治的境界から近隣の生活インフラの延長へと変わります。

しかし、この「隣人」は激しい衝撃ももたらしました。

勝利路の一側面では、営区の塀と民家が間近に隣り合っていました。駐車スペースが不足していたため、地元の人々は塀側を「勝利路大駐車場」と呼んでいました。この景観が最も衝撃的な出来事を経験したのは2007年5月のことです。シンガポール軍「星光部隊」所属のF-5F戦闘機が任務中に湖口営区に墜落し、大きな爆発音と立ち昇る黒煙は、今日まで勝利路の住民たちが語り継ぐ衝撃的な記憶として残っています。

湖口営区が台湾社会に与えた衝撃は、2013年に頂点に達しました。

湖口装甲542旅団に所属する下士・洪仲丘が、退役直前の不当な管教により死亡しました。この悲劇は25万人の「白衫軍（白衣のデモ隊）」を凱ダグラン大通りに集結させ、《軍事審判法》の大幅改正をもたらしました。湖口の人々にとって、それは塀の内側で最も重い影であり、勝利路という名称が全国ニュースの中で皮肉な響きを持つものとなりました。

> 📝 キュレーター注記：歴史は湖口に二つの痕跡を残しました。一つは1964年の権力への挑戦、もう一つは2013年の人権への覚醒です。

今日、勝利路を歩くと、映画館とプールはすでに歴史の塵の中に消えています。2000年代初頭、陳総統が湖口を親臨して国防展示行事を主宰した際、列をなして走る戦車と旋回するヘリコプターを見学した高校生たちは大いに興奮しました。

現在もここは台湾装甲兵の心臓部ですが、よりどころ静かな巨獣のように、勝利路の民家と隣り合って暮らしています。「反乱の発祥地」から「地域の駐車場」へと、湖口営区の塀は台湾が戒厳令、発展、そして民主化への転換を経てきたすべての矛盾と温かさを記録しています。
