# 台湾原住民族の歴史と正名運動

1987年5月15日、台北の刑場。法医が19歳の若者のもとへ歩み寄り、麻酔薬を打とうとした。その若者は首を横に振った。「自ら罪を犯したのだから、この痛みを覚醒したまま受けなければならない」と言った。

彼の名は湯英伸（タン・インシェン）。ツォウ族の出身で、阿里山（アルイシャン）の特富野（トフシャ）部落の出身だった。一年前、彼は新聞のレストラン求人広告を見て、学生のような顔つきのまま山から台北へ出て行き、ある洗濯店に騙り込まれた。紹介料3,500元、雇用主は彼の身分証明書を没収し、深夜まで働かせ続けた。1986年1月25日、勤務9日目、彼は酒を飲み、争いの末に雇用主一家3人を殺害した。2歳の女の子も含まれる。その後、彼は派出所に出頭した。

その銃弾が殺したのは、犯罪を犯した若者だけではなかった。

## 四百年の名、四百年の鎖

この運動を理解するには、「一つの名が一つの鎖である」ということを先に理解しなければならない。

1624年、オランダ東インド会社が台湾に到着し、島の住民を「帰化蕃」と「野蕃」に分類した。教化し、従わせ、組み込む——これが植民地支配の三段階の論理である。清朝が統治を引き継いだ後も分類は続き、「熟番」と「生番」の基準は辮髪を切り、漢姓に改め、税を納めることだった——漢人に似れば似るほど「文明化」されているとされた。日本植民地時代（1895〜1945）にはさらに体系化され、「蕃人」と呼ばれ、「理蕃」警察が駐在し、集団移住が推進された。部落は故郷の地から引き離され、移住させられ、改名させられた。

1930年、セデック族（賽德克族）の頭目モーナ・ルード（莫那・魯道）は300人の族人を率いて霧社（ブショウ）で蜂起した。日本は飛行機と毒ガスを出動させ、部落はほぼ壊滅した。これは英雄譚ではない。ある民族が崖っぷちに追い詰められた後の反乱と、その反乱がいかに徹底的に消し去られたかの残酷な記録である。

1945年、国民政府が台湾を接収し、新たな名を与えた：「山胞（サンポー）」。聞こえは柔らかいが、その実態は「山に住む漢人同胞」という暗示であり、原住民族の主体性を意図的に曖昧にし、同化政策に進歩的な外衣をまとわせるものだった。族語の使用禁止、漢姓への強制変更、山地平地化の推進、部落への道路建設、集団移住。台湾原住民族はこのようにして、自らの故郷で最も見知らぬ人となった。

> 📝 **五つの名、五つの鎖**
>
> 蕃人（日本統治期）→ 山地人（戦後初期）→ 山胞（制度化）→ 原住民（1994年憲法改正）→ 原住民族（2005年原基法）。最初の三つは、他者が名付けたもの。後の二つは、自ら取り戻したものである。

## ウフォンの嘘、そしてその銅像

湯英伸が銃殺された後、台湾社会は異例の自己反省に陥った。

カトリック花蓮教区の単国璽司教が先頭に立ち、蔣経国総統に「銃の下に人を留めよ」と請願する隊列を率いた。主流メディアは初めて原住民の都市における境遇を大々的に報じた：就職詐欺、低賃金搾取、身分証明書の没収。ツォウ族の青年の事件が、長年にわたって厳重に覆われていた傷口を切り開いた。

湯英伸の故郷の郷は、「ウフォン郷（吳鳳鄉）」と呼ばれていた。この名は清代に広く伝えられた故事に由来する：漢人のウフォンが自らをいけにえとし、首狩りの原住民を感化したという話である。この故事は日本統治時代にも国民政府時代にも国家に利用され、教科書に載り、銅像が建てられた。伝えられたメッセージは明確だった——原住民は開化されるべき野蛮人だ、と。

1988年、一群の原住民の若者たちが嘉義駅前に赴き、ウフォンの銅像を引き倒した。1989年3月1日、ウフォン郷は阿里山郷に改名された。1990年、ウフォンの故事は教科書から削除された。湯英伸の一族は「ツォウ族」という名を取り戻した（日本統治時代には「ツァオ族」と強制的に改称されていた）。

これは歴史の破壊ではない。数十年にわたって国家に認定されてきた嘘を正すことである。

同年の1988年、数千人の原住民が台北の街頭に繰り出し、「還我土地（土地を返せ）」と叫び、伝統領域の返還を要求した。これは正名運動が初めて集団的なアイデンティティをもって政治の舞台に登場した瞬間であり、四年後の憲法改正の前奏でもあった。

## 1984年：原住民か、山胞か？

正名運動制度的な起点を探るなら、1984年12月に遡るべきである。

台湾原住民族権利促進会（原権会）が設立された。台湾初の全国規模の原住民組織である。設立趣旨の一つが正名の推進だった。「我々は山胞ではない。我々は台湾原住民である」。この言葉を当時の台湾で口にすることは、確固たる勇気を要した。

湯英伸事件はこの訴えに社会的なエネルギーを与えた。原住民の都市における苦境、就職搾取、文化の根の喪失が、初めて主流の世論に正面から認識された。それは同情ではなく、ある社会が自らの歴史に対する負債に不安を覚め始めたのである。

## 1994年：憲法に二文字が刻まれる

1993年、数千人の原住民がケダグラン大通りに集結し、各民族の伝統衣装を着て行進した。翌年、この訴えは実現した。1994年、『中華民国憲法増改条文』が改正され、「山胞」が正式に「原住民」に改められた。

「原住民」の二文字が国家の最高法規に記されたのは、これが初めてである。

しかし、まだ一文字足りなかった。「原住民」は個人のアイデンティティの認定であり、「原住民族」は集団的主体であり、民族全体の言語、土地、自治の主張を承認するものである。この一文字の差を埋めるのに11年を要した。2005年、『原住民族基本法』が可決され、「族」の字が正式に法に組み込まれ、自治権、土地権、文化継承義務の法的枠組みが確立された。

> 📝 **「族」を一文字加える政治的重み**
>
> 「原住民」は個人のラベルであり、「原住民族」は民族の主体である。2005年の改正は、集団的な土地主張、言語復興義務、そして政府と原住民族の「対等な交渉」の仕組みを法的に承認したものである。それは一文字ではなく、一連の憲政的約束であった——ただし、実現にはなお距離がある。

## 9族から16族へ：埋もれた顔が、一つずつ戻る

「正名」は多くの民族にとって、「山胞」からの改名だけではなく、他者の民族籍の中から見出されることでもあった。

戦後に公式に認定された原住民族は9族のみであり、多くの民族は近隣の大民族に組み込まれ、言語と祭儀は沈黙の中で消融していった。1990年代から正名申請が相次いで行われた：

| 年   | 民族                             | 備考                                                    |
| ---- | -------------------------------- | ------------------------------------------------------- |
| 2001 | **シャオ族（邵族）**             | 南投日月潭、現在総人口900人未満                         |
| 2002 | **カバラン族（噶瑪蘭族）**       | 花蓮、かつてアミ族の戸籍に隠れていた                    |
| 2004 | **タロコ族（太魯閣族）**         | タイヤル族系に属していた                                |
| 2005 | **サキライヤ族（撒奇萊雅族）**   | 1878年の「加礼宛事件」後、100年以上にわたり名前を隠した |
| 2008 | **セデック族（賽德克族）**       | 霧社事件の民族、78年を待った                            |
| 2014 | **ラアルアワ族（拉阿魯哇族）**   | ツォウ族系に属していた、人口300人未満                   |
| 2014 | **カナカナフ族（卡那卡那富族）** | ツォウ族系に属していた、ラアルアワ族と同年に正名        |

2014年までに、公式認定の原住民族は16族に達した。サキライヤ族はアミ族の身分の下に130年以上身を隠してから、ようやく自らの名を取り戻した。一つひとつの正名の背後には、数年、あるいは数十年にわたる文化の自救活動があった：長老の語音を採集し、祭儀を再建し、族譜をたどり、そして政府に書類を提出し、裁定を待つ。

## 2016年：総統府の謝罪

2016年8月1日、台湾原住民族の日。蔡英文は総統府の経国庁において、政府を代表して全原住民族に正式に謝罪した。中華民国の史上初めて、国家元首が原住民族に対する四百年にわたる歴史的不正義を認めたのである。

謝罪文では責任が列挙された：オランダ政権と鄭成功政権による平埔族の虐殺、清朝の流血弾圧、日本の理蕃政策、戦後の山地平地化、族語使用の禁止、そしてヤミ族（雅美族）の知らぬ間に蘭嶼（ランユ）に核廃棄物が貯蔵されたこと。

彼女は謝罪文中でタイヤル族の言葉を引用した：

> 「タイヤル族の言語において、『真実』は Balay という。そして『和解』は Sbalay という。すなわち、Balay の前に S の音を加えるのだ。真の和解は、真実に誠実に向き合うことによってのみ達成される。」
> — 蔡英文、2016年8月1日

同日、彼女は「原住民族歴史正義と転換正義委員会」（原転会）の設立を発表し、総統自らが召集人を務め、毎年8月1日に全国に進捗を報告すると約束した。

原住民の代表者たちの謝罪への反応はさまざまであった：感動した者もいれば、具体的な約束が欠けていると考える者もいた。「謝罪に意味はない、土地を返せ」と言う者もいた。伝統領域の画定は現在も進展が遅く、原基法における自治条項の実現も困難を極めている。Sbalay——それは真実を語った後にのみ起こりうることであり、その真実は今もなお語られ続けている。

## 言語は最後の防壁

正名が成功した後も、問題は減らなかった。

台湾の16族には42種の方言があり、その半数以上がユネスコにより「危機」または「深刻な危機」の言語に指定されている。シャオ族語の流暢な話者は、2020年代の統計で10人を下回っている。カナカナフ族の族語教室において、完全な継承資格を有する長老は数えるほどしかいない。言語が死ねば、民族の記憶も死ぬ。復興は正名よりも難しく、家庭、学校、部落、政策が同時に連携し、若者が日常生活でほとんど役に立たない言語を口にすることを望まなければならない。

---

1984年の原権会設立から、1994年の正名の憲法への明記、2005年の原基法、2016年の謝罪に至るまで、各段階は前段階よりも困難を増していった。

しかし、この道の起点を探すなら、それは1987年5月15日、台北、19歳のツォウ族の青年が刑場で麻酔薬を拒否した時であろう。

彼は自ら罪を犯したのだから、その痛みを覚醒したまま受けなければならないと言った。

彼の一族が外で泣き叫び、彼の名が社会に広がり、そして一つの運動が始まった。彼の故郷は後に阿里山郷という名を取り戻し、彼の一族はツォウ族という名を取り戻し、彼の一族の名は憲法に、基本法に記され、三十年遅れの国家の謝罪がなされた。

あなたは今日、彼の名を覚えているだろうか？
