# キーボード上の文明衝突：東アジア文字入力法の百年の進化

## 30秒でわかる概要

世界中のコンピューターキーボードはQWERTY配列で、1870年代に英語タイプライター用に設計されたレイアウトです。しかし、東アジアには20億人以上が使用する文字システム（漢字、仮名、ハングル、タイ文字、ミャンマー文字）があり、これらはそもそも表音文字ではありません。彼らはどうしたのでしょうか。答えは、それぞれの文明が独自の「翻訳層」を発明したことです——それが入力法です。これらの入力法は単なる技術ツールではなく、アイデンティティの戦場でもあります。台湾は注音を使い、中国は拼音を使い、日本はローマ字を使い、韓国は文字を直接分解します。その一つの選択の裏には、デジタル化に直面する文明ごとの異なる哲学があります。

---

## 問題の本質：26のアルファベット vs 数万の文字

英語話者は「入力法」を必要としたことがありません。キーボードに26のアルファベットがあり、打てばそのまま出てきます。しかし漢字は50,000字以上あり、常用のものでも3,000〜5,000字あります。5,000個のキーがあるキーボードを作ることは不可能です。

これは、東アジアの文明が根本的な問題を解決しなければならないことを意味します。**限られたキーで無限の文字をどう表現するか。**

それぞれの文明がまったく異なる答えを出し、それらの答えは言語構造、教育体系、さらには政治的選択を深く反映しています。

---

## 🇹🇼 台湾：注音記号（「発音」で文字を探す）

### 注音の歴史的背景

台湾の主流の入力法は**注音入力法**で、37の注音記号（ㄅㄆㄇㄈ⋯）を使って発音を表記します。「台湾」を打ちたければ、`ㄊㄞˊ ㄨㄢ`を入力し、システムが同音字を一覧表示して選択させます。

注音記号自体は1913年の「読音統一会」で誕生し、章太炎（しょうたいえん）などの学者が漢字の偏旁から簡略化したものです。これは**ラテン文字から完全に独立した発音体系**であり、この点が極めて重要です。

### 台湾が注音にこだわる理由

台湾が注音を守り続けるのには、相互に強化し合う四つの理由があります。教育体系が根本です。小学校1年生の最初の10週間は注音の教育に全力を注ぎ、これは台湾人にとって最も根強い識字ツールであり、切り替えるコストが高すぎます。アイデンティティが原動力です。注音記号は繁体字中国語圏に特有の表記システムであり、ラテン文字を使用しないことが中華文化の伝統の継続と見なされています。技術的には、注音は中国語の四声（さらに軽声も）を正確に表記でき、漢語拼音では完全に実現が難しい面があります。最後に、台湾のキーボードは各アルファベットの横に対応する注音記号が併記されており、デュアル表記を形成し、このシステムがハードウェアレベルでも根強く定着しています。

### 注音の限界

注音の最大の問題は**同音字が多すぎる**ことです。中国語（国語）には約1,300の異なる音節しかないのに、数万の漢字に対応しなければなりません。「ㄕˋ」と入力すると「是、事、式、室、市、試、視、適、勢、世⋯⋯」など数十個の文字が表示されることがあります。使用者は候補リストから文字を選択しなければならず、入力速度が遅くなります。

近年、スマート注音入力法（マイクロソフト新注音、RIMEなど）がAIによる文脈予測で精度を大幅に向上させましたが、選字という本質的な問題は依然として残っています。

### 倉頡：もう一つの道

1976年、「中国語コンピューターの父」と呼ばれる**朱邦復（しゅほうふく）**が**倉頡入力法**を発明しました。これは発音に一切頼らず、**字形を分解する**方法です。各漢字が1〜5つの「字根」に分解され、キーボード上の25のキー（AからYまで、Zキーは除く[^2]）に対応します。

例えば「明」= 日 + 月 = `A` + `B`。

倉頡の利点は**一字一コード**で、選字が不要なことです。熟練した倉頡使用者は注音を上回る速度を出せます。朱邦復は後に倉頡の特許権を放棄し、オープンソースの中国語入力法の先駆けとなりました。これはオープンソースソフトウェア運動よりも20年も早いことでした[^1]。

倉頡は香港で非常に普及しており（コンピューター使用者の過半数）、台湾では常に少数派にとどまっています。主な理由は学習曲線が急峻であることです。

### 行列入力法

廖明德（りょうめいとく）が発明した**行列入力法**は、もう一つの台湾発の方式です。数字キーを基盤として字形を分解し、「多くの字根を覚えなくてよい」という設計哲学を持っています。これは台湾が入力法分野で継続的に革新を続けていることを示しています。

## 🇨🇳 中国：漢語拼音（ラテン文字で中国語を綴る）

### 拼音の選択

中国本土の主流の入力法は**漢語拼音入力法**で、26のアルファベットを直接用いて漢字の読みを綴ります。「台湾」を入力するには`taiwan`と打ち、システムが簡体字中国語に変換します。

この選択には深い歴史的背景があります。

1. **1958年の漢語拼音方案公布**：それまでの注音字母（中国では「注音符号」と呼ぶ）およびウェード式ローマ字に取って代わりました
2. **簡略化文字改革**：1956年から簡体字が推進され、拼音入力と補完し合う関係になりました。拼音を学ぶ→拼音で打字する→簡体字を出力する
3. **国際化の考慮**：拼音はラテン文字を使用するため、外国人が中国語を学びやすく、中国語話者が標準的なキーボードで入力しやすくなります

### 拼音 vs 注音：気づかないかもしれない文化的分岐

表面上、注音と拼音はどちらも「発音で文字を探す」方法です。しかし、深い差異があります。

|                | 台湾注音                     | 中国拼音                       |
| -------------- | ---------------------------- | ------------------------------ |
| 記号体系       | 独立記号（ㄅㄆㄇ）           | ラテン文字（bpmf）             |
| 文化的根源     | 漢字の偏旁に由来             | ラテン化運動に由来             |
| 学習の前提     | 英語を先に学ばない           | アルファベットを知る必要がある |
| キーボード要件 | 注音表記付きキーボードが必要 | どの英語キーボードでも可       |
| 文字との関係   | 「発音を記述する」           | 「ラテン文字に翻訳する」       |

この差異は技術的なものにとどまらず、両岸の「中国語は国際社会とどう接すべきか」という根本的な分岐を反映しています。台湾は西洋から独立した記号体系を維持することを選び、中国はラテン化を受け入れることを選びました。

### 五筆字型：中国の「倉頡」

注目すべきは、中国にも字形入力法があることです。代表が**五筆字型**（王永民、1983年）です。そのロジックは倉頡と同様で、漢字を筆画に分解してキーボードに対応させます。五筆は1990年代の中国のオフィスで非常に普及しましたが、拼音入力法のスマート化と携帯電話の普及に伴い、使用率が急激に低下しました。現在、中国の95%以上の使用者が拼音入力を使用しています。

## 🇯🇵 日本：ローマ字→仮名→漢字の三段変身

### 日本語入力の独特な課題

日本語は世界で最も複雑な書記体系の一つで、同時に三つの文字を使用しています。

- **平仮名**（ひらがな）：46の基本音節記号
- **片仮名**（カタカナ）：46個、主に外来語に使用
- **漢字**（かんじ）：常用約2,000〜3,000字

日本語入力法の標準的な方法は「**ローマ字入力**」です。

1. アルファベットを入力 → 自動的に平仮名に変換：`ka` → `か`、`n` → `ん`
2. 入力を続けると、システムが単語を組み立てる：`kanji` → `かんじ`
3. スペースキーを押して漢字に変換：`かんじ` → `漢字`

これは**三層変換**のプロセスです。アルファベット→仮名→漢字。各層で使用者の判断が必要です。

### 日本が仮名直接入力ではなくローマ字を使う理由

日本には**かな入力**（仮名直接入力）の選択肢もあり、キーボードの各キーが一つの仮名に対応しています。しかし、50以上のキー配置を覚える必要があり、日本の教育体系では英語教育の中でローマ字をすでに教えているため、大多数の人がアルファベットを使う方が便利だと感じています。

現在の日本の大多数の使用者はローマ字入力を使用しています（比率は調査方法により異なりますが、約80〜90%と推定されます[^6]）。高齢者や専門のタイピストの一部のみが仮名直接入力を使用しています。

### 日本語入力の文化的含意

日本語の漢字変換には興味深い文化的影響があります。若者が**漢字を手書きする方法を忘れ始めている**ことです。入力法が正しい漢字を自動的に表示するため、使用者は「どう読むか」を知っていればよく、「どう書くか」を覚える必要がありません。この現象には日本語で専門的な用語があります：「**漢字忘れ**」（かんじわすれ）。

## 🇰🇷 韓国：二ボル式（最も洗練されたキーボード設計）

### ハングルの天才性：キーを直接対応できる文字

ハングル（한글）は1443年に世宗大王（せそうだいおう）の命により創製されたアルファベット体系であり、明確な発明者がいる世界でも極めて稀な文字です。14の子音（ㄱㄴㄷㄹ⋯）と10の母音（ㅏㅓㅗㅜ⋯）で構成され、これらの文字が音節ブロックに組み合わされます。

ハングルの子音と母音を合わせた基本文字はわずか24個で、QWERTYキーボードの26キーにぴったり収まります。

### 二ボル式（두벌식, Dubeolsik）：左手に子音、右手に母音

韓国の標準入力法**二ボル式**（両手式）の設計は極めて直感的です。

- **左手**が子音を打つ：ㄱ(r) ㄴ(s) ㄷ(e) ㄹ(f) ㅁ(a)⋯
- **右手**が母音を打つ：ㅏ(k) ㅓ(j) ㅗ(h) ㅜ(n) ㅡ(m)⋯

打字時、両手が交互に動き、リズム感が非常に良く、**選字が不要**で、打てばそのまま出力されます。

これは東アジアの全入力法中、**唯一候補文字リストが不要なもの**です。ハングルの音節ブロックはリアルタイムで組み合わされます。`ㅎ` + `ㅏ` + `ㄴ` = 한、`ㄱ` + `ㅡ` + `ㄹ` = 글。プロセス全体がゼロ遅延、ゼロ選字です。

### なぜハングル入力法が最も洗練されているのか

ハングル自体が「書きやすさ」のために設計されたからです。世宗大王の設計哲学は「智者不終朝而会、愚者可浹旬而学」[^3]（賢者は朝までに習得し、愚者も十日で学べる）でした。600年後、この設計はデジタル時代でも完璧に適合しています。24の文字がキーボードにぴったり収まり、子音と母音が左右の手に分かれ、変換も選字も不要です。

## 🇹🇭 タイ：Kedmanee（タイプライター時代から続く配列）

### タイ文字の課題：44の子音 + 声調記号

タイ文字には44の子音記号、15の母音記号（組み合わせで28の母音形を形成）、4つの声調記号があり、合計60以上の文字があります。これは標準キーボードのキーをはるかに超えます。

解決策は**Kedmanee配列**（เกษมณี）です。Suwanprasert Ketmaneeが1920〜1930年代にタイプライター用に設計しました[^4]（ウィキペディアによると、この配列は1932年頃に定型化されたとされています）。最もよく使われる文字をShiftなしの位置に配置し、あまり使わないものをShift層に配置しています。

### タイ文字入力の特徴

タイ文字は**表音文字**ですが、その表記規則は極めて複雑です。母音は子音の前、後、上、下に現れることがあります。例えば เ（e）は子音の前に書かれますが、読み時には後ろになります。これは、打字順序と読解順序が必ずしも一致せず、使用者は「先に母音を打ってから子音を打つ」というケースに慣れる必要があることを意味します。

タイ文字入力は選字が不要（ハングルと同様）ですが、通常層とShift層の2層のキー配置を覚える必要があります。

## 🇲🇲 ミャンマー：Unicodeの戦い

### Zawgyi vs Myanmar Unicode：デジタル内戦

ミャンマー文字入力法の物語は東アジアで最もドラマチックなものです。ミャンマー文字には33の子音と複雑な組み合わせ規則がありますが、本当の問題は入力法自体ではなく、**フォントエンコーディング**にあります。

2000年代、ミャンマーのエンジニアZaw Htutが**Zawgyiフォント**を開発しました。これはUnicode標準に準拠していませんが、使いやすさから急速に普及しました。2010年代までに、ミャンマーの携帯電話の約90%がZawgyiを使用していました。

問題は、ZawgyiとUnicodeが互換性がないことです。同じ文章が両方のシステムでまったく異なる表示になり、大量のコミュニケーション混乱を引き起こしました。

2019年、ミャンマー政府は**Myanmar Unicode**への全面移行を正式に宣言しました[^5]。Facebookも同年、ミャンマーの使用者をZawgyiからUnicodeへの切り替えを強制しました。この移行は2,000万人以上の使用者に影響を与え、一国のデジタルインフラの大移転に相当する規模でした。

## モバイル時代：新たな戦場

スマートフォンは入力法の生態系を一変させました。台湾の注音入力（テンキーやフルキーボード）はモバイルでも主流ですが、手書き入力と音声入力の使用率が急速に上昇しています。中国はAI主導の方向に進んでいます。搜狗拼音、百度入力法が主流となり、「スライド入力」が拼音の効率を大幅に向上させました。日本では**フリック入力**が発達し、指でテンキー上でスワイプして仮名の方向を選択します。アルファベットは一切必要ありません。韓国には**千字文入力法**（천지인, Cheonjiin）があり、ㅣ ㆍ ㅡ（天地人）の三つの基本筆画ですべてのハングルを組み合わせます。小さな画面に極めて適しています。

モバイル時代が一つの興味深い現象をさらに顕著にしました。**若い世代が手書き能力を失いつつある**ことです。これは漢字文化圏で特に深刻です。入力法がすべての漢字を覚えてくれるとき、あなたの手は忘れてしまうのです。

## AI時代：入力法の終焉？

音声認識とAI対話技術の進歩に伴い、根本的な疑問が浮上しています。**私たちはまだ入力法を必要とするのでしょうか。** 音声入力は多くの場面で打字に取って代わっており、中国のWeChatにおける音声メッセージの使用率は特に高いです。AI予測により入力法はますます「賢く」なり、数文字打つだけで文章全体を予測できるようになりました。手書き認識技術の進歩も「指で画面に文字を書く」ことを実用的なものにしています。

しかし、入力法は消えません。なぜなら、それは単なるツールではなく、**文化の記憶の担い体**だからです。台湾の子供たちが注音を学ぶ10週間、日本人がキーボード上でローマ字を漢字に変換するその瞬間、韓国人が左手に子音、右手に母音で打つリズム感。これらはすべて、デジタル時代における各文明と自らの文字との親密な対話なのです。
