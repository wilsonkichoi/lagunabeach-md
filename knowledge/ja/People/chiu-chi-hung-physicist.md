> **30 秒概要**：交通大学博士の邱継弘、ニックネーム「大河馬（ダカワウマ）」。幼い頃からソフトウェアプログラミングに強い興味を示し、高校時代は全国科学展で第三位を受賞し、「特殊才藝」推薦で国立交通大学情報工学部に入学した [^4]。2006年、Web 2.0 の波に乗ってパートナーとコミュニティサイト funP（推推王）を設立したものの、商業モデルが不明確のため 2007 年に失敗を宣言し、3年間で約2000万円を費やした [^8] [^10]。2009年、その廃墟から cacaFly 聖洋科技を設立し、Facebook 台湾広告総代理として出発し、その後 Google、LINE、Microsoft、Spotify などの国際プラットフォームの代理権を獲得した。adHub、TenMax、BeingDigital などのサブブランドを設立し、台湾で最も完全なローカルデジタル広告エコシステムを構築した [^9] [^11]。商業的な業績だけでなく、邱継弘は政治の世界にも積極的に進出し、台北市政府市政顧問を務め、2019年には時代力量決策委員への立候補を登記した。「左派経営者」としてのテクノロジーの思考で政治を変えようとしたのである [^20] [^21]。近年は、AI マーケティングと小売メディアネットワーク（RMN）への注力を強め、台湾 MarTech 産業の転換をリードし続けている [^1] [^2] [^3]。

## 早年の啓蒙と「新交大グループ」の技術的遺伝子

邱継弘は 1976 年に生まれた。そのエンジニアの道は、偶然の代理参加から始まった。小学6年生のとき、兄に代わってプログラミングコースに参加したことが、ソフトウェアへの強い興味を開くきっかけとなった [^4]。この才能は高校時代に実証され、優れたプログラミング作品で全国科学展第三位を受賞し、「特殊才藝」推薦で国立交通大学情報工学部に無事入学した [^4]。

交通大学在学中、邱継弘は袁賢銘教授の「分散型システム研究室」（DCSLab）に参加した。この研究室は台湾のソフトウェア起業史において重要な地位を占めており、核心メンバーには後に曜碩科技を設立した郭栄昌・呉英豪、Ubitus 創業者の沈上謙、奇群科技創業者の宋牧奇などが含まれていた [^5] [^24]。メディアに「新交大グループ」と称された若者たちは、初期シリコンバレーの起業家精神を持ち、台湾を「ハードウェアメーカー」の思考から「ソフトウェアサービス」へと転換させようとしていた。2004年、博士課程在学中の邱継弘はチームを率いてトレンドマイクロ「百万プログラミングコンテスト」で最優秀賞を獲得し、米国 IBM Silicon Valley Lab への経験も積んだ。これらの学術・業界での深い蓄積が、後に技術駆動で広告業界を変える基盤となった [^4] [^6] [^7]。

📝 **キュレーターノート**：邱継弘の早年の経験、特に「新交大グループ」の集団効果は、台湾のテクノロジー人材が学術から起業へと進む独特の道筋を物語っている。DCSLab は技術的な揺りかごであると同時に、「ソフトウェアサービス」の思考を孵化する温床であり、当時の台湾の主流であるハードウェア製造業と鮮やかな対比をなしていた。彼のエンジニアの遺伝子は、当初から伝統的な広告業者とは異なる方法でデジタルマーケティング産業を解体し再構築することを運命づけていた。

## 台湾 Web 2.0 で最も高い授業料：funP 推推王

2006年、世界の Web 2.0 ブームの影響を受け、邱継弘は4人のパートナーと共にコミュニティサイト「funP.com」（推推王）を設立した [^8]。funP は Digg に類似したソーシャルフィルタリングアルゴリズムを採用し、「集合知」とソーシャルシェアを重視した、当時台湾では比較的大規模な Web 2.0 の実践例であった [^8]。翌年、彼は台湾チームを代表してシリコンバレー DEMO Show のステージに立ち、米国聴衆に向けて Web 2.0 の概念を説明した [^4]。

技術的な先行は商業的な成功には結びつかなかった。funP は3年間で約2000万円の資金を費やし、複数の困難に直面した。当時の台湾の広告主は Yahoo などのポータルサイトに大きく依存しており、Web 2.0 のトラフィックへの信頼が低かった。2009年に Facebook が台湾に本格上陸すると、そのソーシャルつながりと API エコシステムは funP をはるかに超え、funP のトラフィックは雪崩的に減少した [^10]。2009年、商業モデルが維持不可能と判断した邱継弘は、製品開発を諦め、「案件完了で即金が入る」広告サービス業への転身を決意した [^10]。この失敗経験を通じて、彼は痛感した。ネットワークの世界では、強力な技術とトラフィックがあっても、商業モデルがなければ存続できないということを。

📝 **キュレーターノート**：funP 推推王の失敗は、邱継弘の起業家人生における重要な「成人式」であった。巨額の資金を費やしただけでなく、純粋な技術的理想主義者から、商業モデルの重要性を深く理解する実践者へと彼を変えた。この経験は後に cacaFly を設立する基盤となり、新しい技術の波に直面した際にも、技術と商業価値をどう結びつけるかをより現実的に考える力を与えた。

## エンジニアが広告業界を変える：cacaFly の技術的台頭

2009年に設立された cacaFly 聖洋科技は、当初から強いエンジニアの遺伝子を持っていた。伝統的な広告会社がクリエイティビティと人脈を重視するのに対し、cacaFly のチームの半分はプラットフォームアルゴリズムに精通したプログラマーだった。邱継弘はアルゴリズムを解析可能なツールと捉え、大量のテストを通じてロジックを把握し、クライアントのコスト削減に貢献した [^12]。

2011年、cacaFly は Facebook 台湾公式広告代理店となり、これが同社の成長における転換点となった [^9]。その後、Google、Microsoft、LINE、Spotify などの国際プラットフォームの代理権を次々と獲得し、adHub や TenMax などのサブブランドを展開した。特に TenMax は台湾ローカルのリアルタイム入札（RTB）システムとデータ管理プラットフォーム（DMP）の構築に注力し、台湾の広告産出価値を国内に留め、完全なローカルデジタル広告エコシステムを確立することを目指した [^10] [^11]。

📝 **キュレーターノート**：cacaFly の台頭は、広告業界における邱継弘の「エンジニア的思考」の勝利である。彼は広告をデータ分析とアルゴリズム最適化が可能なシステムとして捉えた。この技術志向の戦略により、cacaFly は急速に国際プラットフォームのローカル巨人となった。また、TenMax などのサブブランドを通じて台湾に独立したデジタル広告インフラを構築しようとしたのは、「デジタル主権」への思考の表れでもある。

## 政治の世界へ：時代力量決策委員と市政顧問

邱継弘の視野は商業にとどまらなかった。柯文哲が台北市長を務める期間中、邱継弘は台北市政府市政顧問に任命され、デジタルマーケティングとスマートシティの発展に関する専門的な助言を行った [^21]。

2019年1月、邱継弘は時代力量第二届決策委員への立候補を登記した [^20]。立候補声明において、彼は自らを「左派経営者」と称し、今回の立候補は党の運営モデルの変革に関わるものであり、IT 産業のイノベーションのエネルギーを政党の運営に注入したいと強調した [^20]。政党は「データ分析力」を備え、技術的手段を通じて有権者のニーズを把握すべきだと主張した。当時、時代力量は路線議論の重要な時期にあり、邱継弘の参加は中世代のテクノロジーリーダーが制度内参加を通じて台湾の政治的意思決定の科学性とデジタル力を高めようとする試みと見なされた。

📝 **キュレーターノート**：邱継弘の政治参加は、台湾の新世代テクノロジーエリートに対する公共事務への関心を反映している。彼はエンジニアの「システム最適化」の論理を政治の分野に持ち込み、データ分析と合理的な意思決定を強調した。これは彼自身のアイデンティティの転換であると同時に、台湾社会におけるテクノロジー統治への期待と試みを象徴している。

## 広告代理店の政治的地雷と中立性の課題

Facebook 在台湾の公式窓口として、cacaFly は政治的な論争に巻き込まれることは避けられなかった。2017年、国民党シンクタンクは「Facebook が緑になった」と批判し、蔡英文政権が代理店を通じて言論統制を行っていると疑った [^22] [^23]。この疑惑に対し、邱継弘は何度も明確にした。cacaFly が担当するのは広告業務（Business）のみであり、コンテンツ審査（Policy）には関与しない。コンテンツポリシーは Facebook 米国本社が直接管理していると [^22]。

この論争は、台湾の特殊な政治的エコシステムにおけるデジタル広告代理店の課題を浮き彫りにした。各政党の顧客にサービスを提供しつつ、技術的な中立性をいかに維持するか。邱継弘は「産業建設者」の立場で対応し、DMA 理事长を務めることで、情報セキュリティ、個人情報保護、データ活用、詐欺対策などの議題を推進し、デジタル広告環境をより透明で規範的な発展の軌道に導こうとした [^15] [^16]。

📝 **キュレーターノート**：台湾の政治環境において、デジタル広告代理店はしばしば嵐の中心に立たされる。邱継弘は「Facebook が緑になった」という告発に対し、産業リーダーとしての危機管理能力を発揮した。論争を政治的イデオロギーから技術的中立性と産業規範の次元に引き戻し、DMA 理事长の立場を通じてすべての参加者が信頼できるデジタル広告の秩序を構築しようとした。これは商業的配慮であると同時に、産業倫理への取り組みでもある。

## AI 転換と RMN：MarTech 第二波をリード

2020年以降、邱継弘は cacaFly の主要戦場を AI マーケティングと小売メディアネットワーク（RMN）へと転換した。同社が立ち上げた **Alchemy AI クリエイティブ生成プラットフォーム**は、「レイヤー思考」（Layer Thinking）を導入し、AI 生成素材における製品の歪みという課題を解決し、商用価値を大幅に向上させた [^1] [^18]。

RMN の分野では、cacaFly は「上山採薬」などのブランドがデータ駆動のソリューションを構築するを支援し、関連技術を東南アジア市場へも展開した [^2] [^13] [^19]。彼は業界フォーラムで何度も強調している。AI 時代の鍵は「データガバナンス」（Data Governance）にあり、高品質なファーストパーティデータを掌握してこそ、Cookie が消滅した後のデジタル広告市場で生き残れると [^14]。2024年、cacaFly は Google 優秀代理店と LINE ダイヤモンド級パートナーの認定を同時に獲得した [^11]。

📝 **キュレーターノート**：Web 2.0 から AI 時代へ、邱継弘は常に技術の波の最先端に立っている。AI マーケティングと RMN への取り組みは、商業戦略の転換であると同時に、将来のデジタル広告エコシステムに対する予測でもある。彼が「データガバナンス」を強調していることは、プライバシー保護とデータ倫理への理解を示している。これはポスト Cookie 時代における MarTech 産業の核となる競争力である。

## 結語：失敗の中で設計された「失敗しないシステム」

funP の2000万円という授業料から cacaFly のデジタル広告王国へ。邱継弘の起業家の歴史は、失敗から絶えず反復的に学び続けるプロセスである。彼はエンジニアのテストと反復の論理で、広告業界の伝統的な直観と賭けに取って代わった。彼のニックネーム「大河馬」は、まさに彼そのものの特質を表している。外見は穏やかだが、デジタル波の水面下では、極めて高いスピードで正確に前進する。

📝 **キュレーターノート**：邱継弘の物語は、一つの核心的な命題に帰着する。変動し続けるデジタル世界の中で、どうやって「失敗しないシステム」を設計するか。このシステムの定義は「間違いから学び、迅速に反復し、技術と商業モデルを深く融合させる」ことであり、決して間違いを犯さないことを追求するものではない。彼の「大河馬」哲学は、この堅牢かつ機敏な起業家精神の最高の体現であり、台湾のテック起業家たちに貴重な示唆を与えている。
