# 九合一選挙とは何か？

> **30秒でわかる概要：** 「九合一」という言葉が広く使われるようになったのは2014年のことです — 台湾で初めて「九種の地方公職人員選挙」が同日に投票されました。しかし、この構図が2014年に突然出現したわけではありません。1967年の台北市昇格、1979年の高雄市昇格、1994年の直轄市長初の民選、2010年の五都昇格、2014年の桃園昇格、2014年の山地原住民區長新設という六つの転換点が積み重なった結果です。一枚の投票用紙で九つの職位を選ぶという形態の裏には、台湾の三十年にわたる地方自治制度の完全な進化の歴史があります。この記事で伝えたいのは「2026年どう投票するか」ではなく、「なぜ九合一という構図が大切にされるべきか」ということです。

---

## 「九合一」という言葉の由来

2014年11月29日、台湾で初めて九種の地方公職人員選挙が同日に投票されました。メディアが「九合一選挙」という言葉を広め始めました — それ以前の選挙でも複数の職位が同日に投票されていましたが、組み合わせが異なっていました [^1]。

「九合一」という言葉の構成はとてもシンプルです。九つの職位を合わせて一日に投票するという意味です。しかし、このシンプルな構図の裏には、台湾の1947年の憲法制定以来、最も完全な地方自治の進化史があります。2014年以前、最も複雑な地方選挙は「七合一」や「五合一」で、職位数は都市の昇格に伴い徐々に増えていきました [^2]。

| 年   | 地方選挙の名称     | 同日職位数       | 主な制度的転換点                                     |
| ---- | ------------------ | ---------------- | ---------------------------------------------------- |
| 1994 | 第一屆直轄市長選挙 | 台北・高雄で別々 | 直轄市長初の民選                                     |
| 2002 | 五合一             | 5種              | 縣市長 + 縣市議員 + 鄉鎮市長 + 鄉鎮市民代表 + 村里長 |
| 2009 | 五合一             | 5種              | 五都昇格前の構図を継続                               |
| 2010 | 五都選挙           | 4種              | 五都昇格後初の直轄市長選挙                           |
| 2014 | **九合一**         | 9種              | 桃園昇格 + 山地原住民區長新設                        |
| 2018 | 九合一             | 9種              | 国民投票と大選の同時実施初（後に法改正で分離）       |
| 2022 | 九合一             | 9種              | 国民投票は大選と分離                                 |
| 2026 | 九合一             | 9種              | 中央選挙委員会 11/28 投票日 [^3]                     |

## 九つの職位とは何か

《公職人員選舉罷免法》および《地方制度法》の規定に基づき、2026年に同日投票される九つの公職は以下の通りです [^4]：

### 1. 直轄市長（6席）

台北、新北、桃園、台中、台南、高雄の六都の首長。任期4年、再選による連続一期まで。直轄市の行政、教育、社会福祉、警察などの事務を担当します。

直轄市長という職位の歴史は1967年（台北市昇格）と1979年（高雄市昇格）に遡りますが、1967年から1994年までは官任命で、1994年に初めて民選となりました。詳細な歴史と制度の進化については、「2026 九合一選舉（2026年九合一選挙）」を参照してください。

### 2. 直轄市議員（約380席、議会の席次数による）

六都の議会代表。任期4年、再選制限なし。予算審議、市政監督、立法を担当します。

議員選挙は1980年代に設計された「複数選挙区単記非譲渡式（SNTV）制度」を継続して使用しています — 2008年に小選挙区比例代表並立制に改正された立法委員とはまったく異なる方向性です。詳細については「議員制度（議員制度）」を参照してください。

### 3. 縣（市）長（16席）

非直轄市の16の縣市の首長 — 基隆市、新竹市、嘉義市の三市、および宜蘭、新竹、苗栗、彰化、南投、雲林、嘉義、屏東、花蓮、台東、澎湖、金門、連江の十三縣 [^5]。任期4年、再選による連続一期まで。

### 4. 縣（市）議員（約532席）

非直轄市の縣市の議会代表。任期4年、再選制限なし。同様にSNTV制度を採用しています。

### 5. 鄉（鎮、市）長（約198席）

非直轄市内の鄉、鎮、縣轄市の首長。任期4年、再選による連続一期まで。鄉鎮市長は台湾の末端の民選首長です — 縣市長よりも地域社会に近く、村里長よりも実質的な行政権を持ちます。

### 6. 鄉（鎮、市）民代表（約2,148席）

鄉鎮市の民意代表会の代表。任期4年、再選制限なし。

### 7. 直轄市山地原住民區長（6席）

新北烏來、桃園復興、台中和平、高雄那瑪夏、桃源、茂林。任期4年。

この職位は2014年の桃園昇格時に新設されました — 昇格という技術的な手続きによって原住民族の部落の自治権が剥奪されないようにするためです。詳細については「直轄市山地原住民區長（直轄市山地原住民區長）」を参照してください。

### 8. 直轄市山地原住民區民代表（約50席）

六つの山地原住民區の民意代表会。

### 9. 村（里）長（7,748席）

台湾で最も末端の民選首長。直轄市の「里」、非直轄市の鄉鎮市の「村」または「里」 — 全台で合計7,748。任期4年、再選制限なし。

村里長は台湾の選挙の中で唯一「無党籍が圧倒的に多い」職位です — 規模が小さすぎて（一つの里の平均人口は2,000〜5,000人）、政党の立場よりも一世帯ずつ知っていることのほうが重要です。詳細については「村里長制度（村里長制度）」を参照してください。

## なぜ「九」なのか、「八」や「十」ではないのか

それぞれの職位にはそれぞれの歴史があります。しかし「九」という数字は1947年の憲法制定時に決まったものではなく、1949年から2014年までの連続する65年間の積み重ねの結果です。

**1967年** — 台北市が直轄市に昇格。台湾で初めて「直轄市長」という官職が誕生しました（当時は官任命でした）。しかし、鄉鎮市長や村里長などの末端民選職位は1950年代からすでに存在していました。

**1979年** — 高雄市が直轄市に昇格。台北市と高雄市が並ぶ直轄市となりましたが、両市の市長は依然として官任命でした [^6]。

**1994年11月26日** — 《省縣自治法》と《直轄市自治法》の可決後、台北市長と高雄市長が初めて民選となりました。陳水扁が台北市長に、吳敦義が高雄市長に当選しました。これは台湾の地方自治史上の重要な転換点です [^7]。

**1999年** — 《地方制度法》が分立していた《省縣自治法》と《直轄市自治法》に代わり、全国の地方自治に関する法源を統一しました。九合一の構図の法的基盤がここで確立されました。

**2010年12月25日** — 五都昇格：新北市、台中市、台南市、高雄市（県と合併）、台北市が同時に直轄市として運営を開始しました。台湾の都市地理は「五都」の構図へと再編されました [^8]。同時に、もともと五つの県に属していた「山地鄉」の郷長の民選の地位が消滅し（昇格後に区長は官任命に変更）、原住民族の自治権をめぐる議論が起こりました。

**2014年** — 桃園が直轄市に昇格し、台湾は「六都」の時代に入りました。同時に立法院は《地方制度法》改正案を可決し、「直轄市山地原住民區長 / 區民代表」という職位を新設しました — 2010年の昇格で失われた原住民族の民選自治権の一部を回復させるためです [^9]。

**2014年11月29日** — 初めての「九合一選挙」が同日に投票されました。九つの職位の構図はこの日以降固定されました。

## 制度設計における緊張関係

九合一という構図は一見自然に見えますが、その裏には多くの構造的な緊張関係があり、それぞれが現在もなお進化を続けています。

### 昇格 vs 地方自治

「昇格」は台湾の都市統治における核心的な概念です — 省轄市から直轄市へと変わることで、予算規模、人事編成、行政階層がすべて変化します。しかし、昇格は「民選首長の喪失」という副作用ももともたらします。もともと縣轄市だった「市長」の民選の地位が昇格後に消滅します（官任命の区長に変わる）。

2010年の五都昇格時にこの副作用が初めて大規模に発生しました。特に高雄県の三つの山地鄉（那瑪夏 / 桃源 / 茂林）の郷長の民選の地位が剥奪されました。2014年の桃園昇格時にも同様の議論が再び起こりました（復興鄉）。立法院は2014年の《地方制度法》改正により「直轄市山地原住民區長」という職位を新設し、この問題に対処しました。

しかし、平地の鄉鎮市長の民選の地位は現在も昇格によって消滅したままです — この構造的な問題はまだ広く解決されていません [^10]。

### 直轄市議員 vs 立法委員の制度的分歧

立法委員は2008年に「小選挙区比例代表並立制」（区域 + 比例代表）に改正されましたが、縣市議員は現在も1980年代に設計された「複数選挙区単記非譲渡式（SNTV）」のままです — 同じ民選代表でありながら、制度設計はまったく異なります。

この分歧は18年間にわたって維持されています。その背景には地方派閥と政党の利益の絡み合いがありますが、SNTVが小党・第三勢力・多様な代表性に与える影響は、議員レベルにおいて立法委員よりもはるかに深刻です。詳細については「議員制度（議員制度）」を参照してください。

### 村里長のスケールのパラドックス

全台で7,748人の村里長がいます — これはセブンイレブンの6,600店舗以上よりも多い数です。四年ごとに7,748件の小規模な選挙が同時に実施されます。しかし、一つの里の平均有権者数はわずか2,000〜5,000人であり、この職位の政治経済学は他の八つの職位とはまったく異なります — 無党籍が圧倒的に多く、再選率は80%以上、月額補助4.5万台湾ドルは給与ではなく、派閥や個人の関係が政党よりも重要です [^11]。

詳細な議論については「村里長制度（村里長制度）」を参照してください。

### 原住民族區の二重のアイデンティティ

直轄市山地原住民區は法的には「直轄市の行政区」であると同時に「原住民族の自治単位」でもあります — この二重のアイデンティティが、資源配分、文化政策、土地管理における多重の緊張関係を生み出しています。區長は直轄市政府、中央原住民族委員会、部落会議の三つの層の間で調整を行う必要があり、一般的な「區長」の業務とはまったく異なります。

詳細については「直轄市山地原住民區長（直轄市山地原住民區長）」を参照してください。

## 一枚の投票用紙で九つの職位を選ぶ具体的な場面

2026年11月28日土曜日午前8時、全台22の縣市で投票所が同時に開設されます [^12]。有権者が投票所に入ると、以下の流れを経ます。

1. **受付** — 身分証明書と印鑑を提示。工作人員が選挙人名簿と照合します。
2. **投票用紙の受け取り** — 戸籍所在地に応じた投票用紙を受け取ります。直轄市の有権者は：直轄市長 / 直轄市議員 / 里長 の三枚の投票用紙を受け取ります（六都の人口 = 約1,500万人の有権者）。非直轄市の有權者は：縣市長 / 縣市議員 / 鄉鎮市長 / 鄉鎮市民代表 / 村長または里長 の五枚の投票用紙を受け取ります（その他の縣市 = 約800万人の有権者）。
3. **記票所に入る** — 各投票用紙で一名の候補者にチェックを入れます。村里長の候補者は3〜10名、議員の候補者は30〜50名（複数選挙区）の場合もあり、記票時間は平均5〜15分です。
4. **投票** — 投票用紙をそれぞれ対応する投票箱に入れます。

九つの職位のうち、各有権者が実際に受け取るのは3〜5枚の投票用紙です（戸籍が直轄市にあるか、山地原住民區にあるか、離島にあるかによって異なります）。原住民族の有権者はさらに原住民族議員 / 區長 / 區民代表の投票用紙を追加で受け取ります。

実際の投票用紙の色：直轄市長（赤）/ 縣市長（赤）/ 直轄市議員（黄）/ 縣市議員（黄）/ 鄉鎮市長（白）/ 鄉鎮市民代表（白）/ 村里長（緑）/ 山地原住民區長（青）/ 山地原住民區民代表（青）— 色分けにより間違った投票箱への投入を防ぎます [^13]。

## 国際比較：「九合一」は台湾特有の産物か

世界の民主主義諸国における「同日投票」の設計はさまざまです。台湾の「九合一」は比較的複雑ですが、独自というわけではありません。

**アメリカ：** 連邦 / 州 / 地方の選挙は異なる年に分散して行われることが多いです。しかし、特定の選挙日（11月の第一週の火曜日）には、大統領 + 議会 + 州知事 + 地方の国民投票 + 地方議員 + 市長 + 裁判官（一部の州では裁判官も民選）が同日に投票されることがあります — テキサス州では単日に20項目以上を投票することもあります [^14]。

**日本：** 統一地方選挙は4年ごとに一度、知事 / 都道府県議会議員 / 市町村長 / 市町村議会議員が同日に投票されます。しかし、これら四つの職位の選挙は全国で統一された日付ではありません — 各都道府県が異なる投票日を選ぶことができます。「統一」の度合いは台湾より低いです [^15]。

**韓国：** 「전국동시지방선거」（全国同時地方選挙）は4年ごとに一度、市道知事 / 教育監 / 市道議員 / 區市郡長 / 市郡區議員 / 校務監察委員 / 比例代表議員 / 教育委員など複数の職位が同日に投票されます — 台湾の九合一と同様の論理ですが、独立した教育監選挙（教育長が行政首長から独立している）がある点が異なります [^16]。

**イギリス：** 地方選挙は分散しており、市議会 / カウンシル / 警察委員の選挙は異なる日に行われることが多いです。同日に投票される職位数は通常3種未満です。

**ドイツ：** 連邦制の下、各州の地方選挙の日付は州が独自に決定します。同日に投票される職位数は州によって異なります（通常1〜3種）。

国際比較から見ると、台湾の九合一の構図は韓国モデルに近いです — 複数の職位が同日に投票されるが全国で統一された日付であるという点。異なる点は、台湾に独立した「直轄市山地原住民區」の職位があることです — これは台湾の原住民族の自治権に対する具体的な制度的回答であり、大多数の国にはこれに対応する設計はありません。

## なぜこの構図が大切にされるべきか

九合一は1947年の憲法制定時に計画された最終的な設計ではなく、1950年から2014年までの連続する65年間の段階的な積み重ねの結果です。それぞれの職位にはそれぞれの歴史があり、それぞれの制度的転換点にはそれぞれの時代の選択が対応しています。

**有権者にとって**、この構図は一枚の投票用紙の裏に完全な地方自治の体系があることを意味します — 村里長から直轄市長まで、鄉鎮市民代表から縣市議員まで、四年ごとに人民がすべての末端の民選首長と代表を同時に再構成することができます。これは世界的に見ても珍しいことです。

**候補者にとって**、この構図は多様な選挙戦略があることを意味します — 直轄市長は政策議論で勝負し、議員はSNTVの配票で勝負し、村里長は地域密着の活動で勝負し、原住民族區長は部落のネットワークで勝負します。それぞれの職位の政治経済学はまったく異なりますが、同日投票によりすべての候補者が同じ選挙サイクルを共有します。

**民主主義の基盤整備にとって**、この構図は中央選挙委員会、各縣市の選挙委員会、戸政司、警察署、調査局、監察院、メディア、市民団体、g0vのエンジニア、ファクトチェックセンター — 一整套の機関が四年ごとの同日のプレッシャーテストの中で自らの能力を検証することを意味します。

2026年11月28日の選挙は、台湾で六回目の九合一の同日投票です。そのたびにこの基盤整備のプレッシャーテストが行われます。2026年の選挙の詳細な背景と外部圧力の分析については、「2026 九合一選舉（2026年九合一選挙）」を参照してください。
