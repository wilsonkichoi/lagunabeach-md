> **30 秒概覽：**
> 每天早上，台灣人的 LINE 群組總會被那種彩虹漸層、暴力加粗新細明體的「早安圖」洗版。
> 這些被年輕人戲稱為「長輩圖」的影像，其實源自 2010 年代初期地方政府推動的「市民免費電腦課程」。
> 它們不僅是長輩們在螢幕前瞇著眼，一字一字敲出的數位祝福，更在近年來成為假訊息傳播的載體，同時也被年輕世代轉化為獨特的迷因文化。

「陳老師，這個字要怎麼變大？那個蓮花可以放左邊一點嗎？」2016 年，在新北市政府舉辦的免費電腦課程教室裡，講師陳怡慧正被一群平均年齡超過 60 歲的學生包圍。這群長輩不是在學 Excel 報表，而是在練習如何把一張風景照配上「平安喜樂」四個大字。[^1]

這就是「長輩圖」的誕生現場。對年輕人來說，這些圖片是審美災難；但對長輩來說，這是他們跨越數位鴻溝、努力擠進晚輩世界的門票。

### 從 PhotoCap 到 LINE Camera：數位落差下的美學

長輩圖的「獨特審美」——高飽和度的風景、彩虹漸層的文字，以及被暴力加粗的新細明體——其實是程式限制與生理需求的產物。早期長輩們多使用台灣開發的免費影像處理程式 **PhotoCap**，這款程式功能強大但操作介面直覺，內建的多種外框與文字特效，正好符合長輩們「越鮮豔越喜氣」的審美觀。[^2]

隨著智慧型手機普及，製作工具轉向了 **LINE Camera** 或各式手機應用程式。但無論工具如何更迭，那種「圖要大、字要亮」的風格始終如一。這背後其實隱含著生理上的必然：隨著視力退化，只有對比強烈、字體巨大的圖片，才能讓他們在小小的手機螢幕上看得清楚。

📝 策展人筆記：長輩圖的審美不是崩壞，而是生理機能與程式預設值的最大公約數。

### 蓮花、佛祖與太陽：長輩的數位表情符號

在長輩圖的語境裡，圖像並非隨機拼湊，而是具有如表情符號（Emoji）般的象徵意義。

| 圖像元素      | 象徵意義                                     |
| :------------ | :------------------------------------------- |
| **蓮花**      | 代表聖潔、不煩憂，希望將清淨的祝福帶給你。   |
| **佛祖/觀音** | 代表祈求神明保佑平安，是最厚重的守護。       |
| **太陽/風景** | 分享朝氣與溫暖的精神，象徵新的一天充滿希望。 |
| **咖啡/茶杯** | 邀約共飲的意象，代表「我掛念著你」。         |

比起打字，長輩們往往花更多時間在精挑細選一張意義正確的圖片。[^3] 對他們而言，傳送一張佛祖圖，等同於在廟裡為你點一盞平安燈。

### 「我還在，我掛念你」：社群平台上的生存確認

為什麼長輩這麼愛傳長輩圖？心理學家分析，這是一種「尋求認同與歸屬感」的行為。[^4] 對於退休後社交圈縮小的銀髮族來說，LINE 群組是他們與社會連結的重要管道。

「其實對老人家來說，打字很慢、視力不好，打一句『你好嗎』可能要花五分鐘。」一位在 Threads 上分享母親心聲的網友提到，那張充滿鮮花的圖，背後的意思其實是：「我今天還活著，而且我還掛念著你。」[^5]

傳送訊息的那一刻，他們參與晚輩世界的渴望就已經完成了。至於你有沒有回覆，或者回覆了什麼，對他們來說往往是次要的。

📝 策展人筆記：長輩圖不是訊息，是數位時代的「平安信」。

### 假訊息的溫床：長輩圖的灰色地帶

然而，長輩圖的普及也帶來了意想不到的挑戰。由於其視覺吸睛、文字簡潔的特性，加上長輩們對資訊來源的辨識能力可能較弱，長輩圖逐漸成為假訊息與認知作戰的溫床。許多不實的健康偏方、政治謠言，甚至詐騙訊息，都常以長輩圖的形式在社群平台廣為流傳。[^8] 這些假訊息利用長輩圖的親和力與信任感，滲透到家庭群組中，不僅造成資訊混亂，更可能影響社會穩定。[^9]

📝 策展人筆記：當情感連結被惡意利用，長輩圖的溫柔也可能成為資訊戰的破口。

### 挑戰與轉變：從嘲諷到迷因的文化融合

長輩圖曾是世代衝突的導火線。年輕人抱怨群組被洗版，甚至出現了「長輩圖終結者」等反制手段。[^6] 然而，隨著時間推移，這種風格反而被年輕世代「迷因化」。

現在，我們可以看到許多「惡搞長輩圖」，用同樣的審美風格寫著諷刺的文字，或是政治人物用長輩圖風格製作宣傳海報。這種文化的融合，讓原本嚴肅的世代隔閡，轉化成了一種帶有幽默感的共同語言。年輕人透過模仿與再創作，不僅消解了長輩圖的刻板印象，也讓這種獨特的視覺符號在台灣網路文化中獲得了新的生命。[^10]

### 餘韻：下次收到時，或許能多一份溫柔

2017 年，當齊柏林導演墜機的消息傳出時，許多長輩群組傳的是他鏡頭下的台灣美景配上哀悼文字。[^7] 那些圖片依然有著粗糙的邊框與鮮豔的字體，但在那一刻，文字背後的哀傷與祝福是真實的。

下次當你的手機在清晨叮咚響起，看到那張閃爍著彩虹光的「早安」時，或許可以試著忽略那暴力的新細明體。因為在那張圖的背後，可能是一個長輩瞇著眼、戴著老花眼鏡，在螢幕前努力了半小時，才終於把那份「平安」送到了你的手上。
