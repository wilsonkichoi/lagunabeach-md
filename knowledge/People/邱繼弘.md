> **30 秒概覽**：交通大學資工博士邱繼弘，綽號「大河馬」。他自小展現對軟體程式的濃厚興趣，高中曾獲全國科展第三名，並以「特殊才藝」推甄進入國立交通大學資訊工程系 [^4]。2006 年，他受 Web 2.0 浪潮影響，與夥伴創辦社群網站 funP（推推王），雖風光一時卻因商業模式不明於 2007 年宣告失敗，三年燒掉近兩千萬 [^8] [^10]。2009 年，他從廢墟中轉型，創立 cacaFly 聖洋科技，從 Facebook 台灣廣告總代理起步，陸續拿下 Google、LINE、Microsoft、Spotify 等國際平台代理資格，並建立 adHub、TenMax、BeingDigital 等子品牌，打造台灣最完整的本土數位廣告生態系 [^9] [^11]。除了商業成就，邱繼弘亦積極走入政壇，曾任台北市政府市政顧問，並於 2019 年登記參選時代力量決策委員，試圖以「左派老闆」的科技思維改造政治 [^20] [^21]。近年來，他更積極佈局人工智慧（AI）行銷與零售媒體聯播網（RMN），持續引領台灣 MarTech 產業轉型 [^1] [^2] [^3]。

---

## 早年啟蒙與「新交大幫」的技術基因

邱繼弘出生於 1976 年，其工程師之路始於一段偶然的頂替。國小六年級時，他因替兄長參加程式設計課程，意外開啟了對軟體的濃厚興趣 [^4]。這份天賦在高中時期得到驗證，他憑藉卓越的程式作品獲得全國科展第三名，並順利以「特殊才藝」推甄進入國立交通大學資訊工程系 [^4]。

在交大求學期間，邱繼弘加入了袁賢銘教授的「分散式系統實驗室」（DCSLab）。這個實驗室在台灣軟體創業史上佔有重要地位，核心成員包括後來創辦曜碩科技的郭榮昌、吳英豪，以及優必達（Ubitus）創辦人沈上謙、奇群科技創辦人宋牧奇等人 [^5] [^24]。這群被媒體譽為「新交大幫」的年輕人，具備早期矽谷的創業思維，試圖將台灣從「硬體代工」的思維轉向「軟體服務」。2004 年，仍在攻讀博士班的邱繼弘帶領團隊奪得趨勢科技「百萬程式競賽」首獎，並曾前往美國 IBM Silicon Valley Lab 實習，這些學術與業界的深厚累積，成為他日後以技術驅動廣告業的基石 [^4] [^6] [^7]。

📝 **策展人筆記**：邱繼弘的早年經歷，特別是「新交大幫」的群體效應，揭示了台灣科技人才從學術走向創業的獨特路徑。DCSLab 既是技術搖籃，也是孵化「軟體服務」思維的溫床，跟當時台灣主流的硬體製造業形成鮮明對比。他的工程師基因，從一開始就註定了他將以不同於傳統廣告人的方式解構並重塑數位行銷產業。

## 台灣 Web 2.0 最昂貴的學費：funP 推推王

2006 年，受全球 Web 2.0 浪潮影響，邱繼弘與四位夥伴共同創辦了社群網站「funP.com」（推推王）[^8]。funP 採用了類似 Digg 的社交過濾演算法，強調「群眾智慧」與社交分享，是當時台灣較具規模的 Web 2.0 實踐之一 [^8]。隔年，他代表台灣隊登上矽谷 DEMO Show 舞台，向美國觀眾闡釋 Web 2.0 的概念 [^4]。

技術上的領先並未轉化為商業成功。funP 三年間燒掉近兩千萬資金，面臨多重困境：當時台灣廣告主仍高度集中在 Yahoo 等入口網站，對 Web 2.0 流量缺乏信心；2009 年 Facebook 強勢進入台灣，其社群連結度與 API 生態遠超 funP，導致 funP 流量雪崩式下跌 [^10]。2009 年，眼見商業模式難以支撐，邱繼弘決定放棄產品開發，轉向「結案就能拿到錢」的廣告服務業 [^10]。這段失敗經歷讓他深刻體悟到：在網路世界，擁有強大的技術與流量若無商業模式支撐，終將難以為繼。

📝 **策展人筆記**：funP 推推王的失敗，是邱繼弘創業生涯的關鍵「成人禮」。它燒掉巨額資金，也讓他從純粹的技術理想主義者，轉變為深刻理解商業模式重要性的實踐者。這段經歷為他日後創辦 cacaFly 奠定了基礎，使其在面對新技術浪潮時能更務實地思考如何將技術與商業價值結合，設計出「不失敗系統」。

## 工程師重塑廣告業：cacaFly 的技術崛起

2009 年創立的 cacaFly 聖洋科技，從一開始就帶有濃厚的工程師基因。不同於傳統廣告公司強調創意與人脈，cacaFly 團隊中有一半是精通平台演算法的程式設計師。邱繼弘將演算法視為可解析的工具，透過大量測試抓取邏輯，為客戶節省成本 [^12]。

2011 年，cacaFly 正式成為 Facebook 台灣官方廣告代理，這成為其發展的分水嶺 [^9]。此後，公司陸續拿下 Google、Microsoft、LINE、Spotify 等國際平台代理權，並發展出 adHub 與 TenMax 等子品牌。其中 TenMax 致力於建立台灣本土的即時競價（RTB）系統與資料管理平台（DMP），目標是把台灣廣告產值與資料留在本土，建立完整的在地數位廣告生態系 [^10] [^11]。

📝 **策展人筆記**：cacaFly 的崛起是邱繼弘「工程師思維」在廣告業的勝利。他把廣告視為一個可被數據分析、演算法優化的系統。這種技術導向的策略讓 cacaFly 迅速成為國際平台的在地巨頭，也透過 TenMax 等子品牌試圖在台灣建立一套獨立的數位廣告基礎設施，展現了其對「數位主權」的思考。

## 走入政壇：時代力量決策委員與市政顧問

邱繼弘的視野並未侷限於商業。柯文哲擔任台北市長期間，邱繼弘受聘為台北市政府市政顧問，在數位行銷與智慧城市發展上提供專業建議 [^21]。

2019 年 1 月，邱繼弘登記參選時代力量第二屆決策委員 [^20]。在參選聲明中，他以「左派老闆」自居，強調這次參選關乎黨務運作模式的改造，希望將 IT 產業的創新動能注入政黨運作 [^20]。他主張政黨應具備「資料分析力」，透過技術手段了解選民需求。當時時代力量正值路線辯論的關鍵期，邱繼弘的加入被視為中世代科技領袖試圖透過體制內參與，提升台灣政治決策的科學性與數位力。

📝 **策展人筆記**：邱繼弘的政治參與反映了台灣新一代科技菁英對公共事務的關懷。他試圖把工程師的「系統優化」邏輯帶入政治領域，強調數據分析與理性決策。這既是他個人身份的轉變，也象徵著台灣社會對科技治理的期待與嘗試。

## 廣告代理商的政治地雷與中立挑戰

作為 Facebook 在台灣的官方窗口，cacaFly 難免捲入政治爭議。2017 年，國民黨智庫指控「臉書變綠了」，質疑蔡英文政府透過代理商進行言論審查 [^22] [^23]。面對質疑，邱繼弘多次澄清：cacaFly 僅負責廣告業務（Business），不涉及內容審核（Policy），內容政策由 Facebook 美國總部直接掌控 [^22]。

這場爭議凸顯數位廣告代理商在台灣特殊政治生態中的挑戰：如何在服務各政黨客戶的同時維持技術中立的形象。邱繼弘選擇以「產業建制者」的高度回應，透過擔任 DMA 理事長，推動資安、個人資料保護、資料應用與打擊詐騙等議題，試圖把數位廣告環境導向更透明、更規範的發展軌跡 [^15] [^16]。

📝 **策展人筆記**：數位廣告代理商在台灣的政治環境中，往往身處風暴中心。邱繼弘面對「臉書變綠了」的指控，展現了其作為產業領袖的危機處理能力。他把爭議從政治意識形態拉回技術中立與產業規範的層面，透過 DMA 理事長的身份建立一套能讓所有參與者信任的數位廣告秩序，這既是商業考量也是對產業倫理的維護。

## 人工智慧轉型與 RMN：引領 MarTech 第二波浪潮

2020 年後，邱繼弘將 cacaFly 的核心戰場轉向人工智慧（AI）行銷與零售媒體聯播網（RMN）。他推出的 **Alchemy AI 創意生成平台**，引入「圖層思維」（Layer Thinking），解決 AI 生成素材中產品失真的痛點，大幅提升商用價值 [^1] [^18]。

在 RMN 領域，cacaFly 協助品牌如「上山採藥」構建資料驅動的解決方案，並將相關技術輸出至東南亞市場 [^2] [^13] [^19]。他多次在產業論壇中強調，AI 時代的關鍵在於「資料治理」（Data Governance），掌握高品質的第一方資料才能在 Cookie 消失後的數位廣告市場中生存 [^14]。2024 年，cacaFly 同時獲得 Google 卓越代理商與 LINE 鑽石級經銷夥伴認證 [^11]。

📝 **策展人筆記**：從 Web 2.0 到 AI 時代，邱繼弘始終走在技術浪潮的前沿。他對 AI 行銷與 RMN 的佈局是商業戰略轉型，也是對未來數位廣告生態的預判。他對「資料治理」的強調，顯示其對隱私保護與數據倫理的理解——這會是 MarTech 產業在後 Cookie 時代的核心競爭力。

## 結語：失敗中設計的「不失敗系統」

從 funP 的兩千萬學費到 cacaFly 的數位廣告王國，邱繼弘的創業史是一段不斷從失敗中迭代的過程。他用工程師的測試與迭代邏輯，取代了廣告業傳統的直覺與豪賭。他的綽號「大河馬」也正如他本人的特質：外表穩重，但在數位浪潮的水面下，卻能以極高的速度精準前行。

📝 **策展人筆記**：邱繼弘的故事歸結於一個核心命題：如何在變動不居的數位世界中設計一個「不失敗系統」。這個系統的定義是「能從錯誤中學習、快速迭代、把技術與商業模式深度融合」，而非追求永不犯錯。他的「大河馬」哲學正是這種穩健敏捷創業精神的最佳寫照，為台灣的科技創業家提供了寶貴啟示。
