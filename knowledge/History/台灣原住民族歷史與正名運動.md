# 台灣原住民族歷史與正名運動

1987年5月15日，台北刑場。法醫走向一個19歲的年輕人，準備施打麻醉藥。那個年輕人搖了搖頭。他說：「自己罪有應得，所以必須接受這個刑痛。」

他叫湯英伸，鄒族人，來自阿里山特富野部落。一年前，他看到報紙上的西餐廳徵人廣告，帶著一張學生臉從山上來到台北，被騙入一家洗衣店。介紹費3,500元，雇主扣了他的身分證，深夜還要他繼續工作。1986年1月25日，工作第九天，他喝了酒，在衝突中殺死雇主一家三口，包括一個2歲的女孩，然後走去派出所投案。

那把槍打死的，不只是一個犯了罪的年輕人。

## 四百年的名字，四百年的鎖

要理解這場運動，先要理解「一個名字，就是一把鎖」是什麼意思。

1624年荷蘭東印度公司抵台，把島上的住民分成「歸化蕃」與「野蕃」：教化、順從、納入，這是殖民管理的三步邏輯。清朝接手後延續分類，「熟番」與「生番」的標準是剪辮、改漢姓、繳稅——越像漢人，越文明。日本殖民（1895—1945）更系統化：稱「蕃人」，設「理蕃」警察駐守，推行集體移住，把部落從世居地拆散、遷移、重新命名。

1930年，賽德克族頭目莫那·魯道率三百名族人在霧社起義。日本出動飛機與毒氣鎮壓，整個部落幾近滅絕。這不是英雄史詩；這是一個族群被逼到懸崖後的反撲，以及那反撲如何被徹底清除的殘酷記錄。

1945年，國民政府接收台灣，給了一個新名字：「山胞」。聽起來親切，其實暗示「住在山上的漢人同胞」——刻意模糊了原住民族的主體身份，讓同化政策多了一層進步的外皮。禁說族語、強制漢姓、推動山地平地化、在部落開路、遷村。台灣原住民族就這樣成了自己家鄉最陌生的人。

> 📝 **五個名字，五把鎖**
>
> 蕃人（日治）→ 山地人（戰後初期）→ 山胞（官方制度化）→ 原住民（1994入憲）→ 原住民族（2005原基法）。前三次，是別人替你命名；後兩次，是你自己搶回來的。

## 吳鳳的謊言，和那尊銅像

湯英伸被槍決後，台灣社會陷入少見的集體自省。

天主教花蓮教區主教單國璽帶頭聲援，帶隊向蔣經國請願「槍下留人」。主流媒體第一次大篇幅報導原住民的都市處境：求職陷阱、低薪剝削、身分證被扣。一個鄒族青年的案件，拆開了一道被嚴實遮蓋多年的傷口。

湯英伸的家鄉那個鄉，叫「吳鳳鄉」。這個名字來自清代一則廣傳的故事：漢人吳鳳主動以自身為祭品，感化了嗜殺人頭的原住民。這個故事在日治和國民政府時代都被國家徵用，寫入教科書，立銅像，傳遞的訊息很清楚：原住民是需要被開化的野蠻人。

1988年，一群原住民青年走進嘉義車站前，把吳鳳銅像推倒。1989年3月1日，吳鳳鄉正名為阿里山鄉。1990年，吳鳳的故事從教科書刪除。湯英伸的族人叫回了鄒族（日治時代被強稱「曹族」）。

這不是毀壞歷史。這是糾正一則被國家認證了幾十年的謊言。

同一年1988年，數千名原住民走上台北街頭，喊出「還我土地」，要求政府歸還傳統領域。這是正名運動第一次以集體身份登上政治舞台的時刻，也是四年後那場憲法修正的前奏。

## 1984：原住民，還是山胞？

要找正名運動的制度性起點，要回到1984年12月。

台灣原住民族權利促進會（原權會）成立，是台灣第一個全國性的原住民組織。成立宗旨之一，就是推動正名：「我們不是山胞，我們是台灣原住民。」這句話在當時的台灣說出口，需要很確定的勇氣。

湯英伸事件讓這個訴求有了社會能量。原住民的都市困境、就業剝削、文化失根，第一次被主流輿論正視。那不是同情，是一個社會開始對自己的歷史欠帳感到不安。

## 1994：憲法寫進兩個字

1993年，數千名原住民聚集在凱達格蘭大道，穿著各族傳統服飾遊行。隔年，這個訴求落地：1994年，《中華民國憲法增修條文》修正，正式將「山胞」改為「原住民」。

這是第一次，「原住民」兩個字寫進國家最高法律。

但還差一個字。「原住民」是個人身份認定；「原住民族」是集體主體，承認的是整個族群的語言、土地、自治主張。這一個字的差距，花了11年才補上。2005年，《原住民族基本法》通過，「族」字正式入法，確立了自治權、土地權、文化傳承義務的法律框架。

> 📝 **加一個「族」字的政治份量**
>
> 「原住民」是個人的標籤；「原住民族」是民族的主體。2005年的改動，在法律上承認的是集體性的土地主張、語言復振義務，以及政府與原住民族「對等協商」的機制。那不是一個字，是一整套憲政承諾——雖然落實至今仍有距離。

## 從9族到16族：被埋名的臉孔，一一回來

「正名」對許多族群而言，不只是從「山胞」改名，而是從別人的族籍裡被找回來。

戰後官方認定的原住民族只有9族，許多族群被歸入鄰近大族，語言和祭儀在沉默中消融。自1990年代，正名申請陸續展開：

| 年份 | 族群             | 備注                                 |
| ---- | ---------------- | ------------------------------------ |
| 2001 | **邵族**         | 南投日月潭，目前全台人口不足900人    |
| 2002 | **噶瑪蘭族**     | 花蓮，曾隱入阿美族戶籍               |
| 2004 | **太魯閣族**     | 原屬泰雅族系                         |
| 2007 | **撒奇萊雅族**   | 1878年「加禮宛事件」後被迫隱名逾百年 |
| 2008 | **賽德克族**     | 霧社事件的族群，等了78年             |
| 2014 | **拉阿魯哇族**   | 原屬鄒族系，人口不足300人            |
| 2014 | **卡那卡那富族** | 原屬鄒族系，與拉阿魯哇族同年正名     |

到2014年，官方認定原住民族達到16族。撒奇萊雅族在阿美族身份下隱藏了超過130年，才重新叫回自己的名字。每一個正名，背後是幾年、甚至幾十年的文化自救：採集耆老語音、重建祭儀、追溯族譜，然後向政府遞交文件，等待裁決。

## 2016：總統府的道歉

2016年8月1日，台灣原住民族日，蔡英文在總統府經國廳，代表政府向全體原住民族正式道歉。這是中華民國史上第一次，國家元首承認對原住民族四百年來的歷史不公。

道歉文歷數責任：荷蘭與鄭成功政權屠殺平埔族、清朝流血鎮壓、日本理番政策、戰後山地平地化、禁說族語、以及在雅美族人不知情下將核廢料儲存蘭嶼。

她在道歉文中引用泰雅族語：

> 「在泰雅族的語言裡，『真相』，叫做 Balay。而『和解』叫做 Sbalay，也就是在 Balay 之前加一個 S 的音。真正的和解，只有透過誠懇面對真相，才有可能達成。」
> — 蔡英文，2016年8月1日

她同日宣布成立「原住民族歷史正義與轉型正義委員會」（原轉會），以總統身份親自擔任召集人，承諾每年8月1日向全國報告執行進度。

原住民族代表對道歉的反應不一：有人動容，有人認為缺乏具體承諾，有人說「道歉沒有意義，要還土地」。傳統領域劃設至今進展遲緩，原基法中的自治條文落地困難。Sbalay，是說完真相之後才可能發生的事——而那份真相，仍在說。

## 語言是最後一道防線

正名成功之後，問題並沒有減少。

台灣16族有42種方言，超過半數被聯合國列為「瀕危」或「嚴重瀕危」語言。邵族語的流利使用者，2020年代的統計不足10人。卡那卡那富族的族語課，族內具備完整傳承資格的耆老屈指可數。語言死了，族群記憶就死了。復振比正名更難，它需要家庭、學校、部落、政策同步配合，需要年輕人願意開口說一個在日常生活幾乎派不上用場的語言。

---

從1984年原權會成立，到1994年正名入憲，到2005年原基法，到2016年道歉，每一站都比前一站走得更難。

但如果你要找這條路的起點，大約是1987年5月15日，台北，一個19歲的鄒族青年在刑場拒絕了麻醉藥。

他說他罪有應得，必須清醒地承受那個痛。

他的族人在場外哭喊，他的名字在社會裡流傳，然後一個運動開始了。他的家鄉後來叫回阿里山鄉，他的族叫回鄒族，他的族人的名字被寫進憲法、寫進基本法，有了一篇遲到三十年的國家道歉。

你今天還記得他的名字嗎？
